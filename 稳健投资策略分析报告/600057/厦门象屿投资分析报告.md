## 决策理由

1. **商业模式清晰但护城河较窄**：大宗供应链行业竞争激烈，CR5市场份额仅5.42%，厦门象屿市场份额1.35%，护城河主要来自国企背景和全球网络，但技术壁垒较低。

2. **盈利质量承压**：2025年归母净利润同比下降8.87%，扣非净利润虽大幅增长339.13%，但主要系异常减值减少所致，主业盈利能力仍需观察。ROE从2022年的18.06%降至2025年的5.01%，盈利能力下滑明显。

3. **现金流质量堪忧**：2025年自由现金流为-9.62亿元，连续两年为负，经营现金流虽为正但主要用于存货和应收账款增加，现金转化效率下降。

4. **股息率尚可但分配能力存疑**：当前股息率约3.01%，但分红支付率从2022年的44.6%降至2024年的约36.5%，分配意愿有所减弱。2025年分红预案每股0.22元，需关注实际支付能力。

5. **财务风险较高**：资产负债率70.43%，短期债务占比97.21%，流动性压力较大。广义净现金仅8.49亿元，现金保护能力弱。

## 风险提示

1. **大宗商品价格波动风险**：公司主营大宗商品供应链，商品价格剧烈波动直接影响盈利，2025年新能源业务期现毛利为-18.03亿元，套期保值操作风险较大。

2. **流动性风险**：短期债务占比97.21%，短期偿债压力大。虽然经营现金流为正，但自由现金流连续为负，需关注债务滚动能力。

3. **盈利能力下滑风险**：ROE从2022年的18.06%降至2025年的5.01%，毛利率仅2.08%，盈利空间极薄，抗风险能力弱。

4. **存货跌价风险**：存货从2024年的261.42亿元增至2025年的319.47亿元，增幅22.21%，在商品价格下行周期面临较大跌价风险。

1. **境外业务爆发式增长**：2025年境外业务营收1649.12亿元，同比增长154.75%，占总营收比例从23.7%提升至40.2%，国际化布局成效显著。

5. **永续债利息侵蚀利润**：2025年归母净利润中包含永续债利息5.07亿元，扣除后实际归母净利润仅7.86亿元，盈利质量进一步下降。

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 维持性Capex系数 | 0.8 | 制造业/供应链行业标准 |
| 维持性资本开支估算 | 8 亿元 | 折旧摊销 × 系数 |
| Owner Earnings | **25.85 亿元** | 扣非归母净利润 + 折旧摊销 − 维持性资本开支 |
| 支付率锚定值 | 40% | 历史均值 |

| 股息税率 | 10% | 持股1个月至1年 |

**粗算公式**：
```
粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值
                = [25.85 × 40% × (1-10%) + 8] / 228.67
                = [9.31 + 8] / 228.67
                = 17.31 / 228.67
                = 7.57%
```

**修正粗算**（考虑自由现金流为负）：
由于2025年自由现金流为-9.62亿元，Owner Earnings计算可能过于乐观。采用保守口径，假设Owner Earnings = 扣非归母净利润 × 0.7 = 12.50亿元。

```
修正粗算穿透回报率 = [12.50 × 40% × (1-10%) + 8] / 228.67
                    = [4.50 + 8] / 228.67
                    = 12.50 / 228.67
                    = 5.47%
```

- 分配能力验证：OCF 100.97亿 > Owner Earnings 25.85亿 ✅
- 分配意愿：**中** — 分红支付率从44.6%降至36.5%，分配意愿有所减弱
- 税务泄漏：股息税率 = 10%

> **粗算穿透回报率 R = 4.82%**（保守口径） — ✅ 通过（门槛值 = 3.75%）
