### 4.1 经营回顾

**2025年核心经营数据：**

| 指标 | 2025年 | 同比变动 |
|------|-------:|---------|
| 营业收入 | 10,502亿元 | +0.9% |
| 主营通服业务收入 | 8,955亿元 | +0.7% |

| EBITDA | 3,389亿元 | +1.6% |
| EBITDA率 | 32.3% | +0.2pct |
| 算力服务收入 | 近900亿元 | +11%+ |
| 智算服务收入 | - | +279% |
| 国际市场收入 | 293亿元 | +28.5% |

**收入结构分析：**
- 有线宽带业务收入1,415.69亿元，同比增长8.7%，成为传统通信业务中的增长亮点
- 无线上网业务收入3,690.92亿元，同比略有下滑
- 语音、短彩信等传统通信业务收入持续收缩
- 算力服务、智能服务是业绩增长的新引擎，2025年收入分别同比增长11.1%/5.3%

**关键运营指标：**
- 千兆宽带客户达到1.1亿户
- 家庭宽带网络收入同比增长7.1%
- 物联网卡客户数达到约15亿
- AIoT平台成为全球最大的连接管理平台
- 公司重点布局超50款自研行业大模型
- 6G国际标准立项数居全球运营商第一阵营

### 4.2 前瞻指引

**2026年展望：**
- 2026年是"十五五"规划的起步之年
- 公司目标：收入稳健增长、利润协同增长
- 资本开支计划1,366亿元（同比下降约9.5%）
  - 通信网络资本开支792亿元，同比下降20%
  - 算力网络资本开支378亿元，同比增长62%
  - 智能网络资本开支89亿元，同比增长20%
  - 算力网络和智能网络投资占三大网络投资比重将超过37%
- 所需资金主要来自经营活动现金流

### 4.3 资本配置意图

- 2025年资本开支约1,509亿元，同比下降8%
- 投向结构明显优化：从传统通信网络向算力网络和智能网络倾斜
- 持续加大分红力度，2025年全年股息每股5.27港元
- 资金主要来源于经营活动现金流

### 4.4 风险因素

1. **外部环境变化影响加深**：宏观经济不确定性增加
2. **行业新旧动能转换关键期**：传统通信业务增长放缓，新兴信息服务增长动能尚在培育
3. **电信服务增值税政策调整**：套餐收入分拆纳税对利润产生一次性影响
4. **应收账款快速增长**：政企业务扩张带来应收账款和坏账风险上升
5. **经营现金流下降**：2025年经营现金流2,329亿元，同比下降26.2%
