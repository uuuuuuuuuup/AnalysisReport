## 四、MD&A 深度信息

> 数据来源：mx_finance_search + 2025年年报公告

### 经营回顾

1. **销量表现**：2025年全年销售汽车291.3万辆，同比增长8.5%，创近9年新高，连续六年实现同比正增长
2. **新能源突破**：新能源销量约111万辆，同比增长51%，首次突破百万大关
3. **海外拓展**：海外销量63.7万辆，同比增长18.9%，海外板块毛利率19.49%，显著高于国内
4. **营收增长**：实现营业收入1,640亿元，同比增长2.67%
5. **盈利结构**：归母净利润40.75亿元，同比下降44.34%（主要因非经常性损益大幅回落）；扣非归母净利润27.95亿元，同比增长8.03%
6. **毛利率改善**：全年综合毛利率15.54%，同比提升0.60个百分点，呈现逐季向上态势；Q4单季毛利率16.8%，创近两年新高

### 前瞻指引

1. **2026年销量目标**：总体销量330万辆（同比+13.3%）；新能源销量140万辆（同比+26.2%）；海外销量75万辆（同比+17.7%）
2. **产品规划**：全年投放10余款全新及改款产品，包括长安启源Q06、新款深蓝S05、阿维塔06T等新能源车型
3. **全球化布局**：从"国内为主+辐射海外"转向全球一体化布局，覆盖欧洲、东南亚、中东非洲、中南美洲等市场

### 资本配置意图

1. **2026年计划投资额**：144.7亿元
   - 固定资产投资：44.7亿元（核心研发能力、造型、前瞻及固态电池试验能力、新能源产能结构调整、数字化转型、海外业务拓展）
   - 长期股权投资（含基金）：100亿元（夯实智能化基础能力、做大做强长安科技、新能源品牌建设、动力电池制造能力、海外投资平台、新兴产业如飞行汽车/机器人）

2. **研发投入**：2025年研发投入125.76亿元（含资本化），占营收比例7.67%，研发投入强度居行业前列；资本化率从35.96%升至43.08%

### 风险因素

1. **行业竞争加剧**：汽车行业价格战持续，盈利空间受挤压
2. **新能源转型风险**：深蓝汽车仍亏损（2025年净亏损8.99亿元，较上年15.7亿元收窄），阿维塔品牌盈利尚需时间
3. **合资板块拖累**：合营企业投资收益由正转负（2024年3.7亿元 → 2025年-4.3亿元），长安福特/长安马自达持续衰退
4. **汇率波动风险**：2026Q1汇兑损益对利润产生显著影响
5. **现金流压力**：经营现金流从484.9亿降至183.6亿，主因主动缩短供应商账期（应付账款从438亿降至298亿，提前支付140亿）
