### 因子 2：穿透回报率粗算

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.7 | 服务业标准 |
| Owner Earnings（5年均值口径） | **~150 百万元** | 极度宽松假设（含非经常性损益） |
| Owner Earnings（扣非口径，2025） | **负值** | 扣非净利润-50百万元 |
| 支付率锚定值 | 9.69% | 2025年度实际值 |
| 年均回购金额 | 0 | 无回购 |
| 股息税率 | 0% | A股持股>1年免征 |

**粗算公式**：

`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

- 宽松口径：`150 × 9.69% × 1 / 15,620 = 14.54 / 15,620 = 0.09%`
- 5年均值支付率口径：`150 × 23.84% × 1 / 15,620 = 35.76 / 15,620 = 0.23%`

- 分配能力验证：5年均值OCF 3.5 百万元 << Owner Earnings 150 百万元 ❌（现金转化极差）
- 分配意愿：**弱** — 支付率从30%+骤降至9.69%，且无回购
- 税务泄漏：持股>1年税率为0%

> **粗算穿透回报率 R = 0.23%（极度宽松口径）** — ❌ 否决（门槛值 = 3.746%）
>
> 即使使用最宽松的假设（5年均值归母净利润+5年均值支付率），回报率仍仅为0.23%，远低于门槛值3.746%。若使用扣非口径，Owner Earnings为负值，回报率直接为负。
