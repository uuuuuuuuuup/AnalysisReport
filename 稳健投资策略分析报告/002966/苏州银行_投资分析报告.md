## 决策理由

1. **区域优势显著**：苏州银行立足苏州、深耕江苏、全面融入长三角，区域经济发达，需求稳定。苏州市财政局为实际控制人，地方政府支持力度大。

2. **资产质量优良**：不良贷款率0.83%，显著低于商业银行平均水平（约1.6%）；拨备覆盖率420.59%，风险抵补能力强；核心一级资本充足率9.79%，符合监管要求。

3. **盈利质量高**：净利率43.14%，盈利能力强；扣非净利润与净利润差异<1%，非经常性损益影响小；ROE维持在9-12%区间，盈利稳定性好。

4. **安全边际较小**：精算穿透回报率4.11%，仅超额门槛值0.344 pct，安全边际较小。当前股价8.81元高于目标买入价7.69元，建议等待回调。

5. **分配意愿保守**：派息率30%左右，低于银行业平均水平（35-40%）；未实施股份回购；但2025年管理层增持60万股，显示对公司前景信心。

## 风险提示

1. **利率政策风险**：LPR超预期下调可能导致息差额外收窄，影响盈利能力。2025年央行持续降息，银行业息差承压。

2. **区域经济风险**：苏州银行深耕长三角，区域经济波动可能影响资产质量和盈利能力。需关注房地产及小微尾部风险。

3. **竞争压力**：省内面临江苏银行、南京银行等大行竞争，全国性银行也在长三角布局，竞争压力较大。

4. **监管政策风险**：银行业受银保监会严格监管，资本充足率、不良贷款率等指标受监管要求，政策变化可能影响经营。

### 因子 3：精算穿透回报率 + 现金质量审计

**银行业特殊性说明**

苏州银行作为商业银行，财务报表结构与一般企业不同：
- 无应收账款、无存货
- 收款模式：存款即时到账，贷款按期收息
- 现金流特点：经营现金流波动较大，主要受存贷款规模变化影响

**真实可支配现金结余估算**

由于银行业特殊性，采用扣非净利润作为真实可支配现金结余的保守估算：
- 扣非净利润：50.39亿元
- 非经常性损益影响小（<1%）
- **真实可支配现金结余 = 50.39亿元**

**现金储备质量**

| 项目 | 金额（亿元） | 说明 |
|:---|---:|:---|
| 广义现金 | 189.50 | 现金及等价物 |
| 受限现金 | 约151.60 | 银行业监管要求（估算80%） |
| 可自由支配现金 | **37.90** | 广义现金−受限现金 |
| 派息可持续年限 | 2.48 年 | 可自由支配现金 / 年派息总额 |

**精算关键步骤说明**

- **步骤1 真实现金收入**：银行业收款模式为预收+现结，无应收账款，真实现金收入 ≈ 营业收入
- **步骤3 非经常项分类**：扣非净利润与净利润差异<1%，非经常性损益影响小
- **步骤4 经营现金支出**：银行业经营现金支出主要为利息支出、员工薪酬等，OCF为109.69亿元
- **步骤5 资本扣除**：资本开支4.55亿元，主要为IT系统和网点建设

```
精算穿透回报率 = [50.39 × 30% × (1−0%) + 0] / 368.38
```
