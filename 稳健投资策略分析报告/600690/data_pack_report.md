## 四、MD&A 深度信息

> 数据来源：AI-Tools MCP GetFinancialReport + WebSearch 公开资讯

### 4.1 经营回顾

**2025年全年经营业绩：**
- 营业收入3,023.47亿元，同比+5.71%，首次突破3,000亿元
- 归母净利润195.53亿元，同比+4.39%
- 扣非归母净利润186.04亿元，同比+4.49%
- 毛利率26.66%，同比下降4.17个百分点
- 净利率6.67%，同比下降2.51个百分点
- ROE（加权）16.98%，同比下降3.58个百分点

**分品类表现：**
- 空调同比+9.55%，冰箱同比+1.11%，厨电同比+0.51%
- 水家电、洗衣机、装备部品及渠道综合服务业务保持稳健

**分区域表现：**
- 北美：2H25美洲地区收入同比-1.5%，4Q25北美收入同比持平略降，利润率因需求疲软、行业促销力度大及关税抬升成本而下滑
- 新兴市场：2H25南亚/东南亚/中东非收入同比+1%以上，延续较快增长
- 欧洲：一次性费用拖累利润

**Q4单季波动：**
- Q4收入682.9亿元，同比-6.7%
- Q4归母净利润21.8亿元，同比-39.2%
- Q4扣非净利润17.1亿元，同比-45.1%
- Q4毛利率24.8%，同比-5.1pct，受原材料上涨、行业竞争及关税等拖累

### 4.2 费用管控

- 2025年销售费用率11.2%，同比-0.6pct
- 国内市场加速落地统仓ToC模式并持续推进营销数字化变革
- 海外市场推进终端零售创新、整合全球资源，提升运营效率
- 管理费用率4.55%，同比+0.23pct
- 财务费用-0.51亿元（净收益），上年为9.72亿元，主要因汇率变动

### 4.3 前瞻指引

**四大战略布局，打开长期增长空间：**

1. **全面推进TC变革与AI全流程提效**：TC变革缩短交易环节、降低库存，空调业务经销商终端收入同比增长超130%，库存周转效率提升50%以上。未来数字化与AI深度落地，将持续释放效率红利。

2. **重构大暖通一体化，打造新增长曲线**：整合家用空调、智慧楼宇、水联网业务，从单一产品转向全屋气候解决方案。

3. **AI及数据中心高速发展带动冷却系统需求升级**：液冷方案及高功率密度散热技术成为重要增量市场。

4. **全球化布局深化**：新兴市场延续较快增长，持续拓展南亚、东南亚、中东非等区域。

### 4.4 资本配置意图

- **分红规划**：发布《未来三年（2026年-2028年）股东回报规划》，预计2026年现金分红比例不低于58%，2027-2028年不低于60%
- **股份回购**：拟以自有资金/自筹资金回购30-60亿元A股股份，股份将用于员工持股计划
- **研发投入**：2025年研发投入107.17亿元，占营业收入3.54%，研发人员25,913人

### 4.5 风险因素

- 原材料成本大幅上涨风险
- 外需及关税不及预期风险
- 行业竞争加剧、价格战风险
- 汇率波动风险（公司通过外汇衍生品进行对冲）
- 商誉账面价值273.00亿元，管理层评估无需计提减值准备
