### 决策理由

1. **估值严重偏高**：PE(TTM) 50.14倍，对应盈利收益率仅1.99%，远低于门槛值3.756%，即使100%分配也无法达标
2. **经营现金流恶化**：2024FY OCF为-6.19亿元，2023FY仅1.33亿元，利润无法转化为现金，应收账款从32.28亿元飙升至52.10亿元
3. **股息分配意愿极弱**：近3年平均支付率仅约11%，公司倾向于将利润投入研发和扩张而非回报股东
4. **资本消耗强度高**：年资本开支14-20亿元，在建工程从8.50亿元增至23.49亿元，属于重资产扩张期
5. **框架适配性问题**：联影医疗属于高增长医疗科技股，处于产能扩张期，非稳健投资策略的最佳适用标的

### 风险提示

1. **估值回归风险**：PE 50倍处于历史较高分位，若增长不及预期，估值压缩空间巨大
2. **应收账款坏账风险**：应收账款周转天数从83天延长至146天，回款速度显著放缓，坏账风险上升

4. **海外地缘风险**：海外收入占比22%且持续提升，关税政策和贸易保护主义可能增加运营成本

### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元）**

| 变量 | FY2021 | FY2022 | FY2023 | FY2024 | FY2025E |
|:---|---:|---:|---:|---:|---:|

| FCF = OCF−Capex | 688 | -4,006 | -1,432 | -1,863 | ⚠️待披露 |

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 维持性Capex系数 | 0.70 | 制造业标准，公司处于扩张期取中值 |
| 维持性资本开支估算 | 3.15 亿元 | 折旧摊销4.50亿 × 0.70 |
| Owner Earnings | **19.23 亿元** | 扣非净利润17.88 + 折旧摊销4.50 − 维持性Capex3.15 |
| 支付率锚定值 | 10% | 基于历史最低水平 |
| 年均回购金额 | 4.5 亿元 | 2023-2024回购均值 |
| 股息税率 | 0% | A股自然人持股超1年暂免 |

**粗算公式**：
```
粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值
               = [19.23 × 10% × (1-0%) + 4.5] / 946.5
               = [1.923 + 4.5] / 946.5
               = 6.423 / 946.5
               = 0.679%
```

- 分配能力验证：OCF -6.19亿(2024) < Owner Earnings 19.23亿 ❌（OCF严重不足）
- 分配意愿：**极弱** — 支付率仅10%，回购主要用于员工激励而非股东回报
- 税务泄漏：股息税率 = 0%（A股长期持有）

**EV口径双轨制检查**：
- 广义净现金/市值 = 40.94/946.5 = 4.33% < 40%，不触发EV口径
- 即使强制计算EV口径：EV = 946.5 - 40.94 = 905.56亿元
- R_EV = 6.423 / 905.56 = **0.709%**

> **粗算穿透回报率 R = 0.679%** — ❌ 否决（R = 0.679% < Rf = 1.756%，EV口径 R_EV = 0.709% 仍 < Rf）

**否决依据**：
1. R (0.679%) < Rf (1.756%) → 检查EV口径 → R_EV (0.709%) 仍 < Rf → **否决**
2. R (0.679%) < 门槛值×0.5 (1.878%) → 检查EV口径 → R_EV (0.709%) 仍 < 1.878% → **否决**
