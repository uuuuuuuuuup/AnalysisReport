## 决策理由

1. **穿透回报率严重不足**：粗算穿透回报率1.71%，低于十年期国债收益率1.762%，意味着以当前价格买入，即使公司100%分配利润，股东获得的现金回报也不及无风险利率，这是不可接受的。

2. **估值透支增长预期**：当前PE(TTM)约27.75倍，PB约4.02倍，而2024年营收增速仅3.02%、扣非净利润同比下降11.82%。市场定价隐含了持续高增长的预期，但实际增长已明显放缓。

3. **EV口径仍不达标**：公司广义净现金约38.7亿元，净现金/市值仅4.29%，EV口径回报率1.79%仍低于门槛值半数（1.88%），说明问题不在资本结构而在绝对估值水平。

4. **扣非净利润首次下滑**：2024年扣非净利润30.99亿元，同比下降11.82%，这是上市以来首次下滑，反映出核心盈利能力面临压力。

5. **商誉悬顶构成隐性风险**：商誉101.69亿元占资产总额30.58%，若行业持续低迷，商誉减值将对利润造成重大冲击。

---

## 风险提示

1. **商誉减值风险**：商誉101.69亿元（占资产30.58%），为审计关键事项，若并购医院业绩不达预期，可能触发大额减值。

2. **行业增速放缓**：2024年眼科行业受消费需求不足影响增速明显放缓，DRG医保支付改革和人工晶体集采政策可能进一步压缩利润空间。

3. **合规风险持续**：2025年至今累计超50家子公司收到监管罚单，涉及医保违规、税务等问题，反映内控体系存在薄弱环节。

4. **毛利率下行趋势**：2024年毛利率48.12%，同比下降2.67个百分点，若持续下行将侵蚀盈利能力。

## 后续跟踪事项

1. **2025年年报披露（预计2026-04-24）**：关注全年营收增速是否回升、扣非净利润是否恢复增长、商誉减值测试结果。

2. **毛利率走势**：跟踪2025年各季度毛利率是否企稳回升，若持续低于48%则需重新评估盈利模型。

3. **股价回落至安全区间**：若股价回落至4.43-5.70元区间（对应粗算回报率3.76%-4.83%），可重新启动分析。

## 关键发现

1. 爱尔眼科是民营眼科绝对龙头，市占率约28%（是第二名的3倍以上），护城河宽阔，品牌+规模+人才+牌照构成多重壁垒。

2. 2024年扣非净利润首次下滑（-11.82%），而归母净利润仍增长5.87%，差异主要来自4.13亿元公允价值变动收益等非经常性项目。

3. 公司有息负债率极低（约4.49%），财务结构稳健，广义净现金约38.7亿元。

4. 当前股价9.67元处于近5年低位区间（5年最低8.81元），但低位不等于低估——穿透回报率视角下仍不具备安全边际。

5. 2025Q1业绩超预期（收入+15.97%，扣非+25.78%），若全年恢复增长，估值压力可能缓解，但需等待全年数据验证。

### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元）**

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 维持性Capex系数 | 0.7 | 医疗服务业标准，设备更新+维护为主 |
| 维持性资本开支估算 | 7.00 亿元 | 折旧摊销10亿 × 系数0.7 |
| Owner Earnings | **38.56 亿元** | 归母净利润35.56 + 折旧摊销10.00 − 维持性资本开支7.00 |
| 支付率锚定值 | 40% | 近2年稳定在41%+，取40%偏保守 |
| 年均回购金额 | ~0 亿元 | 回购可忽略 |
| 股息税率 | 0% | A股持股>1年免征 |

**粗算公式**：

`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

= [38.56 × 0.40 × 1.0 + 0] / 901.77

= 15.42 / 901.77

= **1.71%**

**否决门检验**：

| 检验项 | 条件 | 结果 | 判定 |
|:---|:---|:---:|:---:|
| R vs Rf | R < Rf? | 1.71% < 1.762% | ⚠️ 触发，检查EV口径 |
| R_EV vs Rf | R_EV < Rf? | 1.79% > 1.762% | ✅ EV口径通过Rf检验 |
| R vs 门槛值×0.5 | R < 1.881%? | 1.71% < 1.881% | ⚠️ 触发，检查EV口径 |
| R_EV vs 门槛值×0.5 | R_EV < 1.881%? | 1.79% < 1.881% | ❌ EV口径仍不通过 |

- 分配能力验证：OCF 48.82亿 > Owner Earnings 38.56亿 ✅
- 分配意愿：**中** — 近2年支付率提升至41%+，但2022年仅24.22%，长期一致性待观察
- 税务泄漏：股息税率 = 0%（A股持股>1年）

> **粗算穿透回报率 R = 1.71%** — ❌ 否决（R < 门槛值×0.5，EV口径1.79%仍低于1.881%）
