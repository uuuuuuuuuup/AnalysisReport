## 四、MD&A 深度信息

### 4.1 经营回顾

**2025年度经营概况：**
- 营业收入188.06亿元，同比下降0.71%
- 归母净利润52.72亿元，同比增长0.7%
- 扣非净利润50.45亿元，同比下降0.64%
- 经营活动现金流净额55.49亿元，同比增长7.68%
- 公司及合营/联营企业全年完成货物吞吐量7.22亿吨，同比增长

**分业务板块表现：**

| 业务板块 | 营收（亿元） | 同比 | 分部业绩 | 毛利率 |
|---|---|---|---|---|
| 集装箱处理及配套服务 | 27.00 | +60.90% | 31.56亿元（+32.1%） | 75.19% |
| 干散杂货 | - | -2.70%（吞吐量） | 下降29.1% | - |
| 液体散货 | - | -4.10%（吞吐量） | 下降22.8% | - |
| 物流及港口增值服务 | - | -6.40% | 增长1.4% | 提升2.62pct |

> 集装箱业务成为核心增长引擎，2025年集装箱分部实现净利润26.8亿元，同比+26.8%，利润占比46.8%，较去年提升9.9pct。干散货受粮食、煤炭等货种业务量减少及投资收益减少影响；液体散货受外部因素及国内炼厂开工率较低影响。

### 4.2 前瞻指引（2026年经营计划）

- 新增航线不少于10条
- 推进"矿石超市"建设
- 持续巩固中国北方第一大油品外贸口岸地位
- 加快前湾港区北岸集装箱升级改造工程
- 推进董家口相关重点项目建设

### 4.3 资本配置意图

**重大资本开支项目：**

| 项目 | 投资估算 |
|---|---|
| 董家口港区琅琊台湾作业区东部集装箱码头一期工程 | 约90.97亿元 |
| 董家口港区湾底通用码头工程 | 约66.15亿元 |
| 合计 | 约157.12亿元 |

> 公司明确推进港口基础设施、智慧化和绿色化提前布局，2025年投资活动现金流净额-31.12亿元，较上年-0.39亿元大幅增加，主要因结构性存款及三个月以上定期存款净流出12.70亿元，以及购建固定资产、在建工程等支出较上年增加。

### 4.4 风险因素

1. **市场风险**：全球贸易格局波动、地缘政治冲突可能影响货物吞吐量，尤其是液体散货、干散杂货等对外部环境敏感的货种
2. **行业竞争风险**：环渤海港口群竞争加剧，周边港口在航线布局、腹地货源争夺上的竞争可能分流公司业务
3. **经营风险**：港口基础设施投资规模大、周期长
4. **同业竞争风险**：公司与山东港口集团及其部分子公司主营业务存在一定同业竞争，山东港口集团承诺5年内解决同业竞争问题（承诺窗口期正在缩短）
5. **地缘政治波动风险**：国际地缘政治波动对原油进口造成一定扰动
