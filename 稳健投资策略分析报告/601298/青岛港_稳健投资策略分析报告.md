### 决策理由

1. **护城河极宽但估值已反映**：青岛港拥有深水良港自然垄断、"一省一港"政策壁垒、北方第一航线密度等多重护城河，但当前PE~10.9倍、PB~1.26倍已部分反映这些优势，精算回报率仅勉强超过门槛值
2. **分红稳定但支付率偏低**：43%支付率、3.92%股息率提供一定收益支撑，但支付率偏低限制了穿透回报率的上行空间；累计派现150.6亿远超融资55.95亿，分配意愿中等偏强
3. **安全边际极薄**：精算回报率3.95%仅超门槛值0.19pct，收入下降5%即跌破门槛值，抗风险缓冲不足
4. **资本开支压力**：董家口港区两大项目合计约157亿元投资，未来可能推高有息负债、压缩自由现金流

### 风险提示

1. **安全边际不足**：精算回报率仅超门槛值0.19pct，收入下降约5%即跌破门槛值，当前价格缺乏足够安全垫
2. **资本开支风险**：董家口港区157亿重大投资可能推高有息负债（2025年已同比增长41.3%），挤压自由现金流和分红空间
3. **同业竞争待解**：山东港口集团承诺5年内解决同业竞争问题，窗口期缩短，可能引发资产范围变化
4. **干散货/液体散货承压**：2025年干散货分部业绩下降29.1%，液体散货下降22.8%，若宏观经济持续放缓将拖累整体盈利

### 后续跟踪事项

1. **2026年中报关注点**：集装箱业务增速是否维持，干散货/液体散货是否企稳回升（2026年8月）
2. **董家口项目进展**：资本开支节奏及融资方案，关注有息负债率变化（持续跟踪）
3. **同业竞争解决进展**：山东港口集团承诺窗口期及可能的资产注入/剥离方案（2027年前）

## 关键发现

1. 青岛港拥有**极宽护城河**——胶州湾深水良港不可复制、北方唯一可停靠40万吨级矿石船的港口、"一省一港"政策强化区域垄断
2. **集装箱业务成为核心增长引擎**：2025年集装箱处理及配套服务营收27亿元（+60.9%），分部净利润26.8亿元（+26.8%），利润占比46.8%
3. **安全边际极薄**：精算穿透回报率3.95%仅超门槛值0.19pct，临界收入倍数0.95×（收入下降5%即不达标），韧性弱
4. **广义净现金103.3亿元**但占市值仅18%，未触发EV口径双轨制，现金保护等级为"无保护"
5. **157亿重大资本开支**将显著推高有息负债，2025年有息负债已同比增长41.3%至41.56亿元

### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元）**

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.7 | 港口行业标准，固定资产占比高但维持性投入相对可预测 |
| 维持性资本开支估算 | 7.70 亿元 | 折旧摊销11亿 × 系数0.7 |
| Owner Earnings | **53.75 亿元** | 扣非归母净利润50.45 + 折旧摊销11 − 维持性资本开支7.70 |
| 支付率锚定值 | 43% | 取2025年度实际支付率 |
| 年均回购金额 | 0 亿元 | 仅有授权，尚无实施 |
| 股息税率 | 0% | A股持股超过1年免征红利税 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`
= [53.75 × 43% × 100% + 0] / 572.52
= 23.11 / 572.52
= **4.04%**

- 分配能力验证：OCF 55.49 > Owner Earnings 53.75 ✅
- 分配意愿：**中等偏强** — 43%支付率，累计派现150.6亿远超融资55.95亿，派现融资比2.69倍
- 税务泄漏：股息税率 = 0%（A股持股>1年免征）

> **粗算穿透回报率 R = 4.04%** — ⚠️ 边际通过（门槛值 = 3.762%，超额仅0.28pct）
