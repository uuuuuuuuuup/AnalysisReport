## 决策理由

1. **穿透回报率严重不足**：粗算穿透回报率仅1.69%，远低于门槛值3.75%，差距达2.06个百分点。即使考虑经营现金流改善，盈利基数过小（归母净利润仅0.75亿元）难以支撑合理回报。

2. **应收账款风险突出**：2025年应收账款35.50亿元，同比增长126.57%，远高于营收增速46.64%。应收账款周转天数虽从98.67天降至74.34天，但绝对金额激增意味着回款压力加大，现金质量存疑。

3. **盈利质量存疑**：2025年扭亏为盈主要依赖：①资产减值准备大幅减少；②西北旺新村D1地块清算、太阳宫D区地块收储等非经常性项目贡献。扣非净利润0.75亿元与经营现金流15.43亿元差异巨大，主要因存货减少释放现金42.29亿元（不可持续）。

4. **业绩波动剧烈**：近5年净利润分别为-9.42亿（2021）、1.52亿（2022）、-3.94亿（2023）、-6.33亿（2024）、0.75亿（2025），呈现"一年盈两年亏"的周期性特征，盈利稳定性差。

5. **广义净现金为负**：广义净现金 = 货币资金12.34亿 − 有息负债17.83亿 = -5.49亿元，现金保护等级为"无保护"，安全边际折扣需30%。

## 风险提示

1. **应收账款回收风险**：应收账款增速（+126.57%）远超营收增速（+46.64%），若下游客户经营恶化或回款周期延长，可能面临大额坏账损失。

2. **盈利可持续性风险**：2025年扭亏为盈依赖非经常性项目（地产清算、土地收储）和存货释放，核心电磁线业务毛利率仅9.74%，盈利能力薄弱。

3. **控股股东股权风险**：控股股东福建丰榕投资持股35.42%，2025年部分股份（1,400万股）遭司法拍卖，股权稳定性受关注。

4. **周期性波动风险**：公司业绩受原材料价格（铜价）和下游需求（电网投资、新能源汽车）影响大，呈现强周期性特征。

## 关键发现

1. **股息率行业第一但盈利支撑不足**：当前股息率5.22%（行业排名第1），但ROE仅1.30%，盈利能力难以支撑持续高分红。2025年分红率54.77%，但归母净利润基数过小（0.75亿元），实际派现仅0.41亿元。

2. **经营现金流与净利润严重背离**：2025年经营现金流15.43亿元，归母净利润仅0.75亿元，差异主要来自存货减少42.29亿元。这种"去库存释放现金"的模式不可持续。

3. **电磁线业务增长但毛利承压**：电磁线产销量创历史新高，收入增长46.64%，但毛利率从2022年的16.04%降至2025年的9.74%，反映行业竞争加剧或成本压力。

4. **战略转型进行中**：公司积极推进"去地产化"，聚焦电磁线主业，但转型效果需时间验证。房地产业务仍贡献非经常性收益，未来可持续性存疑。

5. **新能源汽车扁线技术领先**：公司新能源汽车扁线已批量供货比亚迪、汇川技术等头部电机厂，技术护城河中等，但行业竞争激烈，技术优势能否转化为持续盈利待观察。

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 维持性Capex系数 | 0.7 | 制造业标准 |
| 维持性资本开支估算 | 0.40 亿元 | 资本开支 × 系数（估算） |
| Owner Earnings | **0.75 亿元** | 扣非归母净利润（保守口径） |
| 支付率锚定值 | 54.77% | 2025年分红率 |
| 年均回购金额 | 0.75 亿元 | 2024年回购计划中值 |
| 股息税率 | 0% | 持股>1年免税 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`  
= [0.75 × 54.77% × (1−0%) + 0.75] / 53.88  
= [0.41 + 0.75] / 53.88  
= 1.16 / 53.88  
= **2.15%**

**修正说明**：由于年均回购金额0.75亿元为2024年一次性事件，不应作为持续性回报，修正后：
= [0.75 × 54.77% × (1−0%)] / 53.88  
= 0.41 / 53.88  
= **0.76%**

**进一步修正**：考虑到公司历史分红不稳定（近5年仅2024年、2025年分红），支付率锚定值应保守估计为30%：
= [0.75 × 30% × (1−0%)] / 53.88  
= 0.23 / 53.88  
= **0.42%**

> **粗算穿透回报率 R = 0.42% ~ 2.15%**（取中值 **1.69%**）— ❌ 否决（门槛值 = 3.75%）

**分配能力验证**：
- OCF 15.43亿 > Owner Earnings 0.75亿 ✅
- 但OCF与净利润差异巨大，主要因存货减少释放现金（不可持续）
