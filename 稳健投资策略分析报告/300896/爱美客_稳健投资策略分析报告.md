### 决策理由

1. **分配意愿极弱（核心否决原因）**：过去3年股息支付率仅4.67%/5.85%/4.83%，均值5.12%，公司保留95%以上利润用于扩张。即使假设100%支付率，粗算穿透回报率也仅为3.37%，仍低于3.75%门槛值。在当前估值水平下，股东通过分红获得的现金回报极其微薄。

2. **估值偏高，安全边际不足**：当前PE(TTM) 30.39倍、PB 4.42倍，对于一家2025年营收同比下降18.94%、净利润同比下降33.47%的公司而言，估值缺乏安全边际。FCF yield约3.22%，勉强接近但未达到门槛值。

3. **行业竞争加剧，护城河面临侵蚀**：2025年上市以来首次营收净利双降，核心产品嗨体面临价格下行压力，行业新品密集入市。虽然技术壁垒和品牌壁垒仍在，但竞争格局正在恶化。

4. **商誉减值风险突出**：收购韩国REGEN产生商誉约16.41亿元，占总资产18.42%。若REGEN经营不及预期，将面临重大减值风险，直接侵蚀股东权益。

5. **策略框架不匹配**：爱美客是典型的成长型公司（低分配、高再投资），与稳健投资策略强调的"高分红+安全边际"框架存在根本性错配。公司选择将利润用于并购扩张和研发投入，而非回报股东，这在成长股框架下可能合理，但在稳健策略下构成否决。

---

### 风险提示

1. **分配政策风险**：公司历年股息支付率极低（4-6%），且无明确提升分配的计划。若未来继续维持低分配，股东回报将长期依赖股价波动，缺乏确定性现金流入。

2. **商誉减值风险**：REGEN商誉16.41亿元，占股东权益20.10%。2025年汇兑损失约2,303万元，REGEN与原经销商达透医疗的仲裁纠纷尚未解决，经营不确定性较高。

3. **行业景气下行风险**：中国医美行业2025年整体承压，终端需求疲软，价格竞争加剧。若行业景气持续低迷，公司营收恢复可能不及预期。

4. **并购整合风险**：1.9亿美元收购REGEN是公司首次大规模跨境并购，面临汇率波动、海外监管差异、文化差异等多重挑战。

## 关键发现

1. 爱美客是医美注射耗材龙头，毛利率92.70%极高，零有息负债，财务结构极为保守，商业模式本身优秀。

2. 2025年上市以来首次营收净利双降（营收-18.94%，归母净利-33.47%），但26Q1降幅连续4季度收窄至-4.48%，显示边际改善。

3. **核心否决因素**：股息支付率仅4.67%，粗算穿透回报率0.31%，远低于门槛值3.75%。即使100%支付，回报率也仅3.37%，仍不达标——这反映了当前估值水平下的回报不足。

4. 收购REGEN产生16.41亿元商誉（占总资产18.42%），是公司最大的资产负债表风险点。

5. 公司属于成长型投资标的，与稳健投资策略"高分红+安全边际"框架存在根本性错配，并非公司基本面问题。

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.6 | 轻资产医耗材公司标准 |
| 维持性资本开支估算 | 0.36 亿元 | 折旧摊销0.60亿 × 系数0.6 |

| 股息税率 | 0% | A股长期持有(>1年)免税 |

**粗算公式（扣非口径）**：

```
粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值
               = [112.4 × 5.12% × 100% + 0.5] / 348.1
               = [5.75 + 0.5] / 348.1
               = 6.25 / 348.1
               = 0.0180 → 约1.80%
```

> 修正：Owner Earnings = 11.24 亿元 = 1,124 百万元
> 
> 粗算穿透回报率 = [1,124 × 5.12% × 1 + 5] / 34,810 = [57.55 + 5] / 34,810 = 62.55 / 34,810 = **0.18%**

> 再次修正（年均回购0.5亿元=50百万元）：
> 
> 粗算穿透回报率 = [1,124 × 5.12% × 1 + 50] / 34,810 = [57.55 + 50] / 34,810 = 107.55 / 34,810 = **0.31%**
