### 决策理由

1. **护城河宽但正在收窄**：11张III类医疗器械注册证（国内最多）构成强合规壁垒，但华熙生物已打破嗨体颈纹独家垄断，童颜针赛道从少数玩家增至9款，竞争格局显著恶化
2. **业绩首度双降且趋势未明**：2025年营收-18.9%、归母净利润-34.1%，为上市以来首次下滑，核心产品量价齐跌，新增长点尚未形成规模
3. **当前估值不提供安全边际**：粗算穿透回报率2.14%远低于门槛值3.76%，即使以2024年常态化盈利计算，回报率亦仅3.22%，勉强接近门槛值
4. **大额商誉悬顶**：收购REGEN形成商誉16.41亿元（占总资产18.4%），叠加AestheFill经销权仲裁索赔16亿元，存在显著下行风险
5. **现金储备雄厚提供缓冲**：广义净现金41.30亿元，几乎零有息负债，为度过行业低谷提供充足弹药

### 风险提示

1. **竞争持续加剧**：核心产品嗨体垄断被打破，童颜针赛道玩家激增，若价格战持续则毛利率可能从92.7%进一步下滑
2. **商誉减值风险**：REGEN商誉16.41亿元，若AestheFill国内推广受阻（仲裁限制）或海外业绩不及预期，可能触发大额减值
3. **AestheFill仲裁案**：原经销商达透医疗索赔16亿元，仲裁期间不得自行在中国内地销售该产品，直接影响REGEN国内变现
4. **消费需求疲软**：宏观消费降级影响医美消费频次和客单价，行业景气度下行趋势短期难逆转

### 后续跟踪事项

1. **2026Q1/Q2业绩拐点确认**：关注Hutox肉毒素上市后放量情况及核心产品售价趋稳信号（2026年4-8月）
2. **AestheFill仲裁裁决**：跟踪深圳国际仲裁院裁决结果及对公司国内销售策略的影响
3. **REGEN并表完整年度表现**：2026年为REGEN首个完整并表年度，关注收入贡献及商誉减值测试结果（2026年年报）

| 参数 | 值 | 说明 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

= [13.11 × 50% × 100% + 0] / 347.69 = 6.555 / 347.69 = **1.88%**（市值口径）

= 6.555 / 306.39 = **2.14%**（EV口径，EV = 347.69 − 41.30）

- 分配能力验证：OCF 13.24 > Owner Earnings 13.11 ✅
- 分配意愿：**中等** — 2024年支付率58.5%体现较强意愿，但2025年降至46.7%，且无持续回购计划
- 税务泄漏：股息税率 = 0%（持股>1年）

> **粗算穿透回报率 R = 2.14%（EV口径）** — ⚠️ 边际不达标（门槛值 = 3.76%，R > 门槛值×0.5 = 1.88% 但 R < 门槛值）
