## 四、MD&A 深度信息

> 数据来源：WebSearch - 爱美客2025年报管理层讨论与分析、券商研报

### 4.1 经营回顾

**2025年度经营概况**：
- 营业收入24.53亿元，同比下降18.94%——上市以来首次营收下滑
- 归母净利润12.91亿元，同比下降34.05%——上市以来首次净利润下滑
- 扣非净利润10.99亿元，同比下降41.30%
- 毛利率92.70%，较上年下降1.94个百分点
- 研发投入3.60亿元，同比增长18.47%，研发投入占营收比14.67%

**分产品收入结构**：
| 产品线 | 2025收入 | 同比变动 |
|--------|--------:|---------|
| 溶液类注射产品 | 12.65亿元 | -27.48% |
| 凝胶类注射产品 | 8.90亿元 | -26.82% |
| 冻干粉类注射产品 | 2.08亿元+ | 新增并表 |

**业绩下滑原因**：
1. 医美行业景气度变弱，终端需求疲软
2. 行业新品密集入市，价格竞争加剧
3. 核心产品（嗨体、濡白天使等）面临价格下行压力
4. 费用扩张影响盈利能力（销售费用3.87亿元，同比+39.76%）

### 4.2 前瞻指引

- **产品管线**：重组透明质酸酶、司美格鲁肽注射液、去氧胆酸注射液三款在研产品已获临床试验批件
- **新品上市**：米诺地尔搽剂已获药品注册证书；注射用A型肉毒毒素已于2026年1月获批
- **国际化**：收购韩国REGEN公司85%股权，获得AestheFill与PowerFill两款全球产品
- **体重管理赛道**：重点布局司美格鲁肽产品研发，形成"医美+大健康"双轮驱动
- **26Q1改善信号**：营收6.34亿元，同比-4.48%，降幅连续4个季度收窄（25Q2: -25.11% → 25Q3: -21.27% → 25Q4: -9.61% → 26Q1: -4.48%）

### 4.3 资本配置意图

- **并购整合**：1.9亿美元收购韩国REGEN公司85%股权，拓展全球渠道
- **上游延伸**：收购成都众兴美源生物科技，布局医美原料研发及生产制造
- **设备布局**：投资上海维脉，布局医美能源设备领域
- **委托理财**：拟使用最高不超过50亿元闲置自有资金进行委托理财（银行结构性存款、大额存单、收益凭证等）
- **产能建设**：在建工程253.14百万元，主要系昌平生命谷产业基地等项目建设

### 4.4 风险因素

| 风险类型 | 具体描述 |
|----------|---------|
| 行业竞争加剧风险 | 医美行业供给端新产品不断涌现，传统主力产品面临价格竞争及市场份额被分流压力 |
| 产品研发与注册风险 | 医美产品研发周期长、投入大，监管审评标准不断提高，在研项目存在研发失败或注册进度不及预期风险 |
| 商誉减值风险 | 收购REGEN后商誉激增至16.41亿元（同比增长近5倍），若被收购资产经营不及预期将面临减值 |
| 汇率波动风险 | 国际化业务增加，外币结算比重上升，境外投资购汇产生汇兑损失（2025年汇兑损失约2,303万元） |
| 应收账款风险 | 客户信用风险需持续关注 |
| 新品商业化风险 | 新品推广需要阶段性费用率偏高，短期利润承压 |
