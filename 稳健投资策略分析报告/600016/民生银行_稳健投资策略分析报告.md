## 决策理由

1. **价值陷阱风险触发**：ROE连续5年下行（6.59%→4.93%）、支付率从~30%降至~27%、NIM从1.91%压缩至1.40%，三项价值陷阱特征同时存在，按框架规则应排除
2. **安全边际极薄**：修正后安全边际仅+0.43 pct，几乎无缓冲空间，任何盈利波动都可能使回报率跌破门槛值
3. **分红可持续性存疑**：2025年末期股息仅0.053元/股（10派0.53），较2023年2.16元/10股大幅缩水75%，反映资本补充压力下的分配能力约束
4. **盈利能力趋势性恶化**：归母净利润从2021年343.81亿降至2025年305.63亿，5年CAGR为-2.88%，营收虽在2025年转正（+4.82%），但信用减值损失大增18.64%吞噬利润
5. **低PB不等于安全**：市净率0.29倍看似极具吸引力，但银行PB深度折价反映市场对资产质量的持续担忧——损失类贷款同比+23.58%、房地产不良率仍达3.61%

---

## 关键发现

1. **营收拐点初现但利润仍在探底**：2025年营收+4.82%终结四年连降，但归母净利润-5.37%连续两年下滑，信用减值损失539.50亿（+18.64%）是核心拖累
2. **房地产风险大幅出清**：房地产不良贷款余额降幅近30%，不良率压降至3.61%，是年报中最大的积极信号
3. **分红断崖式缩水**：2025年度末期股息仅10派0.53元，全年合计0.189元/股，支付率约27%，远低于2021-2023年约30%的稳定水平
4. **ROE持续下行至4.93%**：远低于银行业平均（约10%），也低于自身资本成本，意味着股东价值正在被侵蚀
5. **股权高度分散无实控人**：第一大股东大家保险系持股仅16.79%，2027年董事会换届可能带来治理变数

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | ~1.0 | 银行业D&A≈物理Capex，大致相抵 |
| 维持性资本开支估算 | ~100 亿元 | 折旧摊销×系数 |
| Owner Earnings | **305.63 亿元** | 采用归母净利润（D&A-Capex影响<3%可忽略） |
| 支付率锚定值 | 25% | 保守选取，近3年均值27.6%但趋势下行 |
| 年均回购金额 | 0 亿元 | 历史上无回购注销 |
| 股息税率 | 0% | A股个人持股>1年免征 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值` = [305.63 × 0.25 × 1 + 0] / 1,637 = **4.67%**

- 分配能力验证：2025年实际分红82.74亿 < Owner Earnings×支付率 = 76.41亿 ❌（实际分红超过锚定分配额，因实际支付率27.1%>锚定25%）
- 分配意愿：**弱** — 支付率从~30%降至~27%，末期股息大幅缩水，资本补充优先于股东回报
- 税务泄漏：股息税率 = 0%（A股个人持股>1年）

> **粗算穿透回报率 R = 4.67%** — ✅ 通过（门槛值 = 3.738%），但超出幅度仅0.93 pct，属于边际通过

### 因子 3：精算穿透回报率 + 现金质量审计
