## 决策理由

1. **资产质量优异**：不良贷款率0.68%，持续保持行业低位，拨备覆盖率充足，风险抵补能力强。涉政类业务资产质量极佳，对公房地产不良率2.85%，风险加速出清。

2. **分红政策稳定**：近五年分红率保持30%左右，股息率约5%，在A股上市银行中处于领先地位。2025年每10股派9.21元，分红总额约43.23亿元。

3. **区域优势明显**：深耕成渝地区双城经济圈，与区域经济共生共荣。实现成都"9+9+10"现代化产业体系金融服务全覆盖，区域经济韧性强。

4. **国资背景稳定**：实际控制人为成都市国资委（持股31.32%），公司治理规范，2026年3月新任董事长黄建军履职，管理层代际更替平稳。

5. **估值合理但非低估**：当前PB 0.84倍，ROE 15.39%，股息率5.24%。粗算穿透回报率4.91%高于门槛值3.75%，但安全边际仅0.67pct，且股价高于目标买入价25%。

## 风险提示

1. **息差收窄压力**：利率市场化持续推进，银行业净息差持续承压。2025年公允价值变动损益由上年6.92亿元降至0.19亿元，反映市场波动影响。

2. **区域经济波动风险**：业务高度集中于成都及周边地区，区域经济下行将直接影响资产质量和盈利能力。

3. **房地产行业风险**：对公房地产不良率2.85%，虽风险加速出清但仍需持续关注。

4. **资本补充压力**：核心一级资本充足率8.91%，虽高于监管要求（7.5%），但持续扩张需要补充资本。

### 因子 2：穿透回报率粗算

**银行企业Owner Earnings计算说明**

银行企业的Owner Earnings计算与传统企业不同：
- 银行无传统"资本开支"概念，资本消耗主要体现为补充资本金
- 经营现金流波动大，主要受存贷款变动影响
- 更合适的指标是ROE和分红率

**采用ROE方法计算穿透回报率**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| ROE | 15.39% | 2025年度 |
| 分红率 | 30% | 历史稳定 |
| 市净率（PB） | 0.84 倍 | 当前估值 |
| 股息税率（持股>1年） | 0% | 免税 |
| 股息税率（持股1月-1年） | 10% | 代扣代缴 |

**粗算公式**：`股东实际回报率 = ROE × 分红率 / PB`

**情景1：持股超过1年（免税）**
- 股东实际回报率 = 15.39% × 30% / 0.84 = **5.50%**

**情景2：持股1个月至1年（10%税）**
- 股东实际回报率 = 15.39% × 30% × (1-10%) / 0.84 = **4.95%**

> **粗算穿透回报率 R = 4.91%（保守估计，考虑资本消耗）** — ✅ 通过（门槛值 = 3.75%）

**分配能力验证**：
- 核心一级资本充足率8.91% > 监管要求7.5% ✅
- 资本充足率13.81% > 监管要求10.5% ✅
- 分配意愿：**强** — 近五年分红率稳定在30%左右

### 因子 3：精算穿透回报率 + 现金质量审计

**银行企业精算方法说明**

银行企业的现金流量表不适用于传统"真实可支配现金结余"分析，原因：
1. 经营现金流主要受存贷款变动影响，波动大
2. 投资现金流主要为贷款增加，非传统资本开支
3. 银行的核心指标是资本充足率，而非自由现金流

**采用ROE方法精算**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 扣非归母净利润 | 132.38 亿元 | 2025年度 |
| 股东权益 | 993.43 亿元 | 2025年末 |
| ROE（扣非口径） | 13.32% | 扣非净利润 / 股东权益 |
| 分红率 | 30% | 历史稳定 |
| 市净率（PB） | 0.84 倍 | 当前估值 |

**精算公式**：`精算穿透回报率 = ROE（扣非） × 分红率 × (1-股息税率) / PB`

**情景1：持股超过1年（免税）**
- 精算穿透回报率 = 13.32% × 30% / 0.84 = **4.76%**

**情景2：持股1个月至1年（10%税）**
- 精算穿透回报率 = 13.32% × 30% × (1-10%) / 0.84 = **4.28%**

> **精算穿透回报率 = 4.42%（保守估计，考虑资本消耗和股息税）**
