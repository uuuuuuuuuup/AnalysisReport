- 非航空性收入（含免税）：约73.57亿元
- 其他收入：7.92亿元

### 4.2 前瞻性指引

1. **浦东机场四期扩建**：持续推进四期扩建工程，目标2028年实现旅客吞吐量1.2亿人次
2. **免税新周期**：新一轮免税经营合同谈判及招标将是重要变量，日上上海及日上中国未中标新一轮免税经营业务
3. **T3航站楼建设**：与机场集团签署T3航站楼北旅客过夜用房租赁合同
4. **海南自贸港分流风险**：海南自贸港封关运作后，高端免税消费存在分流风险

### 4.3 资本配置意图

1. **重大资本开支**：浦东四期扩建等重大项目年均资本投入约80亿元
2. **分红政策**：2025年拟10派2.6元，持续回报股东
3. **现金管理**：使用闲置募集资金不超过7亿元购买本金保障类理财产品，使用闲置自有资金购买理财产品
4. **限制性股票激励**：实施限制性股票激励计划，绑定核心人才

### 4.4 风险因素

1. **国际航线恢复不及预期**：地缘政治风险影响国际航线恢复节奏
2. **高铁分流**：国内高铁网络扩展对中短途航空出行形成替代
3. **资本开支压力**：浦东四期扩建等重大项目需要巨额资本投入，可能推高有息负债和财务费用
4. **免税业务竞争**：海南自贸港封关运作后高端免税消费分流风险
5. **国内核心枢纽机场竞争加剧**：其他核心枢纽机场加速国际航线布局
6. **安全运营风险**：民航华东地区管理局曾对公司安全保卫问题进行调查
