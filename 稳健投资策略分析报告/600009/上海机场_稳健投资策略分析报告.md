## 决策理由

1. **穿透回报率严重不足**：粗算穿透回报率1.03%（扣非口径+35%支付率），远低于门槛值3.756%，即使使用2025年年化数据+50%支付率的乐观假设，也仅达2.01%，仍低于门槛值
2. **估值偏高**：当前PE 31.4倍、股息率仅1.855%，市场定价已充分反映国际航线恢复预期，缺乏安全边际
3. **免税业务结构性承压**：人均免税客单价从2019年350元骤降至2024年120元，离岛免税与跨境电商持续分流，新合同虽稳定但增长空间有限
4. **资本开支压力**：浦东机场四期扩建总投资约500亿元，2027年后折旧费用预计增至25亿元，将显著侵蚀自由现金流

---

## 风险提示

1. **国际航线恢复不及预期**：中美航班量仅恢复至2019年50%，地缘政治风险可能延缓复苏节奏
2. **免税消费持续低迷**：客单价恢复乏力，若无法重构免税商品竞争力，非航业务毛利率或进一步承压
3. **四期扩建成本超预期**：总投资500亿元，投产后折旧和运营成本将大幅增加
4. **高铁分流**：短途航线面临高铁替代效应，国内航线增长受限

---

## 后续跟踪事项

1. **2025年报披露**：关注全年扣非净利润及新免税合同实际贡献（预计2026年3-4月披露）
2. **国际航线恢复节奏**：关注中美/中欧航班量恢复进度，若恢复至2019年80%以上将显著提振业绩
3. **四期扩建进展**：关注资本开支节奏及融资安排，评估对自由现金流的影响

## 关键发现

1. 上海机场拥有极强的区域垄断护城河，浦东机场国际旅客量占全国40%+，不可替代
2. 公司现金储备极为充裕（广义现金171.87亿元，有息负债仅7亿元），资产负债表健康
3. 当前估值（PE 31.4x）已充分反映恢复预期，从稳健投资策略角度看缺乏安全边际
4. 2020-2023年连续4年未分红，2024年恢复分红但支付率仅28.3%，分配意愿仍需验证
5. 新免税合同采用"固定租金+销售提成+49%合资公司分红"模式，降低了波动性但也限制了上行空间

### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元）**

| 变量 | FY2020 | FY2021 | FY2022 | FY2023 | FY2024 |
|:---|---:|---:|---:|---:|---:|
| 归母净利润 | -1,168.58 | -1,341.42 | -2,789.98 | 1,189.10 | 2,225.02 |
| 折旧摊销（估） | 1,800 | 1,900 | 1,950 | 2,000 | 2,000 |
| 资本开支 | 62 | 82 | 259 | 2,834 | 3,250 |
| FCF = OCF−Capex | -1,279.56 | 1,767.30 | -1,358.39 | 1,185.92 | 2,280.50 |

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 维持性Capex系数 | 0.7 | 重资产基础设施类，取中高值 |
| 维持性资本开支估算 | 14.00 亿元 | 折旧摊销20亿 × 系数0.7 |
| Owner Earnings | **20.09 亿元** | 扣非归母14.09 + 折旧摊销20 − 维持性Capex14 |
| 支付率锚定值 | 35% | 保守取值（2019年30.7%，2025年中期50.05%） |
| 年均回购金额 | 0 亿元 | 无显著回购 |
| 股息税率 | 0% | A股直接持有>1年免税 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

= [20.09 × 35% × 100% + 0] / 684 = 7.03 / 684 = **1.03%**

- 分配能力验证：OCF 55.31 > Owner Earnings 20.09 ✅
- 分配意愿：**中等** — 2020-2023年连续4年未分红，2024年恢复但支付率仅28.3%，2025年中期提升至50.05%，趋势向好但持续性待验证
- 税务泄漏：股息税率 = 0%（A股长期持有）

**EV口径检查**：
- 广义净现金/市值 = 164.87/684 = 24.1% < 40%，EV口径非强制
- EV = 684 − 164.87 = 519.13 亿元
- EV口径粗算 = 7.03 / 519.13 = 1.35%

> **粗算穿透回报率 R = 1.03%** — ❌ 否决（R = 1.03% < Rf 1.756%，EV口径1.35%仍 < Rf 1.756%）

**乐观情景（2025年年化+50%支付率）**：
- 2025年前三季度扣非归母净利润16.09亿，年化约21.45亿
- Owner Earnings = 21.45 + 20 − 14 = 27.45亿
- 粗算 = [27.45 × 50% × 100% + 0] / 684 = 13.73 / 684 = **2.01%**
- 门槛值×0.5 = 1.878%，2.01% > 1.878% 但 < 门槛值3.756% → ⚠️ 边际不达标
