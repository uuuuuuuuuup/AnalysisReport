**粗算公式**: 

> **粗算穿透回报率 R = 3.11%** — ❌ 不达标(门槛值 = 3.731%)

**EV口径粗算**:
```
EV = 市值 − 广义净现金 = 55.60 − 33.13 = 22.47亿元

```
