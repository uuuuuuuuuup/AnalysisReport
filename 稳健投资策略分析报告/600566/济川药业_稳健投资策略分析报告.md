## 决策理由

1. **高分红提供安全垫**：2024年度分红率75.7%，股息率TTM约7.5%，在当前低利率环境下具有吸引力，但高分红可持续性取决于业绩恢复
2. **核心产品面临集采压力**：小儿豉翘清热颗粒和蒲地蓝消炎口服液贡献主要收入，若被纳入集采降价50%，预计减少年利润3.2-3.8亿元
3. **业绩处于下行通道**：2024年营收同比-16.96%，2025H1营收同比-31.87%、净利润同比-45.93%，短期拐点尚未确认
4. **资产负债表极其健康**：有息负债接近零，广义净现金约75亿元/市值32.9%，为分红和抵御风险提供强保障

---

## 风险提示

1. **集采政策风险**：核心品种小儿豉翘清热颗粒若纳入集采，价格体系将面临重塑，预计影响利润3.2-3.8亿元/年
2. **业绩持续下滑风险**：2025H1营收同比下降31.87%，若Q3/Q4未能显著恢复，全年业绩可能大幅低于预期
3. **产品集中度风险**：蒲地蓝和小儿豉翘合计收入占比超77%，单一产品风险突出
4. **应收账款回收风险**：应收账款周转天数从89天(2023)拉长至132天(2025Q3)，回款速度放缓

---

## 后续跟踪事项

1. **2025年年报发布**(预计2026年4月底)：关注全年营收是否恢复增长，以及2026年业绩指引
2. **集采政策动态**：关注小儿豉翘清热颗粒是否被纳入省级或全国集采
3. **2025Q3财报**：确认H2是否出现业绩拐点，营收环比是否改善

## 关键发现

1. 济川药业是典型的高ROE+高分红+低负债价值股，但2024-2025年业绩持续下滑削弱了投资吸引力
2. 精算穿透回报率(2024单年)仅3.97%，勉强超过门槛值3.77%，安全边际极薄(+0.20 pct)
3. 广义净现金/市值达32.9%，资产负债表极其健康，但现金主要来自历史积累而非当期经营
4. 2025年6月实际控制人变更为曹龙祥、曹飞父子，触发全面要约收购，治理结构需持续关注
5. 3年均值口径下精算回报率5.39%显著高于门槛值，但2025H1数据暗示均值可能高估

**粗算公式**：粗算穿透回报率 = [Owner_Earnings x 支付率锚定值 x (1-股息税率) + 年均回购] / 当前市值

- 分配能力验证：OCF 21.76 < Owner Earnings 23.44 (2024年经营现金流偏弱)
- 分配意愿：**强** — 分红率从32.3%持续提升至75.7%
- 税务泄漏：股息税率 = 0%(长期持有)

> **粗算穿透回报率 R = 5.26%** — 通过(门槛值 = 3.77%)
