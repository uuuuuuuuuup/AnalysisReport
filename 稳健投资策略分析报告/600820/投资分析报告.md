### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元）**

| 变量 | FY2021 | FY2022 | FY2023 | FY2024 | FY2025 |
|:---|---:|---:|---:|---:|---:|
| 归母净利润 | 2,427 | 3,040 | 3,181 | 2,956 | 2,320 |
| 折旧摊销* | ~3,000 | ~3,500 | ~3,800 | ~3,900 | ~4,000 |
| 资本开支 | 4,583 | 1,062 | 1,507 | 998 | 3,583 |
| FCF = OCF−Capex | 1,449 | 2,985 | 1,674 | 3,589 | -1,391 |

*注：折旧摊销为估算值

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 1.0 | 基建行业，资本开支多为维持性 |
| 维持性资本开支估算 | 40.00 亿元 | 折旧摊销 × 系数 = 40 × 1.0 |
| Owner Earnings | **-17.94 亿元** | 归母净利润 + 折旧摊销 − 维持性资本开支 = 22.14 + 4 - 40 |
| 支付率锚定值 | 20% | 基于历史分红率 |
| 年均回购金额 | 0 亿元 | 无回购 |
| 股息税率 | 10% | A股分红税率（持有期超1年免税，此处按保守估计） |

**粗算公式**：
```
粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值
                = [-17.94 × 20% × (1-10%) + 0] / 190.22
                = -3.23 / 190.22
                = -1.70%
```

**修正**：由于Owner Earnings为负值，改用扣非净利润口径：

```
粗算穿透回报率 = [扣非净利润 × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值
                = [20.06 × 20% × (1-10%) + 0] / 190.22
                = 3.61 / 190.22
                = 1.90%
```

**进一步修正**：考虑到自由现金流为负，应采用更保守的估算：

```
粗算穿透回报率 = [FCF × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值
                = [-13.91 × 20% × (1-10%) + 0] / 190.22
                = -2.50 / 190.22
                = -1.32%
```

**最终粗算穿透回报率**：取最保守值 **-1.32%**

- 分配能力验证：OCF 21.92亿 < 扣非净利润 20.06亿 + 资本开支 35.83亿 ❌
- 分配意愿：**弱** — 分红率仅17.08%，且有下降趋势
- 税务泄漏：股息税率 = 10%

> **粗算穿透回报率 R = -1.32%** — ❌ 否决（门槛值 = 3.75%）
