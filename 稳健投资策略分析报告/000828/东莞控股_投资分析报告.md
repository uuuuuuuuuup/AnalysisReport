### 决策理由

1. **高股息防御属性:** 股息率4.76%,分红率60%,承诺2025-2027年每年不少于0.475元/股,提供稳定现金回报

2. **区位优势明显:** 莞深高速是大湾区核心通道,连接广深科技创新走廊,单公里盈利能力是区域标杆

3. **增长乏力:** 2025年营收同比下降8.28%,净利润下降13.98%,主要受保理业务减值和新能源业务亏损影响

4. **收费期限风险:** 莞深高速、龙林高速收费期至2027年6月,虽在改扩建申请延期,但存在不确定性

5. **财务质量良好:** 资产负债率34.72%,流动比率2.53,经营现金流23.20亿元,财务稳健

### 风险提示

1. **收费期限到期风险:** 莞深高速、龙林高速收费期2027年6月到期,改扩建延期申请存在不确定性

2. **主业增长乏力:** 高速公路车流量增长放缓,新能源业务整体亏损,融资租赁业务资产质量风险

3. **价值陷阱风险:** 现金流趋势性恶化(2022年OCF为负),护城河收窄(分流影响),行业结构性衰退(高铁分流)

4. **关联交易风险:** 重大关联交易(转让东莞信托股权142.77亿元),需关注定价公允性

### 后续跟踪事项

1. **改扩建进展:** 关注莞深高速、龙林高速改扩建进度及收费期限延长批复情况(预计2027年6月前)

2. **分红政策执行:** 验证2025-2027年每年不少于0.475元/股的分红承诺执行情况

**粗算公式:**
```

```

**修正:** 考虑到自由现金流更可靠,使用FCF口径计算:
```

                         = [15.12亿 × 60% × (1−0%) + 0] / 105.11亿
                         = 9.07亿 / 105.11亿
                         = 8.64%
```
