### 决策理由

1. **穿透回报率严重不足**:精算穿透回报率仅0.38%,远低于门槛值3.75%,即使粗算口径(3.04%)也低于门槛值,无法满足稳健投资的基本要求。

2. **盈利质量存疑**:2025年经营现金流8.78亿元,远低于归母净利润9.40亿元,自由现金流仅1.28亿元,盈利现金含量不足,存在"纸面富贵"风险。

3. **分红不可持续**:2025年分红18.26亿元,支付率194%,远超当年净利润,虽然体现管理层回报意愿,但这种高比例分红不可持续,可能透支未来现金流。

4. **毛利率持续下滑**:毛利率从2023年的69.59%降至2025年的57.53%,两年下滑12个百分点,核心产品面临价格压力,盈利能力趋势性恶化。

5. **ROE大幅下降**:ROE从2021年的16.47%降至2025年的8.19%,四年腰斩,股东回报能力显著下降,反映行业竞争加剧和公司竞争力减弱。

### 风险提示

1. **现金流风险**:经营现金流持续下降,2025年同比下降17%,自由现金流仅1.28亿元,现金储备(7.75亿元)不足以支撑持续高分红。

2. **盈利能力下滑**:毛利率、净利率、ROE等核心盈利指标持续下滑,反映行业竞争加剧和产品价格下行压力,盈利能力可能进一步恶化。

3. **分红可持续性风险**:2025年支付率194%不可持续,若未来回归正常水平(30-40%),股息率将从3.08%降至1%以下,对股东回报大幅缩水。

4. **行业政策风险**:血液制品行业受严格监管,医保控费、带量采购等政策可能进一步压缩产品价格空间,影响公司盈利能力。

### 后续跟踪事项

1. **2026年一季报**:关注营收和利润是否企稳回升,毛利率是否止跌,经营现金流是否改善(预计2026年4月30日前发布)。

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.7 | 制药业标准 |
| 维持性资本开支估算 | 2.1 亿元 | 折旧摊销 × 系数 |

**粗算公式**: `粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`  
= [10.3 × 88.3% × (1-10%) + 0] / 269.8  
= 8.19 / 269.8  
= **3.04%**

- 分配能力验证:OCF 8.78 < Owner Earnings 10.3 ❌
- 分配意愿:**强** — 2025年支付率194%,上市21年持续分红
- 税务泄漏:股息税率 = 10%

> **粗算穿透回报率 R = 3.04%** — ⚠️ 边际不达标(门槛值 = 3.75%)

**否决门检查**:
- R = 3.04% > Rf = 1.7548% ✅
- R = 3.04% > 门槛值×0.5 = 1.8774% ✅
- 门槛值×0.5 = 1.8774% ≤ R = 3.04% < 门槛值 = 3.7548% → 标注"边际不达标",继续进入因子 3
