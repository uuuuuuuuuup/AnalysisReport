### 决策理由

1. **穿透回报率严重不足**：粗算穿透回报率仅0.77%，远低于A股门槛值3.75%，即使考虑公司现金储备充裕，也无法弥补估值溢价

2. **现金流质量堪忧**：2024年经营现金流为负0.14亿元，自由现金流为负6.14亿元，应收账款与利润比例高达97.92%，现金回收能力弱

3. **行业面临结构性困境**：全球赤藓糖醇产能超80万吨，需求仅23.8万吨，产能利用率约30%，价格战激烈，行业处于深度调整期

4. **国际贸易壁垒严重**：欧盟征收156.7%反倾销税，美国可能加征224%关税，2024年对美出口占营收20.18%，海外市场面临重大挑战

5. **治理结构存在瑕疵**：2022年、2024年两次更换审计机构，曾因会计处理问题被监管责令整改，董事长75岁高龄

### 风险提示

1. **行业产能过剩风险**：全球赤藓糖醇产能超80万吨，需求仅23.8万吨，产能利用率约30%，价格战可能持续加剧

2. **国际贸易壁垒风险**：欧盟征收156.7%反倾销税（五年），美国可能加征224%关税，海外市场收入面临重大不确定性

3. **现金流风险**：经营现金流连续为负，应收账款与利润比例高达97.92%，存在回款风险

4. **治理风险**：审计机构变更频繁，曾因会计问题被监管关注，董事长高龄

### 后续跟踪事项

1. **行业供需改善**：关注赤藓糖醇行业产能出清进度，若产能利用率回升至50%以上可重新评估

2. **国际贸易谈判**：关注欧盟、美国贸易政策变化，若关税壁垒降低可重新评估

3. **现金流改善**：若经营现金流持续转正且应收账款比例下降，可重新评估

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 维持性Capex系数 | 0.70 | 制造业标准，假设维持性资本开支为折旧摊销的70% |
| 维持性资本开支估算 | 0.35 亿元 | 折旧摊销 × 系数 = 0.50 × 0.70 |
| Owner Earnings | **1.20 亿元** | 归母净利润 + 折旧摊销 − 维持性资本开支 = 1.05 + 0.50 - 0.35 |
| 支付率锚定值 | 6.00% | 取2021-2022年均值，剔除2024年异常值 |
| 年均回购金额 | 0 亿元 | 公司未实施回购 |
| 股息税率 | 0% | A股持股超过1年免税 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值` = **0.77%**

- 分配能力验证：OCF -0.14亿 < Owner Earnings 1.20亿 ❌
- 分配意愿：**存疑** — 2024年分红支付率190.99%异常高，超过当年归母净利润，可持续性存疑
- 税务泄漏：股息税率 = 0%

> **粗算穿透回报率 R = 0.77%** — ❌ 否决（门槛值 = 3.75%）

**注**：由于粗算穿透回报率远低于门槛值，按策略框架应直接否决，不进入因子3精算。但为完整性，仍进行简要分析。
