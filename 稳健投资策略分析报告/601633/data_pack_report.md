## 四、MD&A 深度信息

> 数据来源：AI-Tools MCP GetFinancialReport + WebSearch（长城汽车官网/年报公告/券商研报）

### 4.1 经营回顾

| 项目 | 2025年度 | 同比变动 |
|------|--------:|----------|
| 营业收入 | 222,824.2 | +10.20% |
| 归母净利润 | 9,865.3 | -22.07% |
| 扣非归母净利润 | 6,059.0 | -37.50% |
| 汽车销量 | 132.4万辆 | 再创新高 |
| 单车收入 | 16.83万元 | 同比提升约4,500元 |
| 海外销量 | 50.68万辆 | +11.60%，占总销量38.2% |
| 新能源车型销量 | 创新高 | 占比持续提升 |

**经营要点**：
- 2025年营收创历史新高，近六年持续提升，但利润端明显承压
- "增收不增利"核心原因：销售费用112.7亿元（同比+44%）、研发费用104.3亿元（同比+12%）、直营渠道建设大量投入
- 海外业务毛利率16.70%，较2024年的18.76%下降2.06个百分点，海外竞争烈度上升与地缘贸易摩擦拖累毛利
- 高端车型自营渠道建设投入大量资源，直接侵蚀当期利润率

### 4.2 前瞻指引

- **归元平台新车周期**：公司推出全球首个原生AI全动力汽车平台——归元平台，搭载VLA辅助驾驶大模型，2026年将迎来新车周期
- **新能源转型加速**：Hi4智能四驱电混技术衍生出Hi4-T架构和Hi4-Z架构持续迭代
- **渠道优化**：2026年将在技术、渠道端持续优化，提升直营渠道效率

### 4.3 资本配置意图

- 本集团主要以自有现金及银行借款满足日常经营所需
- 资本开支主要用于国内外工厂建设及产能扩张
- 研发投入持续加大：2025年研发投入149亿元，其中费用化104亿元（费用化率69.8%），资本化45亿元（资本化率30.2%）

### 4.4 风险因素

| 风险类别 | 具体内容 |
|----------|----------|
| 市场竞争风险 | 国内乘用车价格竞争超预期 |
| 海外风险 | 海外市场需求低于预期、国际贸易摩擦风险、海外市场政策变化风险 |
| 汇率风险 | 汇率波动风险（海外业务占比38.2%） |
| 转型风险 | 新能源转型及直营渠道建设短期侵蚀利润 |
| 技术风险 | 智能化、电动化技术迭代速度不及预期 |
