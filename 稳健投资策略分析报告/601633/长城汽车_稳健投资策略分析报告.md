### 决策理由

1. **分配意愿严重不足**：近3年平均支付率仅18.2%（2022年6.2%、2023年31.1%、2024年17.3%），大量利润留存用于研发和渠道扩张，股东实际获得的现金回报极低
2. **"增收不增利"困局**：2025年营收创新高（+10.2%），但归母净利润下降22.1%，扣非净利润更下降37.5%，销售费用同比+44%至112.7亿侵蚀利润
3. **穿透回报率远低于门槛**：即便使用EV口径（扣除广义净现金506.6亿），精算回报率仅1.93%，远低于3.77%门槛值，需要支付率提升至约50%或股价大幅下跌才能达标
4. **强周期属性加大不确定性**：汽车行业强周期性叠加新能源转型阵痛期，利润波动剧烈（2021-2025年净利润从67亿→83亿→70亿→127亿→99亿）
5. **非经常性损益占比过高**：2025年非经常性损益38.06亿元（占归母净利润38.6%），其中政府补助37.48亿元，盈利质量存疑

---

### 风险提示

1. **新能源转型风险**：直营渠道建设+研发投入持续侵蚀利润，转型期可能长达2-3年，期间利润承压
2. **海外业务风险**：海外销量占比38.2%，地缘冲突/贸易壁垒/关税政策变化可能冲击海外收入
3. **竞争加剧风险**：国内乘用车价格战持续，同质化竞争加剧，毛利率从2024年19.51%降至2025年18.04%
4. **应收账款快速增长**：2025年应收账款同比+31.98%至96.0亿元，远超营收增速，回款风险需关注

---

### 后续跟踪事项

1. **2025年度分红方案**：待股东大会公告，若支付率提升至30%以上则需重新评估
2. **2026年Q1/Q2业绩**：关注归元平台新车周期对利润的改善效果
3. **直营渠道效率**：关注销售费用率是否见顶回落

## 关键发现

1. 长城汽车2025年OCF高达403.6亿元（远超净利润98.7亿元），但主要受益于应付账款增加等营运资本变动，可持续性存疑
2. 广义净现金506.6亿元（含理财349.7亿），净现金/市值=28.8%，提供轻度现金保护但不足以弥补回报率缺口
3. 开发支出资本化余额122.12亿元被德勤列为关键审计事项，资本化率30.2%偏高，存在利润调节空间
4. 2025年政府补助37.48亿元占扣非前净利润的38%，盈利对补贴依赖度较高
5. 5年营收CAGR达13.0%，但扣非净利润CAGR仅9.6%，增长质量不及营收增长

**粗算公式（扣非口径）**：

`粗算穿透回报率 = [84.59 × 20% × (1−0%) + 7.30] / 1,761 = [16.92 + 7.30] / 1,761 = 24.22 / 1,761` = **1.38%**

**粗算公式（GAAP口径）**：

`粗算穿透回报率 = [122.65 × 20% × (1−0%) + 7.30] / 1,761 = [24.53 + 7.30] / 1,761 = 31.83 / 1,761` = **1.81%**

- 分配能力验证：OCF 403.6 > Owner Earnings 84.6 ✅（OCF远高于OE，但含大量营运资本变动）
- 分配意愿：**弱** — 近3年平均支付率仅18.2%，大量利润留存用于扩张
- 税务泄漏：股息税率 = 0%

> **粗算穿透回报率 R = 1.38%（扣非口径）/ 1.81%（GAAP口径）** — ⚠️ 边际不达标（门槛值 = 3.77%）
>
> 市值口径下R < 门槛值×0.5（1.885%），检查EV口径：
> - EV口径(扣非) = 24.22 / 1,254.4 = 1.93% > 1.885% → 边际不达标
> - EV口径(GAAP) = 31.83 / 1,254.4 = 2.54% < 3.77% → 边际不达标
