### 决策理由

1. **护城河宽阔**：白电国内份额47.4%，全球智能家电销量第一，年研发投入超200亿元，掌握压缩机/电机/芯片核心技术，品牌+规模+技术三重壁垒
2. **分配意愿极强**：2025年现金分红率73.64%+回购116亿元，综合股东回报率约100%，2025-2027承诺分红比例不低于60%
3. **安全边际薄弱**：精算穿透回报率4.33%仅超门槛值0.57 pct，当前股价处于5年93%分位，估值已充分反映基本面
4. **增长质量存疑**：2025年OCF同比下降11.8%，Q4扣非净利润同比下降23.1%，并购整合短期拖累盈利

### 风险提示

1. **地缘/关税风险**：海外收入占比超42%，"暴风雨比想象猛烈"（方洪波致股东信），关税政策不确定性高
2. **并购整合风险**：Arbonia、东芝电梯、锐珂医疗等并购短期拖累盈利，Q4影响进一步扩大
3. **内需疲弱**：国内家电零售额同比下降4.3%，国补断档后竞争加剧
4. **原材料价格波动**：原材料涨价背景下战略性备货，影响经营现金流

### 后续跟踪事项

1. **2026Q1财报**：验证Q4业绩低点是否为阶段性底部，关注扣非净利润恢复情况（2026年4月底披露）
2. **关税政策落地**：跟踪美国对华关税政策变化，评估对海外收入42%的影响
3. **并购整合进展**：关注Arbonia/东芝电梯/锐珂医疗整合效果，是否在2026年贡献正收益

### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元）**

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 维持性Capex系数 | 0.7 | 制造业标准 |
| 维持性资本开支估算 | 49 亿元 | 折旧摊销70亿 × 0.7 |
| Owner Earnings | **433.67 亿元** | 扣非净利润412.67 + 折旧摊销70 − 维持性Capex49 |
| 支付率锚定值 | 65% | 低于3年均值67.9%，考虑可持续性 |
| 年均回购金额 | 70 亿元 | 2024-2025年均值 |
| 股息税率 | 0% | A股持股>1年免税 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

= [433.67 × 65% × 100% + 70] / 6,052
= [281.89 + 70] / 6,052
= 351.89 / 6,052
= **5.81%**

- 分配能力验证：OCF 533.5 > Owner Earnings 433.67 ✅
- 分配意愿：**强** — 2025年综合股东回报率约100%，2025-2027承诺分红不低于60%
- 税务泄漏：股息税率 = 0%（A股长期持有免税）

> **粗算穿透回报率 R = 5.81%** — ✅ 通过（门槛值 = 3.76%）
