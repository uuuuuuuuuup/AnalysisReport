### 决策理由

1. **穿透回报率不足**：扣非口径粗算回报率仅2.52%，低于A股门槛值3.746%，主要受2025年利润下滑（扣非净利润同比下降16.10%）和销售费用率攀升（22.17%，+4.17pp）影响
2. **现金储备极强但未充分反映在回报中**：广义净现金约70亿元（占市值23.8%），零有息负债，但公司尚未启动回购，现金分配效率有待提升
3. **分红意愿显著增强**：2025年股利支付率跃升至67.65%（vs 2024年44.40%），全年三次分红共10.02亿元，体现管理层对股东回报的重视
4. **业务转型期利润承压**：CGM翻倍增长、海外收入+29.86%是亮点，但线下渠道调整、销售费用前瞻投入导致短期利润下滑
5. **商誉减值风险需持续关注**：商誉10.77亿元占资产6.70%，审计师将其列为关键审计事项

### 风险提示

1. **利润持续下滑风险**：2025年归母净利润同比下降17.94%，扣非净利润下降16.10%，若2026年渠道调整效果不及预期，利润可能继续承压
2. **销售费用率攀升风险**：2025年销售费用率22.17%（+4.17pp），新品推广和品牌建设投入若无法转化为收入增长，将侵蚀利润
3. **商誉减值风险**：商誉10.77亿元，若并购标的（如凯立特、上医集团等）运营不达预期，可能触发大额减值
4. **政府补助波动风险**：2024年其他收益3.52亿元（占净利润19.49%），2025年明显下降，2026Q1其他收益同比下降81.30%

### 后续跟踪事项

1. **2026年中报业绩**：关注渠道调整效果、CGM放量节奏、销售费用率变化，预期2026年8月披露
2. **商誉减值测试**：关注2026年中报是否计提商誉减值，特别是凯立特和上医集团资产组
3. **分红政策延续性**：关注2026年是否维持高频分红（半年度+前三季度+年度），验证67%支付率是否为新常态

## 关键发现

1. **零有息负债运营**：2024-2025年连续两年零有息负债，资产负债率从2022年33.31%降至2025年16.97%，财务结构极为保守
2. **大额定期存款重分类**：2025年末货币资金从66.19亿骤降至5.79亿（-91.25%），实质为银行定期存款重分类至其他流动资产和其他非流动资产，现金类资产总额并未大幅减少
3. **分红力度跃升**：2025年累计分红10.02亿元（支付率67.65%），较2024年的8.02亿元（44.40%）大幅提升，分红频次从年度提升至三次
4. **CGM业务翻倍增长**：血糖管理及POCT板块收入12.81亿元（+24.34%），其中CGM业务翻倍，Anytime 4/5系列获批，瑞银预测2024-2030年CGM收入CAGR达47.1%
5. **Q4业绩断崖式下滑**：2025Q4归母净利润仅0.16亿元（-94.25%），扣非净利润-0.0065亿元，主因线下渠道调整、汇兑损失和资产减值

### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元）**

| 变量 | FY2023 | FY2024 | FY2025 |
|:---|---:|---:|---:|
| 归母净利润 | 2,376.47 | 1,804.27 | 1,481.79 |

| 资本开支 | 2,077.41 | 196.41 | 260.91 |
| FCF = OCF−Capex | 49.57 | 1,619.86 | 1,242.01 |

> 注：2023年Capex异常偏高（含并购），不具代表性。2024-2025年Capex回归正常。

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.7 | 制造业标准，轻资产运营取中值 |
| 维持性资本开支估算 | 1.54 亿元 | 折旧摊销2.20亿 × 0.7 |
| Owner Earnings | **12.35 亿元** | 扣非归母净利润11.69亿 + 折旧摊销2.20亿 − 维持性资本开支1.54亿 |
| 支付率锚定值 | 60% | 取2024-2025年均值偏保守 |
| 年均回购金额 | 0 亿元 | 无回购记录 |
| 股息税率 | 0% | A股直接持有>1年免税 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

= [12.35 × 0.60 × 1.0 + 0] / 293.5 = 7.41 / 293.5 = **2.52%**

- 分配能力验证：OCF 15.03亿 > Owner Earnings 12.35亿 ✅
