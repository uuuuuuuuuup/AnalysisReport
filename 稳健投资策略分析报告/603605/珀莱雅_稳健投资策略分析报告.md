## 决策理由

1. **穿透回报率不足**：当前市值238亿元下，精算穿透回报率仅2.24%，远低于3.76%门槛值。即便以更乐观的Owner Earnings口径粗算，回报率也仅3.07%，仍低于门槛
2. **增长换挡风险**：2025年前三季度营收仅+1.89%，Q3单季同比-11.63%，主品牌首次负增长，高增长逻辑正在被证伪
3. **费用刚性压力**：销售费用率连续6个季度超45%，2024年销售费用51.61亿元（+29.93%），增速高于营收增速（+21.04%），流量成本高企侵蚀利润
4. **分配率有提升空间**：2025年中期分红10派8元（vs 2024年中期10派3.8元），暗示公司可能提升分配率至50%+，若持续则将改善回报率
5. **资产负债表健康**：广义净现金31.91亿元，有息负债仅8.91亿元（主要为珀莱转债），财务风险极低

---

## 风险提示

1. **增长持续放缓**：2025Q3营收-11.63%，若2025全年营收下滑，当前估值将进一步承压
2. **销售费用刚性**：线上流量成本持续上升，销售费用率若维持45%+，利润增速将持续低于营收增速
3. **品牌集中度风险**：主品牌珀莱雅占比近80%，彩棠等子品牌尚未形成独立支撑力
4. **应收账款恶化**：应收账款周转天数从2023年9天升至2024年14天，回款效率下降需关注

**Owner Earnings 历史数据（百万元）**

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| 维持性Capex系数 | 0.7 | 轻资产行业标准 |
| 维持性资本开支估算 | 1.26 亿元 | 折旧摊销1.80亿 × 0.7 |

| 支付率锚定值 | 40% | 含中期分红趋势 |
| 年均回购金额 | 1.01 亿元 | 2025年12月启动 |
| 股息税率 | 0% | A股持股>1年免征 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

= [15.76 × 0.40 × 1.0 + 1.01] / 238.18 = [6.30 + 1.01] / 238.18 = **3.07%**

- 分配能力验证：3年均值OCF 12.29亿 > Owner Earnings 15.76亿 ❌（OCF < OE，因2024年营运资本消耗大）
- 分配意愿：**中等偏强** — 年度支付率稳定30%，含中期分红约40%，2025年中期分红大幅提升暗示加码
- 税务泄漏：股息税率 = 0%（持股>1年）

> **粗算穿透回报率 R = 3.07%** — ⚠️ 边际不达标（门槛值×0.5 = 1.88% < R < 门槛值 = 3.76%），继续进入因子3
