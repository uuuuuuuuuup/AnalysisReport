### 因子 2：穿透回报率粗算（银行股专用方法）

#### 方法论说明

对于银行股，传统的 Owner Earnings 和穿透回报率计算方法不适用，本报告采用以下方法：

**方法一：ROE/PB 法**
- 理论依据：ROE/PB = 净资产收益率 / 市净率 = 理论投资回报率
- 计算公式：ROE/PB = 9.69% / 0.56 = 17.30%

**方法二：股息+增长法**
- 理论依据：股东回报 = 股息率 + 内生增长率
- 内生增长率 = ROE × (1 - 股息支付率)
- 计算公式：5.34% + 9.69% × (1 - 30.15%) = 5.34% + 6.77% = 12.11%

#### 粗算参数汇总

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| ROE | 9.69% | 2025年度 |
| PB | 0.56 | 当前市净率 |
| ROE/PB | 17.30% | 理论回报率 |
| 股息率（TTM） | 5.34% | 过去12个月 |
| 股息支付率 | 30.15% | 2025年度 |
| 内生增长率 | 6.77% | ROE × (1 - 支付率) |
| 预期总回报率 | 12.11% | 股息率 + 增长率 |
| 门槛值 | 3.75% | max(3.5%, Rf+2%) |

#### 粗算结论

**方法一（ROE/PB法）**：17.30% > 门槛值3.75% ✅ 通过

**方法二（股息+增长法）**：12.11% > 门槛值3.75% ✅ 通过

> **粗算穿透回报率 R = 12.11%** — ✅ 通过（门槛值 = 3.75%）
