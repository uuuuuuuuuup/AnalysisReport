## 决策理由

1. **估值严重偏高**：当前PE(TTM)约38.7倍，对应盈利收益率仅2.6%，远低于A股门槛值3.76%；粗算穿透回报率1.18%甚至低于十年期国债收益率1.762%
2. **盈利能力大幅下滑**：归母净利润从2021年96.54亿元降至2025年35.86亿元，降幅62.8%；ROE从37.33%降至6.48%，盈利质量显著恶化
3. **现金分配不足以支撑估值**：2025年全年分红约14.52亿元，股息率仅约1.03%，即便考虑公司提升分红意愿，在当前估值下仍无法提供足够回报
4. **行业竞争格局边际恶化**：免税牌照稀缺性降低，王府井等新进入者分流市场份额，跨境电商价格冲击持续
5. **Q4拐点信号值得关注**：25Q4收入同比+2.81%、归母净利润同比+53.59%（剔除商誉减值后+150.63%），但单季改善不足以扭转估值判断

---

## 风险提示

1. **盈利持续下行风险**：若消费复苏不及预期，归母净利润可能进一步下滑，当前估值将更加不合理
2. **竞争加剧风险**：免税牌照放开、新进入者增加、跨境电商冲击，市场份额可能从~80%继续下滑
3. **政策依赖风险**：海南封关政策落地效果、市内免税新政实际拉动存在不确定性
4. **海外并购整合风险**：2.94亿美元收购DFS大中华区业务，跨境整合存在执行风险

## 后续跟踪事项

2. **海南封关实际效果**：跟踪离岛免税购物金额增速是否维持46.8%的高增长（2026年下半年验证）

### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元）**

| 变量 | FY2021 | FY2022 | FY2023 | FY2024 | FY2025 |
|:---|---:|---:|---:|---:|---:|
| 归母净利润 | 9,653.74 | 5,030.38 | 6,713.69 | 4,267.11 | 3,586.18 |
| 折旧摊销(估) | 350 | 380 | 400 | 400 | 380 |
| 资本开支 | 60 | 875 | 2,923 | 2,499 | 1,273 |
| FCF = OCF−Capex | 8,267.32 | -4,290.25 | 12,203.27 | 5,440.36 | 4,785.56 |

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.7 | 零售行业标准 |
| 维持性资本开支估算 | 2.66 亿元 | 折旧摊销3.80亿 × 系数0.7 |
| Owner Earnings | **37.00 亿元** | 归母净利润35.86 + 折旧摊销3.80 − 维持性Capex2.66 |
| 支付率锚定值 | 45% | 略低于3年均值47.57% |
| 年均回购金额 | 0 亿元 | 授权未实施 |
| 股息税率 | 0% | A股持股>1年免征 |

**粗算公式**：`粗算穿透回报率 = [37.00 × 0.45 × 1.0 + 0] / 1,406.46` = **1.18%**

- 分配能力验证：OCF 60.59 > Owner Earnings 37.00 ✅
- 分配意愿：**中等** — 近3年支付率40-51%，首次中期分红，但绝对分红金额下降
- 税务泄漏：股息税率 = 0%（长期持有）

> **粗算穿透回报率 R = 1.18%** — ❌ 否决（门槛值 = 3.762%）
>
> R = 1.18% < Rf = 1.762% → 检查EV口径
>
> 广义净现金/市值 = 307.06/1,406.46 = 21.8% < 40%（EV口径非强制触发）
>
> EV口径：EV = 1,406.46 − 307.06 = 1,099.40亿元，R_EV = 16.65/1,099.40 = 1.51%
>
> R_EV = 1.51% < Rf = 1.762% → EV口径仍不通过
>
> R = 1.18% < 门槛值×0.5 = 1.881% → ❌ 否决
