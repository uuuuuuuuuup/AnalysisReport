## 四、MD&A 深度信息

> 数据来源：AI-Tools MCP GetFinancialReport、mx_finance_search Skill 搜索年报及新闻

### 4.1 经营回顾

**2025年经营概况**：
- 营业收入 680.49 百万元，同比下降 10.36%
- 归母净利润 64.25 百万元，同比下降 17.42%
- 扣非归母净利润 58.61 百万元，同比下降 8.79%
- 毛利率 34.31%，同比上升 1.22 个百分点
- 经营活动现金流净额 60.97 百万元，同比增长 26.31%

**收入下滑原因**：下游化工、石油天然气、冶金、轻工建材等行业有效需求不足，行业投资下滑、新建项目延期或停滞，导致公司结转合同减少；国内工业自动化仪表市场竞争加剧。

**分产品收入**：
| 产品 | 2025年收入（百万元） | 同比变动 |
|---|---:|---:|
| 工业自动化仪表及装置 | 598.80 | -11.90% |
| 复合材料 | 66.90 | +4.57% |
| 电子器件 | 10.00 | +0.64% |

**投资收益影响**：联营企业横河川仪下游需求不足，营业利润大幅下降50.58%，公司投资收益85.90百万元，同比下降44.97%，为利润下滑的重要拖累因素。

### 4.2 前瞻指引

**2026年经营目标**：
- 营业收入目标 71.0 亿元（同比+4.3%）
- 利润总额目标 7.3 亿元（同比+2.1%）

**券商一致预期（2026年）**：
- 国金证券：预计归母净利润 6.8 亿元，同比+6.3%
- 中邮证券：预计归母净利润 7.14 亿元，同比+13.24%
- 长江证券：预计归母净利润约 7.08 亿元

### 4.3 资本配置意图

1. **科创基地建设**：2026年3月31日公告，拟投资不超过3.47亿元建设"自主可控智能仪器仪表科技创新基地"项目，计划2026年启动，2028年投产，旨在突破核心材料与部件瓶颈。
2. **研发投入**：2025年研发费用 480.09 百万元，研发费用率 7.06%，研发人员 1,049 人。
3. **银行授信**：2026年拟继续申请银行综合授信额度，用于日常经营周转。
4. **分红政策**：保持连续分红，2025年度现金分红+回购注销合计占归母净利润 40.93%。

### 4.4 风险因素

1. 宏观经济波动和国际形势变化带来的不确定性风险
2. 市场竞争加剧的风险
3. 产品研发的风险
4. 投资收益波动风险（横河川仪联营企业业绩波动）
5. 部分原材料供应及价格波动风险
6. 应收账款资金占用及回收风险
