## 决策理由

1. **核心主业几近亏损**：2025年扣非归母净利润仅1.04亿元，对应147.6亿市值，扣非PE超140倍；近5年营收CAGR为-8.0%，主业持续萎缩
2. **现金流趋势性恶化**：OCF从2021年37.2亿骤降至2025年9.9亿（-73%），2025年真实可支配现金结余仅4.3亿元
3. **现金虽多但"看得见吃不到"**：广义现金141亿中，98.5亿沉淀于关联方百联财务公司（占存款56.1%），受限现金19亿，公司仅派息2.68亿/年，分配意愿极弱
4. **非经常性损益主导利润**：2025年非经常性损益5.28亿占归母净利润83.8%，利润质量极低，可持续盈利能力堪忧
5. **行业结构性衰退**：线下零售受电商持续冲击，超市/百货业态收入连年下滑，仅奥特莱斯维持正增长

---

## 风险提示

1. **价值陷阱风险**：4项价值陷阱特征触发（现金流恶化、护城河收窄、行业衰退、分配意愿弱），当前低PE（4.66倍动态）具有欺骗性
2. **关联方资金安全风险**：98.5亿存放于百联财务公司，该财务公司资产负债率92%，一旦出现流动性问题可能影响资金安全
3. **主业持续亏损风险**：联华超市2025年净利润-1.77亿元，超市业态（占收入77%）持续承压，拖累整体业绩
4. **资产变现不确定性**：公司虽拥有大量核心商圈物业，但REITs扩容和土地收储节奏不可控，无法形成稳定收益预期

### 因子 2：穿透回报率粗算

**Owner Earnings 历史数据（百万元，估算）**

| 变量 | FY2021 | FY2022 | FY2023 | FY2024 | FY2025 |
|:---|---:|---:|---:|---:|---:|

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.7 | 零售业标准，物业维护+门店翻新 |
| 维持性资本开支估算 | 5.6 亿元 | 自有资产折旧摊销8亿 × 0.7 |
| Owner Earnings（扣非口径） | **3.44 亿元** | 扣非归母1.04+折旧摊销8−维持性Capex5.6 |
| Owner Earnings（GAAP口径） | **8.71 亿元** | 归母6.31+折旧摊销8−维持性Capex5.6 |
| 支付率锚定值 | 42% | 3年均值 |
| 年均回购金额 | 0 亿元 | 无回购 |
| 股息税率 | 0% | A股持股>1年免征 |

**粗算公式（扣非口径）**：

`粗算穿透回报率 = [3.44 × 42% × (1−0%) + 0] / 147.6` = **0.98%**

**粗算公式（GAAP口径）**：

`粗算穿透回报率 = [8.71 × 42% × (1−0%) + 0] / 147.6` = **2.48%**

- 分配能力验证：OCF 9.93 > Owner Earnings 3.44 ✅（但OCF本身大幅下滑）
- 分配意愿：**弱** — 141亿广义现金仅派息2.68亿，无回购，大量现金沉淀于关联财务公司
- 税务泄漏：股息税率 = 0%（持股>1年）

**EV口径双轨制检查**：

广义净现金/市值 = 114.81/147.6 = 77.8% > 40% → 触发EV口径

EV = 147.6 − 114.81 = 32.79 亿元

`粗算穿透回报率(EV, 扣非口径) = [3.44 × 42% × 1 + 0] / 32.79` = **4.39%**

> ⚠️ EV口径通过门槛值（4.39% > 3.746%），但需验证现金分配意愿。公司141亿广义现金中98.5亿存放于关联财务公司，实际可自由支配现金远低于账面，分配意愿极弱，EV口径可靠性存疑。

> **粗算穿透回报率 R = 0.98%（扣非口径）** — ❌ 否决（R < Rf = 1.746%，且 R < 门槛值×0.5 = 1.873%）
>
> **粗算穿透回报率 R = 2.48%（GAAP口径）** — ⚠️ 边际不达标（门槛值×0.5 ≤ R < 门槛值）
