## 四、MD&A 深度信息

> 数据来源：AI-Tools MCP GetFinancialReport + mx_finance_search 搜索公告/研报

### 4.1 经营回顾

**2025年度业绩快报核心数据**（发布日期：2026-02-27）：

| 指标 | 2025FY | 同比变动 |
|---|---:|---|
| 营业总收入 | 18,616 百万元 | +55.85% |
| 归母净利润 | 1,360 百万元 | -31.19% |
| 扣非归母净利润 | 1,087 百万元 | -32.90% |
| 2025Q4 营收 | 6,549 百万元 | +32.62% |
| 2025Q4 归母净利润 | 322 百万元 | -36.09% |
| 2025Q4 扣非归母净利润 | 253 百万元 | -41.83% |

**营收增长驱动因素**：
1. 国内"以旧换新"政策带动智能扫地机器人和洗地机业务需求旺盛
2. 新品推出与价格矩阵完善推动业务增长
3. 海外市场通过品牌建设和渠道拓展实现收入稳定增长
4. 智能扫地机器人全球销额及销量份额持续稳居第一（IDC数据）

**利润下滑原因**：
1. 全价位段产品布局及洗地机等新品类快速增长初期对毛利率产生一定影响
2. 更积极的市场策略导致销售费用增加
3. 内销自补政策导致毛利率下降
4. 洗衣机、割草机器人等新业务亏损拖累
5. 境内外销售费用增加，品牌、渠道相关投放显著

### 4.2 前瞻指引

根据多家券商研报汇总：

| 券商 | 2026E 归母净利润（百万元） | 2027E 归母净利润（百万元） | 评级 |
|---|---:|---:|---|
| 开源证券 | 2,410 | 3,200 | 买入 |
| 中金公司 | 2,135 | 2,822 | 跑赢行业 |
| 国联民生 | 2,260 | 3,200 | 推荐 |
| 国泰海通 | - | - | 增持 |
| 长江证券 | - | - | 买入 |

**2026年盈利修复预期**：
1. 内销自补逐步退出，预计减亏2-3亿元
2. 洗衣机业务收缩研发、营销投入，预计减亏2-3亿元
3. 割草机器人新品贡献收入增量同时实现约1亿元减亏
4. 扫地机行业竞争进一步缓和

### 4.3 资本配置意图

1. **产能扩张**：总投资25亿元的智能清洁设备产能扩张项目，截至2025年末累计投资12.3亿元，项目完成进度49.2%
2. **港股上市**：2025年6月27日递交港交所IPO申报，目的为提升全球品牌知名度及竞争力，优化资本结构和股东组成
3. **股份回购**：2024年累计回购股份18.75万股，金额5,599万元；2025年推出5,000万至1亿元回购计划，截至9月底已回购36.90万股，金额7,387万元
4. **研发投入**：2024年研发费用9.71亿元（同比+56.93%），2025年前三季度研发费用10.28亿元

### 4.4 风险因素

1. **关税及外需不确定性**：美国高关税政策下供应链国际化进程加速
2. **市场竞争加剧**：国内外市场仍存在差异化竞争，价格竞争可能影响利润率
3. **渠道拓展不及预期**：欧洲线下渠道较为强势
4. **原材料成本大幅上涨**
5. **新业务不及预期**：洗衣机、割草机器人等新品类尚在培育期
6. **库存高企**：2025Q3存货3,715.8百万元，较2024FY增长149.4%
