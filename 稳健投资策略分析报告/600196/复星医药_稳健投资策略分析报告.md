### 决策理由

1. **穿透回报率严重不达标**：精算穿透回报率2.25%远低于门槛值3.76%，差距1.51个百分点，即使考虑回购贡献仍无法弥补
2. **净负债状态无现金保护**：广义净现金-214亿元，货币资金仅覆盖有息负债的40%，现金保护等级为"无保护"
3. **利润质量存疑**：2025年非经常性损益占归母净利润31%（同比从16%大幅提升），扣非净利润仅23.4亿元vs归母33.7亿元
4. **商誉减值悬顶**：商誉账面价值108亿元，占总资产9%，审计师将其列为关键审计事项
5. **应收账款恶化**：应收账款/营收从18.5%升至22.5%，周转天数从66天延长至75天

### 风险提示

1. **集采政策风险**：药品耗材集采常态化，仿制药收入持续承压，创新药放量速度需快于集采冲击
2. **商誉减值风险**：108亿元商誉余额，若子公司业绩不达预期将触发大额减值
3. **偿债压力**：有息负债355亿元，短期债务约227亿元，货币资金/流动负债仅48.7%
4. **关联方风险**：关联资金往来余额约85亿元，财务公司存款11.4亿元，需关注资金安全性

### 后续跟踪事项

1. **2026年中报关注创新药收入增速**：若创新药占比突破40%且收入增速>15%，可重新评估护城河
2. **有息负债压降进度**：公司计划非核心资产退出回笼资金，关注2026年有息负债是否降至300亿元以下
3. **商誉减值测试结果**：2026年报关注商誉减值测试关键假设（折现率、增长率）变化

1. 复星医药精算穿透回报率仅2.25%，远低于A股门槛值3.76%，当前估值不具备稳健投资安全边际

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.7 | 制药行业偏高端，并购驱动增长模式 |

| 股息税率 | 0% | A股持股>1年免征 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

= [41.21 × 30% × 1 + 3.48] / 698.32 = [12.36 + 3.48] / 698.32 = 15.84 / 698.32 = **2.27%**

- 分配能力验证：OCF 52.13 > Owner Earnings 41.21 ✅
- 分配意愿：**中等** — 支付率稳定在30%但偏低，回购规模有限（3.48亿元），未展现强分配意愿
- 税务泄漏：股息税率 = 0%（A股持股>1年免征）

> **粗算穿透回报率 R = 2.27%** — ⚠️ 边际不达标（门槛值×0.5 = 1.88% ≤ R = 2.27% < 门槛值 = 3.76%），继续进入因子3
