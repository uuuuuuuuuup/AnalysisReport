= [97.99 + 22.5] / 3,162.80
= 120.49 / 3,162.80
= **3.81%**

- 税务泄漏：股息税率 = 0%（长期持有）

> **粗算穿透回报率 R = 3.81%** — ⚠️ 边际达标（门槛值 = 3.76%，超额 +0.05 pct）

> **真实可支配现金结余(3年均值) = 14,664百万元**（约146.64亿元）

**精算关键步骤说明**

- **步骤1 真实现金收入**：2025年营业收入925.08亿，应收净变动-80.98亿（减少），收款比率 >100%；保守起见不调增，真实现金收入 = 925.08亿
- **步骤3 非经常项分类**：保留 V1（资产处置）≈0.2亿，V5（投资收益）≈0.1亿；扣除 应扣除非经常性流入（政府补助非经常性部分6.38亿）
