## 四、MD&A 深度信息

> 数据来源：AI-Tools MCP GetFinancialReport + WebSearch（年报管理层讨论与分析、券商研报）

### 4.1 经营回顾

**整体业绩：**
- 2025年营业收入332.82亿元，同比下降9.38%（上市以来首次年度营收下滑）
- 归母净利润81.36亿元，同比下降30.28%
- 扣非净利润80.69亿元，同比下降29.48%
- 经营性现金流101.45亿元，同比下降18.40%

**分区域表现：**
| 区域 | 营收（百万元） | 同比 | 占比 |
|:---|---:|---:|---:|
| 国内业务 | 15,632.38 | -22.97% | 47% |
| 国际业务 | 17,650.38 | +7.40% | 53% |

- 国际业务收入占比首次超过50%，欧洲市场在2024年高增长基础上进一步实现17%增长
- 国内业务承压核心原因：DRG/DIP支付方式改革、院内采购预算缩减、集采降价、检验结果互认、医疗服务价格治理等多重政策叠加

**分业务表现：**
| 业务板块 | 营收（百万元） | 同比 | 占比 |
|:---|---:|---:|---:|
| 体外诊断 | 12,241.00 | - | 36.78% |
| 生命信息与支持 | ⚠️未找到 | - | - |
| 医学影像 | ⚠️未找到 | - | - |
| 新兴业务 | 5,378.00 | +38.85% | 16% |

- 体外诊断已成为公司最大产线，在国内营收结构中占比达48%
- 新兴业务（微创外科、微创介入、动物医疗）展现强劲"第二增长曲线"潜力

**季度趋势：**
- 2025Q4营业收入74.48亿元，同比增长2.86%（单季度营收转正）
- 2025Q4归母净利润5.66亿元，同比下降45.15%

### 4.2 前瞻指引

- 管理层在MD&A中将业绩下滑归因于"国内公立医院招投标活动推迟"、"医疗设备更新项目落地节奏缓慢"等外部行业因素
- 2024年财报指引"加速国产替代"和"向全球医疗器械领导品牌迈进"，2025年国内替代进程受行业环境影响有所放缓
- 多家券商研报预期国内业务2026年有望正增长，国际业务持续增长
- H股发行募集资金将用于全球研发投入

### 4.3 资本配置意图

- 2025年研发投入39.29亿元，占营收11.80%，占比持续提升
- 公司累计专利数量超12,000件
- 2024年完成对惠泰医疗控制权的收购（26.54%股权），2025年投资活动现金流出大幅下降
- 2025年收回定期存款及理财产品增加（投资活动现金流入同比增长185.38%）
- 高比例现金分红政策持续：2025年度累计现金分红53.10亿元

### 4.4 风险因素

1. **商誉减值风险（高）**：商誉余额114.04亿元，占净资产26.5%，占总资产19.24%。被并购标的（如控股75%的DiaSys、参股26.54%的惠泰医疗）未来业绩直接决定商誉减值压力
2. **国内行业政策风险**：DRG/DIP改革、集采降价、医疗反腐等政策持续影响国内业务
3. **毛利率下行风险**：三大业务毛利率均因价格下降出现不同程度下降，2025年整体毛利率60.32%（同比下降约2.8个百分点）
4. **汇率风险**：国际业务占比53%，汇率波动对业绩影响加大
5. **竞争加剧风险**：国内医疗设备市场竞争环境日趋激烈
