## 四、MD&A 深度信息

> 数据来源：2025年年度报告公告 + 国信证券研报

### 4.1 经营回顾

1. **收入端**：2025年公司实现营业收入36.66亿元，同比下降3.17%，主要系：(1) 生物质项目技改导致部分停机；(2) 出售鱼台生物质及中山垃圾发电等项目；(3) 集中供热项目的供热汽价随煤价联动下调。

2. **利润端**：归母净利润2.80亿元，同比增长29.33%，创近五年来最佳业绩。利润增长主要得益于：(1) 燃料成本下降；(2) 出售资产带来1.28亿元的非流动性资产处置收益。

3. **毛利率**：2025年综合毛利率22.06%，同比提升2.58个百分点。电力业务毛利率19.62%（+4.37pct），热力业务毛利率23.94%（+0.71pct）。

4. **费用端**：管理费用率6.72%，财务费用率6.52%。财务费用同比大幅下降17.93%，主要系可转债利息及贷款利息支出下降所致。

5. **现金流**：经营性现金流8.11亿元，同比增长37.13%，主要系收回的补贴电费增加。出售子公司股权收回现金净额5.83亿元，有效缓解了资金压力。

6. **资产处置**：2025年出售鱼台生物质项目、中山垃圾发电项目（交易对价合计4.94亿元）。

7. **减值**：明水长青、宁安长青两家项目公司分别计提资产减值损失6,277.78万元和7,917.55万元，合计约1.42亿元。

### 4.2 前瞻指引

1. **CCER碳交易**：2025年8月15日，生态环境部对第三批CCER方法学正式启动征求意见，生物质发电供热是其中之一。公司预计将在第三批CCER方法学正式发布后参与交易，有望带来潜在的利润增量。公司在手生物质装机容量超过446MW，每年可开发的CCER潜力约180万吨。

2. **茂名项目二期**：2025年9月，公司董事会审议通过投资5.71亿元建设茂名长青热电联产项目二期及配套管网工程，以满足客户扩产带来的蒸汽需求。

3. **业务转型**：公司正推动明水长青、宁安长青等早期投产项目由以发电收入为主，向以供热供暖等市场化业务为主进行战略转型。

4. **数字化智能化**：2025年2月与中科信控签署《战略合作协议》，为公司在行业内的数字化、智能化转型打下坚实基础。

5. **"十五五"规划对接**：公司将业务战略从单纯的"供热服务商"升级为"园区零碳热能综合服务商"，通过深度融合超低排放、生物质掺烧及碳捕集利用等前沿技术，积极利用中央预算内投资及绿色金融工具，主动参与碳市场交易。

### 4.3 资本配置意图

1. **高分红回报股东**：2025年全年合计派发现金3.26亿元，股利支付率116.4%。
2. **债务去杠杆**：资产负债率由年初73.23%大幅下降至62.35%，有息负债规模持续压缩。
3. **核心项目投资**：茂名项目二期投资5.71亿元。
4. **低风险理财**：公司及控股子公司拟使用自有闲置资金进行低风险理财产品投资，投资额度不超过3亿元。

### 4.4 风险因素

1. 原材料（生物质燃料/煤炭）价格上涨风险
2. 生物质电价补贴政策变动风险
3. 应收账款回收不及预期风险
4. 短期债务压力持续增长，融资渠道趋紧
5. 长短债务比持续增长（0.40 -> 0.41 -> 0.52）

### 9.3 理财产品管理制度要点

3. 须具有与理财产品保证金相匹配的自有资金

5. 融资部为理财具体经办部门，审计部负责审计与监督
