## 十一、关键发现与风险提示

### 11.1 关键发现

1. **财务结构稳健**:
   - 资产负债率仅33.93%,处于较低水平
   - 无有息负债,财务费用为负
   - 流动比率大幅提升至1.84

2. **盈利质量分化**:
   - 归母净利润增长12.37%,但扣非净利润下降24.03%
   - 非经常性损益占净利润13.54%,需关注可持续性

3. **现金流压力**:
   - 经营现金流下降16.47%
   - 自由现金流大幅为负,主要因投资支付现金增加

4. **分红政策稳定**:
   - 连续三年维持高分红比例
   - 2025年分红金额12.72亿元,分红比例超60%

### 11.2 风险提示

1. **营收持续下滑**: 营业收入同比下降9.25%,需关注行业景气度
2. **扣非净利润下降**: 扣非净利润下降24.03%,核心盈利能力承压
3. **现金流压力**: 自由现金流大幅为负,需关注投资回报
4. **数据完整性不足**: 部分关键数据缺失,影响深度分析
