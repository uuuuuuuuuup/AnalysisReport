## 四、MD&A 深度信息

> 数据来源：AI-Tools MCP GetFinancialReport + WebSearch

### 经营回顾

1. **营收利润双增长**：2025年实现营业收入36.99亿元，同比增长13.78%；归母净利润6.85亿元，同比增长17.49%；扣非净利润6.05亿元，同比增长20.09%
2. **海外市场高速增长**：海外业务实现收入12.67亿元，同比增长35.16%，收入占比提升至34.26%，海外业务毛利率达76.56%，显著高于国内业务水平
3. **农机导航与机器人板块**：营收7.52亿元，同比大增41.49%，成为增速最快的板块
4. **毛利率持续提升**：整体毛利率由2024年的58.13%提升至2025年的60.98%，提升4.89个百分点
5. **研发投入持续加码**：2025年研发投入5.58亿元，同比增长18.97%，研发投入占营业收入比例达15.08%，研发人员879人，占员工总数37.68%

### 前瞻指引

- 公司披露2026年经营目标：预计实现归母净利润7.88亿元，同比增长约15%
- 拟动用不超过20亿元闲置自有资金进行现金管理（期限2026年5月1日至2027年4月30日），投资品种包括协定存款、低风险理财产品、结构性存款、通知存款、大额存单、固定收益凭证等

### 资本配置意图

- 持续加大研发投入，聚焦高精度定位芯片、乘用车自动驾驶高精度定位产品、激光雷达等研发项目
- 新一代星地融合芯片StellaX正式发布，集成1892个通道
- 全球化布局加速，已在12个国家设立分支机构

### 风险因素

1. **政策依赖风险**：公司业务受益于北斗产业化、智慧农业、数字基建等政策支持，若相关政策调整，可能对业务增长产生影响
2. **技术创新和行业变化风险**：高精度导航定位技术与AI、自动驾驶等新兴行业融合加速，若公司不能准确把握技术趋势、持续研发升级，将丧失市场竞争力
3. **核心技术人员流失风险**：行业技术密集型特征显著
4. **地信板块下滑风险**：2023-2024年有万亿国债、地质灾害专项等集中政策/资金驱动带来一次性高基数；2025年专项预算收紧、项目进入收尾/验收期
