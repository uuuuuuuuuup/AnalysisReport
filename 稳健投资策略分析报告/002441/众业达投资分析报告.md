| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.7 | 分销行业标准，资本开支主要用于仓储物流 |

| 年均回购金额 | 0 亿元 | 近年无回购 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`  
= [2.30 × 90% × (1−0%) + 0] / 51.0 = **4.06%**

**修正**：考虑到2025年实际分红2.72亿元，使用实际分红计算：  
`粗算穿透回报率 = 实际分红 / 当前市值 = 2.72 / 51.0 = 5.33%`

但考虑到分红可持续性，使用扣非净利润口径更保守：  
`粗算穿透回报率 = [扣非净利润 × 支付率锚定值] / 当前市值 = [1.80 × 90%] / 51.0 = 3.18%`

综合考虑，取Owner Earnings口径：**粗算穿透回报率 R = 4.06%**

- 分配能力验证：OCF 4.94亿 > Owner Earnings 2.30亿 ✅
- 分配意愿：**强** — 股利支付率144.56%，愿意动用留存收益分红
- 税务泄漏：股息税率 = 0%（持股>1年）

> **粗算穿透回报率 R = 4.06%** — ✅ 通过（门槛值 = 3.75%）
