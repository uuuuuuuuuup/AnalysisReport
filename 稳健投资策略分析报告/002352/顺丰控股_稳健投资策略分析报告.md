### 决策理由

1. **穿透回报率边际不达标**：精算穿透回报率3.70%略低于门槛值3.77%，主要受低净利率（3.01%）和重资产模式制约，需盈利能力持续改善方可达标
2. **护城河深厚但利润率薄**：时效件市占率64.1%构成准垄断壁垒，但快递行业价格战和人工成本上行持续挤压利润空间，毛利率仅4.19%
3. **资本开支拐点已现**：Capex/营收从2021年9.3%降至2025年3.1%，自由现金流从147亿增至180亿，分配能力持续增强
4. **股东回报意愿显著提升**：分红支付率从24.8%提升至40%，A股回购方案上调至30-60亿元用于注销，综合股东回报率达55%

### 风险提示

1. **行业价格战加剧**：通达系单票收入仅2元左右，电商件竞争持续，可能传导至中高端市场挤压顺丰溢价
2. **人工成本刚性上行**：人工成本1,298亿元占营业成本48.6%，增速15.8%远超营收增速8.4%，持续侵蚀利润
3. **应收账款增速高于营收**：2025年应收账款同比增长10.4%高于营收增速8.4%，回款效率下降需关注
4. **货币资金大幅下降**：从340亿降至210亿（-38.2%），虽因分红回购和偿债所致，但现金缓冲减弱

### 后续跟踪事项

1. **2026年一季报验证**：关注扣非净利润增速是否回升至10%+，预计2026年4月底披露
2. **回购执行进度**：30-60亿元回购方案的实际执行节奏，关注是否如期注销
3. **行业价格走势**：关注快递行业单票收入变化，若价格战缓和则利好顺丰盈利改善

**Owner Earnings 历史数据（百万元）**

**粗算参数汇总**

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 维持性Capex系数 | 0.65 | 重资产物流业，但Capex高峰已过，取中偏低值 |
| 维持性资本开支估算 | 65 亿元 | 折旧摊销10,000百万元 × 0.65 |
| Owner Earnings | **127.64 亿元** | 扣非归母净利润92.64 + 折旧摊销100 − 维持性Capex65 |
| 支付率锚定值 | 40% | 公司五年分红规划锚定 |

| 股息税率 | 0% | A股持股>1年免征 |

**粗算公式**：`粗算穿透回报率 = [Owner_Earnings × 支付率锚定值 × (1−股息税率) + 年均回购] / 当前市值`

= [12,764 × 0.40 × 1.0 + 1,500] / 184,700

= [5,106 + 1,500] / 184,700

= 6,606 / 184,700

= **3.58%**

- 分配能力验证：OCF 27,555 > Owner Earnings 12,764 ✅
- 分配意愿：**强** — 支付率从24.8%提升至40%，回购方案上调至30-60亿用于注销，综合股东回报55%
- 税务泄漏：股息税率 = 0%（A股持股>1年免征）

> **粗算穿透回报率 R = 3.58%** — ⚠️ 边际不达标（门槛值×0.5 = 1.89% ≤ R = 3.58% < 门槛值 = 3.77%），继续进入因子3精算
