#### 模块一：资本消耗强度

**评级：中高**

- 公司属于重资产制造业（石膏板生产线），固定资产占总资产约41%
- 2025年在建工程从7.43亿增长至13.66亿，资本开支力度加大
- 维持性资本开支占折旧摊销比例约60-70%
- 嘉宝莉并购后涂料业务资本消耗低于石膏板，但整合期投入较大
- 海外产能布局（泰国、乌兹别克斯坦等）持续消耗资本

#### 模块二：收款模式

**评级：应收为主（偏弱）**

- 应收账款周转天数从31.45天（2024FY）恶化至59.79天（2025Q3）
- 应收账款规模从24.31亿（2024年末）大幅增长至63.84亿（2025Q3）
- 恶化原因：①嘉宝莉并表后涂料业务应收结构不同 ②下游房地产回款放缓
- 收款比率估算约97.7%，接近1但趋势恶化
- ⚠️ 需关注：若房地产持续低迷，应收账款减值风险上升

#### 模块三：竞争格局与护城河

**第一层（非技术性）：宽护城河**

1. **规模优势**：国内石膏板市占率60%+，远超第二名可耐福（~15%），产能覆盖全国
2. **品牌壁垒**："龙牌"高端品牌 + "泰山"中低端品牌，双品牌覆盖全价格带
3. **成本优势**：规模效应显著，脱硫石膏综合利用技术降低原材料成本
4. **渠道网络**：全国性经销网络深度下沉，渠道粘性强
5. **央企背景**：中国建材集团旗下，融资成本低，资源整合能力强

**第二层（技术性）：窄→中等护城河**

1. 石膏板技术壁垒相对不高，产品差异化有限
2. 护面纸、工业副产石膏等原材料供应存在区域约束
3. 高端产品（高强板、抗菌板）有一定技术差异化
4. 防水和涂料业务护城河尚在构建中，竞争力不如石膏板主业

**综合护城河评级：宽**（石膏板主业护城河极宽，但"两翼"护城河尚窄）

#### 模块四：周期性

**评级：强周期**

- 公司营收与房地产竣工面积高度相关（相关系数>0.7）
- 2025年归母净利润同比下降20.31%，反映行业下行周期
- 石膏板需求70%+来自建筑装饰（与地产竣工/装修挂钩）
- 防水业务同样受地产开工影响
- 涂料业务（嘉宝莉）C端占比更高，周期性略弱
- 当前处于房地产下行周期中段，需求端承压

#### 模块五：人力资本依赖

**评级：低**

- 制造业自动化程度较高，人力依赖度低
- 核心竞争力在于规模、品牌和渠道，而非个人能力
- 管理层更替（尹自波→管理）对公司运营影响有限

#### 模块六：管理层与治理结构

**评级：良好**

| 维度 | 评价 |
|------|------|
| 董事长/总经理 | 管理先生，49岁，央企体系内晋升，2025年1月接任董事长 |
| 管理层持股 | 核心管理层持股为0（央企限制） |
| 治理结构 | 央企治理规范，独立董事制度健全 |
| 资本配置 | 主动降杠杆（有息负债-57.3%），财务费用大降61.67% |
| 并购决策 | 嘉宝莉40.74亿收购，业绩承诺未达标，整合效果待观察 |
| 股东回报 | 稳定分红（40%支付率），但无回购 |

**关注点**：
- 管理层无持股，激励不足，但央企薪酬体系提供基本激励
- 嘉宝莉并购是近年最大资本配置决策，目前效果不佳
- 主动降杠杆是积极信号

#### 模块七：监管与政策风险

**评级：中**

1. **房地产调控政策**：影响下游需求，但政策已转向支持
2. **环保政策趋严**：增加合规成本，但利好龙头（小企业退出）
3. **央企改革**：可能带来治理优化，整体正面
4. **美国石膏板诉讼**：历史遗留或有负债，需持续关注

#### 模块八：MD&A 解读

**前瞻评级：中性偏负面**

- **经营回顾**：石膏板主业承压，"两翼"逆势增长但体量尚小
- **前瞻指引**：海外布局推进中，但短期贡献有限
- **资本配置**：降杠杆+维持分红+海外投资，优先级清晰
- **风险因素**：7项风险全面披露，房地产下行为核心风险
- **业绩趋势**：Q4归母净利润仅3.20亿（环比-50%+），需求端持续恶化
