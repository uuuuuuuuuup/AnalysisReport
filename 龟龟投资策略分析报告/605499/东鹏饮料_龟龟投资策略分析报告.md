## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

| 参数 | 值（百万元） | 说明 |
|------|------------|------|
| S = 营业收入 | 20,875.27 | 2025年度 |
| T = 应收账款净变动 | +18.69 | 2025年末~100百万 vs 2024年末81.31百万（增加为正） |
| U = 预收款/合同负债净变动 | +500（估） | 收入增长31.8%，预收款同步增长 |

保守处理（东鹏为先款后货模式，预收款确定性极高，可将净增加加回）：

```
真实现金收入 = S − T净增加 + U净增加（高确定性预收模式）
= 20,875.27 − 18.69 + 500
= 21,356.58 百万元

收款比率 = 21,356.58 / 20,875.27 = 102.3%
```

**收款比率 > 100%，反映先款后货模式下现金收入超过账面收入，极优。**
