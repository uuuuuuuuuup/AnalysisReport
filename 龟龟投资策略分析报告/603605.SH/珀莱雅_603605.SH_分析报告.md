## 三、因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

- 2024年归母净利润 = **15.52亿元**

### 步骤 2：Owner Earnings 粗算

```
Owner Earnings = 归母净利润 + 折旧摊销 - 资本开支
              = 15.52 + 1.00 - 3.00
              = 13.52 亿元
```

### 步骤 3：分配能力验证（一票否决）

```
FCF = OCF - Capex = 11.07 - 3.00 = 8.07 亿元
股息支付 = 4.69 亿元
FCF / 股息 = 8.07 / 4.69 = 1.72x
```

**分配能力验证：✅ 通过**（FCF充分覆盖股息支付）

- **现金上游障碍：无** ✅

- A股直接持有，股息红利税：持股>1年免征
- Q% = 0%

### 步骤 8：粗算穿透回报率

```
════ 主轨：市值口径 ════
R = [Owner Earnings × M × (1 - Q%) + O] / 当前市值
  = [13.52 × 30% × (1 - 0%) + 0.2] / 237.80
  = [4.06 + 0.2] / 237.80
  = 4.26 / 237.80
  = 1.79%

════ 辅轨：EV 口径 ════
EV = 当前市值 - 广义净现金 = 237.80 - 40.82 = 196.98 亿元
R_EV = 4.26 / 196.98 = 2.16%
```

### 粗算否决门判断

| 条件 | 判断 |
|------|------|
| R < Rf (1.81%)? | 1.79% < 1.81% → 是，检查EV口径 |
| R_EV > Rf? | 2.16% > 1.81% → 通过Rf检验 |
| R < II × 0.5 (1.91%)? | 1.79% < 1.91% → 是，检查EV口径 |
| R_EV > II × 0.5? | 2.16% > 1.91% → 通过 |
| R_EV < II (3.81%)? | 2.16% < 3.81% → 是 |

**粗算结论：⚠️ 边际不达标**（II × 0.5 ≤ R_EV < II），继续进入因子3精算
