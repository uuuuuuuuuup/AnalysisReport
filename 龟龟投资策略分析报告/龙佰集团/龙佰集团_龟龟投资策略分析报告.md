### 模块一：资本消耗强度

**评级：高**

- 2024FY资本开支15.59亿元，占营业收入5.7%，占OCF 41.1%
- 固定资产250.86亿元占总资产37.9%，属于重资产行业

- 维持性资本开支估算：折旧摊销14.00亿元×0.6=8.40亿元，占OCF 22.1%
- **结论**：资本消耗强度高，重资产制造业特征明显，需要持续资本投入维持产能

### 模块二：收款模式

**评级：现结为主，应收可控**

- 应收账款周转天数42.96天（2024FY），处于合理范围
- 应收账款35.53亿元，占营业收入12.9%
- 收款比率估算：1 - (42.96/365) ≈ 0.88，即约88%收入当期收到现金
- 预收/合同负债数据缺失，但钛白粉行业以现结为主
- **结论**：收款模式健康，以现结为主，应收风险可控

### 模块三：竞争格局与护城河

**第一层（非技术性）：窄护城河**

- 全球钛白粉CR4约44%，行业集中度较高
- 龙佰全球市占率约17%，国内约26%-29%，龙头地位明确
- 但钛白粉属于大宗化工品，产品差异化有限，价格竞争激烈
- 行业产能过剩，中小企业面临退出压力

**第二层（技术性）：窄偏中护城河**

- **规模优势**：产能151万吨/年，全球第一，规模效应显著
- **产业链一体化**：从钛矿→钛白粉→海绵钛全产业链布局，成本优势突出
- **技术壁垒**：氯化法钛白粉技术国内领先，突破海外技术封锁
- **成本优势**：上游矿资源自给率较高，循环经济降低成本
- **护城河风险**：行业产能持续扩张，海外巨头技术领先，新能源业务尚未验证

**综合评级：窄护城河**（规模+一体化构成一定壁垒，但大宗品属性限制护城河宽度）

## 四、因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

C = 21.69 亿元（2024FY）

### 步骤2：Owner Earnings 粗算

```
D（折旧摊销）≈ 14.00 亿元
G（维持性资本开支系数）= 0.6（重资产制造业典型值）
H = D × G = 14.00 × 0.6 = 8.40 亿元
I = C + D - H = 21.69 + 14.00 - 8.40 = 27.29 亿元
```

Owner Earnings = 27.29 亿元

### 步骤3：分配能力验证

- OCF = 37.97 亿元 > Owner Earnings = 27.29 亿元 ✅
- OCF - 股息 = 37.97 - 21.44 = 16.53 亿元 > 0 ✅
- **分配能力验证通过**

### 步骤4：现金上游障碍

- 广义净现金 = -139.51 亿元（负值，高负债）
- 净债务/市值 = 139.51/422 = 33.1%
- **存在现金上游障碍**：高负债意味着部分现金流需用于偿债，但未达否决程度

### 步骤5：分配意愿

- 2023年度支付率66.1%，2024年度支付率33.0%，2025年度预计64.1%
- 2024年度低支付率主因：回购云南国钛股权导致财务费用激增，年报不派发
- 2025年恢复至10派6元，分红意愿回升
- 回购方案5-10亿元进行中
- **分配意愿评级：中等**（有分红传统，但支付率波动较大）

### 步骤6：可预测性

- 强周期行业，利润波动大（2024年-32.79%，2025Q3 -34.68%）
- 收入相对稳定（个位数波动），但利润对价格高度敏感
- **可预测性评级：中低**

### 步骤7：税务泄漏修正

- A股持股>1年：股息税率Q = 5%
- 无跨境税务泄漏

### 步骤8：粗算穿透回报率

```
M = 60%（支付率锚定值，保守估计）
O = 5.00 亿元（年均回购金额）
Q = 5%

R = [C × M × (1-Q%) + O] / 当前市值
  = [21.69 × 60% × 95% + 5.00] / 422
  = [12.36 + 5.00] / 422
  = 17.36 / 422
  = 4.12%
```

**粗算穿透回报率 R = 4.12%**

### 粗算否决门判断

- R = 4.12% > Rf = 1.762% ✅
- R = 4.12% > II × 0.5 = 1.88% ✅
- R = 4.12% > II = 3.762% ✅（但安全边际仅0.36pct，边际不达标区域）

**粗算结论：边际通过**（R略高于门槛，进入因子3精算验证）
