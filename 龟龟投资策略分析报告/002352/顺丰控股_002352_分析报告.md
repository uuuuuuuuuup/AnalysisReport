## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

#### Step 1: 确定基准参数

**输入数据**:
- 当前股价：37.32 元
- 总股本：49.13 亿股
- 当前市值：1833 亿元
- 2024 年归母净利润：101.70 亿元
- 无风险利率（Rf）：假设 2.5%（十年期国债）
- A 股门槛收益率：max(3.5%, Rf + 2%) = 4.5%

#### Step 2: Owner Earnings 粗算

**公式**: Owner Earnings ≈ 归母净利润 + 折旧摊销 - 维持性资本开支

**估算**:
- 归母净利润：101.70 亿元
- 折旧摊销（D&A）：估计 90 亿元（基于固定资产和在建工程估算）
- 维持性资本开支：估计 60 亿元（假设 Capex 的 60% 为维持性）

**Owner Earnings 粗算**:
```
I = 101.70 + 90 - 60 = 131.70 亿元
```

#### Step 3: 分配能力验证

**历史分红**: ⚠️待补充确切数据

假设:
- 2024 年分红：约 40 亿元（估计支付率 40%）
- 2024 年回购：⚠️待确认

**分配能力**:
- 自由现金流：321 亿元
- 分红 + 回购：估计 40-50 亿元
- 分配覆盖率：321 / 50 = 6.4 倍（安全）

#### Step 4: 粗算穿透回报率

**假设支付率 M = 40%**

**公式**: 
```
R = [归母净利润 × M + 回购] / 当前市值
```

假设回购为 0:
```
R = [101.70 × 40%] / 1833
R = 40.68 / 1833
R = 2.22%
```

**考虑 Owner Earnings 口径**:
```
R_alt = [131.70 × 40%] / 1833
R_alt = 52.68 / 1833
R_alt = 2.87%
```

#### Step 5: 与门槛比较

- 粗算回报率 R：2.22% - 2.87%
- A 股门槛 II：4.5%
- **R < II**，甚至 **R < II × 0.5 (2.25%)**

#### Step 6: EV 口径检查（因净现金/市值比例低，不适用）

- 净现金：约 4 亿元
- 净现金/市值：0.2%（远低于 40% 阈值）
- **不适用 EV 口径**

### 因子 2 结论

**⚠️ 粗算回报率低于门槛**

- 粗算回报率：2.22% - 2.87%
- 门槛收益率：4.5%
- **R < II × 0.5**，触发否决检查

**否决门判断**:
- 若 EV 口径仍不达标 → 否决
- 但 EV 口径不适用（净现金比例太低）

**初步结论**: **因子 2 粗算不达标**，但考虑到：
1. 估算较为保守
2. 支付率假设可能偏低
3. 需要因子 3 精算验证

**建议**: 进入因子 3 精算，进行更准确的现金流分析

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 核心原则

**极端保守假设**，逐行追踪现金流底层

### Step 1: 真实现金收入还原

**2024 年数据**:
- 营业收入 S：2844.20 亿元
- 应收账款变动：2024 年末 277.15 亿 - 2023 年末约 260 亿 = +17.15 亿（增加）
- 预收款/合同负债变动：⚠️待确切数据（估计变动不大）

**保守处理**:
- 应收账款净增加 17.15 亿元 → 扣除（收入未收到现金）
- 预收款变动：假设不变

**真实现金收入**:
```
真实现金收入 = S - 应收账款净增加
真实现金收入 = 2844.20 - 17.15 = 2827.05 亿元
```

**收款比率**:
```
收款比率 = 2827.05 / 2844.20 = 99.4%
```

**结论**: 收款比率很高（99.4%），收入质量优秀

### Step 2: 应收账款质量核查

虽然收款比率>1，仍做基本检查：

- **应收账款周转天数**: 33.4 天（合理）
- **坏账计提**: ⚠️待财报附注（估计 3-5%）
- **关联方应收**: ⚠️待确认
- **收入确认激进度**: 不高（现金流匹配度好）

### Step 3: 非经常性现金流入分类

**2024 年非经常性损益**: 10.24 亿元

**分类**（估计）:
- 资产处置收入 V1：估计 5 亿元（保留项）
- 政府补贴 V2：估计 3 亿元（扣除项）
- 其他 V4：估计 2.24 亿元（扣除项）

**应扣除合计**:
```
V_deduct = V2 + V4 = 3 + 2.24 = 5.24 亿元
```

### Step 4: 经营性现金支出还原

**2024 年数据**:
- 营业成本：2715.60 亿元
- 应付账款变动：估计 +10 亿元（占用供应商资金）
- 支付员工现金：估计 1200 亿元（人工成本）
- 现金税款：估计 25 亿元
- 现金利息：估计 15 亿元

**经营性现金支出 W**（估计）:
```
W ≈ 营业成本 + 人工 + 税款 + 利息 - 应付增加
W ≈ 2715.60 + 1200 + 25 + 15 - 10
W ≈ 3945.60 亿元
```

⚠️ **注意**: 此为粗略估计，精确数据需查阅现金流量表附注

### Step 5: 资本开支与投资的极端保守处理

**2024 年数据**:
- 资本开支总额 E：估计 100 亿元（全额扣除）
- 投资性购买 X1：估计 20 亿元（对外投资）
- 隐性必要支出 X2：估计 10 亿元（研发资本化等）

**总扣除额 Y**:
```
Y = E + X1 + X2 = 100 + 20 + 10 = 130 亿元
```

### Step 6: 准则差异核查

**A 股财报特点**:
- 收入确认：相对保守
- 递延税资产：正常
- 资本化利息：比例不高

**或有负债**: ⚠️待财报附注确认

### Step 7: 真实可支配现金结余

**公式**:
```
真实可支配现金结余 = 真实现金收入 
                    + 资产处置收入 V1
                    - 应扣除非经常性流入 V_deduct
                    - 经营性现金支出 W
                    - 总扣除额 Y
```

**代入**（基于估计）:
```
真实可支配现金结余 = 2827.05 + 5 - 5.24 - 3945.60 - 130
```
