## 四、穿透回报率分析（因子2/3适配）

### 4.1 ETF回报率构成

对于ETF投资，穿透回报率主要由两部分构成：

```
穿透回报率 = 股息率 + 预期增长率

股息率（当前）= 3.88%

预期增长率（保守估计）：
- 行业调整期：假设未来3年增长率为0-5%
- 复苏后：假设恢复正常增长5-10%

保守情景（调整期）：
R = 3.88% + 0% = 3.88%

中性情景（复苏后）：
R = 3.88% + 5% = 8.88%

乐观情景（高增长）：
R = 3.88% + 10% = 13.88%
```

### 4.2 回报率与门槛对比

| 情景 | 穿透回报率 | vs 门槛（3.81%） | 判断 |
|------|------------|------------------|------|
| 保守（调整期） | 3.88% | +0.07% | 边际达标 |
| 中性（复苏后） | 8.88% | +5.07% | 明确达标 |
| 乐观（高增长） | 13.88% | +10.07% | 高回报 |

### 4.3 安全边际计算

```
保守情景安全边际 = 3.88% - 3.81% = 0.07%（边际）

中性情景安全边际 = 8.88% - 3.81% = 5.07%（充足）

周期性修正：
- 当前处于周期底部 → 门槛可下调1pct
- 修正后门槛 = 3.81% - 1% = 2.81%
- 修正后安全边际（保守）= 3.88% - 2.81% = 1.07%
```
