**假设维持性资本开支系数 G = 0.6**（水泥行业重资产特性）

```

- 资本消耗强度：**中等**
- 公司处于资本开支收缩期，有利于改善自由现金流
- 随着资本开支进一步下降，FCF有望持续改善

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
集团净利润 [A] = 89.40亿元
少数股东损益 [B] = 12.44亿元
GAAP归母净利润 [C] = A - B = 76.96亿元
```

### 步骤2：Owner Earnings粗算

```
折旧摊销 [D] = 75.00亿元（估算）
资本开支 [E] = 133.26亿元
维持性资本开支系数 [G] = 0.6（水泥行业重资产特性）
维持性资本开支 [H] = D × G = 75.00 × 0.6 = 45.00亿元

Owner Earnings [I] = C + D - H
                   = 76.96 + 75.00 - 45.00
                   = 106.96亿元
```

### 步骤3：分配能力验证（一票否决）

```
经营现金流 [OCF] = 184.76亿元
资本开支 [Capex] = 133.26亿元
