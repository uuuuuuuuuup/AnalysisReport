## 三、因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤1：归母净利润**
- C = 35.56 亿元

**步骤2：Owner Earnings 粗算**
- C = 35.56
- D = 7.00（折旧摊销估算）
- G = 0.60（维持性资本开支系数）
- H = D × G = 4.20
- I = C + D − H = 38.36 亿元

**步骤3：分配能力验证**
- OCF = 48.82 亿元 > 股息 14.93 亿元 ✅ 通过

**步骤4：现金上游障碍**
- 少数股东权益占比较低（5%），子公司多为全资或控股
- 障碍评级：**低**

**步骤5：分配意愿**
- 支付率均值 41.1%，DPS持续提升
- 回购金额 2.93亿/年
- 意愿评级：**中等偏强**

**步骤6：可预测性**
- 营收增速放缓但为正，医疗需求长期确定
- 可预测性评级：**中等**

**步骤7：税务泄漏修正**
- A股持股>1年免征红利税
- Q = 0%

**步骤8：粗算穿透回报率**

```
R = [C × M × (1 − Q%) + O] / 当前市值
R = [35.56 × 0.4182 × 1 + 2.93] / 885.91
R = [14.87 + 2.93] / 885.91
R = 17.80 / 885.91
R = 2.01%
```

### 粗算否决门判断

| 条件 | 结果 |
|:---|:---|
| R < Rf (1.762%)? | 否（2.01% > 1.762%） |
| R < II × 0.5 (1.88%)? | 否（2.01% > 1.88%） |
| II × 0.5 ≤ R < II? | **是**（1.88% ≤ 2.01% < 3.762%） |

**粗算结论：边际不达标**——R 处于门槛的 0.5× 到 1× 之间，继续进入因子 3 精算。

## 四、因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤1：真实现金收入还原

```
营业收入 S = 209.83 亿元
应收账款净变动 T ≈ +0.94 亿元（2024年AR增加）
预收款/合同负债净变动 U = 未单独披露（估算为0）

保守处理：应收账款净增加 → 扣除
真实现金收入 = S − T = 209.83 − 0.94 = 208.89 亿元
收款比率 = 208.89 / 209.83 = 99.6%
```

### 步骤2：应收账款 footnote 核查

收款比率 99.6% > 99%，无需深入核查。但注意：
- 应收账款周转天数33.39天（合理，含医保结算）
- AR从18.99亿（2024Q3）增至21.25亿（2025Q3），增幅12%

### 步骤4：经营性现金支出还原

```
第一优先级（现金流量表直接取数，2024FY）：
- 供应商支付 = 53.61 亿元
- 员工支付   = 64.89 亿元
- 现金税款   = 11.83 亿元
- 现金利息   = ⚠️未找到（估算 ~2.00 亿元）

OCF倒推法验证：
W = 真实现金收入 − OCF = 208.89 − 48.82 = 160.07 亿元

⚠️ 差异 = 160.07 − 132.33 = 27.74 亿元
差异原因：现金流量表可能包含其他经营性支出未在明细中体现

采用OCF倒推法：W = 160.07 亿元
标注：⚠️ 现金支出明细不足，已采用OCF倒推法估算W
```

### 步骤5：资本开支与投资的极端保守处理

```
(a) 资本开支总额 E = 5.03 亿元（全额扣除）
(b) 投资性购买 X1 = 投资现金流 − Capex = 28.48 − 5.03 = 23.45 亿元
    （含收购39家医院9.63亿、深圳广晟6.50亿等）
(c) 隐性必要支出还原 X2 ≈ 0 亿元（估算）

总扣除额 Y = E + X1 + X2 = 5.03 + 23.45 + 0 = 28.48 亿元
```

### 步骤6：准则差异核查

| 检查项 | 结果 |
|:---|:---|

| 或有负债 | 未披露明细，需关注对外担保和未决诉讼 |

### 步骤7：真实可支配现金结余

```
真实可支配现金结余 = 真实现金收入
                    + 资产处置收入 V1
                    + 其他投资收入 V5
                    − 应扣除非经常性流入 V_deduct
                    − 经营性现金支出 W
                    − 总扣除额 Y

= 208.89 + 0.06 + 1.53 − 0.50 − 160.07 − 28.48
= 21.43 亿元（2024FY）
```

**合理性校验：**

```
AA_fcf = OCF − Capex = 48.82 − 5.03 = 43.79 亿元

比较：AA = 21.43 vs AA_fcf × 1.5 = 65.69
AA < AA_fcf × 1.5 → 校验通过 ✅

AA / AA_fcf = 21.43 / 43.79 = 48.9%
差异原因：总扣除额Y中包含23.45亿并购投资，占OCF的48%
```

**历史均值 AA = 21.43 亿元**（基于2024FY单年数据，保守取值）
