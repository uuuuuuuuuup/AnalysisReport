### 3.2 穿透回报率粗算公式

**粗算公式**:
```
R% = (E / P) + g
```

其中:
- E = 正常化净利润
- P = 当前市值
- g = 长期增长率

- 5 年平均：约 7.0 亿元
- **正常化净利润**: 取 7.0 亿元（保守估计）

**长期增长率 (g)**:
- **历史增速**: 5 年收入 CAGR 约 10-15%，利润 CAGR 约 15-20%
- **行业增速**: 预调酒行业长期增速约 10-15%
- **公司增速**: 龙头地位稳固，增速略高于行业
- **保守估计**: 取 g = 8-10%

**当前价格对应回报率**:
```
R% = (7.0 / 198.5) + 9% = 3.53% + 9% = 12.53%
```
