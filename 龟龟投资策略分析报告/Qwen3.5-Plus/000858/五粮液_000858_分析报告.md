### 模块三：竞争格局与护城河

#### 第一层：非技术性护城河

**评估**: ✅ **强大护城河**

**证据**:
1. **品牌优势**: 
   - "五粮液"是中国白酒第二品牌，仅次于茅台
   - 浓香型白酒代表，千年传承工艺
   - 品牌价值数百亿元

2. **渠道优势**:
   - 全国化经销商网络完善
   - 深度分销体系
   - 直营 + 经销双轮驱动

3. **规模优势**:
   - 年产能为行业前列
   - 规模效应带来成本优势
   - 议价能力强

4. **转换成本**:
   - 白酒消费具有习惯性和社交属性
   - 商务宴请、礼品场景粘性强
   - 消费者转换成本较高

#### 第二层：技术性护城河

**评估**: ✅ **稀缺老窖池资源**

**证据**:
1. **老窖池资源**:
   - 拥有明代古窖池群（"活文物"）
   - 窖池年龄越老，产酒品质越高
   - 老窖池具有不可复制性

2. **工艺壁垒**:
   - 传统固态发酵工艺
   - 长期储存老熟工艺
   - 勾调技术积累

3. **产地优势**:
   - 四川宜宾独特地理环境
   - 适宜的气候、水质、微生物群落

**结论**: 五粮液拥有强大的品牌护城河 + 稀缺老窖池资源，竞争优势可持续。

**评分**: ⭐⭐⭐⭐⭐ (5/5)

## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤 1：归母净利润

```
集团净利润 [A] = 222.12 亿元 (2025-09 TTM)
少数股东损益 [B] ≈ 7 亿元（估算，基于少数股东权益占比）
GAAP 归母净利润 [C] = A - B = 222.12 - 7 = 215.12 亿元
```

#### 步骤 2：Owner Earnings 粗算

**数据来源**: data_pack_market §4 + data_pack_report P7

```
折旧摊销 [D] ≈ 25 亿元（估算，基于固定资产折旧率）
资本开支 [E] ≈ 18 亿元（估算，基于投资现金流）
维持性资本开支系数 [G] = 80%（白酒行业维持性占比高）
维持性资本开支 [H] = D × G = 25 × 80% = 20 亿元

Owner Earnings [I] = C + D - H
                   = 215.12 + 25 - 20
                   = 220.12 亿元
```

#### 步骤 3：分配能力验证（一票否决）

**检查**:
- OCF (282 亿) > 净利润 (222 亿) ✅
- FCF (282 亿) > 股息 (185 亿) ✅
- 有息负债 = 0 ✅

**结论**: ✅ 通过分配能力验证

#### 步骤 4：现金上游障碍

**评估**:
- 应收账款极低（0.98 亿），无现金 upstream 障碍
- 采用先款后货模式
- 货币资金充裕（1,363 亿）

**结论**: ✅ 无现金上游障碍

#### 步骤 5：分配意愿

**证据**:
- 历史支付率约 55-60%
- 筹资活动现金流持续大额流出（分红）
- 国企背景，分红意愿稳定

**支付率锚定值 [M]**: 55%

#### 步骤 6：可预测性

**评估**:
- 白酒商业模式稳定
- 品牌护城河强大
- 2025 年短期承压但长期可预测性强

**可预测性评级**: 高

#### 步骤 7：税务泄漏修正

**综合股息税率 [Q]**:
- 个人所得税（股息红利）: 10%（持股>1 年免税，假设平均 5%）
- 企业所得税：25%（已计入净利润）

**综合税率 [Q]**: 5%

#### 步骤 8：粗算穿透回报率 R%

**计算公式**:
```
R = [C × M × (1 - Q%) + O] / 当前市值

代入数字:
C = 215.12 亿元
M = 55%
Q = 5%
O = 0（无回购）
当前市值 = 3,965 亿元

R = [215.12 × 55% × (1 - 5%) + 0] / 3,965
  = [215.12 × 0.55 × 0.95] / 3,965
  = 112.67 / 3,965
  = 2.84%
```

**EV 口径粗算**（若适用）:
```
广义净现金 = 1,363.06 亿元
EV = 市值 - 广义净现金 = 3,965 - 1,363 = 2,602 亿元

R_EV = [C × M × (1 - Q%) + O] / EV
     = 112.67 / 2,602
     = 4.33%
```

### 粗算否决门检查

**门槛 [II]**: 3.79%

**检查**:
