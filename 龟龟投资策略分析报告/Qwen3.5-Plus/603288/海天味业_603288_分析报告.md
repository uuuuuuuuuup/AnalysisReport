## 因子 2：穿透回报率粗算（Top-Down）

### 数据准备（单位：亿元）

| 参数 | 数值 | 说明 |
|------|------|------|
| AA（正常化净利润） | 55 | 2020-2024 年平均 |
| M（盈利可持续性系数） | 55% | 保守估计 |
| Q%（Capex/净利润） | 60% | 历史平均 |
| O（其他 FCF 来源） | 0 | 无显著其他来源 |
| 市值 | 2210 | 当前市值 |

### 粗算结果

```
GG = [AA × M × (1-Q%) + O] / 市值
GG = [55 × 55% × (1-60%) + 0] / 2210
GG = 12.1 / 2210
GG ≈ 0.55%
```

**结论**：❌ GG ≈ 0.55% < 3.5%，不达标

## 因子 3：穿透回报率精算（Bottom-Up）

- 资本开支 Capex：103 亿（异常高）
- FCF = 54.06 - 103 = **-48.94 亿**

**正常化假设**：
- 正常年份 OCF：70-80 亿（参考 2020-2024 年）
- 正常年份 Capex：35-40 亿
- 正常年份 FCF：35-40 亿

### 精算结果（使用正常化数据）

```
AA（正常化 FCF）= 40 亿
M = 55%

O = 0

GG = [40 × 55% × 90% + 0] / 2210

GG ≈ 0.90%
```

- 2025 年特别分红：10 派 3 元
- 股利支付率：75.24%（2024 年）
- 分红承诺：未来三年不低于 70%

### 安全边际计算

```
门槛回报率 II = 3.5%
穿透回报率 GG = 0.90%

安全边际 = (GG - II) / II
安全边际 = (0.90% - 3.5%) / 3.5%
安全边际 = -74.3%
```

**结论**：❌ 负安全边际，估值偏高

### 目标买入价计算

```
目标买入价_base = 当前市值 × (GG / II) / 总股本
目标买入价_base = 2210 × (0.90% / 3.5%) / 59.1
目标买入价_base = 9.63 元

现金保护调整（极强保护 15% 折扣）：
目标买入价 = 9.63 × 0.85 = 8.19 元
```

**目标买入价**：8.19 元，较当前股价 37.39 元有 **78% 的下行空间**

### 最终结论

**⚠️ 高优先级观察，暂不建仓**

**核心逻辑**：
1. **优质企业**：海天是调味品行业绝对龙头，护城河深，财务健康
2. **短期困境**：2025 年资本开支异常高（103 亿），导致 FCF 为负
3. **估值偏高**：穿透回报率仅 0.90%，远低于门槛 3.5%
4. **负 EV 现象**：现金储备超过市值，市场极度悲观
5. **观察等待**：等待 FCF 转正、资本开支回落、市场情绪修复

### 建仓条件（需同时满足）

1. ✅ 单季度 FCF 转正，且连续 2 季度为正
2. ✅ 资本开支大幅下降至 40-50 亿/年
3. ✅ 股价回调至 30 元以下（对应 GG ≈ 3.5%）
4. ✅ 穿透回报率 GG 回升至 3.5%+

### 风险提示

1. **资本开支持续高企**：如果 2026 年 Capex 继续维持 100 亿+，FCF 将持续为负
2. **竞争加剧**：千禾等品牌崛起，可能侵蚀海天市场份额
3. **成本上涨**：大豆、白糖等原材料价格大幅上涨
4. **食品安全**：食品添加剂标准趋严，可能增加合规成本
