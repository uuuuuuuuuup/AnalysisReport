### 模块一：资本消耗强度

- Capex / 营业收入 = 29.19 / 123.69 = **23.6%**（较高）
- Capex / OCF = 29.19 / 55.31 = **52.8%**（中等）

**维持性 vs 成长性资本开支**:
- ⚠️ 无法精确区分
- **估算**: 维持性 Capex 约占 40-50%（跑道/航站楼维护）
- 成长性 Capex：扩建项目（卫星厅、第三条跑道等）

**自由现金流质量**:
- 2024 年 FCF = 22.80 亿元

**模块一结论**: 🟡 **中等偏好**
- 资本消耗强度较高（行业特性）
- 但 FCF 转化率高，现金流质量好
- 需关注未来资本开支计划（扩建周期）

### 模块三：竞争格局与护城河

#### 第一层：非技术性护城河

**1. 地理位置垄断** ⭐⭐⭐⭐⭐
- 上海浦东机场：中国最大国际航空枢纽
- 上海虹桥机场：国内航线核心枢纽
- **独占上海航空市场**（中国最大航空市场之一）
- 无法复制的区位优势

**2. 行政准入壁垒** ⭐⭐⭐⭐⭐
- 机场建设需要国家发改委审批
- 上海地区不可能新建第二家机场运营商
- **行政垄断地位稳固**

**3. 规模效应** ⭐⭐⭐⭐
- 航线网络效应：航空公司倾向于集中运营
- 旅客规模带来商业价值（免税、广告）
- 固定成本摊薄（跑道、航站楼折旧）

**4. 转换成本** ⭐⭐⭐
- 航空公司转换机场成本高（基地设施、时刻资源）
- 旅客无转换成本（但受航线限制）

#### 第二层：技术性护城河

**1. 运营效率** ⭐⭐⭐
- 浦东机场容量：8000 万人次/年（全球前列）
- 运行效率：国际领先
- 但技术壁垒不高

**2. 数字化能力** ⭐⭐⭐
- 智慧机场建设
- 但非核心竞争优势

**模块三结论**: 🟢 **强护城河**

**护城河评级**: **宽护城河（Wide Moat）**

**核心优势**:
1. 地理位置垄断（不可复制）
2. 行政准入壁垒（政策保护）
3. 规模效应（网络效应）

**风险点**:
- 高铁分流（短途航线）
- 周边机场竞争（杭州、南京）
- 免税政策变化

### 模块四：周期性

**行业周期属性**: 🟡 **弱周期性 + 事件驱动**

**周期驱动因素**:

1. **宏观经济周期**:
   - GDP 增长 → 商务出行/旅游需求
   - 相关性：中等偏高

2. **油价周期**:
   - 影响航空公司盈利能力 → 间接影响机场起降费收入
   - 相关性：中等

3. **汇率周期**:
   - 人民币贬值 → 抑制出境游
   - 相关性：中等

4. **突发事件**:
   - 疫情（2020-2022 年）：毁灭性打击
   - 地缘政治：影响国际航线
   - 相关性：高（但不可预测）

**上海机场周期特征**:
- **国际航线占比高** → 受宏观经济和汇率影响大
- **免税业务** → 受消费意愿影响
- **固定成本高** → 经营杠杆高（盈亏平衡点以上利润弹性大）

**当前周期位置** (2026-03):
- 疫情后恢复期（2024-2025 年强劲反弹）
- 国际航线恢复程度：⚠️ 待确认（vs 2019 年）
- **判断**: 🟡 **恢复中段**（可能尚未完全恢复至 2019 年水平）

**模块四结论**: 🟡 **中等周期性**
- 非典型强周期行业
- 但受宏观和突发事件影响大
- 当前处于疫情后恢复期

1. **宏观经济风险**:
   - 经济放缓 → 商务出行/旅游需求下降
   - 影响：营收增速放缓

2. **国际关系风险**:
   - 地缘政治紧张 → 国际航线受限
   - 影响：国际航线恢复受阻

3. **政策风险**:
   - 免税政策调整（扣点率上限）
   - 航空性收费政府指导价下调
   - 影响：利润率承压

4. **汇率风险**:
   - 人民币贬值 → 抑制出境游
   - 影响：国际航线客流下降

5. **竞争风险**:
   - 高铁网络分流（短途航线）
   - 周边机场竞争（杭州、南京）
   - 影响：客流增长放缓

6. **资本开支风险**:
   - 扩建项目（第三条跑道、卫星厅）
   - 影响：Capex 上升，FCF 下降

### 待验证事项

**高优先级**:

1. ⚠️ **2024 年年报全文**:

## 附录：关键计算公式汇总

### 因子 2 粗算

```
I = C + D - H
  = 归母净利润 + 折旧摊销 - 维持性资本开支

R = [I × M × (1 - Q%) + O] / 市值
```

### 因子 3 精算

```
真实现金收入 = S - T 的净增加 - U 的净减少

真实可支配现金结余 = 真实现金收入 + V1 + V5 - V_deduct - W - Y

GG = [AA × M × (1 - Q%) + O] / 市值
```

### 因子 4 估值

```
安全边际 JJ = GG - II

目标买入价_base = 市值 × (GG / II) / 总股本

目标买入价 = 目标买入价_base × (1 - 现金保护折扣)
```
