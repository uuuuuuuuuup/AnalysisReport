## 因子 2: 穿透回报率粗算（Top-Down）

### 步骤 1: 归母净利润

**C (GAAP 归母净利润)** = 12.41 亿元（2025 年 TTM）

### 步骤 2: Owner Earnings 粗算

```
D (折旧摊销) = 8.0 亿元（估算）
G (维持性资本开支系数) = 0.8（啤酒行业，80% Capex 为维持性）
H (维持性资本开支) = D × G = 8.0 × 0.8 = 6.4 亿元

I (Owner Earnings 粗算) = C + D - H
                        = 12.41 + 8.0 - 6.4
                        = 14.01 亿元
```

**判断**: 可预测性高 ✅

### 步骤 7: 税务泄漏修正

**综合股息税率 (Q)**: 10%（个人投资者通过沪深股通）  

### 步骤 8: 粗算穿透回报率

```
M (支付率锚定值) = 50%
O (回购) = 0

R (粗算穿透回报率) = [C × M × (1 - Q%) + O] / 市值
                   = [12.41 × 50% × (1 - 10%) + 0] / 272.1
                   = [5.58] / 272.1
                   = 2.05%
```

### 粗算否决门检查

**门槛 (II)**: 3.79%（A 股门槛 = max(3.5%, 1.79%+2%)）

**判断**:
- R (2.05%) < II (3.79%) ❌
- R (2.05%) > II × 0.5 (1.90%) ✅

**结论**: ⚠️ **边际不达标**，需进入因子 3 精算

## 因子 3: 穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1: 真实现金收入还原

```
S (营业收入) = 130.59 亿元（2025 年 TTM）

U (合同负债净变动) = 约 0（假设稳定）

真实现金收入 = S - T 的净增加部分
             = 130.59 - 1.69
             = 128.90 亿元

```

**应扣除合计**: 0.5 亿元

### 步骤 4: 经营性现金支出还原

```

           = 98.43 + 0（应付账款稳定）
           = 98.43 亿元
```


### 步骤 6: 准则差异核查

**判断**: ✅ 无重大准则差异

### 步骤 7: 真实可支配现金结余

```
真实可支配现金结余 = 真实现金收入
                    + 资产处置收入 V1
                    + 其他投资收入 V5
                    - 应扣除非经常性流入 V_deduct
                    - 经营性现金支出 W
                    - 总扣除额 Y

                  = 0.53 亿元
```
