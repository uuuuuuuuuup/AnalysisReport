## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

2024 年归母净利润 = 40.39 亿元

### 步骤 2：Owner Earnings 粗算

```
Owner Earnings ≈ 归母净利润 + 折旧摊销 - 维持性资本开支

假设：
- 折旧摊销 ≈ 20 亿元（估计）
- 维持性资本开支 ≈ 30 亿元（估计，基于行业特点）

Owner Earnings ≈ 40.39 + 20 - 30 = 30.39 亿元
```

**股息支付**：
- ⚠️ 未找到详细股息数据

- ⚠️ 尚未实施

**股东回报合计** ≈ 12.12 + 5.27（假设回购中值） = 17.39 亿元

### 步骤 4：现金上游障碍

**受限资金**：⚠️ 未找到  
**假设**：受限资金比例 10-20%

### 步骤 5：分配意愿

**评定**：✅ **中等偏上**
- 央企背景，分红意愿稳定
- 提议回购显示价值认可

### 步骤 6：可预测性

**评定**：⚠️ **中等**
- 受行业周期影响大
- 央企背景提供一定稳定性

### 步骤 7：税务泄漏修正

**假设**：综合税率 10-15%

### 步骤 8：粗算穿透回报率 R%

```
假设：
- Owner Earnings = 30.39 亿元
- 支付率 M = 30%
- 回购 O = 5.27 亿元
- 税率 Q = 10%

可分配现金 = 30.39 × 30% × (1 - 10%) + 5.27
           = 9.12 × 0.9 + 5.27
           = 8.21 + 5.27
           = 13.48 亿元

粗算穿透回报率 R = 13.48 / 893.49 = 1.51%
```

**粗算结果**：R ≈ 1.51%

### 粗算否决门检查

```
门槛 II = 3.85%
II × 0.5 = 1.925%

R = 1.51% < II × 0.5 = 1.925%

→ 需要检查 EV 口径
```

**EV 口径检查**：

```
广义净现金 = 850.27 - 1,427.60 = -577.33 亿元（净负债）
EV = 市值 - 广义净现金 = 893.49 - (-577.33) = 1,470.82 亿元

R_EV = 13.48 / 1,470.82 = 0.92%

R_EV < R < II

→ 粗算不达标
```

### 因子 2 结论

**⚠️ 粗算穿透回报率不达标**

- 市值口径：R ≈ 1.51% < II = 3.85%
- EV 口径：R_EV ≈ 0.92% < II = 3.85%

**但考虑到**：
1. 上述计算基于保守假设
2. 2025 年业绩已回暖（前三季度净利润同比增长 5.22%）
3. 公司持有大量投资性房地产和子公司股权，账面价值可能被低估

**决定**：⚠️ **进入因子 3 精算，使用更精确数据验证**
