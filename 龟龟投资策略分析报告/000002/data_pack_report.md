## 四、MD&A 深度信息

> 数据来源：WebSearch - 证券之星管理层讨论与分析/新浪财经年报解读/东方财富网

### 4.1 经营回顾

| 维度 | 内容 |
|:---|:---|
| 营业收入 | 2025年营收2,334.3亿元，同比下降32.0% |
| 房地产开发结算收入 | 1,701.1亿元，同比降幅39% |
| 销售金额 | 同比下降45.5% |
| 开发业务毛利率 | 2.0%，历史低位 |
| 综合毛利率 | 9.5%，同比下降0.7个百分点 |
| 交付情况 | 2025年完成交付11.7万套，近两年需交付量约70%已完成 |
| 存货规模 | 3,737.4亿元，较2024年底下降28.0% |

### 4.2 亏损原因分析

1. **高地价项目结算**：高地价项目进入结算周期，叠加市场下行，结算毛利率低位徘徊，开发业务结算毛利大幅减少
2. **减值集中释放**：新增计提信用减值损失341.74亿元、资产减值损失219.29亿元，合计561.03亿元，占净亏损总额63.35%
3. **经营性业务亏损**：部分经营性业务扣除折旧摊销后仍亏损，非主业财务投资亏损
4. **资产处置损失**：部分大宗资产和股权交易价格低于账面值
5. **投资收益恶化**：对联营企业和合营企业的投资收益从上年亏损32亿扩大至60亿

### 4.3 前瞻指引与资本配置意图

| 维度 | 内容 |
|:---|:---|
| 2026年战略重心 | 聚焦化险，稳筑经营底盘 |
| 资产处置 | 积极联动政府部门与央国企，与深业集团达成中山湾中新城项目股权交易 |
| 保障性租赁住房 | 响应国家政策，完成杭州彩虹天空之城、大连两个自持租赁住宅交易 |
| 经营性资产退出 | 深化与国内外知名机构投资者合作，在北京、上海、厦门等地完成多个经营性资产退出 |
| 债务管理 | 积极争取金融机构支持，推进债务重组 |

### 4.4 风险因素

1. **流动性风险**：短期有息负债远超现金储备，存在重大不确定性
2. **市场风险**：房地产市场持续下行，销售规模下降
3. **信用风险**：行业风险敞口升高，其他应收款坏账风险大
4. **持续经营风险**：审计报告提示持续经营重大不确定性
