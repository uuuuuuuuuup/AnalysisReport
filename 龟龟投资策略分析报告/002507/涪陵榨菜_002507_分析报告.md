## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤 1：归母净利润
```
C = 6.73 亿元（2025 年 Q1-Q3 TTM）
```

#### 步骤 2：Owner Earnings 粗算
```
D（折旧摊销） = 0.8 亿元（估算）
G（维持性资本开支系数） = 25%
H = D × G = 0.8 × 25% = 0.2 亿元

I（Owner Earnings） = C + D − H
                    = 6.73 + 0.8 − 0.2
                    = 7.33 亿元
```

#### 步骤 3：分配能力验证
```
假设股息总额 = 3 亿元（估算）
支付率 M = 3 / 7.99 = 37.5%

检查：
- 净利润 > 0 ✅
- OCF > 净利润？44.70 > 6.73 ✅
- 有息负债 = 0 ✅

分配能力：**通过**
```

#### 步骤 4：现金上游障碍
```
净现金/市值 = 36.07 / 115.05 = 31.3% < 40%
→ 使用市值口径
```

#### 步骤 5：分配意愿
```
假设支付率 37.5%，属于中等水平
分配意愿：**中等**
```

#### 步骤 6：可预测性
```
- 弱周期行业 ✅
- 盈利波动较小 ✅
- 商业模式稳定 ✅

可预测性：**中等**
```

#### 步骤 7：税务泄漏修正
```
A 股股息税：
- 持股>1 年：免税
- 持股 1 月 -1 年：10%
- 持股<1 月：20%

假设平均税率 Q = 10%
```

#### 步骤 8：粗算穿透回报率
```
R = [C × M × (1 − Q%) + O] / 市值
  = [6.73 × 37.5% × (1 − 10%) + 0] / 115.05
  = [2.27 + 0] / 115.05
  = 1.97%

⚠️ 但上述计算仅考虑股息，未考虑 Owner Earnings 再投资

更合理的计算：
R = I / 市值
  = 7.33 / 115.05
  = 6.37%

但考虑到：
- 毛利率持续下滑
- 应收账款激增
- 投资现金流异常

应给予保守调整，假设实际可分配现金为 Owner Earnings 的 50%：
R_adjusted = 7.33 × 50% / 115.05 = 3.18%
```

**粗算结果**：R = **3.18%**

### 粗算否决门检查

```
Rf = 1.79%
II = 3.79%

检查：
- R < Rf？3.18% > 1.79% ✅ 通过
- R < II×0.5？3.18% > 1.90% ✅ 通过
- R < II？3.18% < 3.79% ⚠️ 边际不达标

结论：进入因子 3 精算
```

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

```

### 步骤 5：资本开支与投资的极端保守处理

**⚠️关键疑点**：现金流量表显示投资支付现金 45.15 亿元，远超合理 Capex

**合理推测**：
- 大部分投资现金流可能是购买理财产品
- 实际 Capex 应远低于此数值

**估算实际 Capex**：

**总扣除额 Y**：
```
Y = E（实际 Capex）+ X1（投资）+ X2（隐性支出）
  = 1.68 + 0 + 0
  = 1.68 亿元
```

### 步骤 6：准则差异核查

**A 股特征**：
- 收入确认可能激进
- 应收账款周转天数从 1.4 天（2024 年末）激增至 21.9 天（2025 年 Q3）
- 需警惕收入质量

### 步骤 7：真实可支配现金结余

```
真实可支配现金结余 = 真实现金收入 − 经营性现金支出 − 总扣除额
                    = 8.72 − 16.14 − 1.68
                    = -9.10 亿元
