#### 步骤6：可预测性

收入和利润波动极大，可预测性 = 低

#### 步骤7：税务泄漏修正

A股持股超1年免税，Q = 0%

#### 步骤8：粗算穿透回报率

```
─── 主轨：市值口径 ───
R = (C × M × (1-Q%) + O) / 市值
  = (35.86 × 40% × 1 + 0) / 1,388
  = 14.34 / 1,388
  = 1.03%

─── 辅轨：EV 口径 ───
广义净现金/市值 = 306.10 / 1,388 = 22.05%（轻度保护）
EV = 1,388 - 306.10 = 1,081.90 亿元
R_EV = 14.34 / 1,081.90 = 1.33%
```

#### 粗算否决门判定

```
R = 1.03% < Rf = 1.78% → 检查 EV 口径
R_EV = 1.33% < Rf = 1.78% → EV 口径仍不通过

R_EV < II × 0.5 = 1.89% → 是
→ 粗算否决
```

**粗算结论：❌ 否决** — 穿透回报率远低于门槛，即使使用EV口径仍不达标。

> ⚠️ 注意：粗算基于2025年周期性低谷盈利。若使用5年平均归母净利润58.50亿元计算：
> R = (58.50 × 40%) / 1,388 = 1.69%，仍低于门槛3.78%。
> 即使使用2021年峰值盈利96.54亿元：R = (96.54 × 40%) / 1,388 = 2.78%，仍低于门槛。
