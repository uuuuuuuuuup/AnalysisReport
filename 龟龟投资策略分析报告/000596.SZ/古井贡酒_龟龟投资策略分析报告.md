### Step 7：税收泄漏修正

- Q = 0%（A股持股>1年免股息税）
- 修正系数 = 1 - 0% = 1.0

### Step 8：粗算穿透收益率

$$R_{粗算} = \frac{AA \times M \times (1 - Q\%) + O}{当前市值}$$

| 参数 | 值 | 说明 |
|------|-----|------|
| AA（Owner Earnings） | 44.43 亿元 | 2024年FCF |
| M（支付率锚定值） | 50% | 考虑提升趋势 |
| Q（税收泄漏） | 0% | A股免税 |
| O（年度回购） | 0 | 无回购 |
| 当前市值 | 556.87 亿元 | 2026-04-14 |

$$R_{粗算} = \frac{44.43 \times 50\% \times 1.0 + 0}{556.87} = \frac{22.22}{556.87} = \textbf{3.99\%}$$

|------|-----|

| 阈值 II | max(3.5%, 1.79% + 2%) = **3.79%** |

### 因子2 结论
