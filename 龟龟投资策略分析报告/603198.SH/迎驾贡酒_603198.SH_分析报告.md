### 模块一：资本消耗强度

**分析**：

迎驾贡酒作为白酒企业，核心生产资产为窖池和储酒设施。2024年固定资产从约18亿增至26亿，在建工程3亿，主要投向数字化产业园和产能扩建。2024年投资现金流-18.97亿，但其中约7亿为交易性金融资产投资，实际运营性资本开支约10-12亿。

资本消耗强度评估需区分维持性和扩张性开支：
- 维持性资本开支：约3-4亿/年（折旧摊销水平）
- 扩张性资本开支：约6-8亿/年（产能扩建、数字化升级）

**资本消耗强度：中等**（扩张性投入较大，但白酒行业资本密度本身不高）

### 模块三：竞争格局与护城河

#### 第一层：非技术性护城河

**品牌力**：迎驾贡酒为徽酒第二品牌，主打"生态酿造"概念，核心产品为生态洞藏系列。品牌在安徽省内具有较高认知度，但全国化程度低，品牌力远弱于茅台、五粮液等全国性名酒。

**渠道力**：省内渠道网络成熟，但省外扩张受阻。2025年一季度省内营收16.33亿，同比下滑7.71%，为近三年首次负增长。

**竞争格局**：
- 古井贡酒：2024年营收235.8亿，省内市占率约30%，是迎驾的3倍以上，绝对龙头
- 口子窖：2024年营收60.15亿，紧追迎驾，2025年Q1已实现正增长反超势头
- 全国性名酒（茅台、五粮液、洋河等）：加速渠道下沉，挤压次高端价格带

#### 第二层：技术性护城河

- 窖池资源：迎驾拥有一定数量的老窖池，但远不如泸州老窖、五粮液等
- 酿造工艺：生态酿造理念有一定差异化，但并非不可复制
- 产能储备：存货50.43亿（占总资产37.67%），基酒储备充足

**护城河评级：窄**（省内品牌力尚可，但正在被侵蚀；省外几乎无护城河）

### 模块四：周期性

**分析**：

白酒行业具有弱周期性特征，与宏观经济和消费信心相关。当前行业处于深度调整期：
- 2022年至今，白酒行业整体增速放缓
- 2024Q4以来，迎驾贡酒连续四个季度营收净利双降
- 消费需求疲软，渠道库存高企

迎驾贡酒作为区域酒企，周期性更为明显：
- 省内市场受经济下行影响更大
- 次高端价格带受挤压最严重

**周期性：弱周期**（但当前处于行业下行周期，需额外谨慎）

## 四、因子 2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

GAAP归母净利润（2024）= **25.89亿元**

### 步骤2：Owner Earnings粗算

```
Owner Earnings = 归母净利润 + D&A − Capex
              = 25.89 + 2.0 − 10.0
              = 17.89 亿元
```

注：此处Capex为估算运营性资本开支（剔除金融投资），D&A为估算值。

### 步骤3：分配能力验证

- 2024年股息支付：12.0亿元
- Owner Earnings：17.89亿元
- 支付率（基于OE）：12.0 / 17.89 = 67.1%
- OCF：19.07亿元 > 股息12.0亿元 ✓

**分配能力验证：通过**（OCF覆盖股息支付）

### 步骤4：现金上游障碍

- 控股股东迎驾集团持股72.22%，为民营企业
- 无明显现金上游障碍（无港股通税务泄漏、无跨境资金限制）
- 少数股东损益仅0.05亿元，可忽略

**现金上游障碍：无**

### 步骤5：分配意愿

- 连续三年提升DPS：0.9→1.1→1.5元/股
- 支付率从38.5%提升至46.35%
- 无回购记录

**分配意愿：中等偏强**（派息持续提升，但无回购补充）

### 步骤6：可预测性

- 2020-2024年营收CAGR约20.7%，但2025年急剧下滑
- 净利润波动较大，2025年前三季度同比-24.69%
- 合同负债下降，经销商打款意愿减弱

**可预测性：中等偏下**（历史增长良好但当前趋势恶化）

### 步骤7：税务泄漏修正

- A股持股，股息税率：持股>1年免征，持股1月-1年10%，持股<1月20%
- 长期投资者税务泄漏Q% ≈ 0%

### 步骤8：粗算穿透回报率

**市值口径**：
```
R = (Owner Earnings × 支付率锚定值 × (1 − Q%) + 回购) / 市值
  = (17.89 × 46.35% × 1 + 0) / 284.88
  = 8.29 / 284.88
  = 2.91%
```

**使用归母净利润口径**：
```
R = (25.89 × 46.35% × 1 + 0) / 284.88
  = 12.00 / 284.88
  = 4.21%
```

**粗算否决门判断**：
- R（OE口径）= 2.91% < II = 3.82% → 未通过
- R（净利润口径）= 4.21% > II = 3.82% → 通过

**需检查EV口径**：

广义净现金 = 19.01亿元
广义净现金/市值 = 19.01 / 284.88 = 6.68% < 40%

EV口径不适用（广义净现金/市值 < 40%）

**粗算结论：边际不达标**（OE口径2.91%低于门槛3.82%，净利润口径4.21%略高于门槛，需进入因子3精算确认）

## 五、因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤1：真实现金收入还原

| 项目 | 2024年（亿元） |
|------|---------------|
| 营业收入 S | 73.44 |
| 应收账款净变动 T | +1.95 − 0.66 = +1.29（增加为正） |
| 预收款/合同负债净变动 U | 5.78 − 7.34 = −1.56（减少为负） |

保守处理：
- 应收账款净增加1.29亿 → 扣除
- 合同负债净减少1.56亿 → 对应未来交付义务已减少，保守不加回

```

            = 73.44 − 1.29 − 0
            = 72.15 亿元

收款比率 = 72.15 / 73.44 = 98.2%
```

**收款比率98.2%，接近1，收款质量优良**（符合白酒行业先款后货特征）
