# 龟龟投资策略分析报告库

基于价值投资理念的股票分析报告展示网站。

## 访问地址

🌐 **GitHub Pages**: https://[你的用户名].github.io/[仓库名]/

## 功能特性

- 📊 **总览仪表板** - 展示报告统计数据和评级分布
- 📑 **报告列表** - 支持筛选、排序、搜索
- ⭐ **收藏功能** - 本地存储收藏的报告
- 🔍 **快速搜索** - Cmd/Ctrl + K 快捷键搜索
- 📱 **响应式设计** - 适配桌面和移动设备
- 🎨 **优雅界面** - 专业的金融风格设计

## 投资策略框架

龟龟投资策略包含四个核心因子：

1. **因子 1A** - 五分钟快筛（一票否决）
2. **因子 1B** - 深度定性分析
3. **因子 2/3** - 穿透回报率计算
4. **因子 4** - 估值与安全边际

## 本地开发

```bash
# 克隆仓库
git clone https://github.com/[你的用户名]/[仓库名].git
cd [仓库名]

# 启动本地服务器
python -m http.server 8000
# 或
npx serve .

# 访问 http://localhost:8000
```

## 添加新报告

1. 在对应股票代码文件夹中创建 `股票名_代码_分析报告.md`
2. 在 `app.js` 的 `reportData` 数组中添加报告信息

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- [Marked.js](https://marked.js.org/) - Markdown 解析
- [Noto Serif SC](https://fonts.google.com/noto/specimen/Noto+Serif+SC) - 中文字体
