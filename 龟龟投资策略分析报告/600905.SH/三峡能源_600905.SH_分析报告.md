### 模块一：资本消耗强度

**评级：高**

- 2024年资本开支316.68亿元，为OCF的1.68倍
- 在建工程709.24亿元（2025Q3），较2024年报556.10亿大幅增长27.5%
- 自由现金流（OCF-总Capex）持续为负：2024年-127.71亿
- 公司处于高速扩张期，资本消耗极大
- **结论**：资本消耗强度极高，公司持续"烧钱"扩张

### 模块二：收款模式

**评级：应收为主**

- 应收账款447.58亿元（2024年报），占营收150.6%
- 应收账款周转天数493天（2024年报），580天（2025Q3），持续恶化
- 主要应收为国家可再生能源补贴，回款周期极长（1-3年）
- 补贴应收最终由国家信用背书，坏账风险低但时间成本高
- **结论**：典型应收为主模式，大量资金被补贴应收占用

### 模块三：竞争格局与护城河

**非技术性护城河**：
- 央企背景，融资成本显著低于民企（AAA评级）
- 风电/光伏项目需政府审批，获取资源的能力是核心竞争力
- 海上风电项目资源稀缺，先发优势明显

**技术性护城河**：
- 海上风电技术积累深厚，国内第一、全球第二（7.05GW）
- 规模效应带来运维成本优势

**护城河评级：窄**
- 资源型护城河存在，但非不可替代
- 行业竞争加剧，平价上网后利润承压
- 新进入者（如其他央企）同样具备资源获取能力

### 模块四：周期性

**评级：弱周期**

- 风电/光伏发电受自然资源（风况/光照）影响，存在年际波动
- 电力需求有一定经济周期性，但新能源优先消纳
- 长期看，新能源替代趋势确定，需求端无周期性萎缩风险
- 电价受政策影响较大，市场化交易比例提升带来价格波动

### 模块五：人力资本依赖

**评级：低**

- 发电运营高度自动化，对人力依赖度低
- 人力投入回报率（HR Input Return）极高（613-762%）
- 员工成本占营收比例极低

### 模块六：管理层与治理结构

**评级：良好**

- 三峡集团控股，央企治理规范
- 2025-2026年管理层换届：朱承军任董事长，刘姿任总经理
- 新任管理层均具备丰富行业经验，无违规处罚记录
- **不足**：分红意愿偏低（31%），DPS连续三年下降
- **不足**：高资本开支+低分红，中小股东利益关注不够

### 模块七：监管与政策风险

**评级：中**

- 新能源行业高度依赖政策支持
- **正面**：双碳目标长期支持新能源发展
- **负面**：补贴退坡已实施，未来补贴可能进一步减少
- **负面**：电力市场化改革可能压低电价
- **负面**：新能源消纳压力增大，弃风弃光风险

### 模块八：MD&A 解读

**评级：中性**

- 管理层对行业前景保持乐观，但承认短期压力
- 2025年前三季度营收微降2.26%，净利润下滑15.31%
- 持续推进大基地项目，但短期利润承压
- 未给出明确的利润改善时间表

## 【因子 2：穿透回报率粗算（Top-Down）】

### 步骤 1：归母净利润

- 2024年报归母净利润 = **61.11 亿元**

### 步骤 2：Owner Earnings 粗算

按照巴菲特公式：
```
Owner Earnings = 归母净利润 + 折旧摊销 − 资本开支
              = 61.11 + 95 − 316.68
              = −160.57 亿元
```

⚠️ **Owner Earnings为深度负值**，表明公司当前盈利完全无法覆盖资本开支。

### 步骤 3：分配能力验证（一票否决）

- OCF 188.97亿 > 股息19.17亿 → 表面通过
- **但** FCF = OCF − Capex = 188.97 − 316.68 = −127.71亿 → **深度为负**
- 公司通过举债支付股息（2024年筹资现金流131.29亿）
- **分配能力验证：⚠️ 勉强通过但质量极差**（分红依赖举债）

### 步骤 4：现金上游障碍

- 应收账款447.58亿，主要为补贴应收
- 补贴由国家信用背书，最终可收回
- 但回款周期1-3年，时间成本高
- **上游障碍：中等**（回款慢但信用风险低）

### 步骤 5：分配意愿

- 支付率31%，远低于50%的合理水平
- DPS连续三年下降：0.78→0.76→0.67元/股
- 公司明确将再投资扩张置于分红之上
- **分配意愿：弱**

### 步骤 6：可预测性

- 发电收入相对稳定（长期购电协议+优先消纳）
- 但补贴政策、电价市场化带来不确定性
- **可预测性：中等**

### 步骤 7：税务泄漏修正

- A股持股>1年免征股息红利税
- **Q% = 0%**

### 步骤 8：粗算穿透回报率 R%

**口径一：归母净利润×支付率**
```
R = (61.11 × 31% × (1−0%) + 0) / 1,175
  = 18.94 / 1,175
  = 1.61%
```

**口径二：OCF×支付率（更乐观）**
```
R = (188.97 × 31% + 0) / 1,175
  = 58.58 / 1,175
  = 4.98%
```
⚠️ 此口径不合理，因为OCF大部分被再投资消耗

**口径三：EV口径**
```
EV = 市值 − 广义净现金 = 1,175 − (−1,706.69) = 2,881.69 亿
R_EV = 18.94 / 2,881.69 = 0.66%
```
EV口径更差（因净负债极大）

### 粗算否决门检查

| 检查项 | 结果 |
|--------|------|
| R < Rf (1.79%)? | **是**（1.61% < 1.79%）→ 检查EV口径 |
| R_EV < Rf? | **是**（0.66% < 1.79%）→ EV口径更差 |
| R < II×0.5 (1.90%)? | **否**（1.61% < 1.90%）→ ⚠️ |

### 粗算结论：❌ 否决

- 粗算穿透回报率1.61%，低于无风险利率1.79%
- EV口径0.66%更差
- **但**：1.61%接近II×0.5=1.90%，标注"边际不达标"，继续进入因子3

### 步骤 7：真实可支配现金结余

```
真实可支配现金结余 = 真实现金收入
                    + 资产处置收入 V1
                    + 其他投资收入 V5
                    − 应扣除非经常性流入
                    − 经营性现金支出 W
                    − 总扣除额 Y

= 274.52 + 0 + 0 − 0 − 264 − 316.68
= −306.16 亿元（2024年）
```

⚠️ **真实可支配现金结余为深度负值**

**使用OCF口径验证**：
```
真实可支配现金结余 = OCF − Capex
                   = 188.97 − 316.68
                   = −127.71 亿元（2024年）
```

两种口径均显示深度负值。

**历史均值 AA**：
- 仅有2024全年数据，AA = −127.71 亿元
- 2025Q3年化：OCF 172.21×(4/3) = 229.61亿，Capex 199.84×(4/3) = 266.45亿
- 2025年化FCF ≈ 229.61 − 266.45 = −36.84亿（有所改善但仍为负）

**AA = −127.71 亿元（2024年），2025年预计 −37 亿元**

```

```
