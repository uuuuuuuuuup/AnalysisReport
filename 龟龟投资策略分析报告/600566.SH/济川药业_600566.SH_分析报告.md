## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1-2：归母净利润与 Owner Earnings

- GAAP归母净利润 C = 25.32 亿元（2024年）
- Owner Earnings OE = C + D&A - Capex + SBC = 25.32 + 3.2 - 4.0 + 0.1 = **24.62 亿元**

### 步骤 3：分配能力验证

- OCF = 21.76亿 < 归母净利润 25.32亿 → 净现比 = 0.86
- **未达一票否决**（净现比>0.8），但利润含金量不足

### 步骤 4-7：障碍/意愿/可预测性/税务

- 现金上游障碍：无（A股公司，Q%=0）
- 分配意愿：**中等偏强**（2024年支付率跳升至75.72%，但可持续性存疑）
- 可预测性：**低**（2025年业绩大幅下滑，核心产品面临集采风险）
- 税务泄漏：0（A股持有>1年免股息税）

### 步骤 8：粗算穿透回报率

**基于2024年数据：**
R = (OE × M + O) / 市值 = (24.62 × 0.5133 + 0.03) / 231.11 = **5.48%**

**基于2025年预估数据：**
- 2025年预估归母净利润 ≈ 13.70亿 → OE ≈ 12.80亿
- R_2025 = (12.80 × 0.5133 + 0.03) / 231.11 = **2.85%**

**粗算否决门判断：**
- 基于2024年：R = 5.48% > II = 3.81% → ✅ 通过
- 基于2025年：R = 2.85% < II = 3.81% → ❌ 否决

**粗算结论：基于2024年数据通过，但基于2025年预估否决。需进入精算进一步判断。**
