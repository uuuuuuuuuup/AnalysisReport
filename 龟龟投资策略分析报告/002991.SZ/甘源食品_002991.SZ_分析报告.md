- 产品同质化风险存在，竞品模仿门槛较低

**护城河评估：**
- 品牌有一定认知度，但行业竞争激烈、转换成本低

**护城河评级：窄（且边际收窄）**

因子2：穿透回报率粗算（Top-Down）
═══════════════════════════════════════

1. **归母净利润** = 3.76亿元（2024年度）

2. **Owner Earnings粗算**：
   - 归母净利润 + D&A - Capex
   - = 3.76 + 0.70 - 1.67 = 2.79亿元

3. **分配能力验证**：
   - 2024年OCF = 3.27亿 < 归母净利润3.76亿 → OCF/净利润 = 87%
   - 2023年OCF = 2.58亿 < 归母净利润3.29亿 → OCF/净利润 = 78.4%
   - 现金流持续低于账面利润，质量一般
   - **未触发一票否决**（OCF/净利润>50%），但需关注

4. **现金上游障碍**：无（A股，无跨境资金问题）

5. **分配意愿**：
   - 3年平均支付率56.4%
   - 2024年支付率下降至44.1%（派息减少）
   - **分配意愿：中等偏弱**

6. **可预测性**：
   - 2025年业绩大幅下滑（净利-55%），盈利可预测性降低
   - 原材料成本波动大
   - **可预测性：中低**

7. **税务泄漏修正**：
   - A股持股>1年免征红利税
   - Q% = 0%

8. **粗算穿透回报率**：
   - R = (Owner Earnings × M × (1-Q%) + O) / 市值
   - R = (2.79 × 56.4% × 1 + 0) / 46.78
   - R = 1.573 / 46.78
   - **R = 3.36%**

**粗算否决门检查：**
- R = 3.36% < II = 3.81% → 未通过主轨
- R = 3.36% > II × 0.5 = 1.905% → 标注"边际不达标"
- **粗算结论：⚠️ 边际不达标，继续进入因子3精算**
