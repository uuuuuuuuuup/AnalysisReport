### 模块二:收款模式

**判断**:现结为主,应收增加

**分析**:
1. **应收账款占比**:2025年末应收账款2亿元,占营收55.19亿元的3.6%,占比不高
2. **应收账款增长**:2025年上半年应收账款激增230%,需关注
3. **收款模式**:公司财报提到"与客户之间的贸易条款以预付款项为主,除了个别客户例外,一般客户都是按照7天至180天的信贷期付款"
4. **风险提示**:2025年上半年应收账款激增,主要因渠道压货、电商平台结算周期延长、线下医院/诊所渠道占比提升

**结论**:收款模式以现结和预收为主,但2025年应收账款激增需关注

### 步骤8:粗算穿透回报率R%

```
Owner Earnings = 21.5亿元
支付率锚定值[M] = 63%
税务泄漏修正系数 = 0.8
年均回购金额[O] = 0.26亿元(人民币)

可分配现金 = Owner Earnings × M × 0.8 + O
           = 21.5 × 63% × 0.8 + 0.26
           = 10.84 + 0.26
           = 11.10亿元

当前市值 = 276.5亿元(人民币)

粗算穿透回报率R = 可分配现金 / 当前市值
                = 11.10 / 276.5
                = 4.01%
```

### 粗算否决门判断

```
Rf = 1.81%
II = 5%(港股门槛)

R = 4.01%
R < II (4.01% < 5%)

检查EV口径:
广义净现金 = 82.8亿元
EV = 当前市值 - 广义净现金 = 276.5 - 82.8 = 193.7亿元

粗算穿透回报率R_EV = 可分配现金 / EV
                    = 11.10 / 193.7
                    = 5.73%

R_EV > II (5.73% > 5%)
```

### 粗算结论

**主轨(市值口径)**:边际不达标(R=4.01% < II=5%)
**辅轨(EV口径)**:通过(R_EV=5.73% > II=5%)

**决策**:继续进入因子3精算

## 【因子3:穿透回报率精算(Bottom-Up)+现金质量审计】

### 步骤1:真实现金收入还原

由于缺乏详细的应收账款变动数据,采用简化计算:

```
营业收入(2025年) = 55.19亿元
应收账款净增加 = 2.0 - 1.41 = 0.59亿元(假设2024年末应收账款1.41亿元)

真实现金收入 = 营业收入 - 应收账款净增加
             = 55.19 - 0.59
             = 54.60亿元

收款比率 = 真实现金收入 / 营业收入 = 54.60 / 55.19 = 98.9%
```
