## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

**Step 1：归母净利润**
```
C = 43.45亿元（2024年）
```

**Step 2：Owner Earnings粗算**
```
D（折旧摊销）≈ 8.5亿元
E（资本开支）≈ 25亿元
G（维持性资本开支系数）≈ 0.4
H = D × G = 8.5 × 0.4 = 3.4亿元（维持性资本开支粗估）
I = C + D - H = 43.45 + 8.5 - 3.4 = 48.55亿元（Owner Earnings）
```

**Step 3：分配能力验证**
```
经营现金流 = 51.55亿元 > 净利润 = 44.92亿元 ✅ 通过
自由现金流 = 51.55 - 25 = 26.55亿元 > 股息30亿元 ⚠️ 需动用现金储备
```

**Step 4：现金上游障碍**
```
无上游障碍，现金可自由支配
```

**Step 5：分配意愿**
```
支付率 = 69%，呈上升趋势（55%→63%→69%）
分配意愿 = 强 ✅
```

**Step 6：可预测性**
```
收入稳定性 = 高（CV约8%）
利润稳定性 = 中高
可预测性评级 = 中高
```

**Step 7：税务泄漏修正**
```
A股股息税率 = 10%（持股超1年免税，假设10%）
Q = 10%
```

**Step 8：粗算穿透回报率**
```
M（支付率锚定值）= 65%
O（年均回购）= 0亿元
R = [C × M × (1-Q%) + O] / 市值
R = [43.45 × 65% × 0.9 + 0] / 861

R = 2.95%
```

**粗算结果**：R = 2.95% < 门槛3.8%

⚠️ **粗算否决门检查**：
- R = 2.95% > Rf（1.8%）✅
- R = 2.95% > II × 0.5（1.9%）✅
- R = 2.95% < II（3.8%）⚠️ **边际不达标，进入因子3精算**
