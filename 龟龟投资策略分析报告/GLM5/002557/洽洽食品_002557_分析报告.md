## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
集团净利润 [A] = 8.50亿元
少数股东损益 [B] ≈ 0亿元
GAAP归母净利润 [C] = A - B = 8.50亿元
```

### 步骤2：Owner Earnings粗算

```
折旧摊销 [D] ≈ 1.50亿元
资本开支 [E] = 5.82亿元
维持性资本开支系数 [G] = 0.60
维持性资本开支粗估 [H] = D × G = 1.50 × 0.60 = 0.90亿元

Owner Earnings [I] = C + D - H
                   = 8.50 + 1.50 - 0.90
                   = 9.10亿元
```

### 步骤3：分配能力验证

```
经营现金流 [OCF] = 10.39亿元
资本开支 [Capex] = 5.82亿元
自由现金流 [FCF] = OCF - Capex = 10.39 - 5.82 = 4.57亿元

实际分红 = 4.97亿元

判断: FCF(4.57亿) < 分红(4.97亿)，分配能力勉强支撑
```

**⚠️ 分配能力警告**: 自由现金流不足以完全覆盖分红，存在消耗现金储备的风险

### 步骤4：现金上游障碍

```
账面现金 = 45.83亿元
有息负债 = 0亿元
净现金 = 45.83亿元

判断: 无有息负债，现金储备充足，无现金上游障碍
```

### 步骤5：分配意愿

```
2024年分红 = 4.97亿元
支付率 = 58.54%
历史支付率均值 = 57%

判断: 分配意愿强，持续分红，支付率稳定在55-60%
```

### 步骤6：可预测性

**⚠️ 可预测性低**:
- 2025年前三季度净利润同比下滑73%
- 行业竞争加剧，盈利能力不稳定
- 毛利率波动大（从30%降至22%）

### 步骤7：税务泄漏修正

```
A股股息税率 [Q] = 10%（持股超1年免税，保守按10%计）
税后可分配 = Owner Earnings × (1 - Q%)
           = 9.10 × (1 - 10%)
           = 8.19亿元
```

### 步骤8：粗算穿透回报率

```
支付率锚定值 [M] = 58.54%
年均回购金额 [O] = 0亿元
当前市值 = 112.40亿元

粗算穿透回报率 [R] = [I × M × (1 - Q%) + O] / 当前市值
                    = [9.10 × 58.54% × (1 - 10%) + 0] / 112.40
                    = [9.10 × 0.5854 × 0.90] / 112.40
                    = 4.79 / 112.40
                    = 4.26%
```

### 粗算否决门判断

```
门槛 [II] = 3.79%
粗算穿透回报率 [R] = 4.26%

判断: R(4.26%) > II(3.79%)，通过粗算否决门
```

**⚠️ 但需注意**: 粗算基于2024年数据，2025年业绩大幅恶化，实际回报率可能远低于粗算值
