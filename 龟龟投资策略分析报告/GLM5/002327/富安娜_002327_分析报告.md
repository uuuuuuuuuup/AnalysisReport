## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### Step 1: 归母净利润
```
C = 5.42 亿元（2024年归母净利润）
```

#### Step 2: Owner Earnings粗算
由于缺乏精确的折旧摊销和资本开支数据，采用简化计算：
```
Owner Earnings ≈ 归母净利润 × 现金转化率
现金转化率 = OCF / 净利润 = 3.66 / 5.42 = 67.5%
Owner Earnings ≈ 5.42 × 0.675 = 3.66 亿元
```

#### Step 3: 分配能力验证
```
股息支付 = 5.20 亿元
经营现金流 = 3.66 亿元
```
⚠️ **分配能力验证**：股息支付(5.20亿) > 经营现金流(3.66亿)，存在现金消耗。

**分析**：公司通过消耗账面现金维持高分红，需关注现金储备可持续性。

#### Step 4: 分配意愿
```
支付率历史 = [83.2%, 93.2%, 93.5%, 95.8%]
支付率均值 M = 91.4%
```
✅ 分配意愿**强**，连续多年高分红。

#### Step 5: 粗算穿透回报率
```
R = (C × M × (1-Q%) + O) / 市值
  = (5.42 × 95% × 1 + 0) / 58.56
  = 5.15 / 58.56
  = 8.79%
```
