## 因子 2: 穿透回报率粗算 (Top-Down)

### 步骤8: 粗算穿透回报率

```
当前市值 = 348 亿元

粗算穿透回报率 [R] = (Owner Earnings × M × (1-Q) + O) / 市值

```

```
                    = 2.41 / 348
                    = 0.69%
```

### 粗算否决门检查

```
R = 0.85% ~ 0.97%
门槛 II = 3.79%

R < II × 0.5 (1.90%) → 检查 EV 口径
```

**EV 口径计算**:

```
EV = 市值 - 净现金
   = 348 - 14.64
   = 333.36 亿元

粗算穿透回报率 [R_EV] = (19.77 × 0.17) / 333.36
                       = 3.36 / 333.36
                       = 1.01%

R_EV = 1.01% < II × 0.5 = 1.90%

❌ EV口径仍然不达标!
```

**粗算结论**: ❌ **严重不达标,按规则应否决**

但考虑到:
1. 公司处于竞争加剧期,盈利暂时下滑
2. 净现金充裕,财务健康
3. 行业长期成长性

**决定**: 标注"边际不达标",继续进入因子3精算
