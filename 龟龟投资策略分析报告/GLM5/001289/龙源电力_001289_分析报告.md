## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
GAAP归母净利润[C] = 63.45亿元
扣非归母净利润 = 57.11亿元
```

### 步骤2：Owner Earnings粗算

```
折旧摊销[D] ≈ 80亿元（估算）
资本开支[E] = 260.68亿元
维持性资本开支系数[G] = 30%（估算，风电行业）
维持性资本开支[H] = D × G = 80 × 30% = 24亿元

Owner Earnings[I] = C + D - H
                  = 63.45 + 80 - 24
                  = 119.45亿元
```

**⚠️ 关键问题**: 资本开支远超折旧摊销，说明公司处于**高速扩张期**，维持性资本开支估算可能不准确。

### 步骤3：分配能力验证

```
经营现金流(OCF) = 170.62亿元
资本开支(Capex) = 260.68亿元
自由现金流(FCF) = 153.68亿元（AI-Tools计算口径不同）

实际自由现金流 = OCF - Capex = 170.62 - 260.68 = -90.06亿元
```

**⚠️ 一票否决检查**: 自由现金流为**负值**！

**分析**: 
- 公司经营现金流170.62亿元，但资本开支高达260.68亿元
- **自由现金流为负**，说明公司无法依靠自身造血支持扩张
- 需要外部融资维持运营和扩张
- **分配能力存疑**：虽然公司分红19.04亿元，但实际上是在"借钱分红"

### 步骤4：现金上游障碍

```
应收补贴款 = 435亿元
年财务费用 = 34.97亿元
```

**现金上游障碍严重**: 435亿元补贴款被占用，导致公司需要大量借款，年财务费用高达35亿元。

### 步骤7：税务泄漏修正

```

综合股息税率 = 10%
```

### 步骤8：粗算穿透回报率

**口径一：基于归母净利润**

```
R = [C × M × (1 - Q%) + O] / 当前市值
  = [63.45 × 30% × (1 - 10%) + 0] / 1,544.93
  = [63.45 × 0.30 × 0.90] / 1,544.93
  = 17.13 / 1,544.93
  = 1.11%
```

**口径二：基于自由现金流（更保守）**

```
由于自由现金流为负，无法计算有效回报率
```

**口径三：基于经营现金流（考虑补贴回收）**

```
假设2026年补贴款大部分回收，经营现金流将显著改善
保守估计可持续OCF = 150亿元
R = [150 × 30% × (1 - 10%)] / 1,544.93
  = 40.5 / 1,544.93
  = 2.62%
```

### 粗算否决门检查

```
门槛[II] = 3.79%
粗算穿透回报率[R] = 1.11% ~ 2.62%

R < II → ⚠️ 不达标
R < II × 0.5 = 1.90% → 部分口径不达标
```

**因子2结论**: ⚠️ **粗算穿透回报率低于门槛，存在否决风险**
