## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### Step 1：归母净利润
```
C = 22.25 亿元
```

#### Step 2：Owner Earnings粗算
```
D（折旧摊销）≈ 15 亿元
E（资本开支）= 29.19 亿元
G（维持性资本开支系数）= 0.6（估算）
H = D × G = 15 × 0.6 = 9 亿元
I = C + D - H = 22.25 + 15 - 9 = 28.25 亿元
```

#### Step 3：分配能力验证
```
经营现金流 = 55.31 亿元
资本开支 = 29.19 亿元
自由现金流 = 26.12 亿元
股息支付 = 7.46 亿元
```
**结论：分配能力充足，自由现金流覆盖股息支付。**

#### Step 4：分配意愿
```
2024年支付率 = 51%
历史支付率（疫情前）≈ 39%
支付率锚定值 M = 40%（保守估计）
```

#### Step 5：粗算穿透回报率
```
R = [C × M × (1 - Q%) + O] / 当前市值
R = [22.25 × 40% × 100% + 0] / 729.4
R = 8.90 / 729.4
R = 1.22%
```

**⚠️ 粗算穿透回报率仅1.22%，远低于门槛3.79%。**

#### Step 6：使用Owner Earnings口径重新计算
```
R_OE = [I × M] / 当前市值
R_OE = [28.25 × 40%] / 729.4
R_OE = 11.30 / 729.4
R_OE = 1.55%
```

**仍远低于门槛，需进入因子3精算。**

### Step 3：安全边际计算
