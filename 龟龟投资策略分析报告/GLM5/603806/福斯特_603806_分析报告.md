## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
集团净利润 A = 12.89亿元（2024年）
少数股东损益 B = 0.18亿元（估算）
GAAP归母净利润 C = A - B = 12.71亿元
```

### 步骤2：Owner Earnings粗算

```
折旧摊销 D = ⚠️未找到具体数据
  - 参考同行业，估算约3-5亿元
  - 保守取值 D = 4亿元

资本开支 E = 34.39亿元（2024年投资现金流流出）

维持性资本开支系数 G = 估算0.3（制造业通常20-40%）
维持性资本开支 H = D × G = 4 × 0.3 = 1.2亿元

Owner Earnings I = C + D - H = 12.71 + 4 - 1.2 = 15.51亿元
```

**⚠️ 问题**：资本开支34.39亿元远超折旧摊销，说明公司处于扩张期或资本开支效率低。

### 步骤3：分配能力验证

```
2024年经营现金流 OCF = 43.89亿元
2024年资本开支 = 34.39亿元
自由现金流 FCF = 43.89 - 34.39 = 9.50亿元

2024年股息支付 = 6.78亿元
FCF覆盖率 = 9.50 / 6.78 = 140%
```

**结论**：2024年分配能力尚可，但2025年现金流恶化，分配能力存疑。

### 步骤8：粗算穿透回报率R

```
当前市值 = 17.81元 × 26.09亿股 = 464.66亿元

Owner Earnings I = 15.51亿元（粗估）
支付率 M = 51.82%
回购金额 O = 0

粗算穿透回报率 R = [I × M × (1 - Q%) + O] / 市值
R = [15.51 × 0.5182 × 0.9 + 0] / 464.66
R = 7.24 / 464.66
R = 1.56%
```

### 粗算否决门检验

```
A股门槛 II = max(3.5%, Rf + 2%) = max(3.5%, 1.79% + 2%) = 3.79%

R = 1.56% < II = 3.79%
R = 1.56% < II × 0.5 = 1.90%
```

**⚠️ 粗算结论**：R < II × 0.5，触发否决门，需检查EV口径。

### EV口径检验

```
广义净现金 = 50.05 - 2.99 = 47.06亿元（2024年）
广义净现金/市值 = 47.06 / 464.66 = 10.13% < 40%

EV口径不适用（净现金/市值 < 40%）
```

**因子2结论**：❌ **粗算穿透回报率1.56% < 门槛3.79%，触发否决门。**

## 因子4：估值与安全边际

```
Rf = 1.79%（十年期国债收益率）

### 步骤2：价值陷阱排查

| 序号 | 陷阱特征 | 判断 | 说明 |
|------|----------|------|------|
| 1 | 现金流趋势性恶化 | ⚠️ **是** | 2025年OCF从43亿降至0.14亿 |
| 2 | 护城河正在收窄 | ⚠️ **是** | 毛利率从15%降至11%，竞争加剧 |
| 3 | 行业结构性衰退 | ❌ 否 | 光伏长期需求向上，当前为周期底部 |
| 4 | 分配意愿存疑 | ❌ 否 | 支付率稳定在50%+ |
| 5 | 管理层损害价值 | ⚠️ **可能** | 实控人持续减持 |

**价值陷阱风险项数 N = 3**，风险等级：**高**

### 步骤3：安全边际与仓位

```
精算穿透回报率 GG = 0.48%（乐观估计）
门槛 II = 3.79%

安全边际 JJ = GG - II = 0.48% - 3.79% = -3.31%

周期性修正：强周期底部 → 门槛下调1pct
修正后门槛 = 3.79% - 1% = 2.79%
修正后安全边际 KK = 0.48% - 2.79% = -2.31%
```

**现金保护等级**：

```
广义净现金/市值 = 47.06 / 464.66 = 10.13% < 20%
现金保护等级 = 无保护
