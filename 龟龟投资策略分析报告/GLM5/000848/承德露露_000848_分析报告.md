
### 粗算否决门检查

```

EV = 市值 - 广义净现金 = 84.9 - 32.56 = 52.34 亿元
R_EV = 3.06 / 52.34 = 5.85%

R_EV > II ✅ 达标
```

**粗算结论**: 
- 市值口径不达标（3.60% < 3.79%）
- EV口径达标（5.85% > 3.79%）
