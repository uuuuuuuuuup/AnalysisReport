## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
归母净利润 [C] = 2.93亿元（2024年）
```

### 步骤2：Owner Earnings粗算

```
折旧摊销 [D] = 0.50亿元（估算）
维持性资本开支 [H] = D × G = 0.50 × 0.10 = 0.05亿元

Owner Earnings [I] = C + D - H
                   = 2.93 + 0.50 - 0.05
                   = 3.38亿元
```

**⚠️ 关键假设**：
- 折旧摊销按固定资产的10%估算（约0.50亿元）
- 维持性资本开支系数G=0.10（假设90%为成长性资本开支）

### 步骤3：分配能力验证

```
2024年经营现金流 = 5.42亿元
2024年股息支付 = 1.79亿元
股息支付/经营现金流 = 33.01%

✅ 分配能力验证通过（<70%）
```

### 步骤4：现金上游障碍

- **无重大上游障碍**
- 内地上市公司，股息分配无特殊限制
- 无重大子公司留存现金问题

### 步骤5：分配意愿

```

历史支付率 = 需补充数据（暂假设稳定）

✅ 分配意愿强（支付率>50%）
```

### 步骤6：可预测性

- **收入可预测性**：中（增速放缓）
- **利润可预测性**：中（波动较大）
- **现金流可预测性**：低（2025年大幅下降）

**判断**：可预测性中等

### 步骤7：税务泄漏修正

```
综合股息税率 [Q] = 10%（A股标准，持股>1年免税，暂按10%计算）
税后Owner Earnings = I × (1 - Q%) = 3.38 × 0.90 = 3.04亿元
```

### 步骤8：粗算穿透回报率

```
支付率锚定值 [M] = 61%
年均回购金额 [O] = 0.00亿元

粗算穿透回报率 [R] = [I × M × (1 - Q%) + O] / 当前市值
                    = [3.38 × 0.61 × 0.90 + 0] / 47.66
                    = 1.85 / 47.66
                    = 3.88%
```

### 粗算否决门检查

```
门槛值 [II] = 3.50%
粗算穿透回报率 [R] = 3.88%

R > II ✅ 通过粗算否决门
```

**但需注意**：
- R仅略高于II（3.88% vs 3.50%），安全边际很小
- 2025年业绩下滑，粗算可能过于乐观
