## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤1：归母净利润

```
[C] = GAAP归母净利润 = 53.22亿元
```

#### 步骤2：Owner Earnings粗算

```
[D] = 折旧摊销 ≈ 15亿元（估算）
[E] = 资本开支 = 16.88亿元
[G] = 维持性资本开支系数 ≈ 0.5
[H] = D × G = 15 × 0.5 = 7.5亿元

[I] = Owner Earnings = C + D - H
    = 53.22 + 15 - 7.5
    = 60.72亿元
```

#### 步骤3：分配能力验证

```
[M] = 支付率锚定值 = 53.44%
可分配利润 = I × M = 60.72 × 53.44% = 32.45亿元
实际股息 = 28.44亿元

验证: 实际股息 < 可分配利润 ✅ 通过
```

#### 步骤4：现金上游障碍

```
现金上游障碍 = 无（A股上市公司，分红可直接到账）
```

#### 步骤5：分配意愿

```
分配意愿 = 强
- 分红比例连续提升
- 明确的分红承诺（55%）
- 2017-2024年每股股利CAGR为10%
```

#### 步骤6：可预测性

```
可预测性 = 中等
- 收入相对稳定
- 受路网分流影响，存在不确定性
- 改扩建项目投资回报不确定
```

#### 步骤7：税务泄漏修正

```
[Q] = 综合股息税率 = 0%（持股超1年免税）
```

#### 步骤8：粗算穿透回报率

```
[R] = [C × M × (1 - Q) + O] / 当前市值
    = [53.22 × 53.44% × 1 + 0] / 698
    = 28.45 / 698
    = 4.08%
```

### 粗算否决门判断

```
门槛 [II] = 3.85%
粗算回报率 [R] = 4.08%

判断: R > II ✅ 通过
      R > II × 0.5 ✅ 通过
```

**结论**: 粗算穿透回报率4.08%超过门槛3.85%，继续进入因子3精算。
