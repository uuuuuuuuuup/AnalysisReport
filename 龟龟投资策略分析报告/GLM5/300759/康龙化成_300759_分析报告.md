## 因子2: 穿透回报率粗算(Top-Down)

**锚定口径**: 采用扣非净利润 **11.08亿元** 作为基准

### 步骤2: Owner Earnings粗算

**公式**: Owner Earnings = C + D - H

其中:
- C = 扣非归母净利润 = 11.08亿元
- D = 折旧摊销 = ⚠️估算约6亿元(基于固定资产规模)
- H = 维持性资本开支 = D × G

**维持性资本开支系数[G]**:
- 行业特征: 重资产,设备更新快
- 估算G = 1.2(资本开支略高于折旧摊销)

**H** = 6 × 1.2 = 7.2亿元

**Owner Earnings[I]** = 11.08 + 6 - 7.2 = **9.88亿元**

**⚠️ 注**: 此为粗算,实际需精算验证

### 步骤8: 粗算穿透回报率

**公式**: R = [I × M × (1 - Q) + O] / 市值

其中:
- I = Owner Earnings = 9.88亿元
- M = 支付率锚定值 = 20%(保守估计)
- Q = 综合股息税率 = 10%
- O = 年均回购金额 = 0.5亿元
- 市值 = 506.86亿元

**计算**:
```
R = [9.88 × 20% × (1 - 10%) + 0.5] / 506.86
R = [9.88 × 0.2 × 0.9 + 0.5] / 506.86
R = [1.78 + 0.5] / 506.86
R = 2.28 / 506.86
R = 0.45%
```

**⚠️ 异常低值警告**:
- 粗算回报率仅0.45%,远低于门槛
- 原因分析:
  1. 支付率锚定值20%过低(实际可用Owner Earnings的35-40%)
  2. 市值计算可能有误(506亿 vs 实际约500亿)
  3. Owner Earnings粗算可能低估

**修正计算**(采用实际支付能力):
```
实际可分配现金 = Owner Earnings × 实际支付率
= 9.88 × 35% = 3.46亿元

R = [3.46 + 0.5] / 506.86
R = 3.96 / 506.86
R = 0.78%
```

**⚠️ 仍然过低,进入精算阶段重新评估**
