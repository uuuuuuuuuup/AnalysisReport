## 因子2:穿透回报率粗算(Top-Down)

### 步骤1:归母净利润

- **GAAP归母净利润 [C]**: 26.48亿元

### 步骤2:Owner Earnings粗算

```
Owner Earnings = C + D − H

其中:
C = 归母净利润 = 26.48亿元
D = 折旧摊销 = 约2.00亿元(估算)
H = 维持性资本开支粗估 = D × G
G = 维持性资本开支系数 = 约1.5(估算)
H = 2.00 × 1.5 = 3.00亿元

Owner Earnings [I] = 26.48 + 2.00 − 3.00 = 25.48亿元
```

| 自由现金流 > 0 | ✅ 是 | 18.85亿元 |
