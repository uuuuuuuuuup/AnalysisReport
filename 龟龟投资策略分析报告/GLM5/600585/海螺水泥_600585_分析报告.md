## 因子 2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
集团净利润 [A] = 89.40亿元
少数股东损益 [B] = 12.44亿元
GAAP归母净利润 [C] = A - B = 76.96亿元
```

**数据来源：data_pack_market §2.1**

### 步骤2：Owner Earnings粗算

```
折旧摊销 [D] = 75.00亿元（估算）
资本开支 [E] = 133.26亿元
维持性资本开支系数 [G] = 0.6（水泥行业重资产特性）
维持性资本开支 [H] = D × G = 75.00 × 0.6 = 45.00亿元

Owner Earnings [I] = C + D - H
                   = 76.96 + 75.00 - 45.00
                   = 106.96亿元
```

## 附录：关键计算公式

### 因子2粗算公式
```
Owner Earnings = 归母净利润 + 折旧摊销 - 维持性资本开支
粗算穿透回报率 = (Owner Earnings × 支付率) / 当前市值
```

### 因子3精算公式
```
真实可支配现金结余 = 经营现金流 - 资本开支
精算穿透回报率 = (真实可支配现金结余 × 支付率) / 当前市值
EV口径穿透回报率 = (真实可支配现金结余 × 支付率) / (市值 - 净现金)
```

### 因子4估值公式
```
安全边际 = 穿透回报率 - 门槛
目标买入价 = 当前市值 × (穿透回报率 / 门槛) / 总股本 × (1 - 安全边际折扣)
```
