## 因子2：穿透回报率粗算（Top-Down）

```
[C] = GAAP归母净利润 = 41.45 亿元 (2024年)
```

### 步骤2：Owner Earnings粗算

```
[D] = 折旧摊销 = 约 120 亿元 (估算)
[E] = 资本开支 = 612.42 亿元 (2024年)
[G] = 维持性资本开支系数 = 0.3 (重资产行业保守估计)
[H] = D × G = 120 × 0.3 = 36 亿元 (维持性资本开支粗估)
[I] = C + D - H = 41.45 + 120 - 36 = 125.45 亿元 (Owner Earnings)
```


### 步骤8：粗算穿透回报率

```
当前市值 = 1,664 亿元
支付率锚定值 [M] = 30%
年均回购金额 [O] = 15 亿元

粗算穿透回报率 [R]:
R = [C × M × (1 - Q%) + O] / 当前市值
R = [41.45 × 30% × (1 - 0%) + 15] / 1,664
R = [12.44 + 15] / 1,664
R = 27.44 / 1,664
R = 1.65%
```

### 粗算否决门检查

```
门槛 [II] = 3.80% (A股门槛)
粗算穿透回报率 [R] = 1.65%

R < II → 检查EV口径

EV = 当前市值 - 广义净现金
EV = 1,664 - (-266.47)
EV = 1,930.47 亿元

R_EV = [C × M × (1 - Q%) + O] / EV
R_EV = 27.44 / 1,930.47
R_EV = 1.42%

R_EV < II → EV口径仍不通过

⚠️ 粗算结果: R = 1.65% < II = 3.80%
```
