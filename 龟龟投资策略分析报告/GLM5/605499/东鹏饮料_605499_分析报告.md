## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### Step 1: 归母净利润
```
C = 33.27 亿元
```

#### Step 2: Owner Earnings粗算
```
D（折旧摊销）≈ 3 亿元（估算）
E（资本开支）= 68.75 亿元
G（维持性资本开支系数）≈ 0.3（快消品行业特征）
H = D × G = 3 × 0.3 = 0.9 亿元

Owner Earnings [I] = C + D - H = 33.27 + 3 - 0.9 = 35.37 亿元
```

⚠️ **注意**：资本开支远超折旧摊销，说明公司处于高速扩张期，Owner Earnings口径需谨慎使用。

#### Step 3: 分配能力验证
```
经营现金流 OCF = 57.89 亿元
股息支付 = 23.00 亿元
OCF覆盖率 = 57.89 / 23.00 = 252% ✅ 通过
```

#### Step 4: 现金上游障碍
- 无重大上游障碍，现金可自由支配

#### Step 5: 分配意愿
- 支付率历史：60%-69%，意愿**强**
- 家族控股高分红倾向明显

#### Step 6: 可预测性
- 营收增长可预测性：中等（依赖渠道扩张和市场竞争）
- 利润可预测性：中等

#### Step 7: 税务泄漏修正
```
A股股息红利税：
- 持有>1年：免税
- 持有1月-1年：10%
- 持有<1月：20%

综合股息税率 Q = 10%（假设持有期适中）
```

#### Step 8: 粗算穿透回报率
```
M = 60%（支付率锚定值）
O = 0（无回购）

R = [C × M × (1 - Q%) + O] / 当前市值
R = [33.27 × 60% × 0.9 + 0] / 1280
R = 17.97 / 1280
R = 1.40%
```
