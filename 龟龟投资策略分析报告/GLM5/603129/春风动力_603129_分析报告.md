### 粗算否决门检验

```
R = 1.53%
II = 3.50%
Rf = 1.79%

检验1：R < Rf？ → 1.53% < 1.79% → ⚠️ 是

需要检查EV口径
```

#### EV口径计算

```
广义净现金 = 71.49亿元

R_EV = 1.87%
```
