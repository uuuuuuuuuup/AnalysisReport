- 假设投资者持股不足1年: Q = 10%

### 步骤8:粗算穿透回报率R%

```
支付率锚定值 [M] = 60%
年均回购金额 [O] = 0亿元
综合股息税率 [Q] = 10%

粗算穿透回报率 [R] = [C × M × (1 − Q%) + O] / 当前市值
                    = [52.35 × 60% × (1 − 10%) + 0] / 596.58
                    = [52.35 × 0.6 × 0.9] / 596.58
                    = 28.27 / 596.58
                    = 4.74%
```

### 粗算否决门检查

```
门槛 [II] = 3.5%
粗算穿透回报率 [R] = 4.74%

检查1: R < Rf? → 4.74% > 1.79% → ✅ 通过
检查2: R < II × 0.5? → 4.74% > 1.75% → ✅ 通过
检查3: II × 0.5 ≤ R < II? → 4.74% > 3.5% → ✅ 通过
```

**结论**: ✅ **粗算穿透回报率4.74%超过门槛3.5%,通过因子2筛选**
