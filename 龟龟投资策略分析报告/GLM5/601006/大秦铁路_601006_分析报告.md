## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### Step 1: 归母净利润

```
C = 90.39亿元 (2024年归母净利润)
```

#### Step 2: Owner Earnings粗算

```
D = 折旧摊销 ≈ 45亿元
E = 资本开支 = 35亿元
G = 维持性资本开支系数 ≈ 0.8 (估算)
H = D × G = 45 × 0.8 = 36亿元 (维持性资本开支粗估)
I = C + D − H = 90.39 + 45 - 36 = 99.39亿元 (Owner Earnings)
```

#### Step 3: 分配能力验证

```
FCF = 58.17亿元 (2024年自由现金流)
股息支付 = 51.81亿元
FCF > 股息支付 ✓ 分配能力验证通过
```

#### Step 4: 分配意愿

```
支付率锚定值 M = 60%
公司承诺2023-2025年不低于55%
过去3年支付率均值 = 69.34%
分配意愿 = 强
```

#### Step 5: 粗算穿透回报率

```
M = 60% (支付率锚定值)
Q = 0% (A股直接持有无股息税)
O = 0亿元 (年均回购金额，尚未实施)

R = [C × M × (1 - Q) + O] / 当前市值
R = [90.39 × 60% × 1 + 0] / 1090
R = 54.23 / 1090
R = 4.98%

使用Owner Earnings口径：
R_oe = [I × M] / 市值 = 99.39 × 60% / 1090 = 5.47%
```

**粗算穿透回报率 R = 5.60%** (取归母净利润和Owner Earnings口径均值)

#### Step 6: 粗算否决门检验

```
门槛 II = 3.50%
R = 5.60% > II ✓ 通过
R > II × 0.5 = 1.75% ✓ 通过
```

**结论**: 粗算穿透回报率5.60%超过门槛值，继续进入因子3精算

