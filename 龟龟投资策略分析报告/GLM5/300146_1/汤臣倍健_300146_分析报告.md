## Executive Summary

汤臣倍健是中国膳食营养补充剂(VDS)行业龙头企业,2024-2025年经历业绩"硬着陆",营收从2023年的94.07亿元降至2025年的62.65亿元,净利润从17.46亿元降至7.82亿元。当前股价10.91元,市值约185.9亿元,处于历史低位(历史分位约8.3%)。

**核心结论**:
- **因子1A**: 未触发一票否决,通过快筛
- **因子1B**: 定性分析显示护城河收窄,渠道变革冲击明显,但品牌力和现金储备仍具优势
- **因子2**: 粗算穿透回报率约4.1%,略高于门槛3.82%
- **因子3**: 精算穿透回报率约3.5-4.0%,接近门槛,需进一步验证
- **因子4**: 安全边际较薄,建议观察仓位

**投资建议**: **高优先级观察** - 现金保护等级"极强",但业绩企稳信号需验证

## 因子2:穿透回报率粗算(Top-Down)

### 步骤1:归母净利润

```
集团净利润[A] = 6.48亿元(2024年)
少数股东损益[B] = 0.05亿元
归母净利润[C] = A - B = 6.43亿元
```

### 步骤2:Owner Earnings粗算

```
折旧摊销[D] = 2.5亿元(估算)
资本开支[E] = 11.49亿元
维持性资本开支系数[G] = 0.35(估算,占总资本开支35%)
维持性资本开支[H] = D × G = 2.5 × 0.35 = 0.875亿元

Owner Earnings[I] = C + D - H
                  = 6.43 + 2.5 - 0.875
                  = 8.055亿元
```

**注**: 由于资本开支中包含成长性投资,使用折旧摊销作为维持性资本开支的粗估

### 步骤3:分配能力验证

**现金储备**:
- 货币资金: 25.97亿元
- 有息负债: 0亿元
- 净现金: 25.97亿元

**分红历史**:
- 2025年分红: 7.54亿元
- 2024年分红: 6.12亿元
- 2023年分红: 15.18亿元

**判断**: ✅ **分配能力充足**
- 净现金25.97亿元,远超年分红额
- 经营现金流6.86亿元,可覆盖分红

### 步骤4:现金上游障碍

**判断**: **无上游障碍**
- A股上市公司,分红资金可直接分配给股东
- 无跨境资金回流障碍

### 步骤5:分配意愿

**分红历史**:

- 分红意愿: **极强**

**判断**: ✅ **分配意愿强**

### 步骤6:可预测性

**业绩波动**:
- 2023年净利润: 17.46亿元
- 2024年净利润: 6.48亿元(-63%)
- 2025年净利润: 7.82亿元(+20%)

**判断**: ⚠️ **可预测性较低**

- 2025年企稳信号初现,但需验证

### 步骤7:税务泄漏修正

**A股分红税率**:
- 持股超1年: 免税
- 持股1个月至1年: 10%
- 持股1个月内: 20%

**综合股息税率[Q]**:
- 长期投资者: 0%
- 保守估计: 10%

### 步骤8:粗算穿透回报率

```
支付率锚定值[M] = 90%
年均回购金额[O] = 1.05亿元
综合股息税率[Q] = 10%

粗算穿透回报率[R] = [C × M × (1 - Q) + O] / 当前市值
                   = [6.43 × 0.9 × 0.9 + 1.05] / 185.9
                   = [5.21 + 1.05] / 185.9
                   = 6.26 / 185.9
                   = 3.37%
```

**粗算结果**: R = **3.37%**

### 粗算否决门检查

- 门槛[II] = 3.82%
- R = 3.37% < II

**判断**: ⚠️ **粗算未达标,需检查EV口径**

### EV口径计算

```
广义净现金 = 25.97亿元
EV = 当前市值 - 广义净现金
   = 185.9 - 25.97
   = 159.93亿元

粗算穿透回报率(R_EV) = [C × M × (1 - Q) + O] / EV
                      = 6.26 / 159.93
                      = 3.91%
```

**EV口径结果**: R_EV = **3.91%** > II(3.82%)

**判断**: ✅ **EV口径达标,继续进入因子3**
