## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤1：归母净利润**
```
2024年归母净利润 [C] = 104 亿元
```

**步骤2：Owner Earnings粗算**
```
假设：
- 折旧摊销 D ≈ 40 亿元（估计，参考同行）
- 维持性资本开支系数 G = 50%
- 维持性资本开支 H = D × G = 40 × 50% = 20 亿元
- Owner Earnings粗算 = C + D - H = 104 + 40 - 20 = 124 亿元
```

**步骤3：分配能力验证**
```
2024年股息 = 22.05 亿元
22.05 < 124 亿元 ✅ 通过
```

**步骤4：分配意愿**
```
历史支付率约30%，2024年支付率21.2%
支付率锚定值 M = 30%
分配意愿：中等（支付率有所下降）
```

**步骤5：粗算穿透回报率**
```
支付率锚定值 M = 30%
综合股息税率 Q = 10%（港股通约20%，直接持有约10%）
年均回购金额 O = 0（未发现2024年回购）

R = [C × M × (1 - Q%) + O] / 当前市值
  = [104 × 30% × (1 - 10%) + 0] / 644
  = [104 × 0.30 × 0.90] / 644
  = 28.08 / 644
  = 4.36%
```

**使用核心溢利重算**：
```
R_核心 = [69.7 × 30% × (1 - 10%) + 0] / 644
       = [69.7 × 0.30 × 0.90] / 644
       = 18.82 / 644
       = 2.92%
```

### 粗算结论

| 口径 | 穿透回报率R | 门槛II=5% | 结果 |
|------|-------------|-----------|------|
| 归母净利润口径 | 4.36% | 5% | ❌ 未达标 |
| 核心溢利口径 | 2.92% | 5% | ❌ 未达标 |

**检查EV口径**：
```
EV = 市值 - 广义净现金 = 644 - (-1269) = 1913 亿元
R_EV = 28.08 / 1913 = 1.47%
```
**EV口径也未达标**

**决策**：⚠️ 边际不达标，但差异较小，继续进入因子3精算验证

## 因子3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤1：真实现金收入还原

**估算收款比率**：
```
真实现金收入 = 营业收入 × 收款比率
假设综合收款比率约85%（开发业务回款慢，运营业务回款快）
真实现金收入 = 1274.7 × 85% ≈ 1083 亿元

```

```
员工支付 ≈ 50亿元（估计）

经营性现金支出 W ≈ 1070 + 25 + 70 + 50 ≈ 1215 亿元
```

### 步骤3：资本开支与投资

```
资本开支 E ≈ 100-150亿元（估计）
投资性购买 ≈ 50亿元（估计）

总扣除额 Y ≈ 150 + 50 = 200 亿元
```

### 步骤4：真实可支配现金结余

```

               = 1083 - 1215 - 200

```
