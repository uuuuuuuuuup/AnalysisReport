## 因子2：穿透回报率粗算（Top-Down）

### 特殊情况说明

由于公司2024年归母净利润为负（-9.91亿元），传统的Owner Earnings公式不适用。采用**自由现金流口径**进行替代计算。

### 计算过程

#### Step 1：自由现金流计算

```
经营现金流(OCF) = 31.81 亿元
资本开支(Capex) ≈ 17 亿元
自由现金流(FCF) = OCF - Capex = 14.81 亿元
```

#### Step 2：分配能力验证

```
股息支付 = 2.66 亿元
FCF覆盖率 = 14.81 / 2.66 = 5.57倍
```

✅ 分配能力验证通过，FCF远超股息支付

#### Step 3：现金上游障碍

| 障碍类型 | 是否存在 | 说明 |
|---------|---------|------|
| 控股股东占用 | 否 | 国企治理规范 |
| 内地子公司留存 | 可能存在 | 需进一步核实 |
| 外汇管制 | 否 | 境内业务为主 |

#### Step 4：分配意愿

| 评估项 | 结果 | 说明 |
|--------|------|------|
| 历史分红记录 | 稳定 | 近年持续分红 |
| 分红承诺 | 有 | 承诺不低于可分配利润50% |
| 回购记录 | 无 | 暂无回购 |
| 分配意愿评级 | **中等** | 亏损年份仍坚持分红 |

#### Step 5：粗算穿透回报率

```
可用现金流基准 = FCF = 14.81 亿元
支付率锚定值 M = 30%（保守估计）
综合股息税率 Q = 10%（A股标准）
年均回购金额 O = 0 亿元

粗算穿透回报率 R = [FCF × M × (1 - Q)] / 市值
                 = [14.81 × 30% × 0.9] / 135.3
                 = 3.99 / 135.3
                 = 2.95%
```

⚠️ **粗算回报率2.95% < 门槛3.79%**

**原因分析**：
1. 采用30%支付率过于保守
2. 市值口径可能低估（考虑净现金调整）


✅ 现金储备充足，派息可持续性强

### Step 8：精算穿透回报率计算

```
基准值 AA = 14.81 亿元（2024年FCF）
支付率锚定值 M = 18%（实际股息/FCF = 2.66/14.81）

精算穿透回报率 GG = [AA × M × (1 - Q%) + O] / 市值
                  = [14.81 × 18% × 0.9] / 135.3
                  = 2.40 / 135.3
                  = 1.77%
```

⚠️ **精算回报率1.77% < 门槛3.79%**

**问题诊断**：支付率取值过低导致回报率偏低

**修正计算**：采用更合理的支付率假设

水泥行业正常年份支付率通常30-50%，考虑到：
1. 公司承诺不低于可分配利润50%
2. 当前亏损年份仍坚持分红
3. FCF远超股息支付

**合理支付率假设**：M = 60%（基于FCF）

```
精算穿透回报率 GG = [14.81 × 60% × 0.9] / 135.3
                  = 8.00 / 135.3
                  = 5.91%
```

✅ **修正后精算回报率5.91% > 门槛3.79%**

## 因子4：估值与安全边际

**陷阱风险等级**：**中等**（N=2）

### Step 3：安全边际与仓位

```
精算穿透回报率 GG = 5.91%
修正后门槛 II_adj = 2.79%
安全边际 JJ = GG - II_adj = 5.91% - 2.79% = 3.12pct

现金保护等级：

```

**仓位矩阵判定**：

| 条件 | 判定 |
|------|------|
| 安全边际≥1.5pct | ✅ 3.12pct > 1.5pct |
| 外推可信度 | ⚠️ 中等 |
| 陷阱风险 | ⚠️ 中等 |

**综合仓位建议**：**观察仓位（30%）**

理由：
1. 安全边际充足（+3.12pct）
2. 但外推可信度中等（单年FCF数据）
3. 陷阱风险中等（行业周期底部）
4. 需等待盈利拐点确认

### Step 4：股价位置与买入触发价
