## 因子2:穿透回报率粗算(Top-Down)

### 步骤1:归母净利润

```
C = 61.04亿元
```

### 步骤2:Owner Earnings粗算

```
D(折旧摊销) ≈ 50亿元(推算)
E(资本开支) = 59.30亿元
G(维持性资本开支系数) = 0.7(汽车行业假设)
H = D × G = 50 × 0.7 = 35亿元
I = C + D - H = 61.04 + 50 - 35 = 76.04亿元
```

**Owner Earnings粗估值**: 76.04亿元

### 步骤3:分配能力验证

**检查**: 经营现金流(48.49亿元) < Owner Earnings(76.04亿元)

**判断**: ⚠️ **分配能力存疑**
- 经营现金流低于Owner Earnings粗估值
- 说明Owner Earnings可能高估,或资本开支偏大
- 需要进一步精算验证

### 步骤4:现金上游障碍

**分析**:
- 广义净现金641.46亿元,占市值1,097.55亿元的58.46%
- 现金储备充足,无上游障碍

**判断**: 无现金上游障碍

### 步骤5:分配意愿

**分析**:
- 2024年分红比例40%,高于历史均值35%
- 正在研究一年多次分红可行性
- 管理层回报股东意愿明确

**判断**: **分配意愿强**

### 步骤6:可预测性

**分析**:
- 汽车行业周期性强,盈利波动大
- 2024年净利润同比下降35.76%
- 新能源转型期,未来盈利不确定性高

**判断**: **可预测性低**

### 步骤7:税务泄漏修正

**分析**:
- A股分红需缴纳20%红利税(持股超1年免征)
- 假设持股超1年,红利税为0
- 综合股息税率[Q] = 0%

### 步骤8:粗算穿透回报率R%

```
M(支付率锚定值) = 40%
Q(综合股息税率) = 0%
O(年均回购金额) = 0亿元
当前市值 = 1,097.55亿元

粗算穿透回报率 R = [C × M × (1 - Q%) + O] / 当前市值
R = [61.04 × 40% × (1 - 0%) + 0] / 1,097.55
R = 24.42 / 1,097.55
R = 2.22%
```

**粗算穿透回报率**: 2.22%

### 粗算否决门检查

**门槛II = 3.5%**

**检查**:
- R(2.22%) < II(3.5%) ❌ 不达标
- R(2.22%) < II × 0.5(1.75%) ❌ 不达标

**判断**: ⚠️ **粗算否决门触发**
- 粗算穿透回报率低于门槛
- 需要检查EV口径是否改善

### EV口径检查

**触发条件**: 广义净现金/市值 = 58.46% > 40% ✅ 触发EV口径

```
EV = 当前市值 - 广义净现金
EV = 1,097.55 - 641.46 = 456.09亿元

R_EV = [C × M × (1 - Q%) + O] / EV
R_EV = [61.04 × 40% × (1 - 0%) + 0] / 456.09
R_EV = 24.42 / 456.09
R_EV = 5.36%
```

**EV口径穿透回报率**: 5.36%

**EV口径检查**:
- R_EV(5.36%) > II(3.5%) ✅ 达标

**判断**: EV口径达标,但需要通过"现金分配意愿检验"

### 现金分配意愿检验

**检查项**:
1. 分红历史稳定: ✅ 连续多年分红,比例超30%
2. 分红比例提升: ✅ 2024年提升至40%
3. 管理层承诺: ✅ 正在研究一年多次分红
4. 现金储备充足: ✅ 广义净现金641.46亿元

**判断**: ✅ **通过现金分配意愿检验**
