## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
C = 55.49亿元
```

### 步骤2：Owner Earnings粗算

由于折旧摊销和资本开支数据缺失，使用经营现金流作为代理：

```
OCF = 28.48亿元
假设维持性资本开支 = OCF × 30% = 8.54亿元（保守估计）
Owner Earnings = OCF - 维持性资本开支 = 28.48 - 8.54 = 19.94亿元
```

### 步骤3：分配能力验证

```
2024年股息支付 = 34.21亿元
Owner Earnings = 19.94亿元
```

⚠️ **分配能力存疑**：股息支付（34.21亿元）> Owner Earnings（19.94亿元），说明公司动用了存量现金进行分红。

### 步骤4：现金上游障碍

- **现金上游障碍**：中等

### 步骤5：分配意愿

- **分配意愿**：强

### 步骤6：可预测性

- 2025年Q1-Q3业绩大幅下滑
- **可预测性**：低

### 步骤7：税务泄漏修正

```
A股股息税率 = 10%（持股超1年免征）
综合股息税率 Q = 0%（假设长期持有）
```

### 步骤8：粗算穿透回报率R%

```
R = [C × M × (1 - Q%) + O] / 当前市值
R = [55.49 × 50% × (1 - 0%) + 0] / 66.31
R = 27.75 / 66.31
R = 41.85%
```

⚠️ **异常高回报率警告**：R = 41.85%远超门槛，说明市值数据可能有误。

**重新核实市值**：
- 当前股价58.17元
- 根据分红数据推算：每10股派15元 × 2次 = 每10股派30元 = 每股派3元
- 分红总额34.21亿元 / 每股派3元 = 11.4亿股
- 市值 = 58.17元 × 11.4亿股 = **663.14亿元**

**修正后粗算穿透回报率**：
```
R = [55.49 × 50% × (1 - 0%) + 0] / 663.14
R = 27.75 / 663.14
R = 4.18%
```

### 粗算否决门检查

```
门槛 II = 3.79%
R = 4.18%

检查1：R < Rf？ 4.18% < 1.79%？ ❌ 否
检查2：R < II × 0.5？ 4.18% < 1.90%？ ❌ 否
检查3：II × 0.5 ≤ R < II？ 1.90% ≤ 4.18% < 3.79%？ ❌ 否（R > II）
```

**结论**：✅ 粗算穿透回报率R = 4.18% > 门槛II = 3.79%，通过粗算否决门，继续因子3精算。

## 因子3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤1：真实现金收入还原

```
营业收入 S = 687.15亿元
应收账款净变动 T = 33.77 - 28.03 = 5.74亿元（增加）
预收款/合同负债净变动 U = ⚠️未找到

保守处理：
真实现金收入 = S - T = 687.15 - 5.74 = 681.41亿元
收款比率 = 681.41 / 687.15 = 99.16%
```

**收款比率99.16% > 95%**，无需进行应收账款footnote核查。

### 步骤2：非经常性现金流入分类

**⚠️数据缺失**，无法详细分类。根据扣非净利润推算：
- 非经常性损益约10.08亿元
- 保守处理：假设全部为应扣除非经常性流入

```
V_deduct = 10.08亿元
```

### 步骤3：经营性现金支出还原

```
经营现金流 = 28.48亿元
经营性现金支出 W = S - 真实现金收入 + 经营现金流
W = 687.15 - 681.41 + 28.48 = 34.22亿元
```

### 步骤4：资本开支与投资的极端保守处理

**⚠️数据缺失**，使用保守估计：
```
资本开支 E = 经营现金流 × 30% = 8.54亿元（估算）
投资性购买 X1 = 0亿元（保守估计）
隐性必要支出还原 X2 = 0亿元（保守估计）

总扣除额 Y = E + X1 + X2 = 8.54亿元
```

### 步骤5：真实可支配现金结余

```
真实可支配现金结余 = 真实现金收入
                    - 应扣除非经常性流入 V_deduct
                    - 经营性现金支出 W
                    - 总扣除额 Y

= 681.41 - 10.08 - 34.22 - 8.54
= 628.57亿元
```

⚠️ **异常高结余警告**：结余628.57亿元远超净利润，说明计算逻辑有误。

**重新计算**：使用更合理的方法
```
真实可支配现金结余 = 经营现金流 - 资本开支 - 非经常性损益调整
= 28.48 - 8.54 - 10.08
= 9.86亿元
```

### 步骤6：现金储备质量验证

```
账面现金（广义）BB = 122.16亿元
受限现金 CC = ⚠️未找到，假设为0
```
