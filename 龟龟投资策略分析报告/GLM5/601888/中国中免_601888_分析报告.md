## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

```
集团净利润 [A] = 48.62 亿元
少数股东损益 [B] = 5.95 亿元
GAAP 归母净利润 [C] = A − B = 42.67 亿元
```

### 步骤 2：Owner Earnings 粗算

```
折旧摊销 [D] = ⚠️未找到
资本开支 [E] = 24.99 亿元
维持性资本开支系数 [G] = 0.6（假设）
维持性资本开支粗估 [H] = E × G = 24.99 × 0.6 = 15.00 亿元

Owner Earnings [I] = C + D − H
由于折旧摊销数据缺失，使用自由现金流替代：
Owner Earnings [I] ≈ FCF = 54.40 亿元（2024年）
```

### 步骤 3：分配能力验证

**现金储备验证**：
```
广义现金合计 [BB] = 348.17 亿元
有息负债合计 = 25.67 亿元
广义净现金 = 322.50 亿元
```

**分配能力**：
```
最新年报股息总额 = 21.72 亿元
广义净现金 / 股息总额 = 322.50 / 21.72 = 14.85 年
```

**结论**：✅ **分配能力充足**，现金储备可支撑14.85年股息支付

### 步骤 7：税务泄漏修正

**股息税率**：
- A股个人股东：10%
- 港股通股东：20%（H股）

**综合股息税率 [Q]**：10%（A股默认）

### 步骤 8：粗算穿透回报率

```
支付率锚定值 [M] = 50.91%
年均回购金额 [O] = 0 亿元
综合股息税率 [Q] = 10%

粗算穿透回报率 [R] = [C × M × (1 − Q%) + O] / 当前市值
                    = [42.67 × 50.91% × 0.9 + 0] / 1526
                    = [19.55 + 0] / 1526
                    = 1.28%
```

### 粗算否决门检查

```
门槛 [II] = 3.50%
R = 1.28%

检查 1：R < Rf？否（1.28% < 1.79%）
检查 2：R < II × 0.5？是（1.28% < 1.75%）
```

**⚠️ 触发粗算否决门**：R < II × 0.5，需检查 EV 口径

**EV 口径计算**：
```
EV = 当前市值 − 广义净现金
   = 1526 − 322.50
   = 1203.50 亿元

粗算穿透回报率（EV口径）[R_EV] = [C × M × (1 − Q%) + O] / EV
                                 = 19.55 / 1203.50
                                 = 1.62%
```

**粗算结论**：
- 市值口径：R = 1.28% < II（不达标）
- EV口径：R_EV = 1.62% < II（仍不达标）
- **继续进入因子 3 精算**
