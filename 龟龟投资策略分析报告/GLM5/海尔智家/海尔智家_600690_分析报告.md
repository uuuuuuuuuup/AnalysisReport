### 步骤8:粗算穿透回报率R%

```
粗算穿透回报率 R = [C × M × (1 - Q%) + O] / 当前市值

其中:
C = 归母净利润 = 195.53亿元
M = 支付率锚定值 = 55%
Q = 综合股息税率 = 10%
O = 年均回购金额 = 6.11亿元(2026年计划45亿,分摊至每年)
当前市值 = 1938.36亿元

R = [195.53 × 55% × (1 - 10%) + 6.11] / 1938.36
R = [195.53 × 0.55 × 0.9 + 6.11] / 1938.36
R = [96.79 + 6.11] / 1938.36
R = 102.90 / 1938.36
R = 5.31%
```

### 粗算否决门检查

- R = 5.31%
- Rf = 1.82%
- II = 3.82%

**判断**:
- R > Rf ✅ 通过
- R > II × 0.5 = 1.91% ✅ 通过
- R > II = 3.82% ✅ 通过

**结论**: ✅ 粗算穿透回报率5.31%,超过门槛值3.82%,继续进入因子3精算
