## 因子2：穿透回报率粗算（Top-Down）


### 步骤2：Owner Earnings粗算

```
折旧摊销 [D] ≈ 50亿元（估计，基于固定资产规模）
资本开支 [E] = 11.68亿元
维持性资本开支系数 [G] = 0.3（估计）
维持性资本开支 [H] = D × G = 50 × 0.3 = 15亿元

Owner Earnings [I] = C + D - H
                   = 126.71 + 50 - 15
                   = 161.71亿元
```
