## 因子 2：穿透回报率粗算（Top-Down）


### 步骤 2：Owner Earnings 粗算

```
折旧摊销 [D] ≈ 50 亿元（估算）
资本开支 [E] = 120 亿元
维持性资本开支系数 [G] = 0.5（估算，物流行业约50%资本开支为维持性）
维持性资本开支 [H] = D × G = 50 × 0.5 = 25 亿元
Owner Earnings [I] = C + D − H = 91.85 + 50 − 25 = 116.85 亿元
```
