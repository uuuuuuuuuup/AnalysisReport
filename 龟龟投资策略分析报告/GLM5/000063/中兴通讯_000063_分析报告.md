```
折旧摊销 [D] = 60 亿元 (估算)
资本开支 [E] = 150 亿元 (估算)
维持性资本开支系数 [G] = 0.4 (通信设备行业估算)
维持性资本开支 [H] = D × G = 60 × 0.4 = 24 亿元
Owner Earnings [I] = C + D − H = 83.26 + 60 − 24 = 119.26 亿元
```

### 步骤8：粗算穿透回报率

```
支付率锚定值 [M] = 35%
年均回购金额 [O] = 0 亿元
综合股息税率 [Q] = 10%

粗算穿透回报率 [R] = [C × M × (1 − Q%) + O] / 当前市值
                    = [83.26 × 35% × 0.9 + 0] / 1,775
                    = 26.23 / 1,775
                    = 1.48%

⚠️ 粗算回报率远低于门槛，需要检查EV口径
```

### EV口径粗算

```
广义净现金 = 34 亿元
EV = 当前市值 − 广义净现金 = 1,775 − 34 = 1,741 亿元

粗算穿透回报率 [R_EV] = 26.23 / 1,741 = 1.51%

仍然远低于门槛3.50%
```

### 粗算否决门判断

```
粗算穿透回报率 R = 1.48%
门槛 II = 3.50%
R < II × 0.5 = 1.75%?

R = 1.48% < 1.75% → 需要检查EV口径

R_EV = 1.51% < 1.75% → 触发粗算否决门
```

**⚠️ 粗算阶段触发否决门**

按照龟龟投资策略，粗算穿透回报率低于门槛的50%应触发否决。但考虑到：
1. 2024年净利润受一次性因素影响下降
2. 公司正处于业务转型期（算力业务增长）
3. 历史净利润水平更高（2021-2023年均在80-93亿）

**决定**: 继续进入因子3精算，使用历史平均净利润进行验证
