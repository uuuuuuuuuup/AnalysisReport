## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤1：归母净利润
- GAAP 归母净利润 [C] = **22.80亿元**

#### 步骤2：Owner Earnings 粗算
```
Owner Earnings = C + D - H
其中：
C = 归母净利润 = 22.80亿元
D = 折旧摊销 ≈ 3.5亿元（估算）
H = 维持性资本开支 = D × G
G = 维持性资本开支系数 ≈ 0.5（估算）
H = 3.5 × 0.5 = 1.75亿元

Owner Earnings = 22.80 + 3.5 - 1.75 = 24.55亿元
```

**修正方法**：使用自由现金流作为Owner Earnings代理
- 自由现金流 (FCF) = 经营现金流 - 资本开支 = 76.10 - 6.62 = **69.48亿元**

⚠️ **数据异常**：自由现金流远高于净利润，可能原因：
1. 经营现金流包含非经常性项目
2. 存货大幅下降释放现金
3. 应付账款增加

**保守估计**：取净利润和自由现金流的较低值
- Owner Earnings 估算值 = **22.80亿元**（以净利润为基准）

#### 步骤3：分配能力验证
- 自由现金流 = 69.48亿元 > 股息支付 = 13.71亿元
- **✅ 分配能力充足**

#### 步骤4：现金上游障碍
- 现金储备 = 107.56亿元
- 有息负债 = 12.46亿元
- 净现金 = 95.10亿元
- **✅ 无现金上游障碍**

#### 步骤5：分配意愿
- 2024年分红率约90%（含中期分红）
- 分红政策大幅提升
- **✅ 分配意愿强**

#### 步骤6：可预测性
- 收入波动率约12%（低）
- 利润波动率较高（2024年下降11.73%）
- **⚠️ 可预测性中等**

#### 步骤7：税务泄漏修正
- A股股息税率：持股超1年免税
- Q = 0%（假设长期持有）

#### 步骤8：粗算穿透回报率

```
粗算穿透回报率 R = [C × M × (1 - Q%) + O] / 当前市值
                = [22.80 × 60% × 1 + 0] / 412.26
                = 13.68 / 412.26
                = 3.32%
```

**修正计算**（考虑净现金）：
```
EV = 当前市值 - 广义净现金 = 412.26 - 95.10 = 317.16亿元

R_EV = [C × M × (1 - Q%) + O] / EV
     = 13.68 / 317.16
     = 4.31%
```

**进一步修正**（使用自由现金流）：
```
R_FCF = [FCF × M × (1 - Q%) + O] / 当前市值
      = [69.48 × 60% × 1 + 0] / 412.26
      = 41.69 / 412.26
      = 10.11%
```

⚠️ **数据异常警告**：自由现金流/净利润比率 = 69.48/22.80 = 305%，异常偏高。

**保守估计**：取净利润口径
- 粗算穿透回报率 R = **3.32%**（市值口径）
- 粗算穿透回报率 R_EV = **4.31%**（EV口径）

**粗算否决门检查**：
- R = 3.32% < Rf = 1.80% → ❌ 不满足
- R = 3.32% < II × 0.5 = 1.75% → ❌ 不满足
- R_EV = 4.31% > II = 3.50% → ✅ **EV口径通过**

**结论**：使用**EV口径**继续分析。
