## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤1：归母净利润

```
C = 153.44 亿元（2024年归母净利润）
```

#### 步骤2：Owner Earnings粗算

```
D（折旧摊销）≈ 50 亿元（估计）
E（资本开支）= 367 亿元
G（维持性资本开支系数）≈ 0.3（互联网平台）
H = D × G = 50 × 0.3 = 15 亿元（维持性资本开支粗估）
I = C + D - H = 153.44 + 50 - 15 = 188.44 亿元（Owner Earnings）
```

```
自由现金流 = OCF - Capex = 298 - 367 = -69 亿元
⚠️ 自由现金流为负，分配能力受限
```

**一票否决检查**：自由现金流为负，但经营现金流为正，不触发否决

#### 步骤7：税务泄漏修正

```
港股通股息税率 Q = 20%（若派息）
当前不派息，Q = 0%
```

#### 步骤8：粗算穿透回报率

```
M（支付率锚定值）= 0%（不派息）
O（年均回购金额）= 18.6 亿元

R = [C × M × (1-Q%) + O] / 当前市值
R = [153.44 × 0% × 100% + 18.6] / 2,800
R = 18.6 / 2,800
R ≈ 0.66%
```

**⚠️ 问题**：由于不派息，粗算穿透回报率极低

**修正计算**（假设合理支付率30%）：

```
假设合理支付率 M = 30%
R = [153.44 × 30% × 100% + 18.6] / 2,800
R = [46.03 + 18.6] / 2,800
R = 64.63 / 2,800
R ≈ 2.3%
```

**粗算结论**：
- 实际R ≈ 0.66%（不派息）
- 假设派息R ≈ 2.3%
- **R < II(5%)，粗算不达标**
