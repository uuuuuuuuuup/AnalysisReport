### 2.5 粗算穿透回报率

```
═══════════════════════════════════════════════════════════
                    粗算穿透回报率 R%
═══════════════════════════════════════════════════════════

采用公式：R = (C × M + O) / 市值

参数：
  C = 归母净利润 = 156.0 亿元
  M = 支付率锚定值 = 30%（保守估计）
  O = 年均回购金额 ≈ 20 亿元（假设）

  主食分红 = 156.0 × 30% = 46.8 亿元
  回购贡献 = 20 亿元
  合计分配 = 66.8 亿元

  粗算回报率 R = 66.8 / 2,006 = 3.33%

⚠️ 上述计算偏保守，未考虑：

```
