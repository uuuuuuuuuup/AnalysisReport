```

[O] 年均回购金额 = 0亿元

```

```
II×0.5 ≤ R < II (1.90% ≤ 2.60% < 3.79%) → 边际不达标，继续进入因子3
```

**判断**：粗算穿透回报率未达门槛，但属于"边际不达标"，继续进入因子3进行精算验证。
