### 步骤10：精算穿透回报率计算

```
AA（真实可支配现金结余基准值）= 55.11亿元
M（支付率锚定值）= 20%
Q（股息税率）= 0%
O（年均回购金额）= 0亿元

─── 主轨：市值口径 ───
精算穿透回报率 GG = [AA × M × (1 - Q%) + O] / 当前市值
                  = [55.11 × 20% × 1.0 + 0] / 3571
                  = 11.02 / 3571
                  = 0.31%
```

**⚠️ 问题**: 精算穿透回报率仅为0.31%，仍远低于门槛。

**重新审视计算逻辑**:

根据龟龟投资策略，穿透回报率应该基于**可分配现金**而非FCF。对于恒瑞医药：

```
归母净利润 C = 63.37亿元
支付率 M = 20%
可分配现金 = C × M = 63.37 × 20% = 12.67亿元

穿透回报率 = 可分配现金 / 当前市值
           = 12.67 / 3571
           = 0.355%
```

**或者使用FCF口径**:

```
FCF = 55.11亿元
假设可分配比例 = 20%（保守）
可分配现金 = 55.11 × 20% = 11.02亿元

穿透回报率 = 11.02 / 3571 = 0.31%
```

**结论**: 无论使用哪种口径，穿透回报率都**远低于门槛3.5%**。
