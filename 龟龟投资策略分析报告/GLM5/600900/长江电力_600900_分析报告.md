## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

**Step 1: 归母净利润**
```
C = 324.96 亿元（2024年归母净利润）
```

**Step 2: Owner Earnings粗算**
```
D（折旧摊销）≈ 120 亿元（水电资产折旧）
E（资本开支）= 107.75 亿元
G（维持性资本开支系数）≈ 0.9（水电资本开支主要用于维持）
H = D × G = 108 亿元（维持性资本开支粗估）
I = C + D - H = 324.96 + 120 - 108 = 336.96 亿元（Owner Earnings）
```

**Step 3: 分配能力验证**
```
经营现金流 = 596.48 亿元
股息支付 = 220 亿元
支付能力覆盖率 = 596.48 / 220 = 2.71倍 ✓
```
**结论：分配能力充足，不会触发一票否决。**

**Step 4: 现金上游障碍**
- 无上游障碍，分红资金可自由支配

**Step 5: 分配意愿**
- 明确承诺70%分红，历史兑现良好
- 分配意愿：强 ✓

**Step 6: 可预测性**
- 来水量有一定波动，但六库联调可平滑
- 电价相对稳定
- 可预测性：高 ✓

**Step 7: 税务泄漏修正**
```
A股持股超1年免税，Q = 0%
```

**Step 8: 粗算穿透回报率**
```
M = 70%（支付率锚定值）
O = 0（无回购）
R = [C × M × (1 - Q%) + O] / 当前市值
R = [324.96 × 70% × 100% + 0] / 6,704
R = 227.47 / 6,704
R = 3.39%
```

### 粗算结论

| 指标 | 数值 | vs 门槛 |
|------|------|---------|
| 粗算穿透回报率 R | 3.39% | < 3.90%（门槛） |
| 粗算否决门 | 未触发 | R > Rf且R > II×0.5 |

**判断：R < II，标注"边际不达标"，继续进入因子3精算。**
