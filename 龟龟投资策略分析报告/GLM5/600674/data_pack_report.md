## 9. 关键财务比率计算

### 9.1 自由现金流计算

```
FCF = 经营现金流 - 资本开支
FCF (2024) = 879.46 - ⚠️未找到资本开支明细
```

**注**：根据现金流量表，投资活动现金流为正（691.16），说明可能有投资收益回收。

### 9.2 净现金计算

```
狭义净现金 = 现金及等价物 - 有息负债
狭义净现金 (2024) = 873.53 - 7,420.47 = -6,546.94 百万元

广义净现金 = 广义现金合计 - 有息负债
广义净现金 (2024) = 873.53 - 7,420.47 = -6,546.94 百万元
```

**分析**：公司净负债约65.47亿元，净负债率 = 65.47 / 421.38 = 15.5%，负债水平较低。

### 9.3 净负债率

```
净负债 = 有息负债 - 现金及等价物
净负债 (2024) = 7,420.47 - 873.53 = 6,546.94 百万元

净负债率 = 净负债 / 归母股东权益
净负债率 (2024) = 6,546.94 / 42,137.95 = 15.5%
```

**分析**：净负债率15.5%，显著低于长江电力等同行，财务结构稳健。
