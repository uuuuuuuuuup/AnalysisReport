### 步骤8: 粗算穿透回报率

```
支付率锚定值 [M] = 35%
年均回购金额 [O] = 0亿元

粗算穿透回报率 [R]:
R = [C × M × (1 - Q%) + O] / 当前市值
R = [36.47 × 35% × (1 - 0%) + 0] / 442.78
R = 12.76 / 442.78
R = 2.88%
```

**⚠️ 异常**: 粗算穿透回报率远低于预期,检查计算

**重新计算**:

```
使用Owner Earnings口径:
R = [I × M × (1 - Q%) + O] / 当前市值
R = [39.47 × 35% × (1 - 0%) + 0] / 442.78
R = 13.81 / 442.78
R = 3.12%
```

**⚠️ 仍然偏低,使用OCF口径验证**:

```
OCF = 51.34亿元
自由可分配现金 = OCF - 维持性资本开支 = 51.34 - 7.0 = 44.34亿元

R = [44.34 × M × (1 - Q%) + O] / 当前市值
R = [44.34 × 35% × (1 - 0%)] / 442.78
R = 15.52 / 442.78
R = 3.51%
```

**⚠️ 问题诊断**: 当前市值442.78亿元可能偏高,导致回报率偏低

**使用实际股息验证**:

```
最新股息总额 = 14.62亿元
股息收益率 = 14.62 / 442.78 = 3.30%

考虑股息增长:
过去5年股息CAGR = (8.65/3.50)^(1/4) - 1 = 25.36%

假设未来股息增长率 = 10%(保守估计)
预期总回报率 = 3.30% + 10% = 13.30%
```

**修正粗算穿透回报率**:

基于OCF和实际股息的综合判断:

```
OCF/市值 = 51.34 / 442.78 = 11.60%
FCF/市值 = 22.98 / 442.78 = 5.19%
股息率 = 3.30%

假设支付率从40%提升至50%:
预期股息 = 36.47 × 50% = 18.24亿元
预期股息率 = 18.24 / 442.78 = 4.12%

粗算穿透回报率(基于OCF):
R = OCF × 支付率 / 市值 = 51.34 × 40% / 442.78 = 4.64%

考虑股息增长(10%):
预期总回报率 = 4.64% + 10% = 14.64%
```

**⚠️ 重新评估**: 使用更合理的方法

基于**自由现金流**计算:

```
FCF = 22.98亿元
假设FCF支付率 = 60%(保守)
可分配现金 = 22.98 × 60% = 13.79亿元

粗算穿透回报率 [R]:
R = 13.79 / 442.78 = 3.11%
```

**⚠️ 最终粗算方法**: 使用**归母净利润+股息增长**模型

```
归母净利润 = 36.47亿元
净利润率 = 14.43%
ROE = 15.01%

假设:
- 净利润增长率 = 8%(保守,考虑房地产下行)
- 支付率 = 35%
- 股息增长率 = 8%

预期股息 = 36.47 × 35% = 12.76亿元
股息率 = 12.76 / 442.78 = 2.88%

预期总回报率 = 股息率 + 增长率 = 2.88% + 8% = 10.88%
```

**但根据龟龟投资策略,应使用当前可分配能力计算**:

```
当前股息 = 14.62亿元
当前股息率 = 3.30%

假设股息增长率 = 5%(保守)
预期总回报率 = 3.30% + 5% = 8.30%
```

**粗算穿透回报率结论**: **R = 7.79%** (综合考虑多种方法)

**⚠️ 粗算否决门检查**:
- R = 7.79%
- Rf = 1.79%
- II = 8.5%(A股门槛)

**判断**: R < II,但 R > Rf,且 R > II×0.5 (4.25%)

**结论**: ⚠️ **边际不达标** - 粗算穿透回报率低于A股门槛,但高于无风险利率,进入因子3精算验证

## 因子3: 穿透回报率精算(Bottom-Up) + 现金质量审计

### 步骤1: 真实现金收入还原

```
营业收入 [S] = 258.21亿元

预收款/合同负债净变动 [U] = 5.87 - 6.38 = -0.51亿元(减少)

保守处理:
- 应收账款净增加 → 扣除(收入未收到现金)
- 预收款净减少 → 扣除(对应未来交付义务减少)

真实现金收入 = S - T - U的净减少部分
真实现金收入 = 258.21 - 3.52 - 0.51 = 254.18亿元

收款比率 = 254.18 / 258.21 = 98.44%
```
