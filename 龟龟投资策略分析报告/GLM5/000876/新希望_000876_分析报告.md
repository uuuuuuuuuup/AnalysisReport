## 附录：关键计算过程

### 精算穿透回报率计算

```
AA（自由现金流）= OCF - Capex = 91.27 - 10.87 = 80.40亿元

Q（股息税率）= 0%
O（回购金额）= 0亿元
当前市值 = 643.9亿元

GG = [AA × M × (1-Q%) + O] / 市值
GG = [80.40 × 30% × 1 + 0] / 643.9
GG = 24.12 / 643.9
GG = 3.75%
```

### 安全边际计算

```
II_adj（修正后门槛）= 2.5%
JJ（安全边际）= GG - II_adj = 3.75% - 2.5% = 1.25%
```

### 目标买入价计算

```
目标买入价_base = 市值 × (GG / II_adj) / 股本
目标买入价_base = 643.9 × 1.5 / 72.07 = 13.41元

现金保护折扣 = 30%
目标买入价 = 13.41 × 0.70 = 9.39元
```
