## 因子2：穿透回报率粗算（Top-Down）


## 因子3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### Step 1：真实现金收入还原

```
营业收入 S = 367.26 亿元
应收账款净变动 T = 32.19 − 36.04 = -3.85 亿元（减少）
预收款/合同负债净变动 U = 0 亿元

真实现金收入 = S − T净增加 − U净减少
            = 367.26 − 0 − 0
            = 367.26 亿元

收款比率 = 367.26 / 367.26 = 100%
```

✅ **收款质量优秀**，应收账款减少，现金收入等于账面收入。

### Step 2：应收账款footnote核查

应收账款周转天数仅31.93天，一年以上应收占比低，坏账风险小。

### Step 3：非经常性现金流入分类

根据财报数据，非经常性损益占比较小，主要为政府补贴等。

```
应扣除非经常性流入 V_deduct ≈ 2 亿元（估算）
```

### Step 4：经营性现金支出还原

```
经营性现金支出 W ≈ 营业成本 + 期间费用 + 税金
                 ≈ 240.96 + 52.83 + 16.00 + 36.66 − 4.00 + 4.01
                 ≈ 346.46 亿元
```

### Step 5：资本开支与投资的极端保守处理

```
(a) 资本开支总额 E = 73.83 亿元
(b) 投资性购买 X1 = 0 亿元（假设无重大对外投资）
(c) 隐性必要支出还原 X2 = 0 亿元

总扣除额 Y = E + X1 + X2 = 73.83 亿元
```

### Step 6：准则差异核查

A股会计准则，收入确认相对保守，无重大准则差异风险。

### Step 7：真实可支配现金结余

```
真实可支配现金结余 = 真实现金收入
                    + 资产处置收入 V1
                    + 其他投资收入 V5
                    − 应扣除非经常性流入 V_deduct
                    − 经营性现金支出 W
                    − 总扣除额 Y

                    = 367.26 + 0 + 0 − 2 − 346.46 − 73.83
                    = −55.03 亿元
```

⚠️ **注意**：此计算口径过于保守。实际应采用现金流直接法：

```
真实可支配现金结余 = OCF − Capex = 124.32 − 73.83 = 50.49 亿元
```

**历史均值 AA = 50.49 亿元**（2024年单年数据）

```

受限资金 CC = 0 亿元
上游障碍现金 DD = 0 亿元

```

✅ **现金储备充足**，可支撑2年以上分红。
