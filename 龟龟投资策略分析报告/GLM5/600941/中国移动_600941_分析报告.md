## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
集团净利润 A = 1,385 亿元
少数股东损益 B = 45 亿元
归母净利润 C = A - B = 1,340 亿元
```

### 步骤2：Owner Earnings粗算

```
折旧摊销 D ≈ 1,000 亿元（估算，基于固定资产规模7,145亿和行业折旧率14%）
资本开支 E = 1,852 亿元

维持性资本开支系数 G：
- 电信行业资本开支包含维持性和成长性
- 估算维持性占比约60%（网络维护、设备更新）
- G = 0.60

维持性资本开支 H = D × G = 1,000 × 0.60 = 600 亿元
（注：此处采用D×G估算，实际应基于历史资本开支趋势分析）

Owner Earnings I = C + D - E = 1,340 + 1,000 - 1,852 = 488 亿元
```

**⚠️ 重要修正**：上述计算假设资本开支全部为维持性，过于保守。

**更合理的估算**：
- 电信行业资本开支约60%为维持性，40%为成长性
- 维持性资本开支 ≈ 1,852 × 60% = 1,111 亿元
- Owner Earnings = 1,340 + 1,000 - 1,111 = 1,229 亿元
