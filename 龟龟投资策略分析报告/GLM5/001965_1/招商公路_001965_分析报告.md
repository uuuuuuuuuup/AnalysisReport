## 因子2:穿透回报率粗算(Top-Down)

### 步骤1:归母净利润

**GAAP归母净利润 C = 53.22亿元**(2024年报)

### 步骤2:Owner Earnings粗算

**公式**: Owner Earnings = C + D - H

**参数**:
- C(归母净利润) = 53.22亿元
- D(折旧摊销) = ⚠️数据缺失,需估算

**折旧摊销估算**:
- 根据行业经验,高速公路折旧摊销约占营业收入的15-20%
- 估算 D = 127.11 × 18% = 22.88亿元

**资本开支**:
- E(资本开支) = 16.88亿元(2024年投资现金流流出)

**维持性资本开支估算**:
- G(维持性资本开支系数) = 0.6(高速公路行业经验值)
- H(维持性资本开支) = D × G = 22.88 × 0.6 = 13.73亿元

**Owner Earnings**:
- I = C + D - H = 53.22 + 22.88 - 13.73 = **62.37亿元**

### 步骤3:分配能力验证

**股息支付能力**:
- 2024年股息支付:28.36亿元
- Owner Earnings:62.37亿元
- **股息支付率 = 28.36 / 62.37 = 45.48%**

**结论**: ✅ **分配能力充足**,股息支付率低于Owner Earnings,可持续

### 步骤4:现金上游障碍

**现金回流情况**:
- 经营现金流:71.21亿元
- Owner Earnings:62.37亿元
- **现金回流比率 = 71.21 / 62.37 = 114.2%**

**结论**: ✅ **无现金上游障碍**,经营现金流超过Owner Earnings

### 步骤5:分配意愿

**分红历史**:
- 2018-2024年分红比例:40.13% → 53.44%
- 分红比例持续提升
- 未来三年股东回报规划:现金分红比例不低于55%

**结论**: ✅ **分配意愿强**,分红政策稳定且持续提升

### 步骤6:可预测性

**业绩稳定性**:
- 2020-2024年营收复合增长率约15%
- 净利润波动较大,但扣非净利润相对稳定
