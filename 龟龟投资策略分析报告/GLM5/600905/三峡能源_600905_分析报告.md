## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤1：归母净利润

```
GAAP归母净利润 [C] = 61.11亿元
```

#### 步骤2：Owner Earnings粗算

```
折旧摊销 [D] = 120亿元（估算，根据行业特征）
资本开支 [E] = 316.68亿元
维持性资本开支系数 [G] = 0.3（估算，新能源行业特征）
维持性资本开支 [H] = D × G = 120 × 0.3 = 36亿元
Owner Earnings [I] = C + D - H = 61.11 + 120 - 36 = 145.11亿元
```

#### 步骤3：分配能力验证

```
支付率锚定值 [M] = 31.37%
可分配金额 = I × M = 145.11 × 31.37% = 45.53亿元
最新股息总额 = 19.17亿元

判断：可分配金额 > 股息总额 ✓
结论：分配能力验证通过
```

#### 步骤4：现金上游障碍

- 应收账款主要来自国家补贴，回收周期长
- 但信用风险低，最终回收概率高
- **判断**：存在上游障碍，但不影响最终分配能力

#### 步骤5：分配意愿

- 公司章程规定分红比例不低于30%
- 2024年分红率31.37%，符合承诺
- 控股股东增持，体现信心
- **判断**：分配意愿强

#### 步骤6：可预测性

- 新能源发电行业，收入相对稳定
- 但电价市场化带来不确定性
- **判断**：可预测性中等

#### 步骤7：税务泄漏修正

```
A股持股超过1年免征股息税
Q = 0%
```

#### 步骤8：粗算穿透回报率

```
R = (I × M × (1-Q) + O) / 市值
R = (145.11 × 31.37% × 1 + 0) / 1282.94
R = 45.53 / 1282.94
R = 3.55%
```

### 粗算否决门检查

| 检查项 | 结果 | 说明 |
|--------|------|------|
| R > Rf | ✓ | 3.55% > 1.79% |
| R > II × 0.5 | ✓ | 3.55% > 1.75% |
| R ≥ II | ✓ | 3.55% ≥ 3.50% |

### 结论

✅ **粗算穿透回报率3.55%，通过粗算否决门**
