## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
C = 55.17亿元（2024年度归母净利润）
```

### 步骤2：Owner Earnings粗算

由于缺乏折旧摊销数据，采用自由现金流法估算：

```
Owner Earnings ≈ FCF + 利息支出
FCF = 29.95亿元
利息支出 ≈ 0（有息负债仅0.42亿元）
Owner Earnings ≈ 29.95亿元
```

**保守调整**：考虑到白酒行业资本开支的必要性，采用更保守口径：

```
Owner Earnings = C × 70% = 55.17 × 70% = 38.62亿元
```

### 步骤3：分配能力验证

- **2024年分红**：31.72亿元
- **Owner Earnings**：38.62亿元
- **分红/Owner Earnings**：82.14%
- **评估**：分红能力有保障，但分红率较高

### 步骤4：现金上游障碍

- **现金储备**：158.94亿元
- **有息负债**：0.42亿元
- **净现金**：158.52亿元
- **上游障碍**：无（现金充裕）

### 步骤5：分配意愿

- **分红率**：57.49%
- **分红历史**：持续强化分红力度
- **分配意愿评级**：**强**

### 步骤6：可预测性

- **营收波动**：2024年+16.41%，2025Q1-Q3 -14%
- **利润波动**：2024年+20.22%，2025Q1-Q3 -17%
- **可预测性评级**：**中等**（行业调整期）

### 步骤7：税务泄漏修正

- **持股渠道**：A股直接持有
- **股息税率**：10%（持股超1年免税，暂按10%计算）
- **综合股息税率 Q**：10%

### 步骤8：粗算穿透回报率R%

```
R = [C × M × (1 - Q%) + O] / 当前市值
R = [55.17 × 57.49% × (1 - 10%) + 0] / 592.67
R = [55.17 × 0.5749 × 0.9] / 592.67
R = 28.54 / 592.67
R = 4.81%
```

**粗算结果**：
- **R = 4.81%**
- **门槛 II = 3.50%**
- **R > II**，粗算通过

### 粗算否决门检查

- R (4.81%) > Rf (1.79%) ✅
- R (4.81%) > II × 0.5 (1.75%) ✅
- R (4.81%) > II (3.50%) ✅

**粗算结论**：**通过**，继续进入因子3精算。

## 因子3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤1：真实现金收入还原

```
营业收入 S = 235.78亿元
应收账款净变动 T = 0.70 - 0.72 = -0.02亿元（减少）
预收款/合同负债净变动 U = ⚠️未找到

保守处理：
- 应收账款净减少 → 加回（收入已收到现金）
- 预收款净变动 → 暂按0计算

真实现金收入 = S - T的净增加部分 - U的净减少部分
真实现金收入 = 235.78 - 0 - 0 = 235.78亿元

收款比率 = 235.78 / 235.78 = 100%
```

**评估**：收款比率100%，收款质量优秀。
