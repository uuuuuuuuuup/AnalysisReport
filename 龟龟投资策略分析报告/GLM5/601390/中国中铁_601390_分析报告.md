### 模块一：资本消耗强度

**分析**：

基建行业属于重资产、高资本消耗行业。根据现金流量表数据：

```
2024年资本开支 = OCF - FCF = 280.51 - (-36.33) = 316.84 亿元
资本开支/净利润 = 316.84 / 307.58 = 103%
资本开支/归母净利润 = 316.84 / 152.15 = 208%
```

**维持性资本开支估算**：

```
折旧摊销 [D] ≈ 150 亿元（估算）
维持性资本开支系数 [G] = 0.7（基建行业特性）
维持性资本开支 [H] = D × G = 150 × 0.7 = 105 亿元
```

**结论**：

- 资本消耗强度：**高**
- 资本开支超过净利润，说明公司需要持续投入大量资本维持运营
- 这是基建行业的典型特征，但也意味着股东可分配现金有限


### 粗算否决门判断

```
门槛 II = 3.79%
粗算穿透回报率 R = 3.05%

判断：
- R < II → 检查 EV 口径
```

**EV 口径计算**：

```
广义净现金 = -755.84 亿元（净负债）
EV = 当前市值 − 广义净现金 = 861 - (-755.84) = 1,616.84 亿元

EV 口径穿透回报率 R_EV = [C × M × (1 − Q%) + O] / EV
                       = 26.27 / 1,616.84
                       = 1.62%
```

**结论**：

- 市值口径 R = 3.05% < II = 3.79% ❌
- EV 口径 R_EV = 1.62% < II = 3.79% ❌

**⚠️ 双口径均未达标，但差距不大，继续精算分析**
