```

```

**因子2结论**: ⚠️ **边际不达标**

| 5 | 管理层损害价值 | 否 | 无明显证据 |

**陷阱风险项数 N = 2** → 风险**中**

### 步骤3：安全边际与仓位

```
GG = 0.35%
II = 3.50%

安全边际 JJ = GG - II = 0.35% - 3.50% = -3.15%

修正后安全边际 KK = -3.15%（无周期性调整）
```

**现金保护等级**:

```
广义净现金/市值 = 453.92 / 1713.22 = 26.5%

→ 轻度保护（25%折扣）
```

**仓位矩阵判定**:

| 条件 | 结果 |
|------|------|
| 安全边际 < 0 | **不建仓** |
| 外推可信度 = 中 | - |
| 陷阱风险 = 中 | - |

**结论**: 安全边际为负(-3.15%)，**不建仓**。

### 步骤4：股价位置与买入触发价评估

#### Step 4-1：目标买入价反推

```
基准目标买入价（市值口径）：
目标买入价_base = 当前市值 × (GG / II) / 总股本
                = 1713.22 × (0.35% / 3.50%) / 54.22
                = 1713.22 × 0.1 / 54.22
                = 3.16元

现金保护调整后目标买入价：
目标买入价 = 3.16 × 0.75 = 2.37元
```

**判断**: 目标买入价(2.37元) << 当前股价(31.60元)，需等待大幅回调。

#### Step 4-2：历史价格数据

#### Step 4-3：分位计算与买入触发评估
