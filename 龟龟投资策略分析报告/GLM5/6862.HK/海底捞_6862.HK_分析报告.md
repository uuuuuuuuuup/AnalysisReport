## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

**Step 1：归母净利润**
```
C = 47.00 亿元
```

**Step 2：Owner Earnings粗算**
```
D = 折旧摊销 = 25.60 亿元
E = 资本开支 = 15.00 亿元
G = 维持性资本开支系数 = 0.6（估算）
H = D × G = 25.60 × 0.6 = 15.36 亿元
I = C + D - H = 47.00 + 25.60 - 15.36 = 57.24 亿元
```

**Step 3：分配能力验证**
```
经营现金流 ≈ 65 亿元
资本开支 = 15 亿元
自由现金流 ≈ 50 亿元
股息支付 = 44.68 亿元
自由现金流 > 股息支付 ✅
```

**Step 4：现金上游障碍**
```
主要运营主体为境内子公司
分红需从境内汇出至香港
港股通股息税率 = 20%
直接持有港股股息税率 = 0%（香港不征股息税）
```

**Step 5：分配意愿**
```
支付率 = 95%（2024年）
过去3年支付率序列 = [42.7%, 90%, 95%]
分配意愿 = 强
```

**Step 6：可预测性**
```
收入波动率（5年CV）= 约15%
利润波动性 = 较高（2021年亏损）
外推可信度 = 中
```

**Step 7：税务泄漏修正**
```
港股通持股：综合股息税率 Q = 20%
直接持有港股：Q = 0%
本分析假设港股通持股，Q = 20%
```

**Step 8：粗算穿透回报率**
```
M = 支付率锚定值 = 95%
Q = 20%
O = 0（无回购）

R = [C × M × (1 - Q%) + O] / 当前市值
R = [47.00 × 95% × 0.8 + 0] / 770
R = 35.72 / 770
R = 4.64%

使用Owner Earnings口径：
R_OE = [I × M × (1 - Q%) + O] / 当前市值
R_OE = [57.24 × 95% × 0.8] / 770
R_OE = 43.50 / 770
R_OE = 5.65%
```

### 粗算结论

| 口径 | 穿透回报率 | vs 门槛 | 判断 |
|------|------------|---------|------|
| 净利润口径 | 4.64% | < 5.0% | 不达标 |
| Owner Earnings口径 | 5.65% | > 5.0% | 达标 |

**粗算否决门判断**：
- R < Rf？否（4.64% > 1.85%）
- R < II×0.5？否（4.64% > 2.5%）
- II×0.5 ≤ R < II？是（2.5% < 4.64% < 5.0%）

**结论**：标注"边际不达标"，继续进入因子3精算。
