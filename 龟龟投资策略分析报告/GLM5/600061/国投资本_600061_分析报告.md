### 模块一：资本消耗强度

**行业特征**: 金融控股平台，资本消耗强度中等

**分析**:
- **资本开支**: 2024年资本开支仅2.89亿元，占营业收入1.92%
- **资本开支/净利润**: 2.89 / 26.94 = 10.7%，资本消耗强度较低
- **维持性资本开支**: 金融行业主要为IT系统投入、办公设备等，维持性资本开支占比较低

**结论**: 资本消耗强度**低**，有利于现金留存

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```

```

### 步骤2：Owner Earnings粗算

```
折旧摊销 [D] = ⚠️未找到，假设为0（金融行业折旧摊销较少）
资本开支 [E] = 2.89亿元
维持性资本开支系数 [G] = 0.3（金融行业假设）
维持性资本开支粗估 [H] = D × G = 0
Owner Earnings [I] = C + D - H = 26.94 + 0 - 0 = 26.94亿元
```

```

支付率 = 0.88 / 26.94 = 3.27%

⚠️ 分配能力否决门检查：
- 支付率 < 20%？ ✅ 是（3.27% << 20%）
- 支付率 < 10%？ ✅ 是（3.27% << 10%）
- 支付率历史序列是否稳定？ ⚠️ 仅获取2023-2024年数据，均为极低水平

```

```
粗算否决门检查：
- R < Rf？ ✅ 是（1.69% < 1.79%）
- R < II × 0.5？ ✅ 是（1.69% < 1.75%）

结论：❌ 粗算穿透回报率远低于门槛，触发粗算否决门
```

---

## 因子 2 结论

**❌ 否决 - 分配能力严重不足**

国投资本虽然盈利能力尚可（2024年归母净利润26.94亿元，ROE 5.17%），但存在以下致命缺陷：

4. **无回购计划**: 公司无股份回购计划，无法通过回购提升股东回报

### A. 粗算穿透回报率计算

```
公式：R = [C × M × (1 - Q%) + O] / 当前市值

参数：
- C（归母净利润）= 26.94亿元
- M（支付率）= 3.27%
- Q（股息税率）= 10%
- O（回购金额）= 0亿元
- 当前市值 = 46.9亿元

计算：
R = [26.94 × 3.27% × (1 - 10%) + 0] / 46.9
  = [26.94 × 0.0327 × 0.9 + 0] / 46.9
  = 0.793 / 46.9
  = 1.69%

结论：粗算穿透回报率 = 1.69%
```

### B. 安全边际计算

```
公式：安全边际 = 粗算穿透回报率 - 门槛值

参数：
- 粗算穿透回报率 = 1.69%
- 门槛值（A股）= 3.50%

计算：
安全边际 = 1.69% - 3.50% = -1.81%

结论：安全边际为负值，不满足投资要求
```
