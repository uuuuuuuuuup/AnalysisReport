## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 1. 归母净利润

```
2024年全年归母净利润 C = 25.83亿元
2025年前三季度归母净利润 C ≈ 10.60亿元（估算）
```

#### 2. Owner Earnings粗算

```
由于缺乏折旧摊销数据，使用简化公式：

Owner Earnings ≈ 净利润 + 折旧摊销 − 维持性资本开支

假设：
- 折旧摊销 D ≈ 资本开支 × 50% = 2.75 × 0.5 = 1.38亿元（估算）
- 维持性资本开支系数 G = 40%
- 维持性资本开支 H = D × G = 1.38 × 0.4 = 0.55亿元

Owner Earnings I = C + D − H
                = 10.60 + 1.38 − 0.55
                = 11.43亿元（2025年前三季度估算）

年化Owner Earnings ≈ 11.43 × 4/3 = 15.24亿元
```

#### 3. 分配能力验证（一票否决）

```
2024年经营现金流 OCF = 31.04亿元
2024年自由现金流 FCF = 28.63亿元
2024年股息支付 = 10.47亿元

FCF / 股息 = 28.63 / 10.47 = 2.73倍

✅ 通过：FCF远大于股息，分配能力充足
```

#### 4. 现金上游障碍

```
现金及等价物 = 32.75亿元
有息负债 = 12.42亿元
净现金 = 20.33亿元

判断：净现金充足，无上游障碍
```

#### 5. 分配意愿

```
公司承诺2023-2025年每年至少40%利润分红
2024年支付率 = 40.55%，符合承诺

判断：分配意愿强
```

#### 6. 可预测性

```
收入波动：2024年-7.55%，2025年前三季度-5.60%
利润波动：2024年-43.31%，2025年前三季度-63.73%

判断：利润波动极大，可预测性差
```

#### 7. 税务泄漏修正

```
A股股息税率 Q = 10%（个人投资者）
税后Owner Earnings = I × (1 − Q%) = 15.24 × 0.9 = 13.72亿元
```

#### 8. 粗算穿透回报率R%

```
当前市值 = 363.43亿元
支付率锚定值 M = 40%
年均回购金额 O = 1.0亿元

粗算穿透回报率 R = [C × M × (1 − Q%) + O] / 当前市值
                 = [25.83 × 0.4 × 0.9 + 1.0] / 363.43
                 = [9.30 + 1.0] / 363.43
                 = 10.30 / 363.43
                 = 2.83%

使用年化Owner Earnings：
R = [15.24 × 0.4 × 0.9 + 1.0] / 363.43
  = [5.49 + 1.0] / 363.43
  = 6.49 / 363.43
  = 1.79%
```

### 粗算否决门

```
门槛 II = 3.79%（A股）
R = 2.83%（基于2024年净利润）
R = 1.79%（基于年化Owner Earnings）

判断：
- R < II → 检查EV口径
- R < II × 0.5 = 1.90% → 若基于年化Owner Earnings，则接近否决线
```

**结论：❌ 粗算穿透回报率未达标，进入因子3精算验证**
