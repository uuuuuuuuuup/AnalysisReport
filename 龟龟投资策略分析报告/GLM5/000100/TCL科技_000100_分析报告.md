## 因子 2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

**2024年归母净利润 [C] = -41.56亿元**

⚠️ **严重问题**：归母净利润为负，无法进行正常的Owner Earnings计算

### 步骤2：Owner Earnings粗算

**公式**：Owner Earnings = 归母净利润 + 折旧摊销 - 维持性资本开支

**估算**：
- 归母净利润 = -41.56亿元
- 折旧摊销 ≈ 200亿元（基于OCF与净利润差异估算）
- 维持性资本开支 ≈ 150亿元（估算，约占固定资产的8-10%）

**Owner Earnings ≈ -41.56 + 200 - 150 = 8.44亿元**

⚠️ **估算不确定性极高**：亏损年份的Owner Earnings计算参考价值有限

### 步骤3：分配能力验证

**2024年股息支付**：10.4亿元

**问题**：
- 亏损年份仍派息10.4亿元
- 股息来源于历年累积利润或融资
- **分配能力存疑**

### 步骤4：现金上游障碍

- 但子公司众多，可能存在资金归集效率问题

### 步骤5：分配意愿

**评估**：🟡 **中等意愿**

**分析**：
- 2024年亏损仍派息，体现一定股东回报意愿
- 但DPS从0.80降至0.50元/10股，派息下降
- 支付率波动大，缺乏稳定政策

### 步骤6：可预测性

**评估**：🔴 **低可预测性**

**分析**：
- 强周期行业，盈利波动剧烈
- 2024年亏损，2025年前三季度已扭亏
- 未来盈利难以预测

### 步骤7：税务泄漏修正

**A股股息税率**：
- 持股>1年：免税
- 综合税率 [Q] = 0%

### 步骤8：粗算穿透回报率R%

**公式**：R = (Owner Earnings × 支付率 × (1-Q%) + 回购) / 市值

**计算**：
```
R = (8.44亿 × 25% × 100% + 0) / 1017亿
R = 2.11 / 1017
R = 0.21%
```

**⚠️ 严重问题**：
- 粗算穿透回报率仅**0.21%**
- 远低于无风险利率1.79%
- 远低于门槛3.5%

### 粗算否决门判断

| 条件 | 结果 | 判断 |
|------|------|------|
| R < Rf (1.79%) | 0.21% < 1.79% | ✅ 是 |
| R < II×0.5 (1.75%) | 0.21% < 1.75% | ✅ 是 |

**结论**：❌ **触发粗算否决门**

**但根据策略**：需检查EV口径

## 因子 4：估值与安全边际

### 步骤1：门槛计算

**A股门槛**：
```
II = max(3.5%, Rf + 2%)
II = max(3.5%, 1.79% + 2%)
II = max(3.5%, 3.79%)
II = 3.79%
```

**周期性修正**：
- 强周期底部 → 门槛下调1pct
- 修正后门槛 = 3.79% - 1% = **2.79%**

**⚠️ 注意**：即使下调后，穿透回报率0.21%仍远低于门槛

### 步骤2：价值陷阱排查

| 序号 | 陷阱特征 | 判断 | 结果 |
|------|---------|------|------|
| 1 | 现金流趋势性恶化 | OCF 295亿，但FCF -584亿 | ✅ 是 |
| 2 | 护城河正在收窄 | 技术迭代压力，竞争加剧 | 🟡 可能 |
| 3 | 行业结构性衰退 | 面板需求长期存在，周期性 | ✅ 否 |
| 4 | 分配意愿存疑 | 亏损仍派息，但DPS下降 | 🟡 中等 |
| 5 | 管理层损害价值 | 创始人持股，治理良好 | ✅ 否 |

**陷阱风险项数**：N = 2

**判断**：🟡 **中等风险**

### 步骤3：安全边际与仓位

**决策口径**：市值口径（GG = 0.21%）

**安全边际**：
```
JJ = GG - II
JJ = 0.21% - 2.79%
JJ = -2.58%
```

**结论**：❌ **安全边际为负**

**现金保护等级**：
```
广义净现金/市值 = -969.92 / 1017 = -95.4%
```

**判断**：🔴 **无保护**（净现金为负）

**仓位决策**：
- 安全边际<0 → **不建仓**

### 步骤4：股价位置与买入触发价评估

#### Step 4-1：目标买入价反推

**基准目标买入价**：
```
目标买入价_base = 当前市值 × (GG / II) / 总股本
                = 1017 × (0.21% / 2.79%) / 208
                = 1017 × 0.0753 / 208
                = 0.37元
```

**⚠️ 问题**：目标买入价远低于当前股价，说明当前估值过高

**现金保护调整**：无保护，折扣30%
```
目标买入价 = 0.37 × 0.70 = 0.26元
```

**结论**：目标买入价0.26元，远低于当前股价4.89元
