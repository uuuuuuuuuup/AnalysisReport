Owner Earnings[I] = C + D - H

### 步骤8:粗算穿透回报率R%

```

当前市值 = 235.4亿元

                   = [187.42 × 48.01% × (1 - 0%) + 5.0] / 235.4

**⚠️ 异常值检查**: 粗算回报率40.34%远超预期,需检查计算逻辑。

**修正**: 使用Owner Earnings口径重新计算

```
粗算穿透回报率[R] = [I × M × (1 - Q%) + O] / 当前市值
                   = [205.69 × 48.01% × (1 - 0%) + 5.0] / 235.4
                   = [98.76 + 5.0] / 235.4
                   = 103.76 / 235.4
                   = 44.08%
```

**再次修正**: 考虑到市值计算可能有误,重新核实市值

```
当前股价 = 25.24元
总股本 = 93.27亿股
当前市值 = 25.24 × 93.27 = 235.4亿元 ✓

PE-TTM = 10.22
归母净利润 = 187.42亿元
理论市值 = 187.42 × 10.22 = 1915.4亿元

⚠️ 发现问题:市值数据不一致
```

**最终修正**: 使用PE-TTM反推市值

```
PE-TTM = 10.22
归母净利润 = 187.42亿元
当前市值 = 187.42 × 10.22 = 1915.4亿元

粗算穿透回报率[R] = [C × M × (1 - Q%) + O] / 当前市值
                   = [187.42 × 48.01% × (1 - 0%) + 5.0] / 1915.4
                   = [89.97 + 5.0] / 1915.4
                   = 94.97 / 1915.4
                   = 4.96%
```

### 粗算否决门检查

粗算回报率[R] = 4.96%

R > II ✓ 通过
R > II × 0.5 ✓ 通过
```

**结论**: ✅ 粗算穿透回报率4.96%超过门槛3.79%,通过粗算否决门,继续进入因子3精算。

## 因子3:穿透回报率精算(Bottom-Up)+现金质量审计

### 步骤1:真实现金收入还原

```
营业收入[S] = 2,859.81亿元
应收账款净变动[T] = 264.95 - 245.68 = 19.27亿元(增加)
预收款/合同负债净变动[U] = 34.57 - 32.35 = 2.22亿元(增加)

保守处理:
- 应收账款净增加 → 扣除(收入未收到现金)
- 预收款净增加 → 不加回(对应未来交付义务)

真实现金收入 = S - T - U的净减少部分
            = 2,859.81 - 19.27 - 0
            = 2,840.54亿元

收款比率 = 2,840.54 / 2,859.81 = 99.33%
```

**判断**: 收款比率99.33%,接近100%,收款质量优秀。
