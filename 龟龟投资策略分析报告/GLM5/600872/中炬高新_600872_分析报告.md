## 因子2：穿透回报率粗算（Top-Down）

### 计算步骤

#### Step 1：归母净利润

```
C = 8.93 亿元（2024年归母净利润）
```

#### Step 2：Owner Earnings粗算

```
D = 折旧摊销 ≈ 1.5 亿元（估算）
E = 资本开支 ≈ 6.0 亿元
G = 维持性资本开支系数 ≈ 0.5（调味品行业估算）
H = D × G = 1.5 × 0.5 = 0.75 亿元

Owner Earnings [I] = C + D - H = 8.93 + 1.5 - 0.75 = 9.68 亿元
```

#### Step 3：分配能力验证

```
经营现金流 = 10.88 亿元
资本开支 = 6.0 亿元

```

#### Step 4：现金上游障碍

```

广义净现金 = 1.92 亿元

上游障碍 = 无（净现金为正）
```

#### Step 5：分配意愿

```
历史支付率序列 = [35%, 36%, 36%]
支付率均值 M = 35%
分配意愿 = 稳定（连续分红）
```

#### Step 6：可预测性

```
收入波动率（5年CV）= 约15%（中等）
利润波动率 = 较大（2025年下滑明显）
可预测性 = 中等偏低
```

#### Step 7：税务泄漏修正

```
Q = 综合股息税率 = 10%（A股持股超1年免税，假设平均税率）
```

#### Step 8：粗算穿透回报率

```
M = 支付率锚定值 = 35%
O = 年均回购金额 = 0.5 亿元
当前市值 = 142 亿元

R = [C × M × (1 - Q%) + O] / 当前市值
R = [8.93 × 0.35 × 0.9 + 0.5] / 142
R = [2.81 + 0.5] / 142
R = 3.31 / 142
R = 2.33%

使用Owner Earnings口径：
R = [I × M × (1 - Q%) + O] / 当前市值
R = [9.68 × 0.35 × 0.9 + 0.5] / 142
R = [3.05 + 0.5] / 142
R = 3.55 / 142
R = 2.50%
```

**调整说明**：考虑到2024年净利润包含非经常性收益（资产处置等），使用调整后净利润估算：

```
调整后归母净利润 ≈ 7.5 亿元（剔除一次性收益）
R_adjusted = [7.5 × 0.35 × 0.9 + 0.5] / 142 = 2.38 / 142 = 1.68%
```

**粗算穿透回报率 R = 2.33% ~ 2.50%**（取中值约2.4%）

### 粗算否决门判断

```
门槛 II = 3.78%
R = 2.4%

R < II → 未达标
R < II × 0.5 = 1.89%? → 否，R > 1.89%
```

**结论**：粗算穿透回报率**未达标**，但未触发否决门，继续进入因子3精算
