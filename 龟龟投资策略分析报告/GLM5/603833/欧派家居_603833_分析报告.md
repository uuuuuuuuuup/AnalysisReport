```
辅轨（EV口径）：不适用（广义净现金/市值 = 24.2% < 40%）
```

### Step 2：价值陷阱排查

| 序号 | 陷阱特征 | 判断标准 | 结果 |
|------|----------|----------|------|
| 1 | 现金流趋势性恶化 | OCF连续下降？ | ⚠️ 需关注（2025Q3 OCF下降） |
| 2 | 护城河正在收窄 | 竞争加剧？ | ⚠️ 是（行业存量博弈） |
| 3 | 行业结构性衰退 | 终端需求萎缩？ | ⚠️ 是（房地产低迷） |
| 4 | 分配意愿存疑 | 分红可持续？ | ✅ 否（支付率提升） |
| 5 | 管理层损害价值 | 利益输送？ | ⚠️ 需关注（高分红质疑） |

**陷阱风险项数**：N = 3（风险高）

**判断**：
- N = 3 ≥ 2 → 风险高
- GG = 6.05% > II × 1.5 = 5.69% → 保留（回报率足够覆盖风险）

### Step 3：安全边际与仓位

```
决策口径：市值口径
[GG] = 6.05%（使用OCF-资本开支口径更保守，取3.59%）
[II] = 3.79%

安全边际 [JJ] = GG - II = 3.59% - 3.79% = -0.20pct

周期性修正：
- 当前处于强周期底部 → 门槛下调1pct
- 修正后门槛 = 3.79% - 1% = 2.79%
- 修正后安全边际 [KK] = 3.59% - 2.79% = +0.80pct

现金保护等级判定：
- 广义净现金/市值 = 24.2% → 轻度保护（25%折扣）
```

### 仓位矩阵判断

| 条件 | 判断 | 仓位建议 |
|------|------|----------|

| 外推可信度中？ | 是 | - |
| 陷阱风险高？ | 是 | - |

**综合判断**：
- 安全边际0.80pct（修正后）
- 外推可信度中
- 陷阱风险高

**仓位建议**：**观察仓位（30%）或等待更好价格**

#### Step 4-1：目标买入价反推
