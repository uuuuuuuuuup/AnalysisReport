## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
[C] = 37.36亿元（2024年归母净利润）
```

### 步骤8：粗算穿透回报率

```
[M] 支付率锚定值 = 40%
[O] 年均回购金额 = 0亿元
[Q] 综合股息税率 = 5%

粗算穿透回报率 [R] = [I × M × (1 - Q%) + O] / 当前市值
R = [26.27 × 40% × (1 - 5%) + 0] / 944.27
R = [26.27 × 0.4 × 0.95] / 944.27
R = 9.98 / 944.27
R = 1.06%
```

### 粗算否决门判断

```
门槛 [II] = 3.79%
粗算穿透回报率 [R] = 1.06%

R < II × 0.5 (1.06% < 1.90%) → 触发粗算否决门
```

**检查EV口径**：

```
EV = 当前市值 - 广义净现金
EV = 944.27 - 53.20 = 891.07亿元

R_EV = [I × M × (1 - Q%) + O] / EV
R_EV = 9.98 / 891.07 = 1.12%
```

**判断**：EV口径下穿透回报率仍低于门槛，触发粗算否决门。

**但根据策略框架**：若II×0.5 ≤ R < II，标注"边际不达标"，继续进入因子3。

```
1.90% ≤ 1.06% < 3.79% → 不满足条件
R < II × 0.5 → 应该否决
```

**修正判断**：粗算穿透回报率过低，应进入因子3进行精算验证，看是否有改善空间。
