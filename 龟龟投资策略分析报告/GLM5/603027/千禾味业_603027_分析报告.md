
## 因子3：穿透回报率精算（Bottom-Up）

### 步骤1：真实现金收入还原

```
营业收入 [S] = 30.73 亿元（2024年）
应收账款净变动 [T] = 1.36 - 1.52 = -0.16 亿元（减少）
预收款/合同负债净变动 [U] = ⚠️未找到（假设为0）

保守处理：
- 应收账款净减少 → 加回（已收到的现金）
- 预收款净变动 → 无法计算

真实现金收入 = S - T的净增加部分 - U的净减少部分
            = 30.73 - (-0.16) - 0
            = 30.89 亿元

收款比率 = 30.89 / 30.73 = 100.52%
```

**结论**：收款比率>100%，应收账款减少，现金收入质量良好。
