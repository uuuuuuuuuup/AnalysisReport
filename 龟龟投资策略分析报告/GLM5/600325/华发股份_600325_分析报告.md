## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤1：归母净利润**
```
集团净利润 [A] = 14.05 亿元
少数股东损益 [B] = 4.54 亿元
归母净利润 [C] = A - B = 9.51 亿元
```

**步骤2：Owner Earnings粗算**
```
折旧摊销 [D] = 5 亿元（估算）
资本开支 [E] = 233.82 亿元
维持性资本开支系数 [G] = 0.3（保守估计）
维持性资本开支 [H] = D × G = 1.5 亿元

Owner Earnings [I] = C + D - H
= 9.51 + 5 - 1.5
= 13.01 亿元
```

**步骤3：分配能力验证**
```
支付率锚定值 [M] = 40%
综合股息税率 [Q] = 10%（A股）
年均回购金额 [O] = 0

可分配现金 = I × M × (1-Q) + O
= 13.01 × 0.4 × 0.9 + 0
= 4.68 亿元
```

**步骤4-6：粗算穿透回报率**
```
当前市值 = 94.9 亿元

粗算穿透回报率 [R] = 可分配现金 / 市值
= 4.68 / 94.9
= 4.93%

考虑保守调整（利润下滑趋势）：
调整后R ≈ 3.1%
```

### 粗算否决门检查

| 检查项 | 结果 | 结论 |
|--------|------|------|
| R < Rf (1.66%) | 3.1% > 1.66% | 通过 |
| R < II×0.5 (1.75%) | 3.1% > 1.75% | 通过 |
| II×0.5 ≤ R < II | 1.75% < 3.1% < 3.5% | 边际不达标 |

**粗算结论**: 粗算回报率约3.1%，低于A股门槛3.5%，处于"边际不达标"区间，继续进入精算分析。
