## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
[C] = GAAP归母净利润 = 21.59亿元
```

**来源**：data_pack_market.md §2.1

### 步骤2：Owner Earnings粗算

**公式**：
```
Owner Earnings [I] = C + D − H
其中：
  D = 折旧摊销
  H = 维持性资本开支粗估 = D × G
  G = 维持性资本开支系数（默认70%）
```

**数据缺失处理**：
- 折旧摊销 [D]：⚠️未找到，需从财报附注获取
- 假设折旧摊销占营收比例约3%（行业经验值）
- D ≈ 209.57亿 × 3% = 6.29亿元

**计算**：
```
H = D × G = 6.29亿 × 70% = 4.40亿元
I = C + D − H = 21.59亿 + 6.29亿 − 4.40亿 = 23.48亿元
```

### 步骤3：分配能力验证（一票否决）

**检查项**：
1. **经营现金流 > 净利润**？
   - 2024年OCF = 23.17亿，净利润 = 21.89亿
   - **是** ✓（OCF/净利润 = 105.9%）

2. **自由现金流 > 0**？
