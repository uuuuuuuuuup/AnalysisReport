### 模块三:竞争格局与护城河

**分析结论**: **护城河极深,垄断地位稳固**

#### 非技术性护城河

**1. 网络效应**:
- 电梯点位规模: 分众拥有约300万+电梯点位
- 覆盖范围: 一二线城市核心楼宇,形成规模效应
- 广告主价值: 点位越多,触达受众越广,广告效果越好

**2. 品牌认知**:
- "分众传媒"已成为电梯媒体代名词
- 广告主心智占领,品牌认知度高

**3. 转换成本**:
- 物业公司合作长期稳定,更换成本高
- 广告主投放习惯形成,转换成本中等

#### 技术性护城河

**1. 点位资源垄断**:
- 市场份额: 74% (2024年) → 88% (收购新潮后预估)
- 点位获取: 与物业公司长期合作,新进入者难以获取优质点位

**2. 数字化能力**:
- 云化投放系统,实时监测广告效果
- 数据中台,精准投放能力

**3. 规模经济**:
- 固定成本摊薄,单位点位成本低于竞争对手
- 销售费用率: 18.5% (2024年),低于行业平均

**竞争格局评估**:
- 现有竞争者: 弱 (市场高度集中,分众为绝对龙头)
- 新进入者威胁: 低 (电梯资源获取困难)
- 替代品威胁: 中 (短视频、直播等分流广告预算,但电梯媒体不可替代)
- 上游议价能力: 低 (物业公司数量众多且分散)
- 下游议价能力: 中 (大型客户议价能力较强)

**数据来源**: data_pack_market §8

## 因子2:穿透回报率粗算(Top-Down)

### 步骤1:归母净利润

```
C = GAAP归母净利润 = 50.71亿元 (2024年)
```

### 步骤2:Owner Earnings粗算

```
D = 折旧摊销 ≈ 2.0亿元 (估算)
E = 资本开支 = 29.17亿元 (2024年,含投资性支出)
G = 维持性资本开支系数 = 0.3 (估算,轻资产模式)
H = D × G = 2.0 × 0.3 = 0.6亿元
I = C + D - H = 50.71 + 2.0 - 0.6 = 52.11亿元
```

**Owner Earnings = 52.11亿元**

