## Executive Summary

### 核心结论

**投资评级**: ⚠️ **观察仓位 / 暂不建仓**

**关键发现**:
1. ✅ **财务健康**: 现金储备充裕(1049亿元),净现金占市值36%,无财务风险
2. ✅ **盈利改善**: 连续两年盈利,经调整净利润206亿元,净利率14.5%
3. ✅ **首次分红**: 2025年首次派息+回购合计82亿港元,股东回报意识增强
4. ⚠️ **现金流恶化**: 经营现金流下降10%,自由现金流下降34%,"利润向上、现金向下"背离
5. ⚠️ **AI投入压力**: 2026年计划新增100亿资本开支,短期利润和现金流承压
6. ⚠️ **增长放缓**: 用户增速降至2.7%,行业渗透率见顶,直播业务停滞
7. ❌ **穿透回报率不足**: 粗算穿透回报率约7.4%,低于港股门槛5%,但精算后仅4.2%,不达标
8. ❌ **安全边际不足**: 当前股价对应穿透回报率低于门槛,需等待回调

**建议**:
- **当前价位(45.60港元)**: 不建议建仓,等待股价回调至35-40港元区间
- **目标买入价**: 约35港元(对应市值约240亿港元,穿透回报率约8%)
- **观察重点**: AI商业化进展、现金流改善情况、用户增长趋势

## 因子2:穿透回报率粗算(Top-Down)

### 步骤1:归母净利润

```
集团净利润 [A] = 186.17 亿元
少数股东损益 [B] = 0.00 亿元
GAAP归母净利润 [C] = A - B = 186.17 亿元
```

### 步骤2:Owner Earnings粗算

```
折旧摊销 [D] = 88.61 亿元
资本开支 [E] = 149.00 亿元
维持性资本开支系数 [G] = 0.6(保守估计)
维持性资本开支 [H] = D × G = 53.17 亿元

Owner Earnings [I] = C + D - H
                   = 186.17 + 88.61 - 53.17
                   = 221.61 亿元
```

**注**: Owner Earnings粗算值221.61亿元,高于经调整净利润206.47亿元
