### 模块一：资本消耗强度

**分析**:
- 电信运营为典型重资产行业，资本开支巨大
- 2024年Capex/营收 = 614/3896 = 15.8%
- 2025年Capex/营收 = 542/3922 = 13.8%
- Capex呈下降趋势（614→542→500亿指引），资本消耗强度改善
- 但绝对金额仍然巨大，5G/算力投资仍需持续投入
- 折旧摊销808.3亿 > Capex 542亿，说明网络投资已过高峰期

**结论**: 资本消耗强度 = **中高**（绝对金额大，但趋势改善）

### 模块二：收款模式

**分析**:
- C端业务（移动/宽带）：预付费为主，收款模式优秀
- B端业务（政企/云服务）：存在应收账款，回款周期较长
- 应收账款周转天数：45天（2024年）→ 59天（2025Q3），恶化明显
- 应收账款从576亿增至709亿（2025Q3），增速远超收入增速
- 合同负债/预收款：⚠️数据缺失，但电信运营商预收款通常较大
- 现金循环周期：-140天（2024年），说明占用供应商资金能力强

**结论**: 收款模式 = **预收+现结为主**（C端优秀，B端需关注）

### 模块三：竞争格局与护城河

#### 第一层：非技术性护城河
- **牌照壁垒**: 电信运营牌照仅3张，新进入者不可能获得
- **规模经济**: 网络建设固定成本高，用户规模决定单位成本
- **转换成本**: 手机号码携号转网便利性提高，但固网宽带转换成本仍较高
- **网络效应**: 用户越多，网络价值越大（但3家运营商均有全国覆盖）

#### 第二层：技术性护城河
- **5G共建共享**: 与中国电信合作共建5G网络，降低投资成本
- **物联网先发优势**: 物联网连接数6.9亿、车联网8,600万，行业第一
- **eSIM独家**: 唯一受支持的iPhone eSIM运营商
- **联通云**: 增速17.1%，但规模远小于天翼云和移动云

**竞争劣势**:
- 移动用户市占率仅18%，远低于中国移动65%
- 5G基站98万，少于移动194万和电信120万
- 盈利能力最弱：ROE 5.4% vs 移动~10% vs 电信~6.5%
- 品牌认知度和用户质量均不及竞争对手

**结论**: 护城河评级 = **窄**（有牌照壁垒，但竞争地位最弱）

### 模块九：控股折价分析

**分析**:
- 600050.SH持有0762.HK约37%股权
- A股公司为纯控股型，自身几乎无经营业务
- 现金流传递路径：0762.HK经营 → 0762.HK分红(61.3%) → 600050.SH(37%) → A股股东分红
- A股公司从香港子公司收到分红约47亿，但支付给A股股东51亿
- 差额约4亿来自A股公司自有现金/其他收入

**控股折价因素**:
1. 现金流传递效率低（37%持股 × 61.3%支付率 = 仅22.7%的香港子公司利润到达A股股东）
2. A股公司自身管理成本
3. 税务摩擦（港股通分红税率20%，但A股公司直接持股税率可能更低）
4. 信息不对称（A股投资者难以直接跟踪香港子公司经营）

**结论**: 控股折价 = **显著**（A股公司实质为香港子公司的"影子股"）

## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

GAAP归母净利润 [C] = 91.27 亿元

### 步骤2：Owner Earnings粗算

由于600050.SH为控股公司，OE计算需特殊处理：

**方法一：直接法（基于A股公司实际可分配现金）**
- 从香港子公司收到分红: ~47亿
- A股公司其他收入: ~4亿
- A股公司自身运营成本: ~2亿
- **OE_直接** = 47 + 4 - 2 = **49亿元**

**方法二：权益法（基于归母净利润调整）**
- 归母净利润: 91.27亿
- 减：非经常性损益: 14.41亿
- 加：折旧摊销归母份额（≈808.3 × 44%）: 355.6亿
- 减：资本开支归母份额（≈542 × 44%）: 238.5亿
- **OE_权益** = 91.27 - 14.41 + 355.6 - 238.5 = **194.0亿元**

**方法二问题**: 权益法高估了A股公司实际可支配现金，因为大部分D&A和Capex发生在香港子公司层面，A股公司无法直接控制。

**选用方法一**作为粗算基准，更保守更真实。

**OE粗算** = **49亿元**

### 步骤3：分配能力验证

- 2025年实际分红: 51.12亿
- OE: 49亿
- 分红/OE = 104.3% > 100%

⚠️ **分配能力验证: 边际不通过**（分红略超OE，但差距不大）

说明：A股公司当前分红略超其从香港子公司收到的分红，差额由自有现金补充。短期内可持续，但长期需香港子公司提升支付率或A股公司增加其他收入。

### 步骤4：现金上游障碍

- 香港子公司支付率61.3%，仍有提升空间
- A股公司持股37%，无法单方面决定分红政策
- 但作为大股东，影响力较大
- **上游障碍**: 中等（需香港子公司配合提升分红）

### 步骤5：分配意愿

- 支付率持续提升: 41% → 47% → 50% → 55% → 56%
- 管理层明确表示将继续提升派息率
- 香港子公司派息率也从50%以下提升至61.3%
- **分配意愿**: 强

### 步骤6：可预测性

- 电信运营收入稳定，波动小
- 分红政策连续提升，可预测性高
- 但A股公司分红受制于香港子公司，存在一定不确定性
- **可预测性**: 中高

### 步骤7：税务泄漏修正

- A股公司从香港子公司收到分红: 企业间分红免税
- A股股东收到分红: 持股>1年免税
- **税务泄漏 Q%** = 0%（长期持股）

### 步骤8：粗算穿透回报率

```
R = (OE × M × (1 − Q%) + O) / 当前市值
R = (49 × 55% × (1 − 0%) + 0) / 1,431.89
R = 26.95 / 1,431.89
R = 1.88%
```

此计算过于保守。让我用实际分红作为更合理的基准：

```
R = 实际分红 / 当前市值
R = 51.12 / 1,431.89
R = 3.57%
```

### 粗算否决门判断

- R = 3.57%
- II = 3.79%
- R < II → **边际不达标**
- 但 R > II × 0.5 = 1.90% → 不否决
- II × 0.5 ≤ R < II → 标注"边际不达标"，继续进入因子3

**粗算结论**: ⚠️ **边际不达标**，继续精算
