## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤1：归母净利润**
- C = 685 百万元 = 6.85 亿元

**步骤2：Owner Earnings 粗算**
- D（折旧摊销）≈ 70 百万元
- G（维持性资本开支系数）= 1.0（保守假设维持性Capex ≈ D&A）
- H = D × G = 70 百万元
- I = C + D - H = 685 + 70 - 70 = 685 百万元 = 6.85 亿元

**步骤3：分配能力验证**
- OCF = 699 百万元 > Owner Earnings = 685 百万元 ✅（轻微缓冲）
- 股息 = 354 百万元 < OCF = 699 百万元 ✅
- **通过**

**步骤4：现金上游障碍**
- A股直接持有，无跨境分红限制 ✅
- 少数股东损益占比极小（~4%） ✅
- **通过**

**步骤5：分配意愿**
- 3年平均支付率：57.81%
- 支付率锚定值 M = 55%（保守取值）
- 无回购计划
- **中等分配意愿**

**步骤6：可预测性**
- 5年收入CAGR ~15-20%，持续增长
- 利润增长稳定
- **中等可预测性**

**步骤7：税务泄漏修正**
- A股长期持有（>1年）：Q = 0%
- 综合股息税率 Q = 0%

**步骤8：粗算穿透回报率**

```
R = [C × M × (1 - Q%) + O] / 当前市值
R = [685 × 0.55 × 1.0 + 0] / 26,935
R = 376.75 / 26,935
R = 1.40%
```

### 粗算否决门检查

| 检查项 | 数值 | 判定 |
|--------|------|------|
| R（粗算穿透回报率） | 1.40% | — |
| Rf（无风险利率） | 1.76% | — |
| II（A股门槛） | 3.76% | — |
| II × 0.5 | 1.88% | — |
| R < Rf？ | 1.40% < 1.76% | **是** |
| 广义净现金/市值 | 14.9/269.35 = 5.5% | < 40%，EV口径不适用 |

**判定：R = 1.40% < Rf = 1.76%，且EV口径不适用（广义净现金/市值仅5.5% < 40%）**

### ⛔ 因子 2 粗算否决

粗算穿透回报率 R = 1.40%，低于无风险利率 1.76%，且EV口径不适用。根据龟龟投资策略规则，**因子 2 触发否决，不进入因子 3 精算和因子 4 估值**。

## 否决原因深度解读

### 核心问题：估值过高

华测导航当前估值水平：

| 估值指标 | 数值 | 龟龟策略可接受范围 | 差距 |

| PE (TTM) | 39.47x | ~10-15x | 2.6-4.0倍过高 |

| 穿透回报率 | 1.40% | ≥3.76% | 2.7倍不足 |

**根本原因**：市场给予华测导航39倍PE的成长股估值，而龟龟投资策略要求的是价值股级别的回报率（≥3.76%）。两者存在根本性冲突。
