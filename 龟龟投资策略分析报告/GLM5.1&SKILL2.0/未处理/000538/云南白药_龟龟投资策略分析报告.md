## 四、因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 51.53 亿元

### 步骤 2：Owner Earnings 粗算

```
D = 折旧摊销 ≈ 3.0 亿元
G = 维持性资本开支系数 ≈ 0.6（低资本消耗企业）
H = D × G = 3.0 × 0.6 = 1.8 亿元
I = C + D - H = 51.53 + 3.0 - 1.8 = 52.73 亿元
```

> 注：对于低资本消耗企业，Owner Earnings ≈ 净利润，因维持性资本开支≈折旧摊销

### 步骤 3：分配能力验证

```
OCF = 46.00 亿元
年度股息总额 = 46.43 亿元
OCF/股息 = 46.00/46.43 = 99.1%
```

⚠️ OCF略低于股息支付，但差额仅0.43亿元，且公司持有91亿现金储备，短期可持续。**不构成一票否决**，但需在因子3中深入检验。

### 步骤 4：现金上游障碍

A股直接持有，无跨境税务障碍，股息持有>1年免征个人所得税。

Q = 0%（综合股息税率）

**可预测性评级：高**

### 步骤 7：税务泄漏修正

Q = 0%（A股长期持有免税）

### 步骤 8：粗算穿透回报率

```
R = [C × M × (1 - Q%) + O] / 当前市值
R = [51.53 × 85% × (1 - 0%) + 0] / 983.31
R = 43.80 / 983.31
R = 4.46%
```

若使用M=90%：
```
R = [51.53 × 90% × 100% + 0] / 983.31
R = 46.38 / 983.31
R = 4.72%
```

### 粗算否决门判断

| 条件 | 判断 |
|------|------|
| R < Rf (1.762%)? | 否（4.46% > 1.762%） |
| R < II × 0.5 (1.881%)? | 否（4.46% > 1.881%） |
| II × 0.5 ≤ R < II? | 否（4.46% > 3.762%） |
| R ≥ II? | 是（4.46% ≥ 3.762%） |

**粗算结论：✅ 通过** — 粗算穿透回报率高于门槛值

安全边际（粗算）= R - II = 4.46% - 3.762% = **0.70 pct**（边际达标）

## 五、因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```
S = 营业收入 = 411.87 亿元
T = 应收账款净变动 = 103.19 - 99.23 = +3.96 亿元（增加，扣除）
U = 预收款/合同负债净变动 ≈ 0（数据缺失，保守假设）

保守处理：
- 应收账款净增加3.96亿 → 扣除（收入未收到现金）
- 预收款变动 → 保守不加回

真实现金收入 = S - T = 411.87 - 3.96 = 407.91 亿元
收款比率 = 407.91 / 411.87 = 99.04%
```

**收款比率99.04%，极高** → 现金收入质量良好

### 步骤 2：应收账款 footnote 核查

收款比率>95%，但应收账款/利润=200.2%偏高，需关注：
