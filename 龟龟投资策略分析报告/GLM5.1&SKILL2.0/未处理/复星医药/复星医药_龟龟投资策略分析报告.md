### 模块一：资本消耗强度

**评级：中高**

- 2025年研发总投入59.13亿元（占营收14.2%），创新药研发投入43.03亿元
- 资本化研发支出19.00亿元，资本化率约32.1%，偏高
- 资本开支相对温和（~9亿元/年），但投资性支出大（收购、参股等）
- 制药行业天然需要持续高研发投入维持竞争力
- **结论**：资本消耗强度中高，研发投入是刚性支出，资本化比例需关注

### 模块二：收款模式

**评级：应收为主，趋势恶化**

- 应收账款周转天数从68.36天（2024）上升至74.17天（2025Q3），回款效率下降
- 2025年末应收账款93.65亿元，同比增长17.76%，远超营收增速1.45%
- 应收账款/营收比值持续上升
- 医药行业医院端回款周期较长是行业共性，但复星医药的恶化趋势值得关注
- **结论**：收款模式偏弱，应收增长快于营收，需警惕回款风险

### 模块三：竞争格局与护城河

**非技术性护城河：窄**
- 全产业链布局形成一定协同效应，但各板块竞争激烈
- 仿制药板块受集采持续压制，利润空间收窄
- 境外收入占比31.15%，国际化能力是差异化优势

**技术性护城河：窄偏中**
- 创新药收入98.93亿元，占制药业务33.16%，转型初见成效
- 17个车间/生产线通过美国、欧盟、WHO等GMP认证
- 但核心品种（如汉利康、汉曲优）面临PD-1红海竞争
- 研发管线丰富但尚未形成不可替代的技术壁垒
- **结论**：护城河评级"窄"，创新转型进行中但尚未建立宽护城河

### 模块四：周期性

**评级：弱周期**

- 医药行业需求相对刚性，受经济周期影响较小
- 但集采政策周期对仿制药板块形成阶段性冲击
- 创新药审批和医保谈判存在政策周期
- **结论**：弱周期，不适用周期性修正

### 模块五：人力资本依赖

**评级：中高**

- 研发人员是核心资产，创新药研发高度依赖人才
- 2025年支付给职工现金112.46亿元，人力成本占比较高
- 核心管理层近期变动：2025年6月新任CEO，2026年4月新任CFO
- **结论**：人力资本依赖中高，管理层变动需跟踪

### 模块六：管理层与治理结构

**评级：一般**

- 控股股东复星高科技质押比例56.34%，为重大负面信号
  - 质押比例>50%表明控股股东资金需求较大
  - 复星国际整体去杠杆背景下，存在被动减持风险
- 核心管理层近期变动：CEO（2025年6月刘毅接任）、CFO（2026年4月黄智接任）
- 实际控制人郭广昌通过复星国际→复星高科技→复星医药的链条控制
- 关联交易频繁，涉及财务公司等
- **结论**：管理层评分"一般"，控股股东高质押是核心风险

### 模块七：监管与政策风险

**评级：中高**

- 集采常态化持续压制仿制药利润
- 创新药医保谈判降价压力
- 医疗器械集采扩面
- 海外业务面临FDA/EMA监管及地缘政治风险
- **结论**：监管风险中高，政策不确定性是行业共性

### 模块八：MD&A 解读

**前瞻：中性偏正**

- 创新药收入占比持续提升（33.16%），转型方向正确
- 2026-2028年重磅产品梯队密集落地预期
- 境外收入占比提升至31.15%，国际化进展积极
- 但扣非净利润增速仅1.12%，盈利质量改善有限
- 资本配置：研发持续加码，非核心资产处置优化
- **结论**：MD&A前瞻中性偏正，但执行验证待观察

### 模块九：控股折价分析

**适用**

- 复星医药为控股型架构，旗下拥有复宏汉霖、重庆药友、Gland Pharma等众多子公司
- 少数股东权益12,673百万元，占股东权益21.1%
- 少数股东损益占集团净利润比例约24.6%
- 控股折价因素：子公司独立上市/分拆预期（复星安特金拟分拆赴港）可能部分释放价值
- **结论**：适用控股折价分析，少数股东占比高，归母利润被稀释

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 3,371 百万元（2025年度GAAP归母净利润）

> ⚠️ 注意：其中含非经常性损益1,030百万元，扣非归母净利润仅2,340百万元

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）= ~2,500 百万元（估算）
G（维持性资本开支系数）= 0.70
H = D × G = 2,500 × 0.70 = 1,750 百万元
I = C + D - H = 3,371 + 2,500 - 1,750 = 4,121 百万元
```

> 注：D&A为估算值，基于固定资产~174亿+无形资产~142亿的规模，按8-10%综合折旧摊销率估算

### 步骤 3：分配能力验证

- OCF = 5,213 百万元 > 归母净利润 3,371 百万元 ✅
- OCF/归母净利润 = 1.55x，现金流覆盖能力充足
- **结论**：分配能力验证通过

### 步骤 4：现金上游障碍

- 控股型架构，少数股东损益占比~24.6%
- 境外子公司（Gland Pharma等）利润汇回可能受限
- **结论**：存在中等现金上游障碍

### 步骤 5：分配意愿

- 2025年度支付率30.54%，较2024年报口径24.22%有所提升
- 2024年含中报分红合计支付率约56.9%（含特殊分红）
- 3年平均支付率约31-35%
- 回购：2025年启动3-6亿元A股回购，已执行约2亿元
- **结论**：分配意愿中等，支付率有提升趋势但不够稳定

### 步骤 6：可预测性

- 营收增速低（1.45%），但波动不大
- 扣非净利润波动较大（2025年前三季度同比-14.32%）
- 非经常性损益占比高（30.56%），降低盈利可预测性
- **结论**：可预测性中等偏低

### 步骤 7：税务泄漏修正

- A股持股>1年免股息税，Q = 0%
- **结论**：无税务泄漏

### 步骤 8：粗算穿透回报率

```
R = [I × M × (1 - Q%) + O] / 当前市值
  = [4,121 × 0.30 × 1.0 + 200] / 69,832
  = [1,236.3 + 200] / 69,832
  = 1,436.3 / 69,832
  = 2.06%
```

### 粗算否决门判断

| 条件 | 计算 | 结果 |
|---|---|---|
| R < Rf? | 2.06% > 1.762% | 否 |
| R < II × 0.5? | 2.06% > 1.881% | 否 |
| II × 0.5 ≤ R < II? | 1.881% ≤ 2.06% < 3.762% | **是** |

**粗算结论：边际不达标** — R = 2.06% 低于门槛 II = 3.762%，但高于 II×0.5 = 1.881%，继续进入因子3精算
