### 模块一：资本消耗强度

**评级：高**

- 房地产是典型高资本消耗行业，需持续投入土地资金维持销售规模
- 存货占总资产56%以上（2025年末6,682亿元），主要为土地储备和在建项目
- 2025年计划土地投资1,000亿元，资本消耗巨大
- 与轻资产行业相比，每单位利润需要数倍资本支撑
- **结论：资本消耗强度高，不符合"低资本消耗"偏好**

### 模块二：收款模式

**评级：预收为主（正面），但存在风险**

- 房地产实行预售制，先收款后交付，收款模式本质上是预收制
- 销售回笼率连续三年超100%，表面收款能力强
- 但需注意：预收款对应未来交付义务，并非"免费现金"
- 合同负债（已售未交付）规模巨大，构成隐性负债
- 应收账款仅49亿元（占总资产0.4%），收款风险低
- **结论：收款模式为预收制，形式上正面，但预收款本质是递延收入**

### 模块三：竞争格局与护城河

**评级：窄护城河，但正在收窄**

**非技术性护城河：**
- 央企信用优势：融资成本2.72%为行业最低，信用资质保障资金链安全
- 品牌壁垒：连续16年行业领导品牌，品牌价值1,425亿元
- 规模效应：行业销冠地位（2,530亿元），市占率持续提升至4.2%

**技术性护城河：**
- 土储质量：92%销售来自核心城市，优质土储保障盈利空间
- 不动产生态圈：从开发向"投资开发+经营+服务"转型

**护城河收窄信号：**
- 行业整体下行，央企份额提升更多是"剩者为王"而非竞争优势扩大
- 毛利率从2021年的~26%持续下滑至2025年的12.75%
- 核心城市市占率提升但利润率下降，说明价格竞争加剧
- **结论：存在窄护城河（央企信用+品牌），但行业下行正在侵蚀护城河价值**

### 模块四：周期性

**评级：强周期，当前处于下行周期中段**

- 房地产是典型强周期行业，与宏观经济、信贷政策高度相关
- 全国商品房销售额从2021年18万亿降至2025年约10万亿，降幅44%
- 保利发展归母净利润从2020年289亿降至2025年10亿，降幅96%
- 当前处于行业深度调整期，尚未明确见底
- **结论：强周期行业，当前处于下行周期，周期底部位置不确定**

### 模块五：人力资本依赖

**评级：低**

- 房地产开发对核心个人依赖度不高
- 央企管理体系成熟，人才梯队完善
- 董事长刘平年薪296万元，持股772万股，利益绑定度一般
- **结论：人力资本依赖度低**

### 模块六：管理层与治理结构

**评级：一般**

**正面因素：**
- 央企治理，保利集团直接持股40.72%，控制力强
- 2025年股权划转消除中间层级，保利发展升级为一级子公司
- 融资成本持续压降（3.10%→2.72%），财务管控有效
- 销售回笼率连续三年超100%，运营效率高

**负面因素：**
- 2025年审计师更换（信永中和→立信），动机需关注
- 归母净利润连续4年大幅下滑，管理层未能有效扭转
- 2025年下半年连续两个季度扣非亏损
- 少数股东权益占比高达42.3%，利润分配结构复杂
- 非经常性损益占归母净利润363%（2025年），盈利质量差
- **结论：管理层评分一般，央企治理有底线保障，但盈利持续恶化令人担忧**

### 模块七：监管与政策风险

**评级：高**

- 房地产是政策高度敏感行业，"房住不炒"长期基调
- 融资监管（三道红线虽已放松，但仍有约束）
- 土地政策、限购限贷政策频繁调整
- 税收政策风险（房产税试点预期）
- **结论：监管与政策风险高**

### 模块八：MD&A 解读

**评级：负面**

- 2025年归母净利润暴跌79.31%，扣非净利润暴跌84.52%
- 大幅计提减值69亿元，未来仍存在进一步减值可能
- 下半年连续两个季度扣非亏损（Q3亏7.82亿，Q4亏8.94亿）
- 前瞻指引：行业恢复节奏不确定，公司聚焦确定性市场
- 风险因素：市场恢复不确定性、存量资产盘活压力、毛利率下行、减值风险
- 投资性房地产计量模式变更，公允价值变动收益减少25.50亿元
- **结论：MD&A前瞻信号负面，行业仍在深度调整中**

### Step 2：Owner Earnings 粗算

**以2024年度（最新完整年度）为基准：**

```
C（归母净利润）= 50.01 亿元
D（折旧摊销）≈ 5.5 亿元

G（维持性资本开支系数）= 0.5（房地产公司capex主要为办公设备等维持性支出）
H = D × G = 5.5 × 0.5 = 2.75 亿元
I = C + D - H = 50.01 + 5.5 - 2.75 = 52.76 亿元
```

> ⚠️ 注意：房地产公司的"资本开支"在现金流量表中仅反映固定资产投资，主要投资（土地购置、项目开发）通过经营活动现金流体现。因此传统Owner Earnings公式对房地产公司适用性有限。

### Step 7：税务泄漏修正

- 无税务泄漏

### Step 8：粗算穿透回报率

**场景一：以2025年度数据为基准（最保守）**

```
C（2025归母净利润）= 10.35 亿元
M（支付率）= 40%
Q = 0%
O（回购）= 5 亿元

R = [C × M × (1 - Q%) + O] / 当前市值
R = [10.35 × 0.40 × 1.0 + 5] / 727
R = [4.14 + 5] / 727
R = 9.14 / 727
R = 1.26%
```

**场景二：以2024年度数据为基准**

```
C（2024归母净利润）= 50.01 亿元
M（支付率）= 40%
Q = 0%
O（回购）= 7.49 亿元

R = [50.01 × 0.40 × 1.0 + 7.49] / 727
R = [20.00 + 7.49] / 727
R = 27.49 / 727
R = 3.78%
```

**场景三：3年平均（2023-2025）**

```
C_avg = (121.01 + 50.01 + 10.35) / 3 = 60.46 亿元
R = [60.46 × 0.40 + 6.25] / 727
R = [24.18 + 6.25] / 727
R = 30.43 / 727
R = 4.19%
```

### 粗算否决门检查

| 场景 | R | vs Rf(1.76%) | vs II×0.5(1.88%) | vs II(3.76%) | 判定 |
|------|---|-------------|-------------------|-------------|------|
| 2025基准 | 1.26% | ❌ R < Rf | ❌ R < II×0.5 | ❌ R < II | **否决** |
| 2024基准 | 3.78% | ✅ R > Rf | ✅ R > II×0.5 | ✅ R ≥ II | 边际达标 |
| 3年平均 | 4.19% | ✅ R > Rf | ✅ R > II×0.5 | ✅ R > II | 通过 |

**EV口径检查：**
- 广义净现金 = 1,229 - 3,412 = -2,183 亿元
- 广义净现金/市值 = -2,183/727 = -300%
- 远低于40%触发线，EV口径**不适用**

### 因子 2 粗算结论

**以最保守的2025年度数据为基准：R = 1.26%**

- R = 1.26% < Rf = 1.76% → 检查EV口径 → EV不适用 → **否决**
- R = 1.26% < II × 0.5 = 1.88% → 检查EV口径 → EV不适用 → **否决**

> ⚠️ 即使考虑强周期底部门槛下调1pct（II_adj = 2.76%），R = 1.26%仍远低于调整后门槛。

> ⚠️ 即使以2024年度数据为基准（R = 3.78%），也仅边际达标（3.78% vs 3.76%），且2024年本身已是盈利大幅下滑的年份，不代表"正常化"水平。

> ⚠️ 3年平均虽然通过门槛，但对于连续5年加速下滑的公司，使用历史平均会高估未来盈利能力，不符合保守原则。
