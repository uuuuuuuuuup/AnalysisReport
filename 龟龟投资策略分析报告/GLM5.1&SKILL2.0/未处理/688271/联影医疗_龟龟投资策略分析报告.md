### 模块一：资本消耗强度

**评估：HIGH（高）**

- Capex/Revenue ≈ 14.6%（2024FY估算），属于资本密集型
- OCF/Capex = -619/15,000 = -4.1%（2024FY），经营现金流无法覆盖资本开支
- 在建工程持续增长（2024FY: 17.70亿 → 2025Q3: 23.49亿），产能扩张投入巨大
- 研发费用率高达17.1%（2024FY），属于高研发投入企业
- **结论**：公司处于重投入期，资本消耗强度高，自由现金流持续为负

### 模块二：收款模式

**评估：应收为主**

- 应收账款/营业收入 = 42.3%（2024FY），占比极高
- 应收账款周转天数 = 132-168天，回款周期长达4.5-5.6个月
- 医院客户账期长，属于行业特性，但公司应收占比偏高
- 2024FY OCF为负（-6.19亿），现金回收严重滞后于收入确认
- **结论**：收款模式以应收为主，现金回收存在显著障碍

### 模块三：竞争格局与护城河

**评估：窄护城河，正在加固**

**非技术性护城河：**
- 国产高端影像设备龙头品牌，客户认知度高
- 政策红利（国产替代、设备更新）提供结构性支撑
- 全产品线覆盖带来一定规模效应

**技术性护城河：**
- "整机-核心部件-底层元器件"全链条自主可控，打破GPS垄断
- CT/MR/PET-CT多个细分市场市占率第一
- 光子计数CT等前沿技术布局领先
- 研发投入持续高增长（17%+费用率）

**护城河趋势**：正向 — 市占率持续提升，技术壁垒加深
**但**：GPS国际巨头仍占据全球市场主导，价格战风险存在

### 模块四：周期性

**评估：弱周期**

- 医疗影像设备需求受医院资本开支周期影响
- 2024年营收下降9.73%反映政策驱动的需求波动（设备更新政策落地节奏）
- 2025年营收恢复增长34.18%，显示周期底部可能已过
- 长期需求由人口老龄化和医疗渗透率提升驱动，终端需求不会萎缩
- **结论**：弱周期性，但政策波动可造成短期显著影响

### 模块五：人力资本依赖

**评估：中高**

- 研发人员占比高，技术人才为核心资产
- 支付给职工现金/营业收入 ≈ 22.5%（2024FY），人力成本占比高
- 核心高管（张强、GUOSHENG TAN、JUN BAO）具有深厚行业背景，关键人风险存在
- **结论**：人力资本依赖度中高，核心技术人员流失将造成显著影响

### 模块六：管理层与治理结构

**评估：良好**

- 董事长张强自公司创立起担任CEO，稳定性高
- 实控人薛敏间接持股24.27%，控制权稳定
- 员工持股平台绑定核心团队利益
- 新任CFO王建保2024年5月上任，财务治理持续完善
- 无重大治理丑闻或关联方资金占用
- **关注点**：总裁JUN BAO为美国国籍，存在地缘政治风险

### 模块七：监管与政策风险

**评估：中**

- 国内：NMPA审批严格但可控，设备更新政策为双刃剑
- 国际：欧盟对中国医疗器械采购调查、美国《生物安全法案》潜在影响
- 国产替代政策为正面因素，但政策执行节奏不确定
- **结论**：监管风险中等，需持续关注国际政策变化

### 模块八：MD&A 解读

**评估：中性偏正**

- 管理层坦诚承认2024年挑战（政策驱动需求下降）
- 2025年业绩强劲恢复验证了业务韧性
- 海外扩张和前沿技术布局为长期增长点
- 风险因素披露充分
- **结论**：管理层态度务实，前瞻指引可信度中等

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 12.62 亿元（2024FY）
C_2025 = 18.88 亿元（2025FY快报）

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）≈ 5.50 亿元
E（资本开支）≈ 15.0 亿元
G（维持性资本开支系数）≈ 0.65（医疗设备制造业典型值）
H = D × G = 5.50 × 0.65 = 3.575 亿元（维持性资本开支粗估）
I = C + D - H = 12.62 + 5.50 - 3.575 = 14.545 亿元（2024FY）
I_2025 = 18.88 + 6.00 - 3.90 = 20.98 亿元（2025FY估算）
```

- 2024FY OCF = -6.19 亿元（**负值！**）
- 2025Q3 OCF = 1.07 亿元（仅微弱转正）
- **结论**：经营现金流无法覆盖资本开支，分配能力严重不足
- 但考虑到2024年为周期底部，2025年已恢复，暂不否决

### 步骤 5：分配意愿

- 公司明确处于成长期，利润主要用于再投资

- **结论**：分配意愿弱，公司优先再投资

### 步骤 6：可预测性

- 2024年营收-9.73%，2025年+34.18%，波动极大
- 利润波动更剧烈（2024年-36%，2025年+50%）
- 政策驱动需求难以预测
- **结论**：可预测性低

### 步骤 7：税务泄漏修正

- A股个人持股>1年：Q = 0%（免红利税）
- QFII：Q = 10%
- **以个人长期投资者口径：Q = 0%**

### 步骤 8：粗算穿透回报率 R%

```
R = [C × M × (1 - Q%) + O] / 市值
  = [12.62 × 0.068 × 1 + 5.66] / 930.5
  = [0.858 + 5.66] / 930.5
  = 6.518 / 930.5
  = 0.70%

使用2025FY数据：
R_2025 = [18.88 × 0.068 × 1 + 5.66] / 930.5
       = [1.284 + 5.66] / 930.5
       = 6.944 / 930.5
       = 0.75%

即使假设支付率提升至30%（远超历史水平）：
R_30% = [18.88 × 0.30 × 1 + 0] / 930.5
      = 5.664 / 930.5
      = 0.61%

使用盈利收益率（Earnings Yield）作为参考：
EY_2024 = 12.62 / 930.5 = 1.36%
EY_2025 = 18.88 / 930.5 = 2.03%
```

**粗算否决门检查：**
- R (0.70-0.75%) < Rf (1.762%)？→ 是，但需检查EV口径
- R < II × 0.5 (1.881%)？→ 是，R远低于II的一半
- R < II (3.762%)？→ 是

**但**：盈利收益率 EY_2025 = 2.03% > Rf (1.762%)，说明公司有盈利能力但几乎不分红

**粗算结论：边际不达标** — R远低于门槛，但EY略高于Rf，继续进入因子3精算
