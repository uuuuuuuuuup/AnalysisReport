- **精密制造能力**：超高精度模具、激光技术等工艺壁垒
- **技术融合平台**：整合声学、光学、微电子跨领域技术
- **负面因素**：技术迭代速度快，需持续高研发投入维持；立讯精密等竞争对手同样具备强大技术能力

**护城河评级：窄护城河** — 技术差异化与客户黏性存在，但低毛利率和客户集中度反映定价权有限，护城河宽度不足

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 26.65 亿元（2024年报归母净利润）

### 步骤 2：Owner Earnings 粗算

```
D = 折旧摊销 ≈ 20 亿元（按固定资产规模和行业惯例估算）
E = 资本开支 ≈ 40 亿元（估算）
G = 维持性资本开支系数 = 0.70（重资产制造业典型值）
H = D × G = 20 × 0.70 = 14.00 亿元
I = C + D − H = 26.65 + 20 − 14 = 32.65 亿元（Owner Earnings）
```

- 无税务泄漏

### 步骤 8：粗算穿透回报率

```
R = [C × M × (1 − Q%) + O] / 当前市值
R = [26.65 × 0.35 × 1.0 + 12] / 855.8
R = [9.33 + 12] / 855.8
R = 21.33 / 855.8
R = 2.49%
```

**粗算否决门判断**：
- R = 2.49%
- Rf = 1.762% → R > Rf ✅
- II × 0.5 = 1.88% → R > II × 0.5 ✅
- II = 3.762% → R < II ❌

**II × 0.5 ≤ R < II → 标注"边际不达标"，继续进入因子 3**
