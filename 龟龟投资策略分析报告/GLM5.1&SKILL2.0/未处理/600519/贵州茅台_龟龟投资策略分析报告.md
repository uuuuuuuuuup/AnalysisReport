### 模块一：资本消耗强度

**评级：低**

- 茅台是典型的轻资本消耗企业
- 2024年Capex/OCF = 57.5/924.6 = 6.2%，2025年约9.8%
- 维持性资本开支极低（G≈0.25），大部分capex为产能扩张
- 固定资产占总资产仅7.4%，无需持续大量再投入
- 核心生产资料（赤水河生态+茅台镇微环境+基酒储备）无需资本化重置

### 模块二：收款模式

**评级：预收/现结（最优等级）**

- 先款后货模式，应收账款仅260.9万元（占总资产0.00009%）
- 应收账款周转天数0.05天，收款比率99%+
- 合同负债80.1亿元，体现经销商预付款
- 销货现金流1,839.9亿元 > 营业收入1,720.5亿元，现金回收极强

### 模块三：竞争格局与护城河

**护城河评级：宽（Wide Moat）**

**非技术性护城河：**
- 品牌壁垒：中国白酒"天花板"品牌，品牌价值超年营收8倍
- 资源壁垒：茅台镇核心产区不可复制，赤水河生态资源独占
- 定价权：单品营收过千亿，全球烈性酒第一品牌，出厂价与批价之间存在巨大缓冲
- 网络效应：社交货币属性，宴请/送礼刚需

**技术性护城河：**
- 工艺壁垒：传统酿造工艺+国家非物质文化遗产，基酒储备优势（5年窖藏周期形成天然产能壁垒）
- 规模经济：产量规模带来的成本优势（虽然毛利率已91%+，边际改善有限）

**护城河趋势**：稳定偏正面。品牌壁垒和资源壁垒依然坚固，但行业调整期竞争加剧，系列酒面临压力。

### 模块四：周期性

**评级：弱周期**

- 高端白酒具有"消费+投资"双重属性，但茅台品牌力使其抗周期能力远超同行
- 2025年首次年度双降（营收-1.2%，净利-4.53%），但茅台酒营收仍微增0.39%
- 历史表现：2022年行业调整期茅台仍保持增长，验证弱周期特征
- 当前处于行业调整中段，非周期顶部

### 模块五：人力资本依赖

**评级：低**

- 生产工艺标准化程度高（传统工艺+现代管理），不依赖个别酿酒师
- 管理层为国企体系任命，稳定性有保障
- 核心竞争力在于品牌和产区，而非个人能力

### 模块六：管理层与治理结构

**评级：良好**

- 董事长陈华2025年11月就任，总经理王莉（代），具备30年+白酒行业经验
- 控股股东茅台集团持股54.04%，实控人贵州省国资委
- 正面信号：首次回购（60亿元）、分红率提升至79%、承诺2024-2026年≥75%
- 关注点：管理层频繁变动（近年多任董事长），但战略连续性尚可
- 关联交易规模可控（2026年预计81.25亿元，占净资产<5%），经专项审计无违规

### 模块七：监管与政策风险

**评级：中**

- "禁酒令"风险：历史上多次出台，对高端白酒消费场景有压制
- 消费税改革：若推进可能影响渠道利润分配
- 国企改革：混合所有制改革、薪酬制度改革可能影响管理层激励
- 但茅台作为贵州省财政支柱（分红+税收贡献巨大），政策面有底线保护

### 模块八：MD&A 解读

- 七大风险因素披露充分，管理层对行业挑战认知清晰

## 三、因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 823.2 亿元（2025年度）

### 步骤 2：Owner Earnings 粗算

```
D = 折旧摊销 ≈ 15 亿元
G = 维持性资本开支系数 = 0.25（茅台大部分capex为扩张性）
H = D × G = 15 × 0.25 = 3.75 亿元（维持性资本开支粗估）
I = C + D - H = 823.2 + 15 - 3.75 = 834.45 亿元（Owner Earnings）
```

### 步骤 3：分配能力验证

- OCF = 615.0 亿元（2025年，受财务公司存款变动影响大幅下降）
- 销货现金流 = 1,839.9 亿元（同比+0.7%，核心现金流健康）
- OCF < Owner Earnings？是，但主因财务公司扰动，非经营恶化
- **结论：分配能力无实质障碍**

### 步骤 4：现金上游障碍

- A股直接持有，无跨境限制
- Q = 0%（持股>1年免征个人所得税）

### 步骤 5：分配意愿

- 2025年分红率79%，远超历史均值
- 公司承诺2024-2026年分红率≥75%
- 首次启动回购（60亿元），用于注销减资
- **分配意愿：强**

### 步骤 6：可预测性

- 2025年首次年度双降，增加短期不确定性
- 但茅台酒营收仍微增0.39%，核心业务韧性仍在
- 行业调整期持续时间不确定
- **可预测性：中-高**

### 步骤 7：税务泄漏修正

- Q = 0%（A股直接持有，持股>1年免征）
- 无税务泄漏

### 步骤 8：粗算穿透回报率

```
M = 75%（承诺下限，保守）
O = 30 亿元（年均回购，保守估计）
Q = 0%

R = [C × M × (1 - Q%) + O] / 当前市值
R = [823.2 × 0.75 × 1 + 30] / 17,622.5
R = [617.4 + 30] / 17,622.5
R = 647.4 / 17,622.5
R = 3.67%
```

**若使用 M = 79%（最新实际支付率）：**

```
R = [823.2 × 0.79 + 30] / 17,622.5
R = [650.3 + 30] / 17,622.5
R = 680.3 / 17,622.5
R = 3.86%
```

### 粗算否决门检查

| 口径 | R | II = 3.762% | 判定 |
|------|---|-------------|------|
| M = 75%（保守） | 3.67% | 3.762% | II×0.5 ≤ R < II → 边际不达标 |
| M = 79%（实际） | 3.86% | 3.762% | R ≥ II → 通过 |

**粗算结论：保守口径边际不达标，实际口径通过。继续进入因子3精算。**

## 四、因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```
S = 1,720.5 亿元（2025年营业收入）
T = 应收账款净变动 ≈ 0 亿元（应收仅260.9万元，可忽略）
U = 合同负债净变动 = 8,010 - 9,690 = -1,680 百万元 = -16.8 亿元（减少）

保守处理：
- 应收账款净增加 → 扣除（但金额≈0，忽略）
- 合同负债净减少16.8亿 → 扣除（预收款下降意味着现金流入减少）

真实现金收入 = S - T的净增加 - U的净减少
= 1,720.5 - 0 - 16.8
= 1,703.7 亿元

收款比率 = 1,703.7 / 1,720.5 = 99.02%
```

**收款比率99.02%，极高，符合茅台先款后货模式。**

### 步骤 2：应收账款 footnote 核查

- 收款比率 ≈ 99% > 95%，无需深度核查
- 应收账款仅260.9万元，周转天数0.05天
- 坏账风险可忽略
- **结论：无需进一步核查**

### 步骤 3：非经常性现金流入分类

```
保留项（真实可分配现金）：
V1 = 资产处置收入 ≈ 0 亿元（2025年非流动性资产处置为损失-2.38百万元）
V5 = 其他投资收入 ≈ 8.5 亿元（财务公司利息收入，2025年财务费用-850百万元估算）

扣除项（非可分配）：
V2 = 政府补贴 = 0.04 亿元（4.27百万元，可忽略）
V3 = 保险赔款 = 0
V4 = 其他一次性非投资类流入 = 0

V_deduct = V2 + V3 + V4 = 0.04 亿元（可忽略）
```

### 步骤 4：经营性现金支出还原

```
⚠️ 核心问题：现金流量表经营流出明细不完整

已知数据（2025年）：
- 支付给职工：123.93 亿元
- 支付税费：591.1 亿元
- 购买商品/劳务：⚠️未找到精确值（2024年约85.95亿元）
- 支付利息：≈0（财务费用为负，利息收入>利息支出）

采用 OCF 倒推法（第二优先级）：
W = 真实现金收入 - OCF

但2025年OCF受财务公司存款变动严重扭曲（同比下降33.46%），
需区分"核心经营OCF"与"报告OCF"：

报告OCF = 615.0 亿元
核心经营OCF ≈ 920 亿元（参考2024年924.6亿，销货现金流同比+0.7%）

方案A（保守，使用报告OCF）：
W_reported = 1,703.7 - 615.0 = 1,088.7 亿元

方案B（调整，使用核心OCF）：
W_adjusted = 1,703.7 - 920.0 = 783.7 亿元

⚠️ 标注：2025年OCF受财务公司因素扭曲，方案A系统性高估W（低估结余质量），
方案B更接近真实经营支出。后续计算同时列出两种口径。
```

### 步骤 5：资本开支与投资的极端保守处理

```
(a) 资本开支总额 E = 60 亿元（全额扣除）
(b) 投资性购买 X1 ≈ 0 亿元（2025年无重大对外投资/收购）
(c) 隐性必要支出还原 X2 ≈ 0 亿元（研发费用资本化比例极低）

总扣除额 Y = E + X1 + X2 = 60 亿元
```

### 步骤 6：准则差异核查

- A股收入确认：茅台采用先款后货，收入确认激进度低
- 递延税资产趋势：正常
- 资本化利息比例：无有息负债，不适用
- 或有负债：未披露重大或有负债
- **结论：无重大准则差异风险**

### 步骤 7：真实可支配现金结余

**方案A（保守口径，使用报告OCF）：**

```

真实可支配现金结余 = 真实现金收入 + V1 + V5 - V_deduct - W - Y
= 1,703.7 + 0 + 8.5 - 0.04 - 1,088.7 - 60
= 563.5 亿元
```

**方案B（调整口径，使用核心OCF）：**

```

```
