### 模块一：资本消耗强度

**评级：高**

- 固定资产占总资产 > 45%，典型重资产制造企业
- 2025年资本开支94亿元，远超估算折旧摊销（~40亿元），资本开支/折旧比 = 2.35x
- 维持性资本开支系数 G 估算 = 0.9（重资产属性决定大部分D&A需再投入）
- 大量资本开支用于扩产（海外基地、新业务产能），但维持性部分仍然很高
- **结论**：资本消耗强度高，自由现金流受资本开支压制严重

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C（GAAP 归母净利润）= 40.18 亿元（2025年度）

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）≈ 40.00 亿元
G（维持性资本开支系数）= 0.9（重资产制造，大部分D&A需再投入）
H = D × G = 40.00 × 0.9 = 36.00 亿元
I = C + D - H = 40.18 + 40.00 - 36.00 = 44.18 亿元
```

**Owner Earnings 粗算值 I = 44.18 亿元**

### 步骤 3：分配能力验证

```
OCF（2025）= 114.65 亿元
Capex（2025）= 94.00 亿元
FCF = OCF - Capex = 20.65 亿元
股息支付 = 29.04 亿元
FCF < 股息支付 → ⚠️ 分配能力存疑
```

> 注：2025年资本开支异常高（94亿 vs 2024年约63亿），主要用于新业务扩张。若正常化Capex约65亿元，FCF ≈ 49.65亿元，足以覆盖分红。但按当期数据，分配能力不达标。

### 步骤 4：现金上游障碍

- A股直接持有，无港股通通道障碍
- 内地注册公司，无跨境资金流动障碍
- **现金上游障碍：无**

### 步骤 5：分配意愿

- DPS持续提升：10派1→2→3→4→4.5元
- 支付率从49.48%提升至72.3%
- 回购金额极小（~0.01亿元/年）
- **分配意愿：强**

### 步骤 6：可预测性

- 营收5年CAGR约15%，但年度波动较大
- 净利润波动大（2021年暴跌57.72%）
- **可预测性：中**

### 步骤 7：税务泄漏修正

- A股持有>1年：免征股息税
- Q（综合股息税率）= 0%（长期持有假设）

### 步骤 8：粗算穿透回报率 R%

```
M（支付率锚定值）= 55%
Q = 0%
O（年均回购）≈ 0.01 亿元

R = [I × M × (1 - Q%) + O] / 当前市值
R = [44.18 × 55% × 1.0 + 0.01] / 1,372
R = [24.30 + 0.01] / 1,372
R = 24.31 / 1,372
R = 1.77%
```

### 粗算否决门检查

```
Rf = 1.78%
II = 3.78%
II × 0.5 = 1.89%

R = 1.77% < Rf = 1.78% → 检查 EV 口径
```

**EV 口径检查：**

```
广义净现金 = 25.67 亿元
广义净现金 / 市值 = 25.67 / 1,372 = 1.87% < 40%
→ EV 口径不适用（广义净现金占比过低，不满足双轨制触发条件）

EV = 市值 - 广义净现金 = 1,372 - 25.67 = 1,346.33 亿元
R_EV = 24.31 / 1,346.33 = 1.81%

R_EV = 1.81% < II × 0.5 = 1.89% → EV 口径仍不通过
```

### ⛔ 因子 2 结论：粗算否决

```
R = 1.77% < Rf = 1.78% → 不达标
R = 1.77% < II × 0.5 = 1.89% → 严重不达标
EV 口径不适用（广义净现金/市值 < 40%）
R_EV = 1.81% < II × 0.5 = 1.89% → EV 口径亦不通过

→ 触发粗算否决门，停止后续因子 3/4 详细分析
```
