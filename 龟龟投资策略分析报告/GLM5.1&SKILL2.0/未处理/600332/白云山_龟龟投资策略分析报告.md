## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C（归母净利润）= 2,983 百万元 = **29.83 亿元**（2025年度）

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）≈ 500 百万元 = 5.0 亿元
G（维持性资本开支系数）= 0.3（医药行业，维持性占比较低）
H = D × G = 5.0 × 0.3 = 1.5 亿元
I = C + D - H = 29.83 + 5.0 - 1.5 = 33.33 亿元
```

> ⚠️ 注意：Owner Earnings 粗算值 33.33 亿元远高于 2025 年 OCF（-2.32 亿元），差异极大。
> 这表明利润质量严重不足，账面利润未能转化为现金。

### 步骤 3：分配能力验证（一票否决）

```
2025年 OCF = -2.32 亿元
2025年股息支付 = 13.82 亿元
OCF < 股息支付 → 分配能力存疑
```

**⚠️ 分配能力验证：未通过**——经营现金流不足以覆盖股息分配，分红资金来源于往年现金储备。

但考虑到2024年OCF为34.42亿元（正常水平），2025年OCF恶化可能包含一次性/周期性因素，不触发一票否决，但标注为**高风险**。

### 步骤 4：现金上游障碍

| 障碍类型 | 评估 |
|---------|------|
| 应收账款占用 | 严重——191.35亿元，同比+21.7% |
| 存货占用 | 中等——110.81亿元 |
| 关联方资金占用 | ⚠️未找到 |
| 子公司现金留存 | 可能存在——医药流通子公司资金需求大 |

**现金上游障碍评级：高**——应收账款大幅增长是OCF恶化的主因

### 步骤 5：分配意愿

### 步骤 6：可预测性

| 现金流可预测性 | 低——2025年OCF断崖下滑，难以预测 |

**可预测性评级：低**

### 步骤 7：税务泄漏修正

```
Q（综合股息税率）= 0%
A股持股>1年免征红利税，假设长期持有
```

### 步骤 8：粗算穿透回报率

```
═════ 主轨：市值口径 ════
R = [I × M × (1 - Q%) + O] / 当前市值
R = [33.33 × 0.40 × 1 + 0] / 370.36
R = 13.33 / 370.36
R = 3.60%
```

> ⚠️ 但此计算基于 Owner Earnings（利润口径），与实际现金流严重背离。
> 以2024年OCF为基准的现金流口径：
> R_OCF = [34.42 × 0.40 × 1 + 0] / 370.36 = 13.77 / 370.36 = **3.72%**
> 
> 以2年平均OCF为基准：
> R_avg = [(34.42 + (-2.32)) / 2 × 0.40] / 370.36 = [6.42] / 370.36 = **1.73%**

### 粗算否决门判断

```
II = 3.762%

R（利润口径）= 3.60% < II → 低于门槛
R_OCF（2024单年）= 3.72% < II → 低于门槛，差距仅0.04pct
R_avg（2年平均）= 1.73% < II × 0.5 = 1.881% → 接近否决线
```

**粗算结论：边际不达标**——利润口径和2024单年OCF口径均略低于门槛，2年平均口径接近否决线。继续进入因子3精算。

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```
S（营业收入）= 77,656 百万元 = 776.56 亿元
T（应收账款净变动）= 19,135 - 15,726 = 3,409 百万元 = 34.09 亿元（增加）
U（预收款/合同负债净变动）= ⚠️未找到，保守假设为0

保守处理：
- 应收账款净增加 → 扣除
- 预收款净变动 → 不加回（数据缺失）

真实现金收入 = S - T净增加 = 776.56 - 34.09 = 742.47 亿元
收款比率 = 742.47 / 776.56 = 95.6%
```
