## 三、因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤1：归母净利润**

C = 12.91 亿元（2025年报归母净利润）

**步骤2：Owner Earnings 粗算**

```
D（折旧摊销）= 0.55 亿元
G（维持性资本开支系数）= 0.5（轻资产模式，保守取值）
H = D × G = 0.55 × 0.5 = 0.275 亿元
I（Owner Earnings）= C + D − H = 12.91 + 0.55 − 0.275 = 13.19 亿元
```

**步骤3：分配能力验证**

OCF（2025全年估）= 14.27 亿元 > 股息 6.05 亿元 ✅
OCF/C = 14.27/12.91 = 1.105 > 1 ✅
现金转换质量良好，分配能力无问题。

**步骤4：现金上游障碍**

收款比率 > 100%，应收账款周转天数22.46天，上游障碍极低。✅

**步骤5：分配意愿**

- 历史支付率均值55.3%，2025年降至46.7%
- DPS从3.80元降至2.00元，分配意愿有所减弱
- 无有意义回购（回购用于股权激励，非股东回报）
- 判定：分配意愿中等偏弱（DPS下降信号负面）

**步骤6：可预测性**

- 2025年营收-18.94%，归母净利润-34.05%，业绩波动大
- 行业竞争加剧，护城河趋势恶化
- REGEN并购增加不确定性
- 判定：可预测性低

**步骤7：税务泄漏修正**

A股长期持股（>1年）免征股息税，Q = 0%

**步骤8：粗算穿透回报率**

```
M = 55.3%（3年支付率均值）
Q = 0%
O = 0

R = [C × M × (1 − Q%) + O] / 当前市值
R = [12.91 × 55.3% × 1 + 0] / 346.6
R = 7.14 / 346.6
R = 2.06%
```

### 粗算否决门判断

```
Rf = 1.762%
II = 3.762%
II × 0.5 = 1.881%

R = 2.06% > Rf = 1.762% ✅
R = 2.06% > II × 0.5 = 1.881% ✅
R = 2.06% < II = 3.762% ❌
```

**粗算结论：边际不达标** — R介于II×0.5与II之间，继续进入因子3精算。
