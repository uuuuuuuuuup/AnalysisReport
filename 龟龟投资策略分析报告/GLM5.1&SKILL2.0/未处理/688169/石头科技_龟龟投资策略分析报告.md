## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

**2024年度基准**：C = 19.77 亿元

**2025年度基准**：C = 13.60 亿元（业绩快报）

> 选用2024年度作为粗算基准（最新完整年度），同时参考2025年度数据

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）≈ 1.50 亿元
G（维持性资本开支系数）= 0.50（硬件制造企业中等水平）
H = D × G = 0.75 亿元
I（Owner Earnings）= C + D - H = 19.77 + 1.50 - 0.75 = 20.52 亿元
```

### 步骤 3：分配能力验证

| 检查项 | 2024年度 | 2025Q3（YTD） | 判断 |
|--------|----------|--------------|------|
| OCF | 17.34 亿元 | **-10.60 亿元** | ⚠️ 2025年恶化 |
| OCF > 0 | ✅ | ❌ | **一票否决风险** |
| OCF/净利润 | 0.88 | N/A | 2024年合格 |

> ⚠️ 2025年前三季度OCF转负，分配能力严重恶化。但2024年度OCF为正，粗算继续。

### 步骤 4：现金上游障碍

- 无有息负债，无重大上游现金障碍
- 但2025年OCF转负意味着现金流入不足以覆盖运营支出

### 步骤 5：分配意愿

| 参数 | 数值 | 说明 |
|------|------|------|
| 历史支付率 | ~10-42% | 近年趋降至~10% |
| 支付率锚定值 M | **15%** | 保守取值，略高于近期实际 |
| 年均回购 O | 1.00 亿元 | 2025年回购约1亿元（用于员工激励） |
| 回购是否注销 | 否 | 用于股权激励，非股东直接回报 |

**分配意愿评定：弱**

- 支付率仅~10%，远低于价值投资要求
- 回购用于员工激励而非注销，不构成真正的股东回报
- 管理层倾向于将利润留存用于扩张

### 步骤 6：可预测性

| 维度 | 评估 |
|------|------|
| 收入可预测性 | 中等（行业增速高但竞争激烈） |
| 利润可预测性 | **低**（2025年利润-31%，战略转型期波动大） |
| 现金流可预测性 | **低**（OCF从+17亿转负） |

### 步骤 7：税务泄漏修正

```
Q（综合股息税率）= 10%（保守估计，持股>1年为0%，<1年为10%）
```

### 步骤 8：粗算穿透回报率

**─── 主轨：市值口径 ───**

```
R = [C × M × (1 − Q%) + O] / 当前市值
R = [19.77 × 0.15 × 0.90 + 1.00] / 303.8
R = [2.67 + 1.00] / 303.8
R = 3.67 / 303.8
R = 1.21%
```

**─── 辅轨：EV 口径 ───**

```
广义净现金/市值 = 11.39 / 303.8 = 3.75%（保守口径）
                  ~40 / 303.8 = 13.2%（中等口径）
```

广义净现金/市值 < 40% → **EV 口径不触发**

**粗算否决门检查：**

| 否决条件 | 数值 | 判断 |
|----------|------|------|
| R < Rf | 1.21% < 1.76% | **是** |
| R < II × 0.5 | 1.21% < 1.88% | **是** |
| EV口径是否适用 | 广义净现金/市值 < 40% | **否** |

---

### 🔴 因子 2 粗算否决结论

**R = 1.21% 远低于门槛 II = 3.76%，且 EV 口径不适用。**

粗算穿透回报率仅为1.21%，不足A股门槛3.76%的三分之一，甚至低于十年期国债收益率1.76%。核心原因：

1. **支付率极低**（~10%）：公司仅将约10%的净利润分配给股东，远低于价值投资所需
