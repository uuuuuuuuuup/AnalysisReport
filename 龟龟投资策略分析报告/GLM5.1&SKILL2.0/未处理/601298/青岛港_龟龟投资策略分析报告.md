### 模块一：资本消耗强度

**评估：低**

- 港口基础设施为长寿命资产（折旧年限20-40年），维护性资本开支相对营收比例低
- 2024年FCF/OCF = 4,392/5,153 = 85.3%，资本开支占经营现金流比例仅14.7%
- 董家口港区157亿元投资为成长性资本开支，不影响维护性资本开支评估
- 维护性资本开支系数 G ≈ 0.4-0.5（港口行业特征）

### 模块二：收款模式

**评估：现结为主，应收可控**

- 应收账款周转天数：38-40天，处于合理水平
- 应收账款/总资产 = 3.2%，占比极低
- 坏账计提比例6.45%，计提充分
- 存货周转天数仅1.29天，港口企业几乎无存货积压
- 收款比率 ≈ 100%（应收账款小幅下降，现金回收良好）

### 模块三：竞争格局与护城河

**非技术性护城河：宽**

| 护城河类型 | 具体表现 |
|-----------|---------|
| 自然垄断 | 深水条件（水深18米+），北方唯一可停靠40万吨矿石船和45万吨油轮的港口 |
| 区位优势 | 北方距离国际主航道最近的港口，连接日韩与欧亚大陆桥 |
| 规模经济 | 集装箱吞吐量全球第5、北方第1，2024年3,217万TEU |
| 监管壁垒 | 港口经营许可证严格管控，新建港口审批极难 |
| 腹地锁定 | 覆盖山东及华北，延伸至西北（陕西、甘肃、新疆），货源粘性强 |

**技术性护城河：中等偏强**

- 自动化码头运营效率全球领先（全自动化集装箱码头）
- 港口大模型应用、AI中试基地
- 技术优势提升运营效率，但非不可替代的核心壁垒

**护城河评级：宽** — 自然垄断+区位+规模+监管四重壁垒

### 模块四：周期性

**评估：弱周期**

- 港口吞吐量与宏观经济/贸易量相关，但青岛港表现出较强韧性
- 2020-2025年营收CAGR ~7.3%，仅2023年微降0.1%
- 集装箱业务（占比最大）受周期影响小于散货
- 液体散货和干散货有一定周期性，但占比可控
- 多元化货种结构（集装箱+散货+液体散货+物流）降低周期波动

### 模块五：人力资本依赖

**评估：低**

- 自动化码头大幅降低人工依赖
- 员工支付占收入比例较低
- 核心竞争力在于自然禀赋和基础设施，而非人力

### 模块六：管理层与治理结构

**评估：良好（有改进空间）**

| 优势 | 不足 |
|------|------|
| 国资控股，经营稳健 | 管理层零持股，缺乏股权激励 |
| 分红政策持续优化（34%→45%） | 董事长/总经理未持有公司股份 |
| 同业竞争承诺明确（2027年前解决） | 回购授权尚未实施 |
| 审计意见持续标准无保留 | 关联方交易定价公允性待验证 |

### 模块七：监管与政策风险

**评估：中**

- 港口费率受政府指导价影响，提价空间有限
- 环保政策趋严（绿色港口要求），增加合规成本
- 中美贸易摩擦可能影响集装箱出口量
- 山东省港口一体化整合带来不确定性

### 模块八：MD&A 解读

**评估：中性偏正面**

- 2025年营收微降0.7%，但归母净利润增长0.7%，毛利率提升3.42pct → 定价权+成本控制
- 集装箱业务韧性突出，液体散货和干散货承压
- 董家口港区157亿投资为长期增长奠定基础，但短期增加资本开支压力
- 分红承诺40%+，中期+年度双分红机制优化股东回报

### 模块九：控股折价分析

**不适用** — 青岛港为运营型公司，非控股型公司

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 52.72 亿元（2025年度）

### 步骤 2：Owner Earnings 粗算

```
D = 折旧摊销 ≈ 12 亿元
G = 维护性资本开支系数 ≈ 0.5（港口行业保守估计）
H = D × G = 12 × 0.5 = 6 亿元
I = C + D − H = 52.72 + 12 − 6 = 58.72 亿元
```

Owner Earnings = **58.72 亿元**

### 步骤 3：分配能力验证

- OCF（2025）= 55.49 亿元
- Owner Earnings = 58.72 亿元
- OCF < Owner Earnings，差异3.23亿元，主要因权益法投资收益未产生等额经营现金流
- 实际可分配现金 = OCF − 维护性Capex = 55.49 − 6 = **49.49 亿元**
- 2025年股息 = 22.42 亿元
- 覆盖倍数 = 49.49 / 22.42 = **2.21×** → 充足

**分配能力验证：✅ 通过**

### 步骤 4：现金上游障碍

- A股公司，股息直接以人民币支付
- 无跨境税务障碍
- **无上游障碍**

- 中期+年度双分红机制已实施
- **分配意愿：强**

### 步骤 6：可预测性

- 5年营收CAGR 7.3%，波动率低
- 合营/联营企业投资收益有一定波动
- **可预测性：高**

### 步骤 7：税务泄漏修正

- Q = 0%

### 步骤 8：粗算穿透回报率

