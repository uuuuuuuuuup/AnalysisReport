### 模块一：资本消耗强度

**评级：低-中**

- 啤酒制造属于资产偏重行业，但青岛啤酒资本消耗强度可控
- 固定资产/总资产 ≈ 22.7%，属于中等水平
- 在建工程增长显著（6.31→12.32亿元），反映产能扩张投入
- 维持性资本开支占折旧摊销比例约60-80%，属正常范围
- 交易性金融资产激增208%至62.28亿元，闲置资金转向理财，说明主业资本消耗不高

### 模块二：收款模式

**评级：预收/现结为主**

- 应收账款周转天数仅1.1天，极短
- 现金循环周期为负值（-8至-12天），说明公司占用上游资金
- 合同负债76.74亿元，反映经销商预付款模式
- 收款比率 ≈ 97.9%，接近100%
- ⚠️ 注意：2025年应收账款同比增长约25%，合同负债下降7.68%，信用政策有放宽迹象

### 模块三：竞争格局与护城河

**第一层（非技术性护城河）：宽**

1. **品牌价值**：百年品牌，2025年品牌价值2,803.55亿元，国际市场最具知名度的中国啤酒品牌
2. **渠道网络**：覆盖全国20个省市，56家生产基地，线上销量连续13年增长
3. **规模经济**：年产销764.8万千升，行业第三，CR5>90%的寡头格局中占据稳固地位

**第二层（技术性护城河）：中-宽**

1. **产品结构升级**：中高端以上产品销量+5.2%，白啤品类行业第一
2. **酿造工艺**：百年配方+现代化生产线，品质壁垒
3. **区域壁垒**：沿黄市场传统优势极强，新进入者难以撼动

**护城河评级：宽**

### 模块四：周期性

**评级：弱周期**

- 啤酒属于大众消费品，需求刚性较强
- 行业总量进入存量阶段（零增长），但高端化趋势持续
- 季节性明显（Q4亏损），但年度层面波动小
- 收入5年CV（变异系数）估计<10%，稳定性高

### 模块五：人力资本依赖

**评级：低**

- 制造业属性，不依赖关键个人
- 国企管理体制，管理层更替不影响运营
- 核心竞争力在品牌和渠道，而非个人能力

### 模块六：管理层与治理结构

**评级：良好**

- 董事长姜宗祥：内部培养，具有丰富供应链和战略管理经验

- ⚠️ 蓬莱公司委托贷款7,197万元已全额计提坏账，风控有瑕疵

### 模块七：监管与政策风险

**评级：低-中**

- 酒类消费政策：暂无重大不利政策

## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤1：归母净利润**

C = 45.88 亿元（2025年归母净利润）

**步骤2：Owner Earnings 粗算**

D = 8.0 亿元（折旧摊销估算）
G = 0.65（维持性资本开支系数，啤酒行业典型值）
H = D × G = 8.0 × 0.65 = 5.20 亿元
I = C + D − H = 45.88 + 8.0 − 5.20 = 48.68 亿元

**步骤3：分配能力验证**

FCF = OCF − Capex = 47.40 − 12 = 35.40 亿元
I = 48.68 > FCF = 35.40 → Owner Earnings高于FCF，需关注现金质量

**步骤4：现金上游障碍**

A股直接持有，无跨境障碍。DD = 0

**步骤5：分配意愿**

M = 69.9%（2025年预案支付率）
O = 0（无回购）
最新派息信号：DPS已提升（2.35 vs 2.20元/股）
分配意愿判定：强

**步骤6：可预测性**

- 收入波动率低（5年CV<10%）
- 利润增长稳定（CAGR约5-7%）
- 现金流季节性明显但年度可预测
- 可预测性：高

**步骤7：税务泄漏修正**

Q = 0%（A股持股>1年免征红利税）

**步骤8：粗算穿透回报率**

R = [C × M × (1 − Q%) + O] / 当前市值
= [45.88 × 69.9% × 100% + 0] / 845.1
= 32.07 / 845.1
= **3.79%**

### 粗算否决门检查

| 检查项 | 条件 | 结果 |
|--------|------|------|
| R < Rf? | 3.79% < 1.762%? | ✅ 否 |
| R < II×0.5? | 3.79% < 1.881%? | ✅ 否 |
| II×0.5 ≤ R < II? | 1.881% ≤ 3.79% < 3.762%? | ✅ 否（R ≥ II） |

**粗算结论：✅ 通过（边际达标，R = 3.79% vs II = 3.762%，仅超出0.03pct）**

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤1：真实现金收入还原

S = 324.73 亿元（2025年营业收入）
T = 应收账款净变动 = 1.40 − 1.02 = +0.38 亿元（增加，扣除）
U = 合同负债净变动 = 76.74 − 83.13 = −6.39 亿元（减少）

保守处理：
- 应收账款净增加0.38亿元 → 扣除
- 合同负债净减少6.39亿元 → 不加回（对应未来交付义务减少）

真实现金收入 = S − T净增加 − U净减少 = 324.73 − 0.38 − 6.39 = **317.96 亿元**

收款比率 = 317.96 / 324.73 = **97.9%** ✅（极高，符合啤酒行业特征）
