### 模块一：资本消耗强度

**评估：中等**

- 三一重工作为工程机械制造商，属于资本密集型行业
- 固定资产占总资产约14%（21,436/156,897），属于中等水平
- 在建工程仅约10亿元，资本开支压力可控
- 研发费用率约5.8%（2025年），属于行业正常水平
- 2024年投资CF仅-11.58亿（含资产处置回收），2025年投资CF约-100亿（含对外投资），资本消耗波动较大
- **结论**：资本消耗强度中等，非轻资产模式但也非重资产消耗型

### 模块二：收款模式

**评估：应收为主，需关注**

- 应收账款周转天数：108天（2025Q3），较2024年的115天有所改善
- 应收账款/营业收入 = 27,318/89,231 ≈ 30.6%（年化）
- 收款比率估算：真实现金收入/营业收入 ≈ 98%（基于应收变动估算）
- 行业特征：工程机械行业普遍存在较长的信用账期，尤其国内基建客户
- 风险点：应收账款绝对金额较大（273亿元），坏账风险需持续关注
- **结论**：收款模式为"应收为主"，但周转天数改善中，需持续监控

### 模块三：竞争格局与护城河

**第一层（非技术性护城河）：窄护城河**

- 规模优势：全球工程机械三强，国内市占率26%（第二，徐工28%排第一）
- 品牌优势：在122个国家和地区至少一类产品市占率第一
- 渠道优势：全球180多个国家和地区业务覆盖
- 转换成本：中等，设备售后服务和配件体系有一定粘性

**第二层（技术性护城河）：正在构建**

- 电动化领先：电动搅拌车、无人摊压机群等领域已实现领先
- 智能化/无人化：无人驾驶工程机械商业化应用
- 研发投入：研发费用率5.8%，行业中等偏上
- **风险**：电动化技术壁垒尚未完全建立，卡特彼勒和小松同样在推进

**综合护城河评级：窄护城河**
- 规模和品牌提供一定保护，但行业竞争激烈
- 电动化转型可能带来护城河加深的机会，但尚不确定
- 国内市场面临徐工、中联重科激烈竞争

## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤 1：归母净利润**
- C = 84.08 亿元（2025年归母净利润）

**步骤 2：Owner Earnings 粗算**
- D（折旧摊销）≈ 32 亿元
- G（维持性资本开支系数）= 0.6（制造业典型值）
- H = D × G = 32 × 0.6 = 19.2 亿元
- I（Owner Earnings）= C + D - H = 84.08 + 32 - 19.2 = 96.88 亿元

**步骤 3：分配能力验证**
- OCF = 199.75 亿元 >> I = 96.88 亿元 → 分配能力充足，通过

**步骤 4：现金上游障碍**
- 少数股东损益占比 ≈ 1.2%（1/85），障碍极小
- A股上市公司，股息分配无结构性障碍
- **结论**：无显著现金上游障碍

**步骤 5：分配意愿**
- 近3年支付率序列：41.13% → 51.23% → 50.84%
- 2025年首次实施中期分红
- 2025年实施10-20亿元股份回购
- **结论**：分配意愿"中等偏强"，支付率提升趋势明确

**步骤 6：可预测性**
- 强周期行业，业绩波动大
- 归母净利润5年CV（变异系数）≈ 51%（极高）
- **结论**：可预测性"低"

**步骤 7：税务泄漏修正**
- A股持股超1年：股息个人所得税暂免 → Q = 0%
- **结论**：无税务泄漏

**步骤 8：粗算穿透回报率**

```
R = [C × M × (1 - Q%) + O] / 当前市值
  = [84.08 × 50% × 1 + 5] / 1,708.56
  = [42.04 + 5] / 1,708.56
  = 47.04 / 1,708.56
  = 2.75%
```

### 粗算否决门判断

- R = 2.75%
- Rf = 1.76%
- II = 3.76%
- II × 0.5 = 1.88%

**判断**：II × 0.5 (1.88%) ≤ R (2.75%) < II (3.76%)

**粗算结论：⚠️ 边际不达标，继续进入因子 3 精算**

> 这表明 W 或 Y 的估算可能偏高，或者经营现金流入被低估。

**替代方案：使用 OCF 倒推法**

```
W = 真实现金收入 - OCF = 87,430 - 19,975 = 67,455 百万元

真实可支配现金结余 = 真实现金收入 + V1 + V5 - V_deduct - W - Y
                    = 87,430 + 0 + 0 - 500 - 67,455 - 10,500
                    = 8,975 百万元 = 89.75 亿元
```

> ⚠️ 使用OCF倒推法估算W，标注：现金支出明细不足，已采用OCF倒推法估算W

**合理性校验：**

```
AA_fcf = OCF - Capex = 19,975 - 5,000 = 14,975 百万元

比较 AA 与 AA_fcf：
AA = 8,975 百万元
AA_fcf = 14,975 百万元
AA / AA_fcf = 0.60

AA < AA_fcf × 1.5 → 校验通过，直接使用 AA
```

**历史均值计算（可用数据有限）：**

| 年份 | OCF(百万元) | Capex(估) | AA_fcf | AA(估) |
|------|-----------|----------|--------|--------|
