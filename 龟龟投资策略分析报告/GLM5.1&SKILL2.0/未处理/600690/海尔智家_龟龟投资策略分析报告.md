## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 19,553 百万元 = **195.53 亿元**

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）= ~3,500 百万元 = 35 亿元
G（维持性资本开支系数）= 0.6（制造业典型值）
H = D × G = 35 × 0.6 = 21 亿元
I（Owner Earnings）= C + D - H = 195.53 + 35 - 21 = 209.53 亿元
```

### 步骤 3：分配能力验证（一票否决）

```
OCF = 26,003 百万元 = 260.03 亿元
Owner Earnings = 209.53 亿元
OCF / Owner Earnings = 1.24 > 1 ✅

OCF > 归母净利润 = 260.03 > 195.53 ✅
→ 分配能力验证通过
```

### 步骤 4：现金上游障碍

|----------|------|

### 步骤 8：粗算穿透回报率

```
R = [I × M × (1 - Q%) + O] / 当前市值
  = [209.53 × 55% × (1 - 10%) + 13] / 1,954
  = [209.53 × 0.55 × 0.9 + 13] / 1,954
  = [103.72 + 13] / 1,954
  = 116.72 / 1,954
  = 5.97%
```

### 粗算否决门检查

| 检查项 | 结果 | 说明 |
|--------|------|------|

| II × 0.5 ≤ R < II？ | 5.97% > 3.762% ✅ | 不适用，R已超过II |

### 因子 2 结论：✅ 通过

**粗算穿透回报率 R = 5.97%**，远超门槛 II = 3.762%，粗算通过。
