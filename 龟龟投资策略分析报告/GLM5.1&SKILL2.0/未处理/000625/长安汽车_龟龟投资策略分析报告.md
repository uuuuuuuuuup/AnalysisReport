### 模块一：资本消耗强度

**评估：高**

- **Capex/Revenue**：2024FY约3.7%，看似不高，但绝对金额59.30亿元，且FCF持续为负
- **R&D/Revenue**：2025FY达7.67%（125.76亿元），且资本化率高达43.08%（54.18亿元资本化）
- **R&D资本化率**：43.08%偏高，存在利润调节空间，实际研发支出对现金的消耗更大
- **FCF**：2024FY为-10.81亿元，2025FY估算约-41.63亿元，持续为负
- **未来投资计划**：2026年计划投资144.7亿元，未来十年计划投入约2,000亿元（年均200亿元）

**结论**：长安汽车处于新能源转型重投资期，资本消耗强度极高。研发资本化率43.08%需警惕利润虚增风险。

---

### 模块二：收款模式

**评估：应收为主（恶化趋势）**

- **应收账款周转天数**：从2024FY的6.55天大幅增加至2025Q3的12.47天，接近翻倍
- **应收账款/归母净利润**：2025年高达102.98%，存在利润"纸面化"风险
- **应收账款集中度**：前五名余额合计203.80亿元，占应收和合同资产合计83.34%
- **收款比率**：估算约99.5%（应收增幅相对营收较小），但趋势恶化明显

**结论**：收款模式从现结向应收偏移，周转天数翻倍、应收/净利润超100%是严重警示信号。

### 模块三：竞争格局与护城河

**评估：窄护城河**

**非技术性护城河**：
- 央企背景提供政策资源和产业支持，但非独占性优势
- 多品牌矩阵（启源+深蓝+阿维塔）覆盖全价格带，但品牌溢价能力有限
- 市场份额约5.9%，行业排名第4，非绝对龙头

**技术性护城河**：
- L3/L4级自动驾驶技术领先，获国内首块L3级自动驾驶正式号牌
- 研发投入126亿元（2025FY），但比亚迪等竞争对手投入更大
- 新能源技术壁垒不高，行业同质化竞争严重

**竞争格局**：
- 比亚迪绝对领先，吉利紧随其后，新势力（理想/零跑/小米）增长强劲
- 价格战常态化，行业利润率持续承压
- 合资品牌份额持续下滑，长安福特/长安马自达贡献递减

**结论**：护城河窄且面临收窄风险。央企背景和多品牌布局提供一定缓冲，但在激烈价格战和技术快速迭代的环境下，竞争优势不够稳固。

### 模块四：周期性

**评估：强周期**

- 汽车行业具有明显周期性，受宏观经济、消费信心、政策补贴影响大
- 2020-2023年归母净利润波动：33→36→78→113亿，波动幅度极大

- 新能源转型期行业格局重塑，周期性叠加结构性变化
- 当前处于新能源渗透率突破50%的关键转型期

**结论**：强周期行业，且处于结构性转型期，盈利波动性极高。

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 40.75 亿元（2025FY）

### 步骤 2：Owner Earnings 粗算

```
D = 折旧摊销 ≈ 37 亿元（估算）
G = 维持性资本开支系数 = 1.0（汽车行业，取保守值）
H = D × G = 37 亿元
I = C + D - H = 40.75 + 37 - 37 = 40.75 亿元
```

> 注：当G=1.0时，Owner Earnings = 归母净利润，即假设全部折旧摊销需再投入。

### 步骤 3：分配能力验证（一票否决）

```
OCF（2025FY）≈ 18.37 亿元
归母净利润 = 40.75 亿元
OCF/归母净利润 = 45.1% < 70%

⚠️ 分配能力验证：不通过
OCF远低于归母净利润，利润质量差，大量利润未转化为现金
```

### 步骤 4：现金上游障碍

- 应收账款同比增长23.4%，周转天数翻倍
- 经营现金流同比-62.15%
- 现金上游障碍：**显著**

### 步骤 5：分配意愿

- 支付率40%，央企有分红要求
- 但OCF不足以覆盖股息支付
- 分配意愿：**中等**（有意愿但能力不足）

### 步骤 6：可预测性

- 汽车行业强周期，盈利波动极大
- 新能源转型期，未来盈利难以预测
- 可预测性：**低**

### 步骤 7：税务泄漏修正

- A股持股>1年免税
- Q = 0%

### 步骤 8：粗算穿透回报率

```
─── 市值口径 ───
R = [C × M × (1 - Q%) + O] / 当前市值
  = [40.75 × 40% × 1 + 0] / 997.2
  = 16.30 / 997.2
  = 1.63%

─── EV口径 ───
广义净现金/市值 = 522.22 / 997.2 = 52.4% > 40% → 触发EV口径
EV = 997.2 - 522.22 = 474.98 亿元
R_EV = [40.75 × 40% × 1 + 0] / 474.98
     = 16.30 / 474.98
     = 3.43%
```

### 粗算否决门判断

```
R = 1.63% < Rf = 1.762% → 检查EV口径
R_EV = 3.43%
II × 0.5 = 1.881% ≤ R_EV = 3.43% < II = 3.762%

→ 标注"边际不达标"，继续进入因子3
```

**粗算结论：⚠️ 边际不达标**（EV口径接近但未达门槛）
