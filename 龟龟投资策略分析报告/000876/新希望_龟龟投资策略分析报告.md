【因子 2 粗算穿透回报率】

**步骤1：归母净利润**
C = 4.74 亿元（2024FY）

**步骤2：Owner Earnings 粗算**
D（折旧摊销）≈ 21.74 亿元
G（维持性资本开支系数）= 0.75（资本密集型行业）
H = D × G = 21.74 × 0.75 = 16.31 亿元

可预测性：极低

**步骤7：税务泄漏修正**
Q = 0%（A股长期持有免税）

**步骤8：粗算穿透回报率**

─── 主轨：市值口径 ───
R = [C × M × (1-Q%) + O] / 市值
  = [4.74 × 0.023 × 1 + 0] / 396.24
  = 0.109 / 396.24
  = **0.028%**

─── 辅轨：EV口径 ───
广义净现金 = 86.99 - 520 = -433.01 亿元（净负债！）
EV = 396.24 - (-433.01) = 829.25 亿元
R_EV = [4.74 × 0.023 × 1 + 0] / 829.25
     = **0.013%**

─── 敏感性测试 ───
| 支付率假设 | 市值口径R | EV口径R_EV | vs门槛3.76% |
|-----------|----------|-----------|------------|
| M=2.3%（实际） | 0.028% | 0.013% | 远不达标 |
| M=10% | 0.12% | 0.057% | 远不达标 |
| M=30% | 0.36% | 0.17% | 远不达标 |
| M=50% | 0.60% | 0.29% | 远不达标 |
| M=100% | 1.20% | 0.57% | 仍不达标 |

**粗算否决门检查：**
- R = 0.028% < Rf = 1.76% → 检查EV口径
- R_EV = 0.013% < Rf = 1.76% → EV口径仍不通过
- R = 0.028% < II × 0.5 = 1.88% → 检查EV口径
- R_EV = 0.013% < II × 0.5 = 1.88% → EV口径仍不通过
- 即使M=100%，R = 1.20% < II = 3.76% → 最慷慨假设仍不达标

**粗算结论：❌ 否决**
