**评估维度**：维持当前盈利水平是否需要持续大量资本投入？

**分析**：

- 折旧摊销可覆盖大部分维持性Capex

**评级：中高** — 扩张期资本消耗较高，但维持性投入可控，且折旧可覆盖大部分

**分析**：

- **B2B合同物流/供应链**：月结客户，存在应收账款 → 先供后收
- **月结客户**：账期约30-45天

- **现金循环周期为-1.38天** → 整体上先收后供

**评级：混合型（偏先收后供）** — B2C预付+B2B月结，现金循环周期为负

## 四、因子2：自上而下回报率

### 4.1 Owner Earnings 计算（标准公式）

**公式**：Owner Earnings = C + D + SBC - E

| 参数 | 值 | 来源/说明 |
|------|-----|---------|
| C（归母净利润） | 111.17 亿元 | 2025年全年 |
| D（折旧摊销） | ~55 亿元 | 估算：固定资产540亿×8%+无形资产摊销 |
| SBC（股权激励） | ~2 亿元 | 估算 |
| E（资本支出） | ~120 亿元 | 2024年投资活动现金流净额（保守替代） |

**Owner Earnings** = 111.17 + 55 + 2 - 120 = **48.17 亿元**

### 4.2 粗算穿透回报率 R

| 参数 | 值 |
|------|-----|
| M（支付率） | 40% |
| Q（税务漏损） | 0% |
| O（年均回购） | 15 亿元 |

**R** = (48.17 × 40% × (1-0%) + 15) / 1,835.8
= (19.27 + 15) / 1,835.8
= 34.27 / 1,835.8
= **1.87%**

### 4.3 粗算否决门检查

| 检查项 | 条件 | 结果 |
|--------|------|------|
| R < Rf? | 1.87% < 1.81%? | ❌ 否（R略高于Rf） |
| R < II × 0.5? | 1.87% < 1.91%? | ✅ 是 |
| II × 0.5 ≤ R < II? | 1.91% ≤ 1.87% < 3.81%? | ❌ 否（R < II×0.5） |

**粗算结果：R = 1.87%，低于II×0.5 = 1.91%**

→ 需要检查EV口径

### 4.4 EV口径检查

| 参数 | 值 |
|------|-----|
| 现金及等价物 | 339.36 亿元 |
| 有息负债（估） | ~96.86 亿元 |
| 净现金（狭义） | 242.50 亿元 |
| 净现金/市值 | 13.2% |

净现金/市值 = 13.2% < 20% → **无现金保护**

EV = 1,835.8 - 242.50 = 1,593.3 亿元

R_EV = 34.27 / 1,593.3 = **2.15%**

### 4.5 粗算结论

R = 1.87%，R_EV = 2.15%，均低于 II = 3.81%

**粗算口径：⚠️ 未达标** — 但粗算公式未充分反映顺丰的经营性现金流优势（负现金循环周期），需进入因子3精确计算
