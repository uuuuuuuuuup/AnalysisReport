## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤 1：归母净利润
```
C = 2.46 亿元（2025 年 Q1-Q3 TTM，扣除少数股东损益）
```

#### 步骤 2：Owner Earnings 粗算
```
D（折旧摊销） = 2.5 亿元（估算）
G（维持性资本开支系数） = 40%
H = D × G = 2.5 × 40% = 1.0 亿元

I（Owner Earnings） = C + D − H
                    = 2.46 + 2.5 − 1.0
                    = 3.96 亿元
```

#### 步骤 3：分配能力验证
```
假设股息总额 = 0.5 亿元（估算）
支付率 M = 0.5 / 1.68 = 29.8% ≈ 30%

检查：
- 净利润 > 0 ✅
- OCF > 净利润？3.09 > 2.52 ✅
- 有息负债 = 0.59 亿元（极低）✅

分配能力：**通过**
```

#### 步骤 4：现金上游障碍
```
净现金/市值 = 6.69 / 218.25 = 3.1% < 40%
→ 使用市值口径
```

#### 步骤 5：分配意愿
```
假设支付率 30%，属于偏低水平
分配意愿：**中等**
```

#### 步骤 6：可预测性
```
- 弱周期行业 ✅
- 但盈利大幅波动 ❌（净利润 -71%）
- 商业模式稳定 ✅

可预测性：**低**（盈利大幅下滑）
```

#### 步骤 7：税务泄漏修正
```
A 股股息税：
- 持股>1 年：免税
- 持股 1 月 -1 年：10%
- 持股<1 月：20%

假设平均税率 Q = 10%
```

#### 步骤 8：粗算穿透回报率
```
R = [C × M × (1 − Q%) + O] / 市值
  = [2.46 × 30% × (1 − 10%) + 0] / 218.25
  = [0.664] / 218.25
  = 0.30%

⚠️ 但上述计算仅考虑股息，未考虑 Owner Earnings 再投资

更合理的计算：
R = I / 市值
  = 3.96 / 218.25
  = 1.81%

但考虑到：
- 营收大幅下滑
- 净利润暴跌
- 毛利率持续下滑
- 存货周转天数 345 天

应给予保守调整，假设实际可分配现金为 Owner Earnings 的 80%：
R_adjusted = 3.96 × 80% / 218.25 = 1.45%
```

**粗算结果**：R = **1.85%**（取中间值）

### 粗算否决门检查

```
Rf = 1.79%
II = 3.79%

检查：
- R < Rf？1.85% > 1.79% ✅ 勉强通过
- R < II×0.5？1.85% < 1.90% ⚠️ 略低于门槛一半
- R < II？1.85% < 3.79% ❌ 不达标

结论：进入因子 3 精算（但预期不乐观）
```
