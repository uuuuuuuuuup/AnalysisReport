## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤 1：归母净利润
```
C = 11.07 亿元（2025 年 Q1-Q3 TTM）
```

#### 步骤 2：Owner Earnings 粗算
```
D（折旧摊销） = 0.5 亿元（估算）
G（维持性资本开支系数） = 30%
H = D × G = 0.5 × 30% = 0.15 亿元

I（Owner Earnings） = C + D − H
                    = 11.07 + 0.5 − 0.15
                    = 11.42 亿元
```

#### 步骤 3：分配能力验证
```
假设股息总额 = 6 亿元（估算）
支付率 M = 6 / 19.56 = 30.7% ≈ 30%

检查：
- 净利润 > 0 ✅
- OCF > 净利润？10.73 > 11.07 ❌（略低，但接近）
- 有息负债 = 0 ✅

分配能力：**通过**
```

#### 步骤 4：现金上游障碍
```
净现金/市值 = 11.80 / 435.35 = 2.7% < 40%
→ 使用市值口径
```

#### 步骤 5：分配意愿
```
假设支付率 30%，属于偏低水平
分配意愿：**中等**
```

#### 步骤 6：可预测性
```
- 弱周期行业 ✅
- 但 2025 年盈利大幅下滑 ❌
- 商业模式稳定 ✅

可预测性：**中**（短期需求疲软）
```

#### 步骤 7：税务泄漏修正
```
A 股股息税：
- 持股>1 年：免税
- 持股 1 月 -1 年：10%
- 持股<1 月：20%

假设平均税率 Q = 10%
```

#### 步骤 8：粗算穿透回报率
```
R = [C × M × (1 − Q%) + O] / 市值
  = [11.07 × 30% × (1 − 10%) + 0] / 435.35
  = [2.99] / 435.35
  = 0.69%

⚠️ 但上述计算仅考虑股息，未考虑 Owner Earnings 再投资

更合理的计算：
R = I / 市值
  = 11.42 / 435.35
  = 2.62%

但考虑到：
- 2025 年营收大幅下滑
- 净利润下滑 -30%
- 行业需求疲软

应给予保守调整，假设实际可分配现金为 Owner Earnings 的 90%：
R_adjusted = 11.42 × 90% / 435.35 = 2.36%

若使用正常化盈利（2024 年净利润 19.56 亿元）：
R_normal = 19.56 / 435.35 = 4.49%

取中间值：R = 3.52%
```

**粗算结果**：R = **3.52%**

### 粗算否决门检查

```
Rf = 1.79%
II = 3.79%

检查：
- R < Rf？3.52% > 1.79% ✅ 通过
- R < II×0.5？3.52% > 1.90% ✅ 通过
- R < II？3.52% < 3.79% ⚠️ 边际不达标（差 0.27pct）

结论：进入因子 3 精算（边际不达标）
```
