### 模块一：资本消耗强度

**资本消耗率 = Capex / OCF = 28.48 / 48.82 = 58.3%**

爱尔眼科的资本消耗强度为 **高**。公司采用"体外产业基金孵化+上市公司收购"的扩张模式，投资现金流中大量用于并购收购（约20-22亿元），维持性资本开支约6-8亿元。

- 维持性资本开支占比：~25%
- 成长性/并购资本开支占比：~75%
- **资本消耗强度评级：高**（并购驱动的增长模式消耗大量现金）

---

### 模块二：收款模式

- 应收账款周转天数：33.39天（2024年），31.80天（2025Q3）
- 收款比率 ≈ 99.3%（估算，基于应收变动）
- 医疗服务中，医保结算存在一定账期，但整体收款情况良好
- **收款模式评级：现结+应收混合**（偏向现结，医保部分有账期）

---

### 模块三：竞争格局与护城河

**第一层：非技术性护城河**
- **规模优势**：全球900+家眼科医院/中心，市占率31.4%，远超第二名华厦眼科（5%）
- **品牌优势**："爱尔"是中国最知名的眼科医疗品牌
- **网络效应**：覆盖亚美欧三大洲，转诊网络完善
- **转换成本**：患者对医院和医生的信任构成一定转换成本

**第二层：技术性护城河**
- **医疗技术**：引进国际先进设备和技术，但非独占性
- **数据优势**：庞大患者数据库支撑临床研究
- **人才培养**：完善的医生培训体系

**护城河评级：窄护城河**

规模和品牌优势显著，但商业模式可复制（华厦、普瑞等竞争对手正在模仿），且护城河主要依赖持续并购维持。商誉占净资产40.8%，一旦并购整合不力，护城河可能被侵蚀。

## 【因子 2：穿透回报率粗算（Top-Down）】
═══════════════════════════════════════

### 1. 归母净利润

GAAP 归母净利润 [C] = 35.56 亿元

### 2. Owner Earnings 粗算

Owner Earnings = OCF − Capex = 48.82 − 28.48 = 20.34 亿元（集团口径）

归母口径 Owner Earnings ≈ 20.34 × (35.56/37.36) ≈ 19.36 亿元

### 3. 分配能力验证（一票否决）

- OCF = 48.82 亿元
- 股息 + 回购 = 14.87 + 2.93 = 17.80 亿元
- OCF / 分配 = 2.74x ✅（分配能力充足）

### 4. 现金上游障碍

- 少数股东损益占比 = 1.80/37.36 = 4.8%
- 大部分子公司为全资或控股
- **现金上游障碍：低**

### 5. 分配意愿

- 支付率持续提升：~28% → ~35% → 41.82%
- 回购计划持续执行
- **分配意愿：中等偏强**（改善趋势明确）

### 6. 可预测性

- 营收增速大幅放缓（3.02%）
- 扣非净利润首次负增长
- OCF 相对稳定
- **可预测性：中等**（增速放缓但现金流稳定）

### 7. 税务泄漏修正

- A股持有>1年：股息免税，Q% = 0%
- A股持有1月-1年：Q% = 10%
- **按长期持有假设：Q% = 0%**

### 8. 粗算穿透回报率 R%

**市值口径：**
R = [归母净利润 × M + O] / 市值
= [35.56 × 40% + 2.93] / 896.67
= [14.22 + 2.93] / 896.67
= 17.15 / 896.67
= **1.91%**

**EV口径：**
EV = 市值 − 广义净现金 = 896.67 − 38.64 = 858.03 亿元
R_EV = 17.15 / 858.03 = **2.00%**

**粗算否决门判断：**
- R = 1.91% > Rf = 1.81% ✅（勉强超过无风险利率）
- R = 1.91% > II × 0.5 = 1.905% ✅（勉强超过半门槛）
- R = 1.91% < II = 3.81% ❌（未达门槛）

**粗算结论：⚠️ 边际不达标**（II×0.5 ≤ R < II，继续进入因子3）
