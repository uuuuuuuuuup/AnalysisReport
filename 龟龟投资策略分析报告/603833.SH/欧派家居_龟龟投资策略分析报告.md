### 3.2 1B定性：商业模式与竞争优势分析

#### 3.2.1 商业模式

欧派家居是中国最大的定制家居企业，主营业务涵盖整体橱柜、定制衣柜、卫浴、木门等全品类家居产品。

**收入结构：**
- 橱柜：核心品类，行业绝对龙头
- 衣柜：第二大品类，增速较快
- 卫浴/木门：配套品类，完善全屋定制矩阵

**渠道模式：**
- 经销商模式为主：截至2025H1拥有7709家门店（欧派品牌5031家+欧铂丽978家+其他）
- 直营模式为辅
- 大宗业务（工程渠道）

**盈利模式：**
- 品牌溢价+规模效应+供应链整合
- 经销商预收款模式，现金流优秀
- 2024年经营现金流净额54.99亿元，远超净利润25.99亿元

### 3.3 2粗算：粗算穿透收益率

**粗算穿透收益率 = EPS / 股价**

使用2024年度数据：
- 2024年EPS = 4.29元
- 当前股价 = 50.57元
- **粗算穿透收益率 = 4.29 / 50.57 = 8.48%**

对比基准：
- 10年期国债收益率 = 1.79%
- 穿透收益率溢价 = 8.48% - 1.79% = **6.69个百分点**

> 粗算穿透收益率8.48%显著高于无风险利率1.79%，溢价6.69个百分点，初步显示具备投资吸引力。

### 3.4 3精算：精算穿透收益率

#### 3.4.1 基于扣非净利润的精算

2024年扣非净利润 = 23.16亿元  
扣非EPS = 23.16 / 6.091529 = 3.80元  
**精算穿透收益率（扣非） = 3.80 / 50.57 = 7.52%**

#### 3.4.2 基于5年平均净利润的精算

5年平均净利润 = (20.6 + 26.6 + 26.88 + 30.36 + 25.99) / 5 = 26.09亿元  
5年平均EPS = 26.09 / 6.091529 = 4.28元  
**精算穿透收益率（5年均） = 4.28 / 50.57 = 8.47%**

#### 3.4.3 基于经营现金流的精算

2024年经营现金流净额 = 54.99亿元  
经营现金流/股 = 9.03元  
**现金流穿透收益率 = 9.03 / 50.57 = 17.86%**

5年平均经营现金流/股 ≈ (5.0 + 6.64 + 3.96 + 8.01 + 9.03) / 5 ≈ 6.53元  
**5年平均现金流穿透收益率 = 6.53 / 50.57 = 12.91%**

#### 3.4.4 基于股息的精算

2024年每股分红 = 2.48元  
**股息率 = 2.48 / 50.57 = 4.90%**

5年平均每股分红 ≈ (1.09 + 1.75 + 1.77 + 2.76 + 2.48) / 5 ≈ 1.97元  
**5年平均股息率 = 1.97 / 50.57 = 3.90%**

#### 3.4.5 精算穿透收益率汇总

| 计算方法 | 穿透收益率 | 可信度 |
|---------|----------|--------|
| 扣非EPS精算 | 7.52% | ★★★★☆ |
| 5年平均EPS精算 | 8.47% | ★★★★★ |
| 当年现金流精算 | 17.86% | ★★★☆☆（波动大） |
| 5年平均现金流精算 | 12.91% | ★★★★☆ |
| 当年股息率 | 4.90% | ★★★★★ |
| 5年平均股息率 | 3.90% | ★★★★☆ |

**综合精算穿透收益率：7.52% ~ 8.47%（取扣非和5年均值区间）**

> 精算穿透收益率7.52%-8.47%，远高于10年期国债收益率1.79%，溢价5.73-6.68个百分点。经营现金流穿透收益率更高（12.91%），反映公司现金创造能力优秀。股息率4.90%在A股市场属于较高水平。
