### 模块一：资本消耗强度

**评级：中等**

- 固定资产/营收比：26/62.65 = 41.5%（较高，生产基地投入大）
- Capex/OCF：5.0/12.14 = 41.2%（资本消耗占经营现金流比例中等）
- 在建工程3.58亿元，仍有持续资本投入需求
- 2021年定增募资31亿元用于生产基地建设，资本消耗周期较长

**结论**：资本消耗强度中等，非轻资产模式，但也不属于重资产行业。

---

### 模块二：收款模式

**评级：现结为主，应收可控**

- 应收账款周转天数：18.39天（2025-09-30），较2024年末10.91天有所上升
- 收款比率估算：~98.6%（基于应收变动调整后）
- 应收账款从2024年末1.65亿增至2025Q3的5.04亿，增幅显著，需关注
- 但绝对金额相对营收仍属可控（5.04/62.65 = 8.0%）
- VDS行业以经销商预付款/现结模式为主

**结论**：收款模式以现结为主，但应收账款上升趋势需持续监控。

### 模块三：竞争格局与护城河

#### 第一层：非技术性护城河

**品牌护城河：窄至中等**
- 汤臣倍健品牌认知度较高，但VDS行业品牌忠诚度有限
- 三大品牌（汤臣倍健/健力多/Life-Space）覆盖不同细分市场
- 行业CR3仅22%，格局分散，无绝对龙头

**渠道护城河：窄**
- 线下药店渠道优势正在被线上渠道侵蚀
- 线上渠道竞争激烈，价格透明度高
- 2024-2025年渠道调整导致收入大幅下滑

**规模护城河：窄**
- 市场份额9.3%为行业第一，但优势不明显
- 规模效应主要体现在生产端，但VDS生产门槛不高

#### 第二层：技术性护城河

**评级：弱**
- 研发费用率仅~2%，远低于销售费用率~39%
- 境内专利462项，但配方专利保护力度有限
- VDS产品同质化程度高，技术壁垒低
- 核心竞争力更多依赖营销而非研发

**综合护城河评级：窄** — 品牌有一定认知度，但行业竞争激烈、产品同质化、转换成本低，护城河不宽。

### 模块四：周期性

**评级：弱周期**

- VDS行业需求相对稳定，受经济周期影响较小
- 但汤臣倍健2024-2025年业绩大幅波动，更多是公司自身因素（渠道调整、库存去化）而非行业周期
- 行业增速~3.7%，稳健增长
- 公司业绩波动性高于行业，反映经营策略风险

**结论**：行业弱周期，但公司自身业绩波动性大，需区分行业周期与公司经营周期。

---

### 模块五：人力资本依赖

**评级：中等**

- VDS行业对核心研发人员有一定依赖，但更多依赖品牌和渠道
- 销售团队是核心资源（销售费用率39%）
- 管理层稳定性较好，梁允超为创始人，长期掌舵
- 人力投入回报率（ROI）从2023年273.6%降至2025Q3的164.2%

**结论**：人力资本依赖中等，核心依赖销售团队和管理层战略决策。

**结论**：管理层利益绑定良好，分红意愿强，但战略执行能力有待验证。

## 【因子 2：穿透回报率粗算（Top-Down）】

### 步骤 1：归母净利润

2025年归母净利润 = **7.82 亿元**

### 步骤 2：Owner Earnings 粗算

```
Owner Earnings = 归母净利润 + D&A − Capex
              = 7.82 + 2.0 − 5.0
              = 4.82 亿元
```

对比 OCF − Capex = 12.14 − 5.0 = 7.14 亿元

两者差异 = 7.14 − 4.82 = 2.32 亿元，主要来自：
- Q4非现金损失（可能含商誉减值或资产减值）
- 营运资本变动

### 步骤 3：分配能力验证

```
FCF（OCF口径）= 7.14 亿元
股息支付 = 7.54 亿元
FCF / 股息 = 7.14 / 7.54 = 94.7%
```

**⚠️ 分配能力边际不足**：FCF几乎全部用于分红，留存极少。若利润进一步下滑，分红可持续性存疑。

### 步骤 4：现金上游障碍

A股直接持有，**无显著现金上游障碍**。

### 步骤 5：分配意愿

- 近3年支付率序列：86.9% → 92.8% → 96.4%
- DPS已提升（10派3.6→10派4.5）
- **分配意愿：强**

### 步骤 6：可预测性

- 营收连续2年下滑（-27.3%, -8.38%）
- 净利润波动极大（-62.6%, +19.8%）
- **可预测性：低**

### 步骤 7：税务泄漏修正

A股直接持有，持股超1年股息免税。
**Q% = 0%**

### 步骤 8：粗算穿透回报率

```
════ 主轨：市值口径 ════

方案A（保守，M=70%）：
R = (7.82 × 70% + 1.5) / 186.6
  = (5.47 + 1.5) / 186.6
  = 6.97 / 186.6
  = 3.73%

方案B（乐观，M=90%）：
R = (7.82 × 90% + 1.5) / 186.6
  = (7.04 + 1.5) / 186.6
  = 8.54 / 186.6
  = 4.58%

════ 辅轨：EV口径 ════
EV = 186.6 − 8.08（狭义净现金）= 178.52 亿元

方案A（M=70%）：
R_EV = 6.97 / 178.52 = 3.90%

方案B（M=90%）：
R_EV = 8.54 / 178.52 = 4.78%
```

### 粗算否决门判断

| 方案 | R（市值口径） | R_EV（EV口径） | vs II=3.81% | 结论 |
|------|-------------|---------------|------------|------|
| A（M=70%） | 3.73% | 3.90% | 市值口径<II，EV口径>II | 边际不达标 |
| B（M=90%） | 4.58% | 4.78% | 均>II | 通过 |

**粗算结论**：
- 保守口径（M=70%）：**边际不达标**，需进入因子3精算验证
- 乐观口径（M=90%）：**通过**，但支付率可持续性存疑
