### 模块一：资本消耗强度

**评级：高**

保利发展作为房地产开发企业，资本消耗强度极高：
- 2024年存货7,986亿元，占总资产59.8%
- 2024年土地投资683亿元，计划2025年土地投资1,000亿元
- 存货周转天数1,119天（2024年），资金占用周期极长
- 经营性现金流虽连续7年为正，但金额从2020年151.5亿元降至2024年62.6亿元

**定性参数**：资本消耗强度 = **高**（房地产开发天然高资本消耗行业）

---

### 模块二：收款模式

**评级：预收为主**

- 房地产以预售模式为主，签约回笼率超100%（2024年）
- 合同负债/预收款规模庞大
- 应收账款仅49.32亿元，占总资产0.37%，占比极低
- 应收账款周转天数5.06天（2024年），收款效率高
- 但需注意：预售模式意味着未来交付义务，已收现金并非可自由支配

**定性参数**：收款模式 = **预收为主**（行业特性，收款效率高但对应未来交付义务）

### 模块三：竞争格局与护城河（两层递进）

#### 非技术性分析
- **品牌优势**：连续15年行业品牌第一，央企信用背书
- **规模优势**：销售金额连续两年行业第一
- **融资优势**：综合融资成本3.1%，行业极低水平
- **土地储备**：总土储12,574万方，资源充足

#### 技术性分析
- **毛利率持续下滑**：从35%降至14%，护城河正在被侵蚀
- **同业对比劣势**：归母净利率1.6%远低于中海8.5%、华润9.2%
- **利润四年暴跌81.7%**：规模第一但盈利能力行业垫底
- **三四线下沉后遗症**：前期冲规模下沉低能级城市，利润被蚕食

**护城河评级：窄且正在收窄**
- 央企信用和融资优势构成窄护城河
- 但盈利能力持续恶化，护城河正在收窄
- 规模优势未转化为利润优势

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

2024年归母净利润 = **50.01 亿元**

### 步骤 2：Owner Earnings 粗算

```
Owner Earnings = 归母净利润 + 折旧摊销 − 资本开支（维持性）
             ≈ 50.01 + 25 − 5
             ≈ 70 亿元
```

**注**：房地产企业维持性资本开支极低（PPE投入少），但土地获取和建安支出通过存货和经营现金流体现，不在此处扣除。

### 步骤 3：分配能力验证

- 2024年经营性现金流62.6亿元 > 股息20.4亿元 ✅
- 但OCF同比下降55%，趋势恶化 ⚠️
- 2025年业绩快报归母净利润仅10.26亿元，分配能力严重削弱

### 步骤 4：现金上游障碍

- 少数股东损益占比高：47.37/97.38 = 48.6%
- 大量合作开发项目，利润分配至少数股东
- OCF归母口径 ≈ 62.6 × (50.01/97.38) ≈ 32.2 亿元

### 步骤 5：分配意愿

- 近3年平均支付率37.3%，偏低
- 2024年利润分配方案延迟审议，分配意愿存疑
- 无回购计划
- **分配意愿评级：弱**

### 步骤 6：可预测性

- 归母净利润4年连续下滑：273.9→183.5→120.7→50.0→10.3（2025E）
- 下滑幅度逐年加大
- 行业底部不确定
- **可预测性评级：极低**

### 步骤 7：税务泄漏修正

- A股股息税：持股>1年免税，≤1年10%-20%
- 假设长期持有，税务泄漏Q% = 0%

### 步骤 8：粗算穿透回报率

```
─── 市值口径 ───
粗算穿透回报率 R = (Owner Earnings归母 × M + O) / 当前市值
                 = (32.2 × 37.3% + 0) / 712.2
                 = 12.0 / 712.2
                 = 1.68%

─── EV口径 ───
EV = 当前市值 + 净负债 = 712.2 + 2,146.3 = 2,858.5 亿元
粗算穿透回报率 R_EV = 12.0 / 2,858.5 = 0.42%
```

### 粗算否决门判断

- R = 1.68% < Rf = 1.82% → **不通过**
- R = 1.68% < II = 3.82% → **不通过**
- R_EV = 0.42% < II × 0.5 = 1.91% → **EV口径也不通过**

**粗算结论：❌ 否决** — 穿透回报率远低于门槛，即使考虑EV口径也不达标。
