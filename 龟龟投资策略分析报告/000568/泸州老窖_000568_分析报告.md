## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤 1：归母净利润

```
集团净利润 [A] = 107.95 亿元 (2025-09 TTM)
少数股东损益 [B] ≈ 1.5 亿元（基于少数股东权益占比）
GAAP 归母净利润 [C] = A - B = 107.95 - 1.5 = 106.45 亿元
```

#### 步骤 2：Owner Earnings 粗算

**数据来源**: data_pack_market §4 + data_pack_report P1

```
折旧摊销 [D] ≈ 35 亿元（估算，基于固定资产折旧率）
资本开支 [E] ≈ 12 亿元（估算，基于投资现金流）
维持性资本开支系数 [G] = 80%（白酒行业维持性占比高）
维持性资本开支 [H] = D × G = 35 × 80% = 28 亿元

Owner Earnings [I] = C + D - H
                   = 106.45 + 35 - 28
                   = 113.45 亿元
```

#### 步骤 3：分配能力验证（一票否决）

**检查**:
- OCF (982 亿) > 净利润 (108 亿) ✅
- FCF (861 亿) > 股息 (85 亿) ✅
- 有息负债 = 222 亿（可控）✅

**结论**: ✅ 通过分配能力验证

#### 步骤 4：现金上游障碍

**评估**:
- 应收账款极低（1.10 亿），无现金 upstream 障碍
- 采用先款后货模式
- 货币资金充裕（3,174 亿）

**结论**: ✅ 无现金上游障碍

#### 步骤 5：分配意愿

**证据**:
- 历史支付率约 60-65%
- 筹资活动现金流持续大额流出（分红）
- 国企背景，分红意愿稳定

**支付率锚定值 [M]**: 60%

#### 步骤 6：可预测性

**评估**:
- 白酒商业模式稳定
- 品牌护城河强大
- 2025 年短期承压但长期可预测性强

**可预测性评级**: 高

#### 步骤 7：税务泄漏修正

**综合股息税率 [Q]**:
- 个人所得税（股息红利）: 10%（持股>1 年免税，假设平均 5%）
- 企业所得税：25%（已计入净利润）

**综合税率 [Q]**: 5%

#### 步骤 8：粗算穿透回报率 R%

**计算公式**:
```
R = [C × M × (1 - Q%) + O] / 当前市值

代入数字:
C = 106.45 亿元
M = 60%
Q = 5%
O = 0（无回购）
当前市值 = 1,544 亿元

R = [106.45 × 60% × (1 - 5%) + 0] / 1,544
  = [106.45 × 0.60 × 0.95] / 1,544
  = 60.68 / 1,544
  = 3.93%
```

**EV 口径粗算**（若适用）:
```
广义净现金 = 2,952.55 亿元
EV = 市值 - 广义净现金 = 1,544 - 2,952 = -1,408 亿元（负 EV）

由于 EV 为负，说明公司现金储备远超市值，这是极端低估信号
```

### 粗算否决门检查

**门槛 [II]**: 3.79%

**检查**:
- R (3.93%) > II (3.79%) ✅ 主轨通过
- R (3.93%) > II×0.5 (1.90%) ✅ 满足条件

**结论**: ✅ **粗算通过**，进入因子 3 精算

### 步骤 1：真实现金收入还原

**数据来源**: data_pack_report P1

