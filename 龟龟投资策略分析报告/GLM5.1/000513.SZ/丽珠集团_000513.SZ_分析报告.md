### 模块一：资本消耗强度

**分析**：
- 医药制造行业属于中等资本消耗行业
- 2024FY Capex/Revenue = 6.60/118.12 = 5.6%

**评定：中** — 资本消耗中等，需持续研发投入但非重资产扩张型

### 模块二：收款模式

**分析**：
- 应收账款周转天数：62天（2025Q3），61.7天（2024FY）
- 医院客户为主，回款周期较长但相对稳定
- 应收账款/营业收入 = 19.36/118.12 = 16.4%（2024FY）
- 合同负债较小（3.8亿元），非预收模式
- 收款比率估算：OCF/Revenue = 29.79/118.12 = 25.2%（含净利润+折旧等非现金项）

**收款比率还原**：
- 真实现金收入 ≈ 营业收入 - 应收账款净增加
- 2024FY：应收账款从~20.0亿降至19.36亿，净减少0.64亿
- 真实现金收入 ≈ 118.12 + 0.64 = 118.76 亿元
- 收款比率 = 118.76/118.12 = 100.5% → 现金回收良好

**评定：应收为主，但质量较好** — 医院客户信用风险低，回款稳定

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

- 2024FY 归母净利润 = **20.61 亿元**
- 2025FY 归母净利润 = **20.23 亿元**
- 2年平均 = **20.42 亿元**

### 步骤 2：Owner Earnings 粗算

```
Owner Earnings = 归母净利润 + D&A − Capex − 营运资本变动

归母净利润 = 20.61 亿元
+ D&A ≈ 3.5 亿元
− Capex ≈ 6.60 亿元（保守口径，含全部投资现金流）
− 营运资本变动 ≈ -1.0 亿元（应收减少、存货减少，为正向贡献）

Owner Earnings ≈ 20.61 + 3.5 − 6.60 + 1.0 = 18.51 亿元
```

**替代验证（OCF口径）**：
```
归母OCF ≈ OCF × (归母净利润/总净利润) = 29.79 × (20.61/23.04) = 26.65 亿元
归母Capex ≈ 6.60 × 0.895 = 5.91 亿元
归母FCF = 26.65 − 5.91 = 20.74 亿元
```

**Owner Earnings 取值**：取两者均值 ≈ **19.6 亿元**，取保守值 **19.0 亿元**

### 步骤 3：分配能力验证（一票否决）

```
OCF = 29.79 亿元
最新年度分红+回购 = 9.84 + 8.31 = 18.15 亿元
OCF / 分配总额 = 29.79 / 18.15 = 1.64x

→ OCF 充分覆盖分配，分配能力验证通过 ✅
```

### 步骤 4：现金上游障碍

- 公司为A股+H股上市，资金主要在境内
- 子公司分红为上游资金主要来源
- 少数股东权益占比上升（7.77→14.04亿元），需关注子公司少数股东分配
- 现金上游障碍：**低** — 境内为主，无显著跨境限制

### 步骤 5：分配意愿

- 连续6年稳定分红，DPS序列：1.25→1.30→1.60→1.35→1.10→1.43
- 2025年DPS提升至1.43元（+30%），分配意愿增强
- 持续回购：2024年8.31亿元，2025年继续执行
- 3年平均支付率57.7%

**分配意愿评定：强** — 高分红+持续回购，对股东友好

### 步骤 6：可预测性

- 收入波动率CV≈6.5%（极低）
- 归母净利润5年复合增速≈3.9%（稳健）
- 毛利率稳定在64-66%区间
- OCF/净利润>1.0（现金质量好）

**可预测性评定：高**

### 步骤 7：税务泄漏修正

- A股持股>1年免征红利税
- Q%（税务泄漏率）= 0%

### 步骤 8：粗算穿透回报率 R%

```
R = (归母净利润 × M × (1 − Q%) + O) / 当前市值
  = (20.61 × 57.7% × 1.0 + 5.5) / 300.6
  = (11.89 + 5.5) / 300.6
  = 17.39 / 300.6
  = 5.79%
```

**粗算否决门检验**：
- R = 5.79% > Rf = 1.81% ✅
- R = 5.79% > II × 0.5 = 1.91% ✅
- R = 5.79% > II = 3.81% ✅

**粗算结论：✅ 通过** — 粗算穿透回报率5.79%显著超过门槛3.81%
