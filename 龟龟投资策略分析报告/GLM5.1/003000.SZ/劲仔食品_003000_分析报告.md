### 模块一：资本消耗强度

**判断：高**

- 2024年Capex约4.0亿元，占营收16.6%，占经营利润113%
- 在建工程持续增加（0.45亿→0.59亿），扩产投资未结束
- 2025年新增长期借款0.90亿元，为扩产融资
- 固定资产从4.28亿增至5.98亿（2025Q3），增幅40%
- **结论**：公司处于重资本扩张期，资本消耗强度高

### 模块二：收款模式

**判断：现结为主（优秀）**

- 应收账款周转天数仅2.16天（2024年度），5.01天（2025Q3）
- 应收账款仅0.20亿元，占营收0.83%
- 收款比率接近100%，下游议价能力弱但对渠道依赖度高
- **结论**：收款模式健康，以现结为主，应收风险极低

### 模块三：竞争格局与护城河

#### 第一层：非技术性护城河
- **品牌效应**：中等偏弱。"劲仔小鱼"在鱼制品细分有知名度，但零食品牌忠诚度低
- **渠道壁垒**：中等偏弱。零食量贩渠道（零食很忙、好想来等）正在颠覆行业格局，该渠道本质"去品牌化"
- **转换成本**：低。零食属冲动消费，消费者忠诚度有限
- **规模效应**：中等。产能扩张带来一定成本优势

#### 第二层：技术性护城河
- **专利/配方**：弱。风味小鱼技术壁垒低，易被模仿
- **网络效应**：无
- **数据优势**：无

**护城河评级：窄**（细分龙头地位存在，但护城河正在被零食量贩渠道侵蚀）

### 模块四：周期性

**判断：弱周期**

- 休闲食品属日常消费品，需求相对稳定
- 但行业竞争格局正在发生结构性变化（量贩渠道崛起）
- 原材料（鱼制品）价格有一定波动性
- **结论**：弱周期，但行业格局变化带来结构性挑战

### 模块五：人力资本依赖

**判断：中**

- 食品制造业对核心技术人员依赖度中等
- 创始人周劲松是公司灵魂人物，存在关键人风险
- 人力投入回报率从94.71%（2024）降至67.95%（2025Q3），效率下降

### 模块六：管理层与治理结构

**判断：一般**

- 创始人周劲松兼任董事长和总经理，权力集中
- 实控人周劲松、李冰玉夫妇持股46.23%，控制权稳固
- 股权质押约2109万股，需关注质押比例变化
- 董事长薪酬67万元，相对公司规模偏低，利益与股东基本一致
- 2024年完成董事会换届，治理结构正常

### 模块七：监管与政策风险

**判断：低**

- 食品行业监管以食品安全为主，公司无重大违规记录
- 无特殊行业监管风险

### 模块八：MD&A 解读

**判断：中性偏负面**

- 2024年业绩亮眼，但2025年明显放缓
- 管理层对行业竞争加剧的认知清晰，但应对策略效果待观察
- "再造一个劲仔"战略遇阻，第二增长曲线增速放缓

### 模块九：控股折价分析

**不适用**（非控股公司型结构）

## 因子 2：穿透回报率粗算（Top-Down）
## ══════════════════════════════════════

### 步骤 1：归母净利润

- GAAP 归母净利润 C = 2.92 亿元

### 步骤 2：Owner Earnings 粗算

```
OE = C + D&A - Capex
   = 2.92 + 0.50 - 4.00
   = -0.58 亿元（负值！）
```

**注**：OE为负，说明公司当前资本开支远超盈利能力，处于重投入扩张期。

改用FCF口径：
```
FCF = OCF - Capex = 5.42 - 4.00 = 1.42 亿元
```

### 步骤 3：分配能力验证（一票否决）

```
FCF = 1.42 亿元 < 股息 1.79 亿元
缺口 = 0.37 亿元
```

**⚠️ 分配能力不足**：FCF不足以覆盖当前分红水平，分红部分依赖现金储备。

### 步骤 4：现金上游障碍

- A股直接持有，无跨境障碍
- Q% = 0%（持股>1年免税）

### 步骤 5：分配意愿

- 持续5年分红，支付率从99%降至61%，趋势向下但仍在分红
- 分配意愿：中等偏强

### 步骤 6：可预测性

- 收入CAGR（2020-2024）≈ 27.5%，但2025年增速骤降至2%
- 利润波动大（2024 +39%, 2025Q3 -20%）
- 可预测性：低

### 步骤 7：税务泄漏修正

- A股长期持有免税，Q% = 0%

### 步骤 8：粗算穿透回报率

```
─── 市值口径 ───
R = (FCF × M + O) / 市值
  = (1.42 × 0.61 + 0) / 46.68
  = 0.87 / 46.68
  = 1.86%

─── EV 口径 ───
EV = 市值 - 广义净现金 = 46.68 - 6.75 = 39.93 亿元
R_EV = 0.87 / 39.93 = 2.18%
```

### 粗算否决门判定

```
Rf = 1.81%
II = 3.81%
II × 0.5 = 1.91%

R = 1.86% > Rf = 1.81% → 第一关通过
R = 1.86% < II × 0.5 = 1.91% → 检查 EV 口径
R_EV = 2.18% > II × 0.5 = 1.91% → EV口径通过否决门
R_EV = 2.18% < II = 3.81% → 边际不达标
```

**粗算结论：⚠️ 边际不达标（EV口径），继续进入因子3**
