### 模块一：资本消耗强度

**分析**：
- 固定资产/总资产 = 335.29 / 6,043.52 = 5.5% → 极轻资产
- 商誉+无形资产/总资产 = 465.90 / 6,043.52 = 7.7% → 中等（主要为库卡商誉214亿）
- 估算Capex/OCF = 150 / 605.12 = 24.8% → 资本消耗强度低
- OCF/Capex = 605.12 / 150 = 4.0倍 → 现金流充裕

**结论：资本消耗强度低**

### 模块三：竞争格局与护城河

**第一层：非技术性护城河**
1. **品牌优势**：美的品牌全球知名度高，多品牌矩阵（美的、COLMO、东芝、小天鹅、华凌）
2. **规模优势**：全球家电营收规模第一，成本领先优势显著
3. **渠道优势**：线上线下全覆盖，全球200+国家和地区销售
4. **网络效应**：智能家居生态，美的美居App用户超5,000万

**第二层：技术性护城河**
1. **研发投入**：2024年研发费用162亿元，营收占比3.97%，行业领先
2. **核心技术**：压缩机（美芝全球份额第一）、电机（威灵）等核心部件自研
3. **专利储备**：全球专利储备超10万件
4. **制造能力**：智能制造灯塔工厂，生产效率行业领先

**护城河评级：宽**（规模+品牌+渠道+技术四重护城河）

## 【因子 2：穿透回报率粗算（Top-Down）】
═══════════════════════════════════════

### 步骤 1：归母净利润

GAAP 归母净利润 C = 385.37 亿元

### 步骤 2：Owner Earnings 粗算

```
OE = C + D&A - Capex - SBC
   ≈ 385.37 + 75 - 150 - 12
   ≈ 298 亿元
```

或使用 FCF 口径：
```
FCF = OCF - Capex
2022: 346.58 - 100 = 246.58
2023: 579.02 - 120 = 459.02
2024: 605.12 - 150 = 455.12
3年均值 ≈ 387 亿元
```

**取 FCF 均值 387 亿元作为 Owner Earnings 粗算基准**

### 步骤 3：分配能力验证

```
OCF = 605.12 亿元
股息 + 回购 = 267.12 + 75 = 342.12 亿元
OCF > 股息 + 回购 → ✅ 分配能力充足
```

### 步骤 4：现金上游障碍

- A股上市公司，分红无上游障碍
- 子公司分红可能存在少量税务成本
- **结论：低障碍**

### 步骤 5：分配意愿

- 支付率从40%持续提升至69.3%，趋势明确向上
- 大额回购计划（50-100亿元/年），70%用于注销
- 2025-2027股东回报规划已发布
- **结论：强**

### 步骤 6：可预测性

- 家电行业需求稳定，美的市占率稳固
- 全球化分散风险
- 营收5年CAGR ≈ 8.5%
- **结论：中高**

### 步骤 7：税务泄漏修正

- A股持有>1年免股息税
- 保守估计 Q% = 5%（考虑部分投资者需缴税）

### 步骤 8：粗算穿透回报率

```
R = [OE × M × (1 - Q%) + O] / 当前市值
  = [387 × 69.3% × 95% + 75] / 5,864
  = [254.87 + 75] / 5,864
  = 329.87 / 5,864
  = 5.63%
```

**粗算穿透回报率 R = 5.63%**

**粗算否决门判断**：
- R = 5.63% > Rf = 1.79% → ✅ 通过
- R = 5.63% > II × 0.5 = 1.90% → ✅ 通过
- R = 5.63% > II = 3.79% → ✅ 通过

**因子 2 结论：✅ 通过**
