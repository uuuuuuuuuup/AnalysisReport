## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 77.11 亿元（2025年度）

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）≈ 10.5 亿元
G（维持性资本开支系数）= 0.50（医药行业，传统Capex中约50%为维持性）
H = D × G = 10.5 × 0.50 = 5.25 亿元
I（Owner Earnings）= C + D - H = 77.11 + 10.5 - 5.25 = 82.36 亿元
```

### 步骤 3：分配能力验证

```
OCF = 112.35 亿元
Owner Earnings = 82.36 亿元
OCF > Owner Earnings ✅ 分配能力充足
```

- 股息税率：持股>1年为0%


### 步骤 7：税务泄漏修正

```
A股直接持有，持股>1年
Q = 0%（免税）
```

### 步骤 8：粗算穿透回报率

```
R = [C × M × (1 - Q%) + O] / 当前市值
  = [77.11 × 21.1% × (1 - 0%) + 1.88] / 3,593
  = [16.27 + 1.88] / 3,593
  = 18.15 / 3,593
  = 0.505%
  ≈ 0.51%
```

### 粗算否决门检查

```
Rf = 1.76%
II = 3.76%
II × 0.5 = 1.88%

R = 0.51% < Rf = 1.76% → 检查EV口径
R = 0.51% < II × 0.5 = 1.88% → 检查EV口径

广义净现金/市值 = 423.20 / 3,593 = 11.78% < 40%
→ EV口径不适用

粗算否决：R < II × 0.5，且EV口径不适用
```

**⚠️ 粗算结论：否决 — 穿透回报率远低于门槛**

> **核心原因分析**：粗算穿透回报率仅0.51%，远低于A股门槛3.76%，主要受两个因素拖累：
> 1. **估值极高**：PE 48倍，市值3,593亿元，分母过大
> 2. **支付率极低**：仅21.1%，公司大量利润留存用于研发，实际分配给股东的现金极少
>
> 即使支付率提升至100%：R_max = [77.11 × 100% + 1.88] / 3,593 = 2.20%，仍低于门槛3.76%

## 附录：分析说明

### 为什么一家"好公司"被否决？

恒瑞医药无疑是中国创新药行业的**优秀企业**：
- ✅ 宽护城河（管线深度+商业化+国际化）
- ✅ 财务极其稳健（零有息负债，现金超400亿）
- ✅ 盈利能力强劲（毛利率86%，ROE 14.7%）
- ✅ 增长前景良好（创新药收入+26%，券商预期CAGR 20%）

**但龟龟投资策略关注的是"股东实际能拿到的现金回报"**，而非企业本身的优秀程度。当前面临两个核心问题：

1. **估值过高**：PE 48倍意味着市场已经为未来5-10年的增长预付了代价。即使公司完美执行，当前价格买入的投资者也难以获得合理回报。

2. **分配意愿不足**：公司仅将17-21%的利润分配给股东，其余留存用于研发。这对公司发展有利，但对追求现金回报的价值投资者而言，未分配的利润只是"纸面财富"。
