# 股票综合分析 Skill

## 功能概述
本 Skill 针对单只股票进行多维度综合分析，整合基本面、技术面、情绪面三大维度数据，输出专业的投资分析结论，帮助投资者判断股票所处阶段、主力资金动向以及后续上涨概率。

## 触发场景
当用户提出以下需求时触发本 Skill：
1. 要求对某只具体股票进行分析/评估
2. 询问某只股票当前处于什么阶段
3. 想了解某只股票的主力资金是否在场
4. 咨询某只股票后续上涨概率及投资建议
5. 需要综合基本面、技术面、情绪面的股票分析报告

## 分析维度
### 1. 基本面分析
- 公司核心业务及商业模式评估
- 财务报表分析（营收、利润、毛利率、净资产收益率等关键指标）
- 行业地位与竞争优势分析
- 估值水平评估（PE、PB、PEG等估值指标对比行业平均）
- 未来成长潜力与风险点分析

### 2. 技术面分析
- 股价走势阶段判断（建仓期、洗盘期、拉升期、出货期）
- 量价关系分析
- 关键技术指标分析（MA、MACD、KDJ、RSI、BOLL等）
- 支撑位与压力位判断
- 趋势分析（短期、中期、长期趋势）

### 3. 情绪面分析
- 市场热度与投资者情绪指标
- 资金流向分析（北向资金、南向资金、主力资金、龙虎榜数据）
- 最新公告、新闻及研报观点汇总
- 板块热度与题材相关性分析
- 舆情监测与市场预期判断

## 输出结论
本 Skill 最终输出包含以下核心内容：
1. **股票所处阶段**：明确判断当前股票处于建仓、洗盘、拉升、出货中的哪个阶段
2. **主力资金分析**：判断主力资金是否在场，近期资金动向如何
3. **上涨概率评估**：给出后续1个月、3个月、6个月的上涨概率百分比，并详细说明理由
4. **投资建议**：基于综合分析给出操作建议（买入、持有、卖出、观望）及风险提示

## 工具调用流程
### 第一步：股票代码确认
调用 `mcp_AI_Tools_invoke_tool` 使用 `StockSearch` 工具，根据用户提供的股票名称/关键词搜索对应的股票代码和全称，确保分析标的准确。

### 第二步：基本面数据采集
1. 调用 `mcp_AI_Tools_invoke_tool` 使用 `GetIncomeStatement` 工具，获取利润表数据，分析营收、利润、毛利率等盈利指标
2. 调用 `mcp_AI_Tools_invoke_tool` 使用 `GetBalanceSheet` 工具，获取资产负债表数据，分析资产质量、负债水平、净资产等
3. 调用 `mcp_AI_Tools_invoke_tool` 使用 `GetCashFlowStatement` 工具，获取现金流量表数据，分析现金流健康状况
4. 调用 `mcp_AI_Tools_invoke_tool` 使用 `GetFinancialIndicators` 工具，获取PE、PB、ROE、毛利率等核心财务指标，进行估值分析
5. 补充调用 `mx_finance_data` 获取完整财务报告全文和更多基本面数据

### 第三步：技术面数据采集
1. 调用 `mcp_AI_Tools_invoke_tool` 使用 `GetDailyKLineData` 工具，获取近3个月日线数据（包含开盘价、收盘价、最高价、最低价、成交量），分析短期走势和量价关系
2. 调用 `mcp_AI_Tools_invoke_tool` 使用 `GetWeeklyKLineData` 工具，获取近2年周线数据，分析中期趋势和关键支撑压力位
3. 调用 `mcp_AI_Tools_invoke_tool` 使用 `GetMonthlyKLineData` 工具，获取近5年月线数据，分析长期趋势和周期位置
4. 实时价格、涨跌幅等最新交易数据通过 `StockSearch` 工具获取
5. 技术指标（MA、MACD、KDJ等）基于获取到的K线数据自行计算分析

### 第四步：情绪面数据采集
（经测试AI-Tools新闻、舆情类接口暂不可用，完全使用东财接口和全网搜索替代）
1. 调用 `mx_finance_search` 获取个股最新公告、新闻、研报、龙虎榜、资金流向、北向资金等完整数据
2. 调用 `WebSearch` 补充获取全网相关舆情信息、行业动态和政策新闻
3. 整合所有公开信息，综合判断市场情绪、资金态度和题材热度

### 第五步：综合分析与报告生成
1. 整合基本面数据，评估公司质地、估值水平、成长潜力
2. 分析技术面数据，判断股票所处阶段（建仓/洗盘/拉升/出货）、主力资金动向
3. 结合情绪面数据，评估市场热度和投资者预期
4. 综合三大维度分析结果，给出不同时间维度的上涨概率及理由
5. 生成最终分析报告，包含投资建议和风险提示

### 补充工具
- 必要时调用 `WebSearch` 获取最新行业政策、宏观经济数据等外部信息
- 调用 `GetMarketHotNews` 了解当前市场热点和板块轮动情况

## 使用示例
### 用户输入
"帮我分析一下贵州茅台这只股票现在怎么样，后续能不能涨？"

### 输出示例
> **贵州茅台（600519）综合分析报告**
> 
> **一、当前所处阶段**：处于震荡上行的中期拉升阶段，估值处于合理区间，近期量能温和放大，资金关注度提升。
> 
> **二、主力资金分析**：主力资金持续在场，近30日北向资金累计净流入28亿元，机构持仓比例稳定在70%以上，筹码集中度较高。
> 
> **三、上涨概率评估**：
> - 1个月内上涨概率：65%，理由：消费旺季临近，业绩确定性强，技术面突破短期压力位
> - 3个月内上涨概率：75%，理由：全年业绩增长预期明确，估值修复空间充足
> - 6个月内上涨概率：80%，理由：长期品牌价值突出，行业龙头地位稳固，提价预期存在
> 
> **四、投资建议**：建议逢低配置，合理仓位15%-20%，止损位设置在1600元。
> 
> **风险提示**：宏观经济下行影响消费需求，行业政策变化风险。

## 注意事项
1. 所有分析结论仅供参考，不构成直接投资建议
2. 投资有风险，入市需谨慎，建议结合自身风险承受能力做出决策
3. 分析结论基于当前可得数据，市场情况变化可能影响分析结果的准确性
