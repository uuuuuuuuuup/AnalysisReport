## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

#### 步骤 1：归母净利润

```
C = GAAP 归母净利润 = 114.05 亿元（2025年前三季度）
```

**年化调整：**
- 2025年前三季度归母净利润：114.05亿元
- 2024年全年归母净利润：122.43亿元
- 2025年全年预估归母净利润：114.05 / 3 × 4 = 152.07亿元

**采用值：** C = 152.07 亿元（年化）

#### 步骤 2：Owner Earnings 粗算

由于折旧摊销（D）和资本开支（E）数据未披露，采用简化计算：

```
Owner Earnings ≈ 归母净利润 × (1 - 资本开支率)

白酒行业资本开支率通常较低，假设为净利润的10%

I = C × (1 - 10%) = 152.07 × 0.9 = 136.86 亿元
```

#### 步骤 3：分配能力验证（一票否决）

**检查项：**
1. ✅ 经营现金流 > 归母净利润（2024年：121.72亿 > 122.43亿，基本持平）
2. ✅ 无有息负债，财务结构稳健
3. ✅ 分红历史稳定，累计派现243.25亿元

**结论：** 分配能力充足，未触发否决

#### 步骤 4：现金上游障碍

**检查项：**
1. ✅ 无有息负债，无债务偿付压力
2. ✅ 应收账款极小，无回款障碍
3. ✅ 省外子公司为主，但无证据显示存在资金回流障碍

**结论：** 无现金上游障碍

#### 步骤 5：分配意愿

**检查项：**
1. ✅ 近三年累计派现167.75亿元
2. ✅ 2025年分红6.06元/股，较2024年提升38.7%
3. ✅ 股息支付率约60%，处于行业较高水平
4. ⚠️ 无股票回购记录

**结论：** 分配意愿强

#### 步骤 6：可预测性

**检查项：**
1. ✅ 商业模式清晰，白酒生产销售
2. ✅ 历史业绩稳定，近5年营收和利润持续增长
3. ✅ 行业地位稳固，清香型龙头
4. ⚠️ 2025年Q3净利润增速放缓至0.48%

**结论：** 可预测性较高

#### 步骤 7：税务泄漏修正

**A股股息税率：**
- 持股期限超过1年：税负为0
- 持股期限1个月至1年：税负为10%
- 持股期限1个月以内：税负为20%

**假设长期持有（>1年）：**
```
Q = 0%（综合股息税率）
```

#### 步骤 8：粗算穿透回报率 R%

```
R = [I × M × (1 - Q%) + O] / 当前市值
  = [136.86 × 60% × (1 - 0%) + 0] / 1693
  = 82.12 / 1693
  = 4.85%
```

### 粗算否决门检查

- ✅ R = 4.85% > Rf = 1.85%
- ✅ R = 4.85% > II × 0.5 = 3.85% × 0.5 = 1.925%
- ✅ R = 4.85% > II = 3.85%

**粗算结论：** ✅ 通过 - 粗算穿透回报率4.85%，超过门槛值3.85%，继续进入因子 3 精算

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```
营业收入 S = 329.24 亿元（2025年前三季度）
应收账款净变动 T = 0.17 - 0.21 = -0.04 亿元（减少）
预收款/合同负债净变动 U = 未披露

保守处理：
- 应收账款净减少 → 加回（收入已收到现金）
```
