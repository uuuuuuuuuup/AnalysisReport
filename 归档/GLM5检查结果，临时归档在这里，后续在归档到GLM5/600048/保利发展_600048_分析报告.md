## 因子2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

```
集团净利润 [A] = 97.4 亿元
少数股东损益 [B] = 47.39 亿元（占比48.64%）
GAAP归母净利润 [C] = A - B = 50.01 亿元
```

### 步骤2：Owner Earnings粗算

```
折旧摊销 [D] = ~15 亿元（估算）
资本开支 [E] = 17.95 亿元
维持性资本开支系数 [G] = 0.8（房地产行业估算）
维持性资本开支 [H] = D × G = 15 × 0.8 = 12 亿元

Owner Earnings [I] = C + D - H = 50.01 + 15 - 12 = 53.01 亿元
```

### 步骤3：分配能力验证

```
经营现金流 OCF = 62.57 亿元
Owner Earnings I = 53.01 亿元

验证：OCF > I → ✅ 分配能力验证通过
```

### 步骤7：税务泄漏修正

```
综合股息税率 [Q] = 10%（A股个人投资者）
税后可分配 = I × (1 - Q%) = 53.01 × 0.9 = 47.71 亿元
```

### 步骤8：粗算穿透回报率

```
支付率锚定值 [M] = 40%
年均回购金额 [O] = 0 亿元（假设）

粗算穿透回报率 [R] = [C × M × (1 - Q%) + O] / 当前市值
                    = [50.01 × 0.4 × 0.9 + 0] / 682.32
                    = 18.00 / 682.32
                    = 2.64%

使用Owner Earnings口径：
R = [I × M × (1 - Q%) + O] / 当前市值
  = [53.01 × 0.4 × 0.9 + 0] / 682.32
  = 19.08 / 682.32
  = 2.80%
```

### 粗算否决门检验

```
门槛 [II] = 3.635%
粗算穿透回报率 [R] = 2.80%

检验：
- R < Rf (1.635%) → 否，R > Rf
- R < II × 0.5 (1.82%) → 否，R > 1.82%
- II × 0.5 ≤ R < II → 是，1.82% < 2.80% < 3.635%

结论：⚠️ 边际不达标，继续进入因子3精算
```

## 附录：关键计算公式

### 穿透回报率计算

```
粗算穿透回报率 R = [归母净利润 × 支付率 × (1 - 税率) + 回购] / 市值

精算穿透回报率 GG = [真实可支配现金结余 × 支付率 × (1 - 率) + 回购] / 市值
```

### 安全边际计算

```
安全边际 JJ = 精算穿透回报率 GG - 门槛 II

目标买入价 = 市值 × (GG / II) / 总股本 × 安全边际折扣
```
