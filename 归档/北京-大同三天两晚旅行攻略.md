### 🗼 应县木塔 ⭐⭐⭐⭐⭐ 重点景点

- **地址**：朔州市应县应元街1号
- **开放时间**：08:00-17:30（17:00停止售票）
- **门票**：50元/人
- **游览时长**：约1.5-2小时
- **距酒店**：约73公里，打车约1小时
- 喜马拉雅讲解

#### 景点简介

应县木塔全称"佛宫寺释迦塔"，建于辽代清宁二年（1056年），是世界上现存最高、最古老的纯木结构楼阁式建筑，高67.31米，与埃菲尔铁塔、比萨斜塔并称"世界三大奇塔"。

#### 游览重点

- **建筑奇迹**：全塔无一颗铁钉，全靠斗拱咬合
- **千年不倒**：历经多次地震、炮火依然屹立
- **辽代塑像**：塔内供奉释迦牟尼佛牙舍利
- **匾额题字**：明清皇帝御笔匾额

#### 注意事项

- 木塔内部目前禁止登塔，只能外观和一层参观
- 建议绕塔一周，从不同角度欣赏
- 可在塔前广场拍照留念

### 🏯 华严寺 ⭐⭐⭐⭐⭐ 重点景点

- **地址**：大同市平城区下寺坡街459号（古城西南隅）
- **开放时间**：旺季08:30-18:00，淡季08:30-17:00
- **门票**：旺季50元/人，淡季40元/人
- **游览时长**：约1.5-2小时
- **距酒店**：约5公里，位于大同古城内

#### 景点简介

华严寺是依据佛教经典《华严经》修建的辽代皇家寺院，距今已有近千年历史。寺内保存着珍贵的辽金建筑和彩塑，是大同必打卡的佛教建筑瑰宝。

#### 游览重点

| 景点 | 特色                                                 | 必看指数   |
| ------ | ------------------------------------------------------ | ------------ |
| **薄伽教藏殿**     | 全国仅存的八大辽代建筑之一，"东方维纳斯"合掌露齿菩萨 | ⭐⭐⭐⭐⭐ |
| **大雄宝殿**     | 现存辽金时期最大佛殿，4.5米高琉璃鸱吻                | ⭐⭐⭐⭐⭐ |
| **华严宝塔**     | 全国第二大纯木榫卯结构木塔（高43.5米）               | ⭐⭐⭐⭐   |
| **千佛地宫**     | 100吨纯铜打造，供奉千尊佛像和高僧舍利                | ⭐⭐⭐⭐   |

#### 游览路线（不走回头路）

1. 上华严寺：山门→普光明殿→大雄宝殿→观音阁→地藏阁
2. 下华严寺：薄伽教藏殿→华严宝塔→藏经阁→地宫

### 🏛️ 善化寺 ⭐⭐⭐⭐⭐ 重点景点

- **地址**：大同市平城区南寺街6号（古城南门附近）

#### 景点简介

善化寺是中国现存规模最大、最为完整的辽金时期建筑群，1961年被列为第一批全国重点文物保护单位。寺内保存着珍贵的金代塑像与清代壁画，相较其他热门景点更为清静，适合静心感受历史。

| **三圣殿**     | 供奉"华严三圣"，金代大斜拱形如花朵       | ⭐⭐⭐⭐⭐ |
| **大雄宝殿**     | 寺内核心辽代建筑，五方佛和二十四诸天塑像 | ⭐⭐⭐⭐⭐ |
| **普贤阁**     | 金代修建的两层楼阁                       | ⭐⭐⭐⭐   |
| **五龙壁**     | 寺前广场明代琉璃五龙壁                   | ⭐⭐⭐⭐   |

#### 游览路线（约1.5小时）

山门（天王殿）→ 三圣殿 → 大雄宝殿 → 普贤阁 → 寺外五龙壁
