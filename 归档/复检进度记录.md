## 目标买入价计算方法

根据龟龟投资策略，目标买入价的计算公式为：

```
基准目标买入价 = 当前市值 × (GG / II) / 总股本

其中：
- GG = 精算穿透回报率
- II = 门槛回报率
- 总股本 = 公司总股数（亿股）
```

然后根据现金保护进行调整（通常乘以0.70-0.75的系数）
