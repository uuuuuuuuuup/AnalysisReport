
## 五、因子 3：穿透回报率精算（Bottom-Up）

### 步骤 1：真实现金收入还原

```
S = 营业收入 = 235.78 亿元（2024年）
T = 应收账款净变动 = 0.70 - 0.71 = -0.01 亿元（减少）
U = 预收款/合同负债净变动 = 35.15 - 29.37 = 5.78 亿元（增加）

保守处理：
- 应收账款净减少 → 加回（已收到现金）
- 预收款净增加 → 不加回（对应未来交付义务）

真实现金收入 = S - T的净增加部分 - U的净减少部分
真实现金收入 = 235.78 - 0 - 0 = 235.78 亿元
收款比率 = 235.78 / 235.78 = 100%
```

- 收款比率100%，现金流质量极高
