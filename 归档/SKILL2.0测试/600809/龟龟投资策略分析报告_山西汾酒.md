
## 因子 3:穿透回报率精算(Bottom-Up) + 现金质量审计

### 步骤1:真实现金收入还原

```
营业收入 S = 360.11亿元(2024年)
应收账款净变动 T = 0.017亿元(增加为正,极小)
预收款/合同负债净变动 U = 0亿元(未找到明细)

保守处理:
- 应收账款净增加 → 扣除(收入未收到现金)
- 预收款净增加 → 不加回(对应未来交付义务)

真实现金收入 = S - T的净增加部分 - U的净减少部分
真实现金收入 = 360.11 - 0.017 - 0 = 360.09亿元
收款比率 = 360.09 / 360.11 = 99.99%
```

**结论**: 收款比率接近100%,现金流质量极高

### 步骤2:应收账款footnote核查
