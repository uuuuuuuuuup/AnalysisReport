### 模块一：资本消耗强度

**资本开支分析**:
- 2024 年 Capex = 28.48 亿元
- Capex / 营业收入 = 28.48 / 209.83 = **13.6%**（中等）
- Capex / OCF = 28.48 / 48.82 = **58.4%**（中等偏高）

**维持性 vs 成长性资本开支**:
- ⚠️ 无法精确区分
- **估算**: 维持性 Capex 约占 20-30%（设备维护）
- 成长性 Capex：新医院建设、并购

**自由现金流质量**:
- 2024 年 FCF = 20.34 亿元
- FCF / 净利润 = 20.34 / 37.36 = **54.4%**（中等）
- 2025 年 Q1-Q3 FCF = 24.45 亿元（优秀）

**模块一结论**: 🟡 **中等偏好**
- 资本消耗强度中等
- FCF 转化率中等（54%）
- 但 2025 年 FCF 改善明显

## 因子 2：穿透回报率粗算（Top-Down）

**⚠️ 重要提示**: 由于关键数据缺失，以下计算为**估算值**

### 步骤 1：归母净利润

```
C = 37.36 亿元（2024 年归母净利润）
```

### 步骤 2：Owner Earnings 粗算

**公式**: I = C + D − H

其中：
- C = 归母净利润 = 37.36 亿元
- D = 折旧摊销 = ⚠️**待补充**（预计 8-12 亿元）

```
I = 37.36 + 10 - 3 = 44.36 亿元
```

### 步骤 8：粗算穿透回报率 R%

**公式**: R = [C × M × (1 − Q%) + O] / 当前市值

```
  = 1.09%
```
